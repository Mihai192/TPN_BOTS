<p>O secvenţă a unui vector se numeşte palindromică dacă primul element ale secvenţei este egal cu ultimul, al doilea cu penultimul, etc.</p>

<h1>Cerinţa</h1>

<p>Se dă un vector cu <code>n</code> elemente, numere  naturale. Determinaţi secvenţa palindromică de lungime maximă.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>secvpal.in</code> conţine pe prima linie numărul <code>n</code>; urmează cele <code>n</code> elemente ale vectorului, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>secvpal.out</code> va conţine pe prima linie numerele <code>p</code> şi <code>u</code>, reprezentând indicii de început şi sfârşit ai secvenţei determinate.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code>;</li>
	<li>numerele de pe a doua linie a fişierului de intrare vor avea cel mult <code>4</code> cifre;</li>
	<li>dacă există mai multe secvenţe palindromice de lungime maximă, se va determina cea mai din stânga;</li>
</ul>

<h1>Explicație</h1>

<p>Secvenţa palindromică de lungime maximă este <code>10 9 8 5 8 9 10</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("secvpal.in");
ofstream fout("secvpal.out");

int main()
{
    int a[1001], n ,st=0 , dr=0;
    fin >> n;
    for(int i = 1 ; i <= n ; ++i)
        fin >> a[i];
    for(int i = 1 ; i <= n ; ++i)
    {
        for(int j = i + 1 ; j <= n ; ++j)
        {
            if(a[i]==a[j])
            {
                int ok=1;
                for(int x=i , y=j ; x<y ; x++ , y--)
                    if(a[x]!=a[y]) ok=0;
                if(ok)
                    if(j-i>dr-st)
                {
                    st=i;
                    dr=j;
                }
            }
        }
    }

    fout << st << ' ' << dr;

    fin.close();
    fout.close();

    return 0;
}</pre>
