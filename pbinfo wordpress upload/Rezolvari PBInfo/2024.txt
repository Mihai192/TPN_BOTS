<h1>Cerința</h1>

<p>Se dă un şir format din <code>n</code> numere naturale nenule. Aflaţi cel mai mic număr natural, diferit de <code>1</code>, care divide un număr maxim de numere din şir.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>divizor112.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale nenule separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>divizor112.out</code> va conține pe prima linie cel mai mic număr natural, diferit de <code>1</code>, care divide un număr maxim de numere din şir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p><code>2</code> divide două numere din şir.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("divizor112.in");
ofstream cout("divizor112.out");
int n , f[1000001] , maxi , rez , x;
int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> x;
        for(int j = 1 ; j * j <= x; j ++)
        {
            if(x % j == 0)
            {
                f[j]++;
                if(j*j != x) f[x/j]++;
            }
        }
    }
    for(int i = 2 ; i <= 1000000 ; i++)
    {
        if(f[i] > maxi)
        {
            maxi = f[i];
            rez = i;
        }
    }
    cout << rez;
}</pre>
