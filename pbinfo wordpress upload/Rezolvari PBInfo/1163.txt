<h1>Cerința</h1>

<p>Cei <code>m</code> cowboys și cei <code>n</code> aliens s-au întâlnit în vestul sălbatic și, păstrând tradiția locului, s-au așezat în șir indian. Cum cowboys erau gazde primitoare și în special foarte precaute, s-au gândit că între doi cowboys consecutivi ar fi bine să fie cel mult un alien (din motive de securitate). De asemenea primul și ultimul din șir să fie cawboys. Dilema care s-a ivit a fost numărul de moduri în care s-ar putea așeza în șir indian ținând cont de condițiile de securitate impuse.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>cowboysandaliens.in</code> conține pe prima linie numerele naturale <code>m</code> și <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>cowboysandaliens.out</code> va conține pe prima linie numărul <code>S</code>, reprezentând numărul de moduri modulo <code>555557</code>, în care s-ar putea așeza în șir indian.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n &lt; m ≤ 500.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Dacă cei trei cawboys ar fi Jake, Percy și Roy pe care-i vom nota J, P, R iar alien ar fi Bronc, notat B, atunci ei s-ar putea așeza în șir indian în următoarele moduri: <code>JBPR , JPBR , JBRP , JRBP , PBJR , PJBR , PBRJ , PRBJ , RBPJ , RPBJ , RBJP , RJBP</code>. În total avem 12 moduri.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>

using namespace std;

ifstream cin("cowboysandaliens.in");
ofstream cout("cowboysandaliens.out");

#define Mod 555557

long long fact(long long x)
{
    long long p = 1;
    for (int i = 1; i <= x; ++ i)
    {
        p *= i;
        p %= Mod;
    }
    return p;
}

long long aranj(long long n, long long k)
{
    long long p = 1;
    for (int i = n - k + 1; i <= n; ++ i)
    {
        p *= i;
        p %= Mod;
    }
    return p;
}

int main()
{
    long long nrc, nra;
    cin >> nrc >> nra;
    cout << (fact(nrc) * aranj(nrc - 1, nra)) % Mod;
    return 0;
}</pre>
