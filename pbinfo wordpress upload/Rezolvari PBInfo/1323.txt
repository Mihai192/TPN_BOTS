<h1>Cerința</h1>

<p>Se citesc două matrice rare și se cere să se calculeze suma lor.</p>

<p>O matrice <code>A(n,m)</code> se numește rară dacă majoritatea elementelor sale sunt egale cu zero (cel puţin jumătate). Datorită numărului mic de numere nenule, o matrice rară <code>A(n,m)</code>, având <code>k</code> elemente nenule, poate fi memorată folosind un șir <code>X</code> conţinând <code>k</code> triplete de forma <code>(linie, coloană , valoare)</code>, corespunzătoare valorilor nenule ale matricei. Elementele șirului <code>X</code> se memorează în ordine lexicografică după <code>(linie,coloana)</code>.</p>

<p>De exemplu matricea cu <code>n = m = 3</code></p>

<pre contenteditable="true" editable="true">1 0 2
0 0 5
0 2 0</pre>

<p>se va memora sub forma sirului <code>X</code> continand <code>4</code> triplete : <code>{(1,1,1) , (1,3,2) , (2,3,5) , (3,2,2)}</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>matrice_rara.in</code> conține pe prima linie , dimensiunile celor două matrice <code>n m</code> – reprezentând numărul de linii şi coloane, şi  <code>N1 N2</code>, numărul de elemente nenule ale matricei <code>A</code>, respectiv matricei <code>B</code>. Apoi următoarele <code>N1</code> linii vor conține triplete – elementele nenule ale matricei <code>A</code> în ordine lexicografică, iar ultimele <code>N2</code> linii vor conține triplete reprezentând elementele nenule ale matricei <code>B</code>, tot în ordine lexicografică .</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>matrice_rara.out</code> va conține pe prima linie numărul de elemente diferite de <code>0</code> din matricea sumă <code>C</code> şi apoi matricea în sine sub forma tripletelor în ordine lexicografică, câte unul pe o linie .</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n , m ≤ 1.000.000</code></li>
	<li><code>1 ≤ N1 , N2 ≤ 300.000</code></li>
	<li><code>-1.000.000.000 ≤ A[i][j], B [i][j]≤ 1.000.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("matrice_rara.in");
ofstream cout("matrice_rara.out");

struct andra
{
    int i , j , val;
};
andra a[300001] , c[300001] , b[300001];

void citire(andra a[] , int n)
{
    for(int i = 1 ; i <= n ; i++)
        cin >> a[i].i >> a[i].j >> a[i].val;
}

void afisare(andra a[] , int n)
{
    cout << n << '\n';
    for(int i = 1 ; i <= n ; i++)
        cout << a[i].i << " " << a[i].j << " " << a[i].val << '\n';
}

int maimic(andra a , andra b)
{
    return a.i < b.i || a.i == b.i &amp;&amp; a.j < b.j;
}

long long  n , m , nc , n1 , n2;

int main()
{
    cin >> n >> m >> n1 >> n2;
    citire(a , n1);
    citire(b , n2);
    int i = 1 , j = 1;
    while(i <= n1 &amp;&amp; j <= n2)
    {
        if(maimic(a[i] , b[j])) c[++nc] = a[i++];
        else if(maimic(b[j] , a[i])) c[++nc] = b[j++];
        else
        {
            if(a[i].val + b[j].val != 0)
            {
                c[++nc] = a[i];
                c[nc].val += b[j].val;
            }
            i++;
            j++;
        }
    }
    while(i <= n1) c[++nc] = a[i++];
    while(j <= n2) c[++nc] = b[j++];
    cout << nc << '\n';
    for(int i = 1 ; i <= nc ; i++)
        cout << c[i].i << " " << c[i].j << " " << c[i].val << '\n';
}</pre>
