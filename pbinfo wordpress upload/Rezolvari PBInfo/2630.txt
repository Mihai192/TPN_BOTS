<h1>Cerința</h1>

<p>Să se scrie funcția cu următorul antet:</p>

<p><code>int PermD(int a[], int n)</code></p>

<p>Funcția primește ca parametru un vector <code>a = (a</code><sub><code>1</code></sub><code>, a</code><sub><code>2</code></sub><code>, ..., a</code><sub><code>n</code></sub><code>)</code> de lungime <code>n</code> care memorează toate valorile distincte din mulțimea <code>{1, 2,..., n-1}</code>, dar exact una din aceste valori apare în vector de două ori. Funcția trebuie să returneze valoarea care apare de două ori.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
	<li>Numele funcției este <code>PermD</code></li>
	<li>Întotdeauna va exista exact o valoare care apare de două ori</li>
	<li>Elementele din vector sunt indexate de la <code>1</code> la <code>n</code></li>
</ul>

<p>Dacă <code>n = 6</code> și <code>a = (3,5,2,1,2,4)</code> atunci <code>PermD(a, n)</code> va returna valoarea <code>2</code>.</p>

<h3>Important</h3>

<p>Soluţia propusă va conţine definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int PermD(int a[], int n)
{
    long long int s1 = 0;
    for(int i = 1 ; i <= n ; ++i)
        s1+=a[i];
    return s1-(1LL * n * (n - 1) / 2);
}</pre>
