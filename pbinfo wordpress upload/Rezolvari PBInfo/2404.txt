<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>test.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații reprezentând elementele șirului <code>A</code> și apoi pe următoarea linie <code>n</code> numere naturale reprezentând elementele șirului <code>B</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>test.out</code> va conține pe prima linie numărul <code>S</code>, reprezentând rezultatul suma cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>4.294.967.296</code></li>
	<li>desigur că dacă îl veți ajuta Alex vă va răsplăti cu o bere sau cu un suc în funcție de vârstă</li>
</ul>

<h3>Explicație</h3>

<p>S-a calculat <code>2</code><sup><code>1</code></sup><code>+3</code><sup><code>2</code></sup><code>+4</code><sup><code>3</code></sup><code>+5</code><sup><code>4</code></sup><code>+6</code><sup><code>5</code></sup><code>=8476</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("test.in");
ofstream cout ("test.out");

long long putere(long long a , long long p , int mod)
{
    if(p == 0) return 1;
    else
    {
       long long x = putere(a , p / 2 , mod);
       if(p % 2 == 0) return x * x % mod;
       else return x * x % mod * a % mod;
    }
}

int main()
{
    unsigned long long a[1005],b[1005] , s = 0;
    int n;
    cin >> n;
    for(int i = 1 ; i <= n ; i++) cin >> a[i];
    for(int i = 1 ; i <= n ; i++) cin >> b[i];
    for(int i = 1 ; i <= n ; i++) s += putere(a[i],b[i] , 6669666);
    cout << s;
}</pre>
