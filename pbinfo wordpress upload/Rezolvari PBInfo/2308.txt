<p>O proprietate interesanta a fracțiilor ireductibile este ca orice fracție se poate obține după următoarele reguli:<br>
- pe primul nivel se afla fracția <code>1/1</code>;<br>
- pe al doilea nivel, în stânga fracției <code>1/1</code> de pe primul nivel, plasam fracția <code>1/2</code> iar în dreapta ei fracția <code>2/1</code>;</p>

<p>nivelul 1: <code>           1/1</code><br>
nivelul 2: <code>     1/2        2/1</code></p>

<p>- pe fiecare nivel k se plasează sub fiecare fracție <code>i / j</code> de pe nivelul de deasupra, fracția <code>i / (i + j)</code> în stânga, iar fracția <code>(i + j) / j</code> în dreapta.</p>

<p>nivelul 1: <code>             1/1</code><br>
nivelul 2: <code>     1/2               2/1</code><br>
nivelul 3: <code> 1/3     3/2       2/3     3/1</code>  </p>

<h1>Cerința</h1>

<p>Dându-se o fracție oarecare prin numărătorul și numitorul său, determinați numărul nivelului pe care se află fracția sau o fracție echivalentă (având aceeași valoare) cu aceasta.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>fractzii.in</code> conține pe prima linie două numere naturale nenule <code>N M</code>, separate printr-un spațiu, reprezentând numărătorul și respectiv numitorul unei fracții.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>fractzii.out</code>  va conţine o ingură linie, pe care va fi scris număr natural nenul, reprezentând numărul nivelului care corespunde fracţiei.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N, M ≤ 2 000 000 000</code></li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>fractzii.in</code></p>

<pre contenteditable="true" editable="true">13 8</pre>

<p><code>fractzii.out</code></p>

<pre contenteditable="true" editable="true">6</pre>

<h1>Exemplul 2:</h1>

<p><code>fractzii.in</code></p>

<pre contenteditable="true" editable="true">12 8</pre>

<p><code>fractzii.out</code></p>

<pre contenteditable="true" editable="true">3</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("fractzii.in");
ofstream cout("fractzii.out");
int cmmdc(int a , int b)
{
    if(b != 0) return a / b + cmmdc(b , a % b);
    else return 0;
}
int main()
{
    int n ,c;
    cin >> n >> c;
    cout << cmmdc(n ,c );
}</pre>
