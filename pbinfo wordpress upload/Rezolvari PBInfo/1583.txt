<h1>Cerința</h1>

<p>Se citesc de la tastatură <code>n</code> numere naturale. Să se determine numărul de numere prime formate din ultimele <code>2</code> cifre ale fiecărui număr.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>2prim.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale din intervalul <code>[10,10000]</code> separate printr-un spațiu.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>2prim.out</code> va conține numărul de numere care au proprietatea de mai sus.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10000</code></li>
</ul>

<h3>Explicație</h3>

<p><code>2</code><u><code>18</code></u> → <code>18</code> nu este prim<br>
<code>43</code><u><code>33</code></u> → <code>33</code> nu este prim<br>
<code>4</code><u><code>17</code></u> → <code>17</code> este prim<br>
<code>5</code><u><code>13</code></u> → <code>13</code> este prim</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
int prim(int n)
{
    if(n==0 || n==1) return 0;
    if(n==2) return 1;
    if(n%2==0) return 0;
    for(int i=3;i*i<=n;i += 2)
    {
        if(n%i==0) return 0;
    }
    return 1;
}
int main()
{
    ifstream fin("2prim.in");
    ofstream fout("2prim.out");
    int n,cate=0,x;
    fin >> n;
    for(int i=1;i<=n;++i)
    {
        fin >> x;
        if(prim(x%100)) cate++;
    }
    fout << cate;
    fin.close();
    fout.close();
    return 0;
}</pre>
