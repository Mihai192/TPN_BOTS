<h1>Cerința</h1>

<p>Gigel a învăţat la matematică despre cel mai mic multiplu comun a două numere şi acum trebuie să determine pentru fiecare valoare <code>x</code> dintr-un set de valori date câte perechi ordonate de numere naturale <code>(a,b)</code> au cel mai mic multiplu comun <code>x</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură număr <code>n</code>, apoi <code>n</code> valori <code>x</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran <code>n</code> valori, separate prin exact un spaţiu; fiecare valoare afișată reprezintă numărul de perechi care au cel mai mic multiplu comun egal cu valoarea <code>x</code> corespunzătoare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000</code></li>
	<li><code>1 ≤ x ≤ 2.000.000.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Cele  <code>15</code> perechi pentru care cel mai mic multiplu comun este <code>12</code> sunt: <code>(1,12)</code>, <code>(2,12)</code>, <code>(3,4)</code>, <code>(3,12)</code>, <code>(4,3)</code>, <code>(4,6)</code>, <code>(4,12)</code>, <code>(6,4)</code>, <code>(6,12)</code>, <code>(12,1)</code>, <code>(12,2)</code>, <code>(12,3)</code>, <code>(12,4)</code>, <code>(12,6)</code>, <code>(12,12)</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n , x;
    cin >> n;
    for(int i = 0 ; i < n ; ++i)
    {
        cin >> x;
        int d = 2;
        int p = 1;
        while(x > 1)
        {
            if(x % d == 0)
            {
                int e = 0;
                while(x % d == 0)
                    e++ , x/=d;
                p*=(2 * e + 1);
            }
            else d++;
            if(x > 1 &amp;&amp; d*d>x)
            {
                p*=3;
                break;
            }
        }
        cout << p << ' ';
    }
    return 0;
}</pre>
