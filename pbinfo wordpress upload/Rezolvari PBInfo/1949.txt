<h1>Cerința</h1>

<p>Un celebru rezolvitor de pe <code>pbinfo</code> împlinește venerabila vârstă de <code>n</code> ani. Ceilalți rezolvitori s-au gândit să îi facă o surpriză și să comande un tort special. Acesta are mai multe nivele, toate având forma pătrată, astfel:</p>

<ul>
	<li>primul nivel este format din <code>n*n</code> prăjituri fiecare având prețul de <code>1</code> leu și având scris numărul <code>1</code> în partea de sus.</li>
	<li>al doilea nivel se suprapune peste primul exact la mijloc și este format din <code>(n-2)*(n-2)</code> prăjituri fiecare având prețul de <code>2</code> lei și având scris numărul <code>2</code> în partea de sus.</li>
	<li>al treilea nivel se suprapune peste al doilea exact la mijloc și este format din <code>(n-4)*(n-4)</code> prăjituri fiecare având prețul de <code>3</code> lei și având numărul <code>3</code> în partea de sus.</li>
	<li>ș.a.m.d.</li>
</ul>

<p>a) Calculați și afișați numărul de nivele pe care le are tortul special.<br>
b) Afișați tortul, așa cum se vede dacă este privit de sus.<br>
c) Calculați și afișați prețul tortului. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code> reprezentând vârsta celebrului rezolvitor.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran pe prima linie numărul de nivele pe care le are tortul special.<br>
Pe următoarele <code>n</code> linii va fi desenat tortul, iar pe următoarea linie va fi scris prețul tortului.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
</ul>

<h3>Explicație</h3>

<p>Tortul are <code>3</code> nivele. Primul nivel este format din <code>25</code> de prăjituri (fiecare cu prețul de <code>1</code> leu și cifra <code>1</code>), al doilea din <code>9</code> prăjituri (fiecare cu prețul de <code>2</code> lei și cifra <code>2</code>), iar al treilea dintr-o singură prăjitură (cu prețul <code>3</code> lei și cifra <code>3</code>). </p>

<p>Prăjiturile de pe primul nivel costă în total <code>25</code> lei, cele de pe al doilea <code>18</code> lei, iar cele de pe al treilea <code>3</code> lei.</p>

    {
        for(int j = 1 ; j <= n ; ++j)
            cout << a[j];

    }
    for(int i = n / 2 ; i >= 1 ; --i)

        for(int j = i + 1 ; j <= n - i ; ++j) a[j]--;
        for(int j = 1 ; j <= n ; ++j)
            cout << a[j];
