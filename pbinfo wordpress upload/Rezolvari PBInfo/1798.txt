<h1>Cerința</h1>

<p>O persoana are de urcat <code>n</code> trepte. Ştiind că poate păşi pe treapta urmatoare sau poate sări peste o treapta, aflaţi in câte moduri poate urca cele <code>n</code> trepte.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>c</code>, reprezentând numărul de moduri in care poate urca cele <code>n</code> trepte.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 30</code></li>
</ul>

<h3>Explicație</h3>

<p>Persoana poate sări direct pe treapta <code>2</code> sau să păşească câte o treapta.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int n , a[101];

int main()
{
    cin >> n;
    if (n == 1) cout << 1;
    else
    {
        a[0] = 1 , a[1] = 2;
        for(int i = 1 ; i <= n ; i++)
            a[i] = a[i - 1] + a[i - 2];
    
        cout << a[n];
    }

    return 0;
}</pre>
