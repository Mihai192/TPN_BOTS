<h1>Cerința</h1>

<p>Se dă un număr natural <code>n</code> format cu cifre distincte nenule. Să se afle câte numere pare se pot obţine din <code>n</code>, prin rearanjarea cifrelor sale.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul numerelor pare care se pot obţine prin rearanjarea cifrelor lui <code>n</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numerele pare ce se pot obţine prin rearanjarea cifrelor lui <code>n</code> sunt: <code>324,342,234,432</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    int n , cnt = 0 , c= 0 , p = 1;
    cin >> n;
    while(n != 0)
    {
        c++;
        if(n % 10 % 2 == 0) cnt++;
        n /= 10;
    }
    for(int i = 1 ; i < c ; ++i) p *= i;
    cout << p * cnt;
    return 0;
}</pre>
