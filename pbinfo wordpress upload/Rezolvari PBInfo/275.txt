<h1>Cerinţa</h1>

<p>Se dă un text în care cuvintele sunt formate din litere mici ale alfabetului englez şi cifre şi sunt separate prin spaţii şi semne de punctuaţie. Să se determine perechea de vocale alăturate din text care apare de cele mai multe ori.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>perechivocale1.in</code> conţine textul dat, situat pe mai multe linii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>perechivocale1.out</code> va conţine pe prima linie perechea de vocale găsită. </p>

<p>Dacă există mai multe asemenea perechi de vocale, se vor afişa toate, în ordine lexicografică şi separate printr-un spaţiu. </p>

<p>Dacă în text nu există nici o pereche de vocale consecutive, se va afişa mesajul “NU”.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>cuvintele din text sunt formate din cel mult <code>40</code> caractere;</li>
</ul>

<h1>Explicație</h1>

<p>Perechile <code>ea</code> şi <code>oa</code> apar de trei ori. Celelalte perechi de vocale din text apar de mai puţine ori.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>

using namespace std;

char pv[ ][3] = {"aa","ae","ai","ao", "au", "ea", "ee", "ei", "eo", "eu", "ia", "ie", "ii", "io", "iu", "oa", "oe", "oi", "ou", "ua","oo", "ue", "ui", "uu", "uo" };
int c[26];
char voc[] = {'a','e','i','o','u'};
char vo[3];

ifstream fin ("perechivocale1.in");
ofstream fout ("perechivocale1.out");

int main()
{
    int n;
    char x[41];
    int vmax=0;
    while (fin >> x)
    {
        n=strlen(x);
        for (int i=0; i<n-1; i++)
        {
            if (strchr(voc, x[i]) &amp;&amp; strchr(voc, x[i+1]))
            {
                vo[0]=x[i];
                vo[1]=x[i+1];
                vo[2]='\0';
                for (int j = 0; j<25; j++)
                {
                    if (strcmp(vo, pv[j])==0)c[j]++;

                }
            }
        }
    }

    for (int i = 1; i<=25; i++)
        if (c[i]>vmax)vmax=c[i];

    for (int i = 1; i<=25; i++)
        if (c[i]==vmax)fout << pv[i] << " ";
    return 0;
}</pre>
