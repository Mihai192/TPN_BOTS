<h1>Cerința</h1>

<p>În arta tipografică <strong>un râu</strong> este o înșiruire de spații care se întinde pe verticală, pe liniile consecutive ale unui text. Mai precis, <strong>un râu</strong> este definit ca o secvență de cel puțin <code>2</code> spații în care oricare <code>2</code> spații de pe linii consecutive sunt adiacente pe verticală sau diagonală. </p>

<p>Să se scrie un program care citește numerele naturale <code>L N</code> și apoi un text formatat pe <code>N linii</code> de cel mult <code>L caractere</code> (vezi  <a class="text-success bg-success" href="/probleme/2682/lungimerandmax">#LungimeRandMax <sup>[100]</sup></a> ), și afișează lungimea celui mai lung râu din text, dacă acesta există sau mesajul <code>NU</code>, dacă nu există niciun râu.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>L</code> și <code>N</code>, iar apoi <code>n</code> rânduri formate din cel mult <code>L</code> caractere.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran lungimea celui mai lung râu, sau <code>NU</code>, după caz.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N ≤ L ≤ 100</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <cstring>
#include <queue>
using namespace std;

char a[101][101];
char s[101];
int cnt = 0, l[101], c, n, b[101][101];

bool inmat(int i, int j){
    return i >= 1 &amp;&amp; i <= n &amp;&amp; j < l[i] &amp;&amp; j >= 0;
}

int di[]={-1,1,-1,1,-1,1};
int dj[]={0,0,1,1,-1,-1};

void fill(int i, int j){
    b[i][j] = 1;
    queue<pair<int, int>> Q;
    Q.push(make_pair(i, j));
    while(!Q.empty()){
        pair<int, int> x = Q.front();
        c++;
        for(int d = 0; d <= 5; ++d){
            int inou = x.first + di[d];
            int jnou = x.second + dj[d];
            if(inmat(inou, jnou) &amp;&amp; a[inou][jnou] == ' ' &amp;&amp; !b[inou][jnou])
                b[inou][jnou] = b[x.first][x.second] + 1, Q.push(make_pair(inou, jnou));
        }
        Q.pop();
    }
}

int main(){
    int k;
    cin >> k >> n;
    cin.get();
    for(int i = 1; i <= n; ++i){
        cin.getline(s, 101);
        int j = 0;
        while(s[j]){
            a[i][j] = s[j];
            j++;
        }
        l[i] = strlen(s);
    }
    int maxi = 0, x , y;
    for(int i = 1; i <= n; ++i)
        for(int j = 0; j < l[i]; ++j)
            if(a[i][j] == ' ' &amp;&amp; !b[i][j])
                fill(i, j);
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j < l[i]; ++j)
            if(b[i][j] > maxi)
                maxi = b[i][j];
    cout << maxi;
    return 0;
}</pre>
