<h1>Cerinţa</h1>

<p>Se citește un număr natural <code>n</code>. Să se determine suma divizorilor pari ai săi.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>S</code>, reprezentând suma divizorilor pari ai lui <code>n</code>.</p>

<h1>Explicație</h1>

<p>Divizorii pari lui <code>30</code> sunt <code>2</code>, <code>6</code>, <code>10</code> și <code>30</code>. Suma lor este <code>48</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main(){
    int n;
    cin >> n;
    long long int sum = 0;
    for(int d = 1; d * d <= n; ++d){
        if(n % d == 0){
            if(d % 2 == 0)
                sum += d;
            if(n / d % 2 == 0)
                sum += n/d;
        }
        if(d * d == n &amp;&amp; d % 2 == 0)
            sum -= d;
    }
    cout << sum;
    return 0;
}</pre>
