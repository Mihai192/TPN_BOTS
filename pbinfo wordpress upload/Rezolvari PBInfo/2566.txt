<p>Moș Crăciun care-i darnic și e bun, văzând că ministerul de resort nu se grăbește să doteze școlile cu tablete și laptop-uri, ia decizia să doteze un număr cât mai mare de școli cu aceste unelte atât de necesare elevilor. El are la dispoziție un număr <code>t</code> de tablete și un număr <code>l</code> de laptopuri. Cum moșul nu vrea să nedreptățească nici o școală, el se gândește să pună în toate pachetele pe care le face același număr de tablete și același număr de laptop-uri.</p>

<h1>Cerința</h1>

<p>Scrieți un program care determină numărul maxim de școli ce vor primi pachete, precum și câte tablete și laptop-uri se găsesc în fiecare pachet.</p>

<h1>Date de intrare</h1>

<p>De pe prima linie din fișierul de intrare  <code>daruri.in</code> se citesc două numere naturale <code>t</code> și <code>l</code> separate printr-un spațiu, având semnificația: <code>t</code> este numărul de tablete și <code>l</code> numărul de laptop-uri pe care le are moșul în stoc.</p>


<h1>Date de ieșire</h1>

<p>Pe prima linie în fișierul <code>daruri.out</code> se va afișa un număr ce reprezintă numărul maxim de școli ce vor primi daruri de la Moș Crăciun, iar pe următoarea linie se vor afișa două numere separate printr-un spațiu reprezentând numărul de tablete și numărul de laptop-uri care intră în componența unui pachet. Dacă moșul nu reușește să facă cel puțin <code>2</code> pachete, el renunță la ideea sa și în fișierul de ieșire se va afișa pe primul rând <code>0</code>, iar pe al doilea rând <code>0 0</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1≤t,l≤1000</code></li>
</ul>

<h1>Exemplul 1</h1>

<p><code>daruri.in</code></p>

<pre contenteditable="true" editable="true">10 15</pre>

<p><code>daruri.out</code></p>

<pre contenteditable="true" editable="true">5
2 3</pre>

<h3>Explicație</h3>

<p>Se pot forma <code>5</code> pachete, fiecare pachet conținând <code>2</code> tablete și <code>3</code> laptopuri.</p>

<h1>Exemplul 2</h1>

<p><code>daruri.in</code></p>

<pre contenteditable="true" editable="true">12 25</pre>

<p><code>daruri.out</code></p>

<pre contenteditable="true" editable="true">0
0 0</pre>

<h3>Explicație</h3>

<p>Moșul nu reușește să facă cel puțin două pachete.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("daruri.in");
ofstream cout("daruri.out");

int main()
{
    int l , t , sc;
    cin >> l >> t;
    int cl=l , ct=t;
    while(t)
    {
        int r=l%t;
        l=t;
        t=r;
    }
    sc=l;
    if(sc>1)
    {
        cout << sc;
        cout << endl;
        cout << cl/sc << ' ' << ct/sc;
    }
    else
    {
        cout << 0 << endl;
        cout << 0 << ' ' << 0;
    }
    return 0;
}</pre>
