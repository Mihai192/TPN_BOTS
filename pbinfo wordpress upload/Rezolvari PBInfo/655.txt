<h1>Cerința</h1>

<p>Se dau două șiruri <code>a</code> și <code>b</code> cu câte <code>n</code> elemente, numere naturale din mulțimea <code>{1,2}</code>. Se dorește transformarea șirului <code>a</code> în șirul <code>b</code>, având la dispoziție următoarele operații:</p>

<ul>
	<li>interschimbarea a două elemente</li>
	<li>înlocuirea unui element <code>1</code> cu <code>2</code> sau a unui element <code>2</code> cu <code>1</code>.</li>
</ul>

<p>Determinați pentru cele două șiruri numărul minim de operații prin care se transformă șirul <code>a</code> în șirul <code>b</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, apoi cele <code>n</code> elemente ale șirului <code>a</code>, apoi cele <code>n</code> elemente ale șirului <code>b</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul minim ce operații <code>C</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
</ul>

<h1>Explicație</h1>

<p>Prin interschimbarea în primul șir a elementului de pe poziția <code>3</code> cu cel de pe poziția <code>5</code> se obține al doilea șir.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int n , a[1005] , b[1005] , cnt1 , cnt2;
int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i ++) cin >> a[i];
    for(int i = 1 ; i <= n ; i ++) cin >> b[i];
    for(int i = 1 ; i <= n ; ++i)
        if(a[i] != b[i])
            if(a[i] == 1) cnt1 ++;
            else cnt2 ++;
    cout << max(cnt1 , cnt2);
    return 0;
}</pre>
