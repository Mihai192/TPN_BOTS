<p>Cristina şi Alina sunt eleve în clasa a V-a şi sunt foarte bune prietene. Le place ca în pauze să se provoace reciproc cu câte o problemă. De data aceasta, e rândul Cristinei să propună o problemă Alinei. Ea îi cere ca dintr-un set de mai multe numere naturale să le găsească pe cele centrale. Bineînţeles că mai întâi îi explică prietenei sale ce este un număr central: un număr care are proprietatea ca, după eliminarea primei şi a ultimei cifre, se obţine un nou număr care conţine numai cifre egale între ele. De exemplu, numărul <code>67771</code> este număr central pentru că, eliminând prima şi ultima cifră, se obţine numărul <code>777</code> care are toate cifrele egale între ele. Alina, care între timp a învăţat să programeze, intră imediat în jocul Cristinei, ştiind că va afla imediat rezultatul corect la problema propusă de prietena ei.</p>

<h1>Cerința</h1>

<p>Având la dispoziţie un set de numere pe care le primeşte pentru verificare, Alina trebuie să spună câte dintre acestea sunt numere centrale.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>centrale.in</code> conţine pe prima linie, numărul natural <code>N</code> care reprezintă numărul de numere ce trebuie verificate. Pe următoarea linie se găsesc cele <code>N</code> numere naturale, separate prin câte un spaţiu.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>centrale.out</code> va conţine o singură linie pe care va fi scris numărul de numere centrale găsite între cele <code>N</code> numere ce trebuie verificate.</p>

<h1>Restricții și precizări</h1>

<p>• <code>1 ≤ N ≤ 100</code><br>
• Fiecare număr din setul dat are cel puţin <code>3 cifre</code> şi cel mult <code>9 cifre</code></p>

using namespace std;
ifstream fin("centrale.in");
ofstream fout("centrale.out");
int pat(int n)
{
    int n1 = n / 10 , pc = n1 % 10 , ok = 0;
    while(n1 > 9)
    {
        if(n1 % 10 != pc) ok = 1;
        n1 /= 10;
    }
    if(ok == 0) return 1;
    else return 0;
}
int main()
{
    long long n , x , cnt = 0;
    fin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        fin >> x;
        if(pat(x)) cnt++;
    }
    fout << cnt;

    fin.close();
    fout.close();

    return 0;
