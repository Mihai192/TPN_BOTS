<h1>Cerinţa</h1>

<p>Să se scrie un program care citește două numere naturale <code>a</code> și <code>b</code> și determină câte numere prime sunt în intervalul închis determinat de <code>a</code> și <code>b</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>a</code> și <code>b</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>n</code>, reprezentând numărul de numere prime din intervalul închis determinat de <code>a</code> și <code>b</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0&lt; a,b &lt;100000</code></li>
	<li>numerele citite nu respectă obligatoriu relația <code>a ≤ b</code></li>
	<li>un număr natural este <strong>prim</strong> dacă are exact doi divizori distincți: <code>1</code> și el însuși.</li>
</ul>

<h1>Explicație</h1>

<p>în intervalul <code>[10,20]</code> sunt <code>4</code> numere prime: <code>11</code>, <code>13</code>, <code>17</code>, <code>19</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int prim(int n);

int main ()
{
    int a,b,cnt=0;
    cin >> a >> b;
    if (a > b)
    {
        int c=a;
        a=b;
        b=c;
    }
    for (int i = a ; i <= b; ++i)
    {
        if (prim(i))
        cnt++;
    }
    cout << cnt;
    return 0;
}
int prim(int n)
{
    int cnt=0; 
    for (int i = 1; i * i <= n; ++i)
    {
        if (n%i==0)
        cnt+=2;
        if (i*i==n)
        cnt--;
    }
    if (cnt==2)
    return 1;
    else
    return 0;
}</pre>
