<p>Se dă un șir de <code>n</code> cuvinte, fiecare format din cel mult <code>30</code> caractere.</p>

<h1>Cerinţa</h1>

<p>Să se determine numărul cuvintelor care conțin doar vocale.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>doarvocale.in</code> conţine pe prima linie numărul <code>n</code>, iar pe <code>n</code> următoarele <code>n</code> linii câte un cuvânt.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>doarvocale.out</code> va conţine pe prima linie numărul <code>X</code>, reprezentând numărul cuvintelor care conțin doar vocale.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 1000</code></li>
	<li>cuvintele vor conține doar litere mici ale alfabetului englez</li>
	<li>se consideră vocale caracterele din mulțimea <code>{a,e,i,o,u}</code></li>
</ul>

<h1>Explicație</h1>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("doarvocale.in");
ofstream cout("doarvocale.out");

int main()
{
    char ch[300];
    int n;
    cin >> n;
    int cnt=0;
    for(int k = 0 ; k < n ; ++k)
    {
        cin >> ch;
        int i = 0;
        bool adev=true;
        while(ch[i]!='\0')
        {
            if(ch[i]!='a' &amp;&amp; ch[i]!='e' &amp;&amp; ch[i]!='i' &amp;&amp; ch[i]!='o' &amp;&amp; ch[i]!='u')
                adev=false;
            i++;    
        }
        if(adev)
            cnt++;
        
    }
    cout << cnt;
    return 0;    
}</pre>
