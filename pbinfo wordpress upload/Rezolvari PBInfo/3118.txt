<h1>Cerința</h1>

<p>Un tort dreptunghiular de dimensiuni <code>MxN</code> trebuie împărţit în porţii pătrate de aceeaşi mărime. Găsiţi numărul minim de porţii care se pot obţine şi dimensiunea <code>L</code> a acestora. Atât dimensiunile dreptunghiului cât şi ale pătratelor în care se împarte sunt numere întregi.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>tort.in</code> va conţine pe prima linie numerele <code>M</code> şi <code>N</code> separate printr-un spaţiu. </p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>tort.out</code> va conţine pe o singură linie, două numere naturale separate printr-un spaţiu, primul fiind numărul minim de porţii, iar celălalt dimensiunea <code>L</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ M ≤ N ≤ 10000</code></li>
</ul>

using namespace std;

ifstream cin("tort.in");
ofstream cout("tort.out");

int cmmdc(int a, int b){
    while(b){
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main(){
    int n, m;
    cin >> n >> m;
    int prod = n * m;
    int cm = cmmdc(n, m);
    cout << prod / (cm * cm) << ' ' << cm;
    return 0;
