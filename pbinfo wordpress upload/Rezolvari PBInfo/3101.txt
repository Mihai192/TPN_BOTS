<h1>Cerința</h1>

<p>Se citește un text de maxim <code>256</code> caractere și o cheie de număr egal de caractere cu textul citit. După aplicarea <code>Encripției XOR</code>, caracterele rezultate se vor afișa în <code>format binar</code> deoarece în majoritatea cazurilor se obțin caractere neprintabile.</p>

<p><code>Encripția XOR</code> presupune aplicarea operatorului pe biți <code>XOR</code> între text și cheie, între fiecare caracter din text și caracterul din cheie de pe aceeași poziție cu cel din text: <code>text[i] XOR cheie[i]</code>. Pentru decriptare, operatorul se aplică între cheie și textul criptat.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>xorcrypt.in</code> conține:<br>
</p><ol>
	<li>Pe prima linie un text format din maxim <code>256</code> caractere.</li>
	<li>Pe a doua linie cheia, de lungime egală cu textul.</li><br>
</ol><p></p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>xorcrypt.out</code> va conține o singură linie, având caracterele în <code>format binar</code> separate prin spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Se va folosi alfabetul englez.</li>
</ul>

#include <cstring>
using namespace std;

ifstream cin("xorcrypt.in");
ofstream cout("xorcrypt.out");

char a[256] , b[256] , rez[8];
int n , x , y;

void schimb(int n)
{
    for(int i = 0 ; i < 8 ; i++)
        rez[i] = '0';

    int ind = 7;
    while(n != 0)
    {
        rez[ind] = n % 2 + '0';
        ind--;
        n /= 2;
    }
    cout << rez << " ";
}

int main()
{
    cin.getline(a , 260);
    cin.getline(b , 260);
    n = strlen(a);
    for(int i = 0 ; i < n ; i++)
    {
        x = int(a[i]) , y = int(b[i]);
        schimb(x^y);
    }
    return 0;
