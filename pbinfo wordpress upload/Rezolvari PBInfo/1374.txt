<h1>Enunt</h1>

<p>Mircea este pasionat de programare. El a început să rezolve probleme din ce în ce mai grele. Astfel a ajuns la o problemă, care are ca date de intrare un tablou pătratic cu <code>n</code> linii şi <code>n</code> coloane, componente tabloului fiind toate numerele naturale distincte de la <code>1</code> la <code>n</code><sup><code>2</code></sup>.  Pentru a verifica programul pe care l-a scris îi trebuie un fişier care să conţină tabloul respectiv. După ce a creat acest fişier, fratele său, pus pe şotii îi umblă în fişier şi îi schimbă câteva numere consecutive, cu numărul <code>0</code>. Când se întoarce Mircea de la joacă constată cu stupoare că nu îi merge programul pentru testul respectiv. După câteva ore de depanare îşi dă seama că programul lui este corect şi că fişierul de intrare are probleme. </p>

<h1>Cerința</h1>

<p>Scrieţi un program care să-l ajute pe Mircea, găsindu-i cel mai mic şi cel mai mare dintre numerele consecutive schimbate de fratele său.</p>

<h1>Date de intrare</h1>

<p>În fişierul <code>numere9.in</code> se dă pe prima linie <code>n</code>, iar pe următoarele <code>n</code> linii elementele tabloului, câte <code>n</code> elemente pe o linie, separate între ele prin câte un spaţiu, după modificările făcute de fratele lui Mircea.</p>

<h1>Date de ieșire</h1>

<p>În fişierul <code>numere9.out</code> se va scrie pe un singur rând cu un singur spaţiu între ele numerele cerute (primul fiind cel mai mic).</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0&lt;n≤500</code></li>
	<li>Fratele lui Mircea schimbă cel puţin un număr în fişier.</li>
	<li>Numerele schimbate de fratele lui Mircea sunt mai mici sau cel mult egale cu <code>60000</code>.</li>
</ul>

<h3>Explicație</h3>

<p>În fişierul de intrare au fost înlocuite cu <code>0</code> numerele <code>2</code>, <code>3</code>, <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("numere9.in");
ofstream cout("numere9.out");
int main()
{
    int n , x , p = 0 , f[60000]={0} , ok = 0 , nr1 , nr2;
    cin >> n;
    for(int i = 0 ; i < n ; ++i)
    {
        for(int j = 0 ; j < n ; ++j)
        {
            cin >> x;
            if(x > p &amp;&amp; x<= 60000) p = x;
            if(x <= 60000)f[x]++;
        }
    }
    if(p < n * n &amp;&amp; p != 60000) p = n * n;
    for(int i = 1 ; i <= p ; ++i)
    {
        if(f[i]==0 &amp;&amp; ok==0)
        {
            nr1 = i;
            ok++;
        }
        if(f[i]==0) nr2 = i;
    }
    cout << nr1 << " " << nr2;
}</pre>
