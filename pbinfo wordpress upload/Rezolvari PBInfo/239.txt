<h1>Cerinţa</h1>

<p>Se dau mai multe numere naturale cu cel mult <code>9</code> cifre fiecare. Să se determine cele mai mari două numere de <code>3</code> cifre care nu se găsesc printre numerele date.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>nrlipsa.in</code> conţine cel mult <code>100000</code> de numere, dispuse pe mai multe linii, numerele de pe fiecare linie fiind separate prin unul sau mai multe spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>nrlipsa.out</code> va conţine pe prima linie cele mai mari două numere de trei cifre care nu apar  în fişierul de intrare, ordonate descrescător, separate prin exact un spaţiu.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>dacă nu există două numere de trei cifre care să nu se afle în fişierul de intrare atunci se va afişa mesajul <code>NU</code>.</li>
</ul>

    int f[1000001]={0};

            f[x]++;
    }

        if(f[i]==0 &amp;&amp; ok < 2)

            {

            }
            if(ok==1)
            {
                b=i;
            }
            ok++;
        }
    }
    if(ok==2)
        fout << a << ' ' << b;
    if(ok==0)
        fout << "NU";
    else if(ok < 2)
        fout << "NU EXISTA";
    return 0;
