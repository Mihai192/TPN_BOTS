<p>Fie șirul Fibonacci dat prin <code>F</code><sub><code>1</code></sub> <code>= 1</code>, <code>F</code><sub><code>2</code></sub> <code>= 1</code> și relația de recurență <code>F</code><sub><code>k</code></sub> <code>= F</code><sub><code>k-1</code></sub> <code>+ F</code><sub><code>k-2</code></sub>, <code>k ≥ 3</code>. Se consideră un număr natural <code>N</code>.</p>

<h1>Cerința</h1>

<p>Să se scrie un program care determină numărul <code>F</code> al fracțiilor diferite ireductibile subunitare, ce se pot forma utilizând primii <code>N</code> termeni ai șirului Fibonacci.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>fibofrac.in</code> conține pe prima linie numărul <code>N</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>fibofrac.out</code> va conține pe prima linie numărul <code>F</code>, cu semnificația de mai sus.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Pentru teste în valoare de <code>24</code> puncte, <code>0 &lt; N &lt; 80</code></li>
	<li>Pentru teste în valoare de <code>40</code> puncte, <code>0 &lt; N &lt; 1101</code></li>
	<li>Pentru teste în valoare de <code>56</code> puncte, <code>0 &lt; N &lt; 50001</code></li>
	<li>Pentru teste în valoare de <code>100</code> puncte, <code>0 &lt; N &lt; 1.000.000</code></li>
	<li>Două fracții ireductibile <code>a / b</code> și <code>c / d</code> sunt diferite dacă <code>a ≠ c</code> sau <code>b ≠ d</code>.</li>
	<li><code>0 ≤ F &lt; 2</code><sup><code>63</code></sup></li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>fibofrac.in</code></p>

<pre contenteditable="true" editable="true">7</pre>

<p><code>fibofrac.out</code></p>

<pre contenteditable="true" editable="true">14</pre>

<h3>Explicație</h3>

<p><code>N = 7</code>; Primii <code>7</code> termeni ai șirului Fibonacci sunt: <code>1</code>, <code>1</code>, <code>2</code>, <code>3</code>, <code>5</code>, <code>8</code>, <code>13</code>. Se pot forma <code>14</code> fracții diferite ireductibile subunitare: <code>1/2</code>, <code>1/3</code>, <code>1/5</code>, <code>1/8</code>, <code>1/13</code>, <code>2/3</code>, <code>2/5</code>, <code>2/13</code>, <code>3/5</code>, <code>3/8</code>, <code>3/13</code>, <code>5/8</code>, <code>5/13</code>, <code>8/13</code>.</p>

<h1>Exemplul 3:</h1>

<p><code>fibofrac.in</code></p>

<pre contenteditable="true" editable="true">500000</pre>

<p><code>fibofrac.out</code></p>

<pre contenteditable="true" editable="true">94988288219</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("fibofrac.in");
ofstream cout("fibofrac.out");
int E[1000001], n;
long long int t;
int main(){
    cin >> n;
    for(int i = 1; i <= n; ++i)
        E[i] = i - 1;
    E[1] = 1;
    for(int i = 2; i <= n; ++i)
        for(int j = 2 * i; j <= n; j = j + i)
            E[j] = E[j] - E[i];
    for(int i = 1; i <= n; ++i)
        if(i <= n / 2)
            t += 2 * E[i];
        else
            t += E[i];
    cout << t - n - 1;
    return 0;
}</pre>
