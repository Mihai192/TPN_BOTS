<h1>A1Z26</h1>

<p>Cifrul A1Z26 este cel mai ușor cifru de substituție: fiecare literă este înlocuită cu numărul ei de ordine în alfabet. Astfel, <code>A</code> devine <code>1</code>, <code>B</code> devine <code>2</code>, ș.a.m.d.</p>

<h1>Cerința</h1>

<p>Se dă un număr <code>p</code>:<br>
1. Dacă <code>p = 1</code>, se dă un șir de caractere <code>s</code> format din litere mari. Să se afișeze numărul de ordine al caracterelor separate printr-un spațiu.<br>
2. Dacă <code>p = 2</code>, se dă un număr <code>n</code> și <code>n</code> numere cuprinse între <code>1</code> și <code>26</code>. Să se afișeze șirul corespunzător numerelor.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>a1z26.in</code> conține pe prima linie numărul <code>p</code>.<br>
Dacă <code>p = 1</code>, pe a doua linie din fișier se află șirul de caractere <code>s</code>.<br>
Dacă <code>p = 2</code>, pe a doua linie din fișier se află numarul natural <code>n</code> iar pe următoarea linie se află <code>n</code> numere.</p>

<h1>Date de ieșire</h1>

<p>Dacă <code>p = 1</code>, fișierul de ieșire <code>a1z26.out</code> va conține pe prima linie numerele de ordine ale caracterelor din șirul <code>s</code>, separate prin câte un spațiu.</p>

<p>Daca <code>p = 2</code>, fișierul de ieșire <code>a1z26.out</code> va conține pe prima linie șirul format din numerele de ordine date, afișate cu litere mari.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ p ≤ 2</code></li>
	<li><code>1 ≤ lungimea șirului s ≤ 1.000.000</code></li>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
	<li><code>1 ≤ numerele de pe a doua linie ≤ 26</code></li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>a1z26.in</code></p>

<pre contenteditable="true" editable="true">1
ANAAREMERE</pre>

<p><code>a1z26.out</code></p>

<pre contenteditable="true" editable="true">1 14 1 1 18 5 13 5 18 5</pre>

<h1>Exemplul 2:</h1>

<p><code>a1z26.in</code></p>

<pre contenteditable="true" editable="true">2
10
1 14 1 1 18 5 13 5 18 5</pre>

<p><code>a1z26.out</code></p>

<pre contenteditable="true" editable="true">ANAAREMERE</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("a1z26.in");
ofstream cout("a1z26.out");

int main()
{
    int p;
    cin >> p;
    if(p == 1)
    {
        char ch;
        while(cin >> ch)
            cout << (int) (ch - 'A' + 1) << ' ';
    }
    else
    {
        int n;
        cin >> n;
        int x;
        for(int i = 1 ; i <= n; ++i)
            cin >> x, cout << (char) (x + 'A' - 1);
    }
    return 0;   
}</pre>
