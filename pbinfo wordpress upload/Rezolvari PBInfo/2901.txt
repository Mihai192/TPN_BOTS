<p>În vacanța de iarnă Adelina a decis să renunțe la teme și a ales să citească o parte din cărțile aduse de Moș Crăciun: cele cu teme istorice sau SF. Pe măsură ce citea își nota în caiet datele calendaristice ale acelor evenimente care au impresionat-o. După vacanță îi povestește colegului de bancă despre aceste evenimente și remarcă faptul că unele date calendaristice au o formă specială, de palindrom: citite de la dreapta la stânga reprezintă aceeași dată ca atunci când se citesc obișnuit, de la stânga la dreapta.</p>

<h1>Cerința</h1>

<p>Pentru cele <code>n</code> date calendaristice din agenda Adelinei numărați și afișați câte date palindromice au fost găsite iar apoi precizați secolele cu cele mai multe date palindromice.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>datapal.in</code> conține următoarele informații :<br>
- pe prima linie un număr <code>n</code> reprezentând numărul de date calendaristice;<br>
- pe fiecare dintre următoarele <code>n</code> linii câte o dată calendaristică de forma <code>zz/ll/aaaa</code> (două cifre pentru zi, două cifre pentru lună, patru cifre pentru an) .</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>datapal.out</code> va conține pe prima linie numărul de date calendaristice palindrom iar pe a doua linie secolele cu cele mai multe date palindrom, în ordine cronologică.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0&lt; n ≤ 1000</code></li>
	<li>toate datele calendaristice sunt valide;</li>
	<li>în colecția de date de test există întotdeauna cel puțin o dată palindromică</li>
	<li>anii aparțin perioadei 10-9999 și sunt d.Hr.</li>
</ul>

<h3>Explicație</h3>

<p>Sunt <code>4</code> date palindromice: două în secolul <code>12</code> și două în secolul <code>22</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <map>
using namespace std;

ifstream cin("datapal.in");
ofstream cout("datapal.out");

string s;

map<string , int> M;

int main()
{
    int n;
    cin >> n;
    int cnt = 0;
    for(int i = 1 ; i <= n ; ++i)
    {
        cin >> s;
        bool ok = true;
        int x = s.length()-1;
        for(int j = 0 ; j <= x ; ++j)
            if(s[j]!=s[x-j])
                ok=false;
        if(ok)
            cnt++ , M[s.substr(4 , 2)]++;
    }
    cout << cnt << endl;
    int fmax = 0;
    for(auto x:M)
        if(x.second > fmax)
            fmax = x.second;
    for(auto x:M)
    {
        int nr = 0;
        string t = x.first;
        int i = 0;
        while(t[i])
        {
            nr = nr * 10 + t[i]-'0';
            i++;
        }
        if(x.second == fmax)
        cout << nr + 1 << ' ';
    }
    return 0;
}</pre>
