<h1>Cerinţa</h1>

<p>Se dau două numere naturale nenule. Să se verifice dacă cele două numere au exact aceiași factor primi, indiferent de puterea acestora.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> și <code>m</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran mesajul <code>DA</code>, dacă cele două numere au exact aceiași factori primi, respectiv <code>NU</code> în caz contrar.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>cele două numere citite vor fi mai mici decât 2 <sup>30</sup></li>
</ul>

<h1>Exemplu 1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">72 48</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">DA</pre>

<h1>Explicație</h1>

<p><code>72</code> și <code>48</code> au exact aceiași factori primi: <code>2 3</code>.</p>


<h1>Exemplu 2:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">72 30</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">NU</pre>

<h1>Explicație</h1>

<p>Factorii primi ai lui <code>72</code> sunt <code>2 3</code>, iar factorii primi ai lui <code>30</code> sunt <code>2 3 5</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    bool aceiasi_factori = true;
    int d = 2;
    while(a > 1){
        int p = 0;
        if(a % d == 0){
            if(b % d != 0){
                aceiasi_factori = false;
            }
            while(a % d == 0)
                a /= d;
            while(b % d == 0)
                b /= d;
        }
        d++;
        if(d * d > a)
            d = a;
    }
    if(aceiasi_factori &amp;&amp; b == 1)
        cout << "DA";
    else
        cout << "NU";
    return 0;
}</pre>
