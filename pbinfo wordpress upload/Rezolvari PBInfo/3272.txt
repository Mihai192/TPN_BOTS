<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Determinați suma divizorilor oglinditelor celor <code>n</code> numere.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>S</code>, reprezentând suma cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Oglinditul lui <code>12</code> este <code>21</code>; suma divizorilor săi este  <code>32</code>. Oglinditul lui <code>23</code> este <code>32</code>; suma divizorilor săi este  <code>63</code>. Oglinditul lui <code>30</code> este <code>3</code>; suma divizorilor săi este  <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;
    unsigned long long suma = 0;
    for(int i = 1; i <= n; ++i){
        int x;
        cin >> x;
        int ogl = 0;
        while(x){
            ogl = ogl * 10 + x % 10;
            x/=10;
        }
        for(int d = 1; d * d <= ogl; ++d){
            if(ogl % d == 0)
                suma += d, suma += ogl/d;
            if(d * d == ogl)
                suma -= d;
        }
    }
    cout << suma;
    return 0;
}</pre>
