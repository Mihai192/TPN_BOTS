<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a subprogramului C++ <strong>recursiv</strong> <code>num</code> care are <code>2</code> parametri: <code>n</code> – prin care primește un număr natural și <code>v</code>, prin care primeşte un tablou unidimensional cu <code>n</code> elemente, numere întregi cu cel mult <code>4</code> cifre.</p>

<p>Subprogramul înlocuieşte cu <code>0</code> fiecare valoare mai mică sau egală cu prima valoare din tablou. Tabloul modificat este furnizat tot prin parametrul <code>v</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>2&lt;n&lt;50</code></li>
	<li>numele subprogramului cerut este <code>num</code></li>
	<li>parametrii sunt, în această ordine: <code>n</code>, <code>v</code></li>
	<li>elementele tabloului <code>v</code> sunt indexate de la zero</li>
	<li><strong>se recomandă realizarea unui subprogram recursiv</strong></li>
</ul>

<p>Dacă <code>n=7</code> şi <code>v=(4, 5, 0, 9, 3, 4, -2)</code>, atunci după apel <code>v=(0, 5, 0, 9, 0, 0, 0)</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void num(int n , int v[])
{
    int val=v[0];
    for(int i = 0 ; i < n ; ++i)
    {
        if(v[i] <= val)
            v[i]=0;
    }
}</pre>
