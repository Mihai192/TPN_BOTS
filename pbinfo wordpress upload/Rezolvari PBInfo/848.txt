<h1>Cerința</h1>

<p>Se dau <code>n</code> șiruri de paranteze rotunde. Să se stabilească, despre fiecare șir, dacă este corect parantezat – adică dacă parantezele se închid corect.</p>

<p>Un șir de paranteze <code>S</code> rotunde este corect parantezat dacă:</p>

<ul>
	<li><code>S</code> este șirul vid, sau</li>
	<li><code>S = (T)</code> și <code>T</code> este corect parantezat, sau</li>
	<li><code>S = AB</code>, iar <code>A</code> și <code>B</code> sunt corect parantezate.</li>
</ul>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>paranteze1.in</code> conține pe prima linie numărul <code>n</code>, pe următoarele <code>n</code> linii câte un șir de paranteze rotunde.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>paranteze1.out</code> va conține <code>n</code> linii: fiecare linie va conține valoarea <code>1</code>, dacă șirul corespunzător de paranteze este corect parantezat și <code>0</code> în caz contrar.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>fiecare șir va avea cel mult <code>255</code> de paranteze</li>
</ul>

#include <cstring>
using namespace std;

ifstream fin("paranteze1.in");
ofstream fout("paranteze1.out");

int main()
{
    int n , k = 0;
    char sir[300];
    int ok=0;
    fin >> n;
    for(int i = 0 ; i < n ; ++i)
    {
        fin >> sir;
        k=0;
        ok=1;
        for(int j = 0 ; j < strlen(sir) &amp;&amp; ok ; ++j)
            if(sir[j]=='(')
                k++;
            else if(sir[j]==')' &amp;&amp; k==0)
                ok=0;
            else
                k--;
        if(ok &amp;&amp; k==0)
            fout << 1 << endl;
        else
            fout << 0 << endl;            
    }
    return 0;
