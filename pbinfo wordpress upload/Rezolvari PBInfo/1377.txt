<h1>Enunt</h1>

<p>Fiind elev în clasa <code>a IX-a</code>, George, îşi propune să studieze capitolul divizibilitate cât mai bine. Ajungând la numărul de divizori asociat unui număr natural, constată că sunt numere într-un interval dat, cu acelaşi număr de divizori.</p>

<p>De exemplu, în intervalul <code>[1, 10]</code>, <code>6</code>, <code>8</code> şi <code>10</code> au acelaşi număr de divizori, egal cu <code>4</code>. De asemenea, <code>4</code> şi <code>9</code> au acelaşi număr de divizori, egal cu <code>3</code> etc. </p>

<h1>Cerința</h1>

<p>Scrieţi un program care pentru un interval dat determină care este cel mai mic număr din interval ce are număr maxim de divizori. Dacă sunt mai multe numere cu această proprietate se cere să se numere câte sunt.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>maxd.in</code> conţine pe prima linie două numere <code>a</code> şi <code>b</code> separate prin spaţiu <code>(a≤b)</code> reprezentând extremităţile intervalului.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire maxd.out va conţine pe prima linie trei numere separate prin câte un spaţiu <code>min nrdiv contor</code> cu semnificaţia:</p>

<ul>
	<li><code>min</code> – cea mai mică valoare din interval care are număr maxim de divizori</li>
	<li><code>nrdiv</code> – numărul de divizori ai lui <code>min</code></li>
	<li><code>contor</code> – câte numere din intervalul citit mai au acelaşi număr de divizori egal cu <code>nrdiv</code></li>
</ul>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ a ≤ b ≤ 1000000000</code></li>
	<li><code>0 ≤ b - a ≤ 10000</code></li>
</ul>

<p><code>maxd.in</code></p>

<h3>Explicație</h3>

<p><code>6</code> este cel mai mic număr din interval care are maxim de divizori egal cu <code>4</code> şi sunt <code>3</code> astfel de numere <code>6</code>, <code>8</code>, <code>10</code>.</p>

<p><code>maxd.in</code></p>

<pre contenteditable="true" editable="true">200 200</pre>

<p><code>maxd.out</code></p>

<pre contenteditable="true" editable="true">200 12 1</pre>

<h3>Explicație</h3>

<p><code>200</code> are <code>12</code> divizori iar în intervalul <code>[200,200]</code> există un singur număr cu această proprietate.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("maxd.in");
ofstream fout("maxd.out");
int nrdiv(int n)
{
        int rez=1;
        int p,d=2;
        while(n>1)
        {
            p=0;
            while(n%d==0)
            {
                p++;
                n/=d;
            }
            rez*=(p+1);
            d++;
            if(d*d>n) d=n;
        }
        return rez;

}

int main()
{
    int a,b,nrmax=0,imax,cnt=0;
    fin>>a>>b;
    for(int i=a;i<=b;i++)
    {
        int nrd=nrdiv(i);
        if(nrd>nrmax)
        {
            nrmax=nrd;
            imax=i;
            cnt=1;
        }
        else if(nrd==nrmax)  cnt++;
    }
    fout<<imax<<" "<<nrmax<<" "<<cnt;
    return 0;
}</pre>
