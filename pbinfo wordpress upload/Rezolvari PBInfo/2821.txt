<h1>Cerinţa</h1>

<p>Se citește un număr natural, <code>n</code> (<code>n≥2</code>) și se cere să se scrie cel mai mic număr natural care are aceiași divizori primi ca <code>n</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran cel mai mic număr natural care are aceiași divizori primi ca <code>n</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 2</code><sup><code>30</code></sup></li>
</ul>

<h1>Exemplu 1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">75</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">15</pre>

<h1>Exemplu 2:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">7</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">7</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int nr = 1;
    int d = 2;
    while(n > 1)
    {
        int p = 0;
        while(n % d == 0)
            n/=d , p++;
        if(p)
            nr = nr * d;
        d++;
        if(d * d > n)
            d = n;
    }
    cout << nr;
    return 0;
}</pre>
