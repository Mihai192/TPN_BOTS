<h1>Cerința</h1>

<p>Se dau două numere naturale <code>a</code> și <code>b</code>. Calculați suma numerelor echilibrate din intervalul <code>[a,b]</code>. Un număr este echilibrat dacă are număr par de cifre si are numărul de cifre pare egal cu numărul de cifre impare. De exemplu <code>3427</code> este echilibrat, iar <code>2333</code> nu este.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerel <code>a</code> și <code>b</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>s</code>, reprezentând suma numerelor echilibrate din intervalul <code>[a,b]</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ a ≤ b ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>În intervalul <code>[56,71]</code> numerele echilibrate sunt <code>56</code>, <code>58</code>, <code>61</code>, <code>63</code>, <code>65</code>, <code>67</code>, <code>69</code>, <code>70</code>, iar suma lor este <code>509</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main(){
    int a, b;
    long long s = 0;
    cin >> a >> b;
    for(int i = a; i <= b; ++i){
        int n = i, cntp = 0, cnti = 0;
        while(n){
            if(n % 2 == 0)
                cntp++;
            else
                cnti++;
            n/=10;
        }
        if(cntp == cnti)
            s += i;
    }
    cout << s;
    return 0;
}</pre>
