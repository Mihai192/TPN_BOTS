<h1>Cerinţa</h1>

<p>Se dă un vector format din <code>n</code> elemente, numere  naturale nenule, şi un număr natural <code>S</code>. Determinaţi, dacă există o secvenţă de elemente din şir cu suma elementelor egală cu <code>S</code>.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>sumainsecv.in</code> conţine pe prima linie numerele <code>n</code> şi <code>S</code>; urmează cele <code>n</code> elemente ale vectorului, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>sumainsecv.out</code> va conţine pe prima linie numerele <code>p</code> şi <code>u</code>, separate printr-un spaţiu, reprezentând indicele de început şi de sfârşit al secvenţei determinate.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>elementele vectorului vor avea cel mult <code>4</code> cifre şi sunt numerotate de la <code>1</code> la <code>n</code></li>
	<li>dacă vectorul nu conţine nici o secvenţă cu suma elementelor <code>S</code>, se va afişa <code>0 0</code></li>
	<li>dacă şirul conţine mai multe secvenţe cu suma elementelor egală cu <code>S</code>, se va determina cea mai din stânga</li>
</ul>

<h1>Explicație</h1>

<p>Există două secvenţe cu suma elementelor egală cu <code>32</code>: <code>10 15 7</code> şi <code>13 19</code>. Dintre acestea, <code>10 15 7</code> este cea mai din stânga.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("sumainsecv.in");
ofstream fout("sumainsecv.out");
int main()
{
    int n , smax , a[105];
    fin >> n >> smax;
    int st = 0, dr = 0;
    for(int i = 1 ; i <= n ; ++i) fin >> a[i];
    for(int i = 1 ; i <= n &amp;&amp; st == 0 ; ++i)
        for(int j = i ; j <= n &amp;&amp; st == 0 ; ++j)
        {
            int sum = 0;
            for(int k = i ; k <= j ; ++k)
                sum += a[k];
            if(sum == smax)
                st = i, dr = j;
        }
    fout << st << " " << dr;
    return 0;
}</pre>
