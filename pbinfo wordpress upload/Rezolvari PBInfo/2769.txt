<h1>Cerința</h1>

<p>Se dă un număr natural <code>n</code> care este produs de trei numere prime distincte. Ştiind că există <code>m</code> numere naturale prime cu <code>n</code>  şi mai mici decât acesta, să se afişeze în ordine crescătoare cele trei numere prime din descompunerea lui <code>n</code>. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> şi <code>m</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele prime din descompunerea lui <code>n</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ m ≤ n ≤ 10</code><sup><code>18</code></sup></li>
</ul>

<h3>Explicație</h3>

<p>Avem <code>66=2•3•11</code> şi există <code>20</code> de numere prime cu <code>66</code>, mai mici decât acesta.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <cmath>
using namespace std;

int main()
{
    long long int x , y , n , m , p , q;
    cin >> x >> y;
    int d = 2;
    while(x % d) d++;
    n=x/d;
    m=y/(d-1);
    long long int delta = (n+1-m) * (n+1-m) - 4*n;
    long long int r = sqrt(delta);
    p = (n + 1 - m - r) / 2;
    q = (n + 1 - m + r) / 2;
    cout << d << ' ' << p << ' ' << q;
    return 0;
}</pre>
