<h1>Cerinţa</h1>

<p>Se dă un șir cu <code>n</code> numere  naturale. Determinați numărul total de cifre al tuturor numerelor prime din șir.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>C</code>, reprezentând numărul total de cifre al tuturor numerelor prime din șir.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Dintre cele <code>6</code> numere citite sunt prime : <code>83 53 401 7</code>. În total, ele au <code>8</code> cifre.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int prim(int n)
{
    if(n==0 || n==1) return 0;
    if(n==2) return 1;
    if(n%2==0) return 0;
    for(int i=3;i*i<=n;i += 2)
    {
        if(n%i==0) return 0;

    }
    return 1;
}
int main ()
{
    int n,x ,nrcif=0;
    cin >> n;
    for (int i=1;i<=n;++i)
    {
        cin >> x;
        if(prim(x))
        {
            while (x != 0)
            {
                nrcif++;
                x=x/10;
            }
        }
    }
    cout << nrcif;

    return 0;
}</pre>
