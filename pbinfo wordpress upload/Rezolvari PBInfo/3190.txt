<h1>Vigenere</h1>

<p>La fel ca <a href="https://ro.wikipedia.org/wiki/Cifrul_Cezar">cifrul Cezar</a>, cifrul Vigenere deplasează literele, dar acesta folosește o deplasare multiplă. Cheia este constituită din câțiva întregi <code>k</code><sub><code>i</code></sub>, unde <code>0 ≤ k</code><sub><code>i</code></sub><code> ≤ 25</code>.</p>

<p>Aceşti întregi pot fi, de exemplu, <code>k = (21, 4, 2 19, 14, 17)</code>. Această cheie ar provoca deplasare primei litere cu <code>21</code>, <code>c1 = p1 + 21 (mod 26)</code>, a celei de-a doua cu <code>4</code>, <code>c2 = p2 + 4 (mod 26)</code>, ş.a.m.d. până la sfârşitul cheii și apoi de la început, din nou. </p>

<p>Pentru o memorare mai ușoară cheia este de regulă dată printr-un cuvânt – cheia de mai sus corespunde cuvântului <code>vector</code>, litera <code>'v'</code>” corespondând numărului <code>21</code> (numerotarea începe de la <code>0</code>), litera <code>'e'</code> cu <code>4</code>, etc.</p>

<p>Un exemplu de cifru Vigenere:</p>

<p>Textul clar: <code>thisisanexampleofthevigenerecipher</code><br>
Cheia: <code>vector</code><br>
Textul cifrat: <code>olklwjvrgqodkpghtkcixbuviitxqzklgk</code></p>

<h1>Cerința</h1>

<p>Se citesc două texte <code>a</code> și <code>b</code> cu lungimea maximă de <code>255</code> de caractere. Să se afișeze textul după aplicarea cifrului Vigenere la textul <code>a</code> cu cheia <code>b</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>vigenere.in</code> conține pe prima linie textul <code>a</code> iar pe a doua linie textul <code>b</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>vigenere.out</code> va conține pe prima linie rezultatul criptării.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>lungimea lui <code>b ≤</code> Lungimea lui <code>a ≤ 255</code>;</li>
	<li>cele două șiruri vor conține doar litere mici din alfabetul englez.</li>
</ul>

<h3>Explicație</h3>

<p>În Cifrul Vigenere începem numărătoarea caracterelor din cheie de la 0, astfel, caracterul <code>'d'</code> din cheie deplasează caractrrul corespunzător din text cu 3 caractere; astfel, <code>'p'</code> devine <code>'s'</code>. Acest procedeu se repetă și pentru restul șirului.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;

ifstream cin("vigenere.in");
ofstream cout("vigenere.out");

char a[260], b[260];
int v[260];

int main(){
    cin >> a >> b;
    int i = 0;
    int n = strlen(b);
    while(a[i]){
        v[i] = a[i] + b[i % n];
        i++;
    }
    for(int k = 0; k < i; ++k){
        v[k] -= 'a';
        if(v[k] > 'z')
            v[k] = v[k] - 'z' + 'a' - 1;
        cout << (char)v[k];
    }
    return 0;
}</pre>
