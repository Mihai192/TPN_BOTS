<h1>Cerinţa</h1>

<p>Se dau cel mult <code>100000</code> de numere naturale, cu cel mult <code>2</code> cifre fiecare. Afişaţi în ordine strict crescătoare valorile impare care se regăsesc printre valorile date, şi în ordine strict descrescătoare valorile pare care se regăsesc printre valorile date.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>pareimpare.in</code> conţine cel mult <code>100000</code> de numere naturale, cu cel mult <code>2</code> cifre fiecare, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>pareimpare.out</code> va conţine pe prima linie şirul valorilor impare, separate printr-un spaţiu, iar pe a doua linie şirul valorilor pare, separate printr-un spaţiu.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>în fişierul de intrare se află cel puţin un număr par şi cel puţin un număr impar;</li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;

ifstream fin("pareimpare.in");
ofstream fout("pareimpare.out");

int main()
{
    int n=1 , p1=1 , p2=1;
    int a[100000], b[100000], c[100000];
    b[0]=-1 , c[0]=-1;
    while(fin >> a[n])
        n++;
    for(int i = 1 ; i < n ; ++i)
    {
        if(a[i]%2==0)
        {
            b[p1]=a[i];
            p1++;
        }
        if(a[i]%2==1)
        {
            c[p2]=a[i];
            p2++;
        }
    }
    sort(c+1 , c+p2);
    for(int i = 1 ; i < p2 ; ++i)
    {
        if(c[i]!=c[i-1])
        fout << c[i] << ' ';
    }
    fout << endl;
    sort(b+1 , b+p1);
    for(int i = p1-1 ; i > 0 ; --i)
    {
        if(b[i-1]!=b[i])
        fout << b[i] << ' ';
    }

    fin.close();
    fout.close();
    
    return 0;
}</pre>
