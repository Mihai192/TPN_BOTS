<p>Fiind dat un număr natural, efectuând suma pătratelor cifrelor numărului dat, apoi repetând însumarea pătratelor cifrelor pentru numerele obţinute ca rezultat, la un moment dat se obţine una dintre valorile <code>1</code> sau <code>4</code>. </p>

<p>De exemplu, pentru numărul natural <code>89</code>, se obţin, pe rând, valorile <code>145</code>, <code>42</code>, <code>20</code>, <code>4</code> iar pentru numărul natural <code>86</code>, valorile rezultate sunt <code>100</code>, <code>1</code>. </p>

<h1>Cerința</h1>

<p>Dat un set de numere naturale, să se determine pentru fiecare dintre ele, numărul de repetări ale calculului sumei pătratelor cifrelor până la obţinerea rezultatului <code>1</code> sau <code>4</code>.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>unupatru.in</code> conţine pe prima linie o valoare naturală <code>n</code>, care reprezintă numărul de numere naturale care vor fi testate. Pe linia a doua a fişierului de intrare se găsesc cele n numere naturale, separate prin câte un spaţiu.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>unupatru.out</code> conţine pe prima linie n numere naturale, care indică, în ordinea din fişierul de intrare, numărul de repetări ale calculului pentru fiecare dintre cele <code>n</code> numere din fişierul de intrare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 &lt;= n &lt;= 10</code></li>
	<li><code>1 &lt;=</code> numerele testate <code>&lt;= 10</code><sup><code>18</code></sup></li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>unupatru.in</code></p>

<pre contenteditable="true" editable="true">2
89 68</pre>

<p><code>unupatru.out</code></p>

<pre contenteditable="true" editable="true">4 2</pre>

<h3>Explicație</h3>

<p>Se obţin, pe rând, pentru <code>89</code>, valorile <code>145 42 20 4</code><br>
Se obţin, pe rând, pentru <code>68</code>, valorile <code>100 1</code></p>

<h1>Exemplul 2:</h1>

<p><code>unupatru.in</code></p>

<pre contenteditable="true" editable="true">3
33 17 638</pre>

<p><code>unupatru.out</code></p>

<pre contenteditable="true" editable="true">10 9 5</pre>

<h3>Explicație</h3>

<p>Valorile obţinute pentru cele 3 valori sunt:<br>
<code>18 65 61 37 58 89 145 42 20 4</code><br>
<code>50 25 29 85 89 145 42 20 4</code><br>
<code>109 82 68 100 1</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("unupatru.in");
ofstream fout("unupatru.out");
int pat(long long n)
{
    long long cnt = 0 , n1 = n;
    while(1)
    {
        if(n == 1 || n == 4) break;
        cnt++;
        long long n1 = n;
        n = 0;
        while(n1 != 0)
        {
            n += (n1 % 10) * (n1 % 10);
            n1 /= 10;
        }
    }

    return cnt;
} main()
{
    long long n , x;
    fin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        fin >> x;
        fout << pat(x) << " ";
    }

    fin.close();
    fout.close();

    return 0;
