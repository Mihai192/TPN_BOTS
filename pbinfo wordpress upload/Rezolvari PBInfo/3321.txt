<p>Peste 3700 de ani lumea a devenit dreptunghiulară, este formată <code>n x m</code> zone și este stăpânită de un rege care are o armată formată din <code>q</code> soldați. În regat se află <code>k</code> turnuri de piatră, ostile armatei regelui, la coordonate cunoscute; fiecare turn atacă zonele de pe linia și coloana sa.</p>

<h1>Cerința</h1>

<p>Regele dorește să afle în câte moduri poate plasa soldații în zonele fără turnuri ale regatului astfel încât aceștia să nu fie atacați de turnuri.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură dimensiunile <code>n</code> și <code>m</code> ale regatului, numărul de turnuri <code>k</code> și numărul de soldați <code>q</code>. Pe următoarele <code>k</code> linii se află câte două valori, reprezentând coordonatele turnurilor.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>p</code>, reprezentând numărul de moduri în care pot fi plasați soldații regelui.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n , m ≤ 4</code></li>
	<li><code>1 ≤ q , k ≤ n*m</code></li>
	<li>în cazul în care nu există niciun mod de plasare a soldaților se va afișa <code>0</code></li>
	<li>soldații sunt diferiți între ei; două moduri de plasare a soldaților pot să folosească aceleași zone, dar să fie diferite prin ordinea în care sunt plasați soldații în aceste zone.</li>
</ul>

<h1>Exemplu 1:</h1>


<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">3 3 1 1
2 2</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">4</pre>

<h3>Explicație</h3>

<p>Zonele neatacate de turnuri au coordonatele <code>(1,1)</code>, <code>(1,3)</code>, <code>(3,1)</code>, <code>(3,3)</code>.</p>

<h1>Exemplu 2:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">4 4 1 9
1 1</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">362880</pre>

<h3>Precizare</h3>

<p>Acesta este cel mai mare test.</p>

<h1>Exemplu 3:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">3 3 3 4
1 1
2 2
3 3</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">0</pre>

<h3>Explicație</h3>

<p>Nu au rămas destule zone neatacate de turnuri.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int n, m, k, p, a[10][10];

int fact(int n){
    if(n == 0)
        return 1;
    else
        return n * fact(n-1);
}

int main(){
    cin >> n >> m >> k >> p;
    int x, y;
    for(int i = 1; i <= k; ++i){
        cin >> x >> y;
        for(int j = 1; j <= m; ++j)
            a[x][j] = 1;
        for(int j = 1; j <= n; ++j)
            a[j][y] = 1;
    }
    int cnt = 0;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            if(a[i][j] == 0)
                cnt++;
    if(cnt < p)
        cout << 0;
    else{
        cout << fact(cnt) / fact(cnt - p);
    }
}</pre>
