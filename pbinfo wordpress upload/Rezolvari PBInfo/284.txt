<h1>Cerinţa</h1>

<p>Se dau două şiruri, cu <code>n</code>, respectiv <code>m</code>, elemente, numere naturale. Primul şir este ordonat crescător, iar al doilea este ordonat descrescător. Să se afişeze, în ordine crescătoare, valorile pare din cele două şiruri.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>interclasare3.in</code> conţine pe prima linie numerele <code>n</code> şi <code>m</code>; urmează <code>n</code> numere naturale separate prin spaţii, dispuse pe mai multe linii, ordonate crescător; urmează <code>m</code> numere naturale separate prin spaţii, dispuse pe mai multe linii, ordonate descrescător.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>interclasare3.out</code> va conţine valorile pare din cele două şiruri, ordonate crescător, câte <code>20</code> de valori pe o linie, valorile de pe aceeaşi linie fiind separate printr-un spaţiu.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n,m ≤ 100000</code>;</li>
	<li>elementele celor două şiruri vor avea cel mult <code>9</code> cifre;</li>
	<li>în cel puţin şir va exista minim un element par;</li>
</ul>

ifstream cin("interclasare3.in");

int main()

    int n, m, k = 0;
    int a[100001], b[100001], c[200001];
    cin >> n >> m;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
    for(int i = 1; i <= m; ++i)
        cin >> b[i];

    int i = 1, j = m;
    while(i <= n &amp;&amp; j >= 1){
        if(a[i] < b[j] &amp;&amp; a[i] % 2 == 0)
            c[++k] = a[i], i++;
        else if(a[i] < b[j])
            i++;
        else if(b[j] % 2 == 0)
            c[++k] = b[j], j--;
        else
            j--;
    }

    while(i <= n){
        if(a[i] % 2 == 0)
            c[++k] = a[i];
        i++;
    }
    while(j >= 1){
        if(b[j] % 2 == 0)
            c[++k] = b[j];
        j--;
    }

    for(int i = 1; i <= k; ++i){
        cout << c[i] << ' ';
        if(i % 20 == 0)
            cout << endl;
    }

    return 0;
