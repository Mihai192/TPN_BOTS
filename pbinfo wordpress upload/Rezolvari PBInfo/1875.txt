<h1>Cerința</h1>

<p>Se consideră un șir de cifre. Să se determine lungimea maximală a unei secvențe din șir formată din cifre egale.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>platou1.in</code> conține pe prima linie cifrele reprezentând termenii șirului, separate prin câte un spațiu.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>platou1.out</code> va conține pe prima linie două numere naturale <code>k x</code>, separate printr-un singur spațiu, <code>k</code> reprezentând lungimea secvenței determinate, iar <code>x</code> reprezentând termenul șirului din care este formată secvența.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>fișierul de intrare conține cel mult <code>1.000</code> de cifre, separate prin câte un spațiu, reprezentând termenii șirului dat;</li>
	<li>dacă șirul conține mai multe secvențe cu lungimea maximală, se va afișa cel mai mic termen ce formează o astfel de secvență.</li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare sunt <code>3</code> secvențe cu lungimea maximă <code>k=4</code>, termenul cel mai mic fiind <code>x=5</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <fstream>

using namespace std;

ifstream fin("platou1.in");
ofstream fout("platou1.out");

int main()
{
int xmin=10, k=0,lg=1,x,y;
   fin>>x; 

while(fin>>y)
     if(x==y)
         lg++; 
    else
      { 
         if(k<lg)
      {
          k=lg; 
           xmin=x; 
      }
else
      if(k==lg &amp;&amp; xmin>x)
          xmin=x;

           lg=1; 
            x=y; 
           }

fout<<k<<" "<<xmin<<endl;

fin.close();
fout.close();

return 0;
} </pre>
