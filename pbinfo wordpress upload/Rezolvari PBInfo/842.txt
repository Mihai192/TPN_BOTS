<h1>Cerința</h1>

<p>Pentru o serie de activități foarte sofisticate, Gigel are nevoie de un fierăstrău special, alcătuit din mai mulţi dinţi. Un fierăstrău de gradul <code>n</code> este format din două fierăstraie de gradul <code>n-1</code>, între care se află un dinte de mărime <code>n</code>. Un fierăstrău de gradul <code>1</code> are un singur dinte, de mărime <code>1</code>.</p>

<p>Afișați un fierăstrău de grad <code>n</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran un șir de numere naturale, separate prin câte un spațiu, reprezentând mărimea fiecărui dinte al fierăstrăului.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code></li>
</ul>

<h1>Explicație</h1>

<p>Un fierăstrău de grad <code>3</code> are la mijloc un dinte de mărime <code>3</code>, între două fierăstraie de grad <code>2</code>.</p>

<p>Un fierăstrău de grad <code>2</code> are la mijloc un dinte de mărime <code>2</code>, între două fierăstraie de grad <code>1</code>.</p>

<p>Un fierăstrău de grad <code>1</code> are un singur dinte, de mărime <code>1</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

void dinti(int n)
{
    if(n == 1)
        cout << 1 << ' ';
    else
    {
        dinti(n-1);
        cout << n << ' ';
        dinti(n-1);
    }
}

int main()
{
    int n;
    cin >> n;
    dinti(n);
    return 0;
}</pre>
