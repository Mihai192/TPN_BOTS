<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a <strong>subprogramului recursiv</strong> <code>numarare</code>, care primeşte prin parametrul <code>v</code> un tablou unidimensional cu cel mult <code>100</code> de elemente întregi, iar prin parametrul <code>n</code> numărul efectiv de elemente din <code>v</code>.</p>

<p>Subprogramul returnează numărul de perechi de elemente vecine din tabloul <code>v</code> care sunt egale.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 100</code></li>
	<li>numele subprogramului cerut este <code>numarare</code></li>
	<li>parametrii sunt, în această ordine: <code>v</code>, <code>n</code></li>
	<li>indicii elementelor tabloului <code>v</code> sunt <code>0</code>, <code>1</code> ,… , <code>n-1</code>.</li>
	<li><strong>se recomandă realizarea unui subprogram recursiv</strong></li>
</ul>

<p>Dacă <code>n=6</code>, <code>v=(5, 6, 6, 3, 5, 5)</code>, rezultatul va fi <code>2</code> – perechile sunt <code>6 6</code> și <code>5 5</code></p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int numarare(int v[100], int n)
{
    int cnt=0;
    for (int i = 0; i < n; ++i)
    {
        cin >> v[i];
        if (v[i]==v[i-1])
        cnt++;
    }
    return cnt;
}</pre>
