<h1>Cerința</h1>

<p>Într-o clasă sunt <code>n</code> elevi, numerotați de la <code>1</code> la <code>n</code>, băieți și fete, pentru fiecare dintre ei cunoscându-se înălțimea, exprimată în centimetri. Să se determine câte fete sunt mai scunde decât cel mai scund băiat și câți băieți sunt mai înalți decât cea mai înaltă fată.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> caracteristicile celor <code>n</code> elevi, astfel: o literă (<code>B</code> sau <code>F</code>), care definește sexul elevului, și înălțimea acestuia, un număr natural nenul.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran două numere <code>A B</code>, separate printr-un spațiu, însemnând: <code>A</code> – numărul de fete care sunt mai scunde decât cel mai scund băiat și <code>B</code> – numărul de băieți care sunt mai înalți decât cea mai înaltă fată.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>înălțimile elevilor vor fi mai mici decât <code>200</code></li>
	<li>se garantează că există cel puțin o fată și cel puțin un băiat</li>
</ul>

<h1>Explicație</h1>

<p>Sunt <code>2</code> fete, cu înălțimile: <code>160 165</code> și <code>3</code> băieți, cu înălțimile: <code>176 180 179</code>. Ambele fete sunt mai scunde decât cel mai scund băiat și toți cei trei băieți sunt mai înalți decât cea mai înaltă fată.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    char c;
    int n , a[1001] , b[1001] , f[1001] , b1 = 0, f1 = 0 , min = 1000 , max = 0 , cf = 0 , cb = 0;
    cin >> n;
    for(int i = 1; i <= n; i++)
    {
      cin >> c >> a[i];
      if(c=='F')
        f[++f1] = a[i];
      else
        b[++b1] = a[i];
    }
    for(int i = 1; i <= f1 ; i++)
        if(f[i] > max)
           max = f[i];
    for(int i = 1; i < b1 ; i++)
        if(b[i] < min)
           min = b[i];
    for(int i = 1; i <= f1; i++)
        if(f[i] < min) cf++;
    for(int i = 1; i <= b1; i++)
        if(b[i] > max) cb++;
    cout<<cf<<" "<<cb;

    return 0;
}</pre>
