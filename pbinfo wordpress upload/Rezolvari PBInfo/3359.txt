<h1>Cerința</h1>

<p>Se dă un șir de <code>n</code> numere naturale. Determinați o pereche de numere consecutive în șir, cu proprietatea că au exact aceiași factori primi, indiferent de exponent. Dacă sunt mai multe asemenea perechi se va determina aceea cu suma numerelor mai mare. Dacă în continuare sunt mai multe perechi, se va determina prima din șir.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran două numere <code>a b</code>, separate prin câte un spațiu, reprezentând perechea determinată. Numerele <code>a b</code> vor fi afișate în ordinea din șir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>numerele din șir vor fi mai mici decât <code>1.000.000.000</code></li>
	<li>dacă nu există nicio pereche care respectă cerința se va afișa <code>NU EXISTA</code></li>
</ul>

<h3>Explicație</h3>

<p>Perechea <code>45 75</code> respectă și ea regula, dar are suma mai mică (<code>45+75 &gt; 80 + 100</code>).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int n, stfin, drfin, a[1001], lmax = 0, l = 1, st = 1, dr = 1, sum, summax;

bool ok(int a, int b){
    int d = 2;
    while(a > 1){
        if(a % d == 0){
            if(b % d != 0)
                return 0;
            else{
                while(a % d == 0)
                    a/=d;
                while(b % d == 0)
                    b/=d;
            }
        }
        d++;
        if(d * d > a)
            d = a;
    }
    return a == b;
}

int main(){
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
    int rez1=-1, rez2=-1;
    for(int i = 2; i <= n; ++i){
        if(ok(a[i-1], a[i]) &amp;&amp; rez1 + rez2 < a[i-1] + a[i]){
            rez1 = a[i-1];
            rez2 = a[i];
        }
    }
    if(rez1 != -1)
        cout << rez1 << ' ' << rez2 << endl;
    else
        cout << "NU EXISTA";
    return 0;
