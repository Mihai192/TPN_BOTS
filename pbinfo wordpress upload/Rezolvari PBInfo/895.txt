<h1>Cerinţa</h1>

<p>Se numește <em>permutare</em> a unei mulțimi o aranjare a elementelor mulțimii în altă ordine. De exemplu, permutările mulțimii <code>{1,2,3}</code> sunt: <code>(1,2,3)</code>, <code>(1,3,2)</code>, <code>(2,1,3)</code>, <code>(2,3,1)</code>, <code>(3,1,2)</code>, <code>(3,2,1)</code>.</p>

<p>Fie <code>(p</code><sub><code>1</code></sub><code>, p</code><sub><code>2</code></sub><code>, ..., p</code><sub><code>n</code></sub>) o permutare a mulțimii <code>{1,2,...,n}</code>. Se numește punct fix al permutării o valoare <code>k</code> din mulțime cu proprietatea că <code>p</code><sub><code>k</code></sub><code> = k</code>.</p>

<p>Scrieţi definiția completă a subprogramului C++ <code>permutare</code> care are <code>2</code> parametri:  <code>a</code>, prin care primeşte un tablou unidimensional  cu maximum <code>100</code> de numere naturale mai mici decât <code>1000</code> și <code>n</code>, numărul efectiv de elemente ale tabloului.</p>

<p>Subprogramul verifică dacă elementele vectorului <code>a</code> reprezintă o permutare fără puncte fixe a mulțimii <code>{1,2,...,n}</code> și returnează valoarea <code>1</code> în caz afirmativ, respectiv <code>0</code> în caz negativ.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt;= 100</code></li>
	<li>numele subprogramului cerut este <code>permutare</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code>, cu semnificația de mai sus</li>
	<li>elementele vectorului <code>a</code> sunt indexate de la zero</li>
</ul>

<p>Dacă <code>n=6</code> și <code>a=(2,3,1,6,4,5)</code> subprogramul va returna valoarea <code>1</code>.</p>

<p>Dacă <code>n=4</code> și <code>a=(2,3,1,4)</code> subprogramul va returna valoarea <code>0</code>, deoarece permutarea are un punct fix.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int permutare(int a[] , int n)
{
    int cnt=0;
    int cn=0;
    int f[100000]={0};
    for(int i = 0 ; i < n ; ++i)
    {
        f[a[i]]++;
    }
    for(int i = 1 ; i <= n ; ++i)
    {
        if(f[i]==1)
            cn++;
    }
    for(int i = 0 ; i < n ; ++i)
    {
        if(a[i]==i+1)
            cnt++;
        if(a[i]>n)
            cn=0;
    }
    if(cnt==0 &amp;&amp; cn==n)
        return 1;
    else
        return 0;
}</pre>
