<p>Să se scrie o funcție C++ care verifică dacă un număr natural transmis ca parametru este <strong>aproape prim</strong>. Un număr natural este <strong>aproape prim</strong> dacă se poate scrie ca produs de două numere prime distincte.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției scrise este <strong>a_prim</strong></li>
	<li>funcția are un parametru, număr natural, a cărui valoare este mai mică decât 2 <sup>31</sup></li>
	<li>rezultatul funcției este <strong>1</strong> dacă valoarea parametrului este aproape prim, respectiv <strong>0</strong> în caz contrar</li>
</ul>

<p>Numărul <code>14</code> este aproape prim, deoarece <code>14=2*7</code>, iar <code>2</code> și <code>7</code> sunt numere prime.</p>

<h1>Important</h1>

<p>Soluția propusa va conține doar funcția cerută. Introducerea în soluție a altor instrucțiuni poate duce la erori de compilare sau de execuție, care vor duce la depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">bool prim(int n){
    int d = 2, cnt = 1;
    while(n > 1){
        int p = 0;
        while(n % d == 0)
            n/=d, p++;
        d++;
        cnt *= (p + 1);
        if(d * d > n)
            d = n;
    }
    return cnt == 2;
}

bool a_prim(int n){
    int cnt = 0;
    for(int d = 2; d * d <= n; ++d){
        if(n % d == 0)
            cnt+=2;
        if(d * d == n)
            cnt--;
    }
    if(cnt == 2){
        int a, b;
        for(int d = 2; d * d <= n; ++d)
            if(n % d == 0)
                a = d, b = n / d;
        if(prim(a) &amp;&amp; prim(b))
            return 1;
    }
    return 0;
}</pre>
