<p>În portul Constanţa sunt ancorate două vapoare pline cu marfă. Ele fac curse repetate către două destinaţii diferite. Se ştie că primul vapor ajunge la destinaţia stabilită după un număr <code>X</code> de săptămâni, iar al doilea vapor după un număr <code>Y</code> de săptămâni. Drumul înapoi ia acelaşi timp. Armatorul celor <code>2</code> vapoare vrea să ştie după câte zile cele <code>2</code> vapoare pleacă din nou concomitent din port. Se mai ştie faptul că pentru manevrarea mărfurilor primului vapor îi sunt necesare <code>z1</code> zile, iar celui de al doilea <code>z2</code> zile.</p>

<h1>Cerința</h1>

<p>Scrieţi un program care determină numărul de zile după care cele <code>2</code> vapoare pleacă din nou concomitent din portul din care au plecat.</p>

<h1>Date de intrare</h1>

<p>De pe prima linie din fişierul de intrare <code>vapoare.in</code> se citesc cele două numere <code>X şi Y</code>, separate printr-un spaţiu, reprezentând numărul de săptămâni necesare ajungerii la destinaţie pentru cele două vapoare. Linia a doua a fişierului de intrare conţine valorile <code>z1 şi z2</code>, separate printr-un spaţiu, reprezentând numărul de zile necesare celor două vapoare pentru manevrarea mărfurilor.</p>

<h1>Date de ieșire</h1>

<p>Pe prima linie a fişierului de ieşire <code>vapoare.out</code> se va afişa numărul de zile după care cele două vapoare se reîntâlnesc în portul de plecare.</p>

<h1>Restricții și precizări</h1>

<p>• <code>0 &lt; X,Y ≤ 100000</code><br>
• <code>0 &lt;= z1, z2 &lt;= 31</code></p>

<h3>Explicație</h3>

<p>Primul vapor are nevoie de 16 zile pentru a junge în portul de unde a plecat, iar al doilea vapor are nevoie de 30 de zile, astfel încât, dupa 240 de zile cele două vapoare se întâlnesc din nou în portul de plecare</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("vapoare.in");
ofstream fout("vapoare.out");

int main()
{
    int n,m;
    fin >> n >> m;
    int z1 , z2;
    fin >> z1 >> z2;
    long long int a = n*2*7+z1;
    long long int b = m*2*7+z2;
    long long int ca=a;
    long long int cb=b;
    while(b)
    {
        int r=a%b;
        a=b;
        b=r;
    }
    fout << ca/a*cb;
    
    fin.close();
    fout.close();
    
    return 0;
}</pre>
