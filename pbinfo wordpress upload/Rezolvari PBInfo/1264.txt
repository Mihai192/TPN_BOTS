<h1>Cerința</h1>

<p> Se dau <code>n</code> numere naturale. Să se afișeze al <code>k</code>-ulea cel mai mic element din șir. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>statisticiordine.in</code> conține pe prima linie numerele <code>n</code> si <code>k</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>statisticiordine.out</code> va conține pe prima linie numărul căutat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ k ≤ n ≤ 4.000.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>4.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p><code>24</code> este al patrulea cel mai mic element din sir.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;

unsigned int a[4000001];

ifstream fin("statisticiordine.in");
ofstream fout("statisticiordine.out");

int main ()
{
    unsigned int n , k;
    fin >> n >> k;
    for (int i = 1 ; i <= n ; ++i) fin >> a[i];
    sort (a+1 , a + n + 1);
    fout << a[k];
    fin.close();
    fout.close();

    return 0;
}</pre>
