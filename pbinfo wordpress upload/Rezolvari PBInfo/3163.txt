<h1>Cerința</h1>

<p>Se dă un șir de <code>n</code> numere naturale și un număr natural <code>val</code>. Determinați lungimea maximă a unei secvențe cu proprietatea că suma numerelor din aceasta este mai mică sau egală cu <code>val</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>secvmaxval.in</code> conține pe prima linie numerele <code>n</code> și <code>val</code>, iar pe a două linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>secvmaxval.out</code> va conține pe prima linie lungimea maximă a unei secvențe care satisface proprietatea dată.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 200.000</code></li>
	<li>numerele de pe a două linie a fișierului de intrare vor fi mai mici decât <code>1.000.000.000</code></li>
	<li><code>1 ≤ val ≤ 2^63</code></li>
</ul>

<h3>Explicație</h3>

<p>O secvență de lungime maximă cu suma elementelor mai mică sau egală cu <code>11</code> este <code>5 2 3</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("secvmaxval.in");
ofstream cout("secvmaxval.out");

long long int a[200001], is = 1, lmax, s;

int main(){
    long long int n, k;
    cin >> n >> k;
    for(int i = 1 ; i <= n; ++i)
        cin >> a[i];
    for(int i = 1 ; i <= n; ++i)
    {
        s += a[i];
        if(s > k)
        {
            if(i - is + 1 > lmax)
                lmax = i - is + 1;
            s-=a[is];
            is++;
        }
    }
    if(n - is + 1 > lmax)
        lmax = n - is + 1;
    if(n - is + 1 == lmax)
        lmax++;
    cout << lmax - 1;
    return 0;
}</pre>
