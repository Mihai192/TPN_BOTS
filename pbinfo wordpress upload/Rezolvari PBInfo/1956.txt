<p>Obținându-se astfel șirurile:</p>

<ul>
	<li><strong>Fibonacci</strong>: <code>1</code>, <code>1</code>, <code>2</code>, <code>3</code>, <code>5</code>, <code>8</code>, <code>13</code>, <code>21</code>, <code>34</code>, <code>55</code>, <code>89</code>, <code>144</code>, …</li>
	<li><strong>Iccanobif</strong>: <code>1</code>, <code>1</code>, <code>2</code>, <code>3</code>, <code>5</code>, <code>8</code>, <code>13</code>, <code>39</code>, <code>124</code>, <code>514</code>, <code>836</code>, …</li>
</ul>

<p>Iccanobif, se întreabă acum, ce număr are mai mulți divizori numere naturale: al <code>n</code>-lea termen din șirul Fibonacci sau al <code>n</code>-lea termen din șirul său.</p>

<h1>Cerințe</h1>

<p>Scrieți un program care să citească un număr natural <code>n</code> și să afișeze:</p>

<p>a) al <code>n</code>-lea termen din  șirul lui Fibonacci și numărul său de divizori <br>
b) al <code>n</code>-lea termen din  șirul lui Iccanobif și numărul său de divizori</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>siruri2.in</code> conține pe prima linie un număr natural <code>p</code>. Pentru toate testele de intrare, numărul <code>p</code> poate avea doar valoarea <code>1</code> sau valoarea <code>2</code>. Pe linia a doua a fișierului se găsește un număr natural <code>n</code>. </p>

<h1>Date de ieșire</h1>

<p><strong>Dacă valoarea lui <code>p</code> este <code>1</code>, se va rezolva numai punctul a) din cerințe.</strong> În acest caz, în fișierul de ieșire <code>siruri2.out</code> se vor scrie al <code>n</code>-lea termen din  șirul lui Fibonacci și numărul său de divizori.</p>

<p><strong>Dacă valoarea lui <code>p</code> este <code>2</code>, se va rezolva numai punctul b) din cerințe.</strong> În acest caz, în fișierul de ieșire <code>siruri2.out</code> se vor scrie al <code>n</code>-lea termen din  șirul lui Iccanobif și numărul său de divizori</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 50</code></li>
	<li>Pentru rezolvarea corectă a primei cerinţe se acordă 50% din punctaj, iar  pentru cerința a doua se acordă 50% din punctaj.</li>
</ul>

<h1>Exemplul 1</h1>

<p><code>siruri2.in</code></p>

<pre contenteditable="true" editable="true">1
8</pre>

<p><code>siruri2.out</code></p>

<pre contenteditable="true" editable="true">21 4</pre>

<h1>Exemplul 2</h1>

<p><code>siruri2.in</code></p>

<pre contenteditable="true" editable="true">2 9</pre>

<p><code>siruri2.out</code></p>

<pre contenteditable="true" editable="true">124 6</pre>

<h3>Explicații</h3>

<p>Pentru primul exemplu: Al optulea termen din șirul lui Fibonacci este <code>21</code>, iar <code>21</code> are <code>4</code> divizori. (<code>p</code> fiind <code>1</code> se rezolvă doar cerința a)</p>

<p>Pentru al doilea exemplu: Al nouălea termen din șirul lui Iccanobif este <code>124</code>, iar <code>124</code> are <code>6</code> divizori. (<code>p</code> fiind <code>2</code> se rezolvă doar cerința b)</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("siruri2.in");
ofstream fout("siruri2.out");
unsigned long long fib(unsigned long long n)
{
    long long f1=1,f2=1,f3;
    if(n==1) return 1;
    else if(n==2) return 1;
    else
    {
        while(n>2)
        {
            f3=f1+f2;
            f1=f2;
            f2=f3;
            n--;
        }
        return f3;
    }
}
unsigned long long ogl(unsigned long long n)
{
    long long ogl=0;
    while(n!=0)
    {
        ogl=ogl*10+n%10;
        n=n/10;
    }
    return ogl;
}
unsigned long long ico(unsigned long long n)
{
    unsigned long long f1=1,f2=1,f3;
    if(n==1) return 1;
    else if(n==2) return 1;
    else
    {
        while(n>2)
        {
            f3=ogl(f1)+ogl(f2);
             f1=f2;
            f2=f3;
            n--;
        }
        return f3;
    }
}
unsigned long long nrdiv(unsigned long long n)
{
    long long cnt=0;
    for(unsigned long long i=1;i*i<=n;i++)
    {
        if(n%i==0) cnt += 2;
        if(i*i==n) cnt--;
    }
    return cnt;
}
int main()
{
    int n,p;
    fin>>p>>n;
    if(p==1)
    fout<<1ull*fib(n)<<" "<<nrdiv(fib(n));
    else fout<<1ull*ico(n)<<" "<<nrdiv(ico(n));
    return 0;
}</pre>
