<h1>Cerința</h1>

<p>Se dau numerele naturale <code>N</code> și <code>X</code>. Aflați dacă <code>N</code> este divizibil cu <code>2</code><sup><code>X</code></sup> și dacă <code>N</code> este divizibil cu <code>5</code><sup><code>X</code></sup>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>nmod25.in</code> conține numărul <code>X</code> și numărul <code>N</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>nmod25.out</code> va conține pe prima linie mesajul <code>DA</code> dacă numărul este divizibil cu <code>2</code><sup><code>X</code></sup> sau <code>NU</code> dacă nu este divizibil, pe a două linie mesajul <code>DA</code> dacă numărul este divizibil cu <code>5</code><sup><code>X</code></sup> sau <code>NU</code> dacă nu este divizibil.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>N</code> are cel mult <code>100.001</code> cifre</li>
	<li><code>0 &lt; X &lt; 5</code></li>
</ul>

<h3>Explicație</h3>

<p><code>50008</code> este divizibil cu <code>2</code> la puterea <code>3</code> dar nu si cu <code>5</code> la puterea <code>3</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;
ifstream cin("nmod25.in");
ofstream cout("nmod25.out");
int put(int a , int b)
{
    int p = 1;
    for(int i = 1 ; i <=a ; ++i)
        p*=b;
    return p;
}
int main()
{
    char c[100001];
    int  x[100001] ,p;
    cin >> p >> c;
    int n = strlen(c);
    for (int i = 0; i < n; i ++)
        x[n - i ] = c[i] - '0';
    int a = x[4]*1000+x[3]*100+x[2]*10+x[1];
    if(a%put(p , 2)==0) cout << "DA" << endl;
    else cout << "NU"<< endl;
    if(a%put(p , 5)==0) cout << "DA";
    else cout << "NU";
    return 0;
}</pre>
