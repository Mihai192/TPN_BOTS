<h1>Cerința</h1>

	<li>numele subprogramului va fi <code>nr_prim</code></li>

<p>Dacă <code>n=28</code>, <code>nr_prim(n)=29</code>. Dacă <code>n=17</code>, <code>nr_prim(n)=19</code>.</p>

<h1>Important</h1>

<p>Soluția propusă va conține doar definiția funcției cerute. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int nr_prim(int n)
{
    n++;
    while(1)
    {
        int cnt=0;
        for(int i = 1 ; i*i<= n ; ++i)
        {
            if(cnt>2)
            break;
            if(n%i==0)
            cnt+=2;
            if(i*i==n)
            cnt--;
        }
        if(cnt==2)
        {
            return n;
            break;
        }
        else
        n++;
    }
}</pre>
