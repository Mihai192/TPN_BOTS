<h1>Cerința</h1>

<p>Se dau n numere naturale. Determinați cel mai mare număr par introdus și numărul său de apariții.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>maximpar.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>maximpar.out</code> va conține pe prima linie numerele <code>maxim</code> si <code>apare</code>, reprezentând maximul par dintre numerele de pe a doua linie a fișierului de intrare și aparițiile acestuia, dacă fișierul de intrare conține cel puțin un număr par, respectiv <code>-1</code>, în caz contrar.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 200</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>1.000</code></li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare sunt <code>6</code> numere, cel mai mare număr par este <code>8</code> și apare de <code>2</code> ori.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("maximpar.in");
ofstream fout("maximpar.out");

int main()
{
    int n, cnt=0 , max=0;
    fin >> n;
    int a[201];
    for (int i = 0 ; i < n; ++i)
    {
        fin >> a[i];
        if (a[i] % 2 == 0 &amp;&amp; a[i] > max)
        max=a[i];
    }
    for (int i = 0 ; i < n; ++i)
    {
        if (a[i]==max)
        cnt++;
    }
    if (cnt==0)
        fout << -1;
    else
    fout << max << ' ' << cnt;
    
    fin.close();
    fout.close();
    
    return 0;
}</pre>
