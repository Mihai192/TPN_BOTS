<h1>Cerința</h1>

<p>Mama vrea să acopere cu covoare o cameră cu lungimea de <code>n</code> metri și lățimea de <code>m</code> metri. Ea are la dispoziție oricâte covoare în formă de pătrat, de orice dimensiune, număr natural și respectă următoarele reguli:</p>

<ul>
	<li>fiecare covor plasat are laturile paralele cu laturile camerei;</li>
	<li>covoarele așezate nu se suprapun;</li>
	<li>de fiecare dată se folosește cel mai mare covor care poate fi ales în acel moment;</li>
	<li>de fiecare dată covorul ales se plasează în așa fel încât zona neacoperită încă să fie un dreptunghi.</li>
</ul>

<p>Știind dimensiunile camerei să se determine care este dimensiunea maximă a unui covor folosit și numărul total de covoare folosite.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> și <code>m</code>, reprezentând dimensiunile camerei.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele <code>L</code> și <code>N</code> reprezentând în ordine dimensiunea maximă a unui covor folosit și numărul total de covoare folosite.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n,m ≤ 1000000000</code></li>
	<li>dimensiunile covoarelor sunt numere naturale nenule</li>
</ul>

<h1>Exemplul 1</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">10 3</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">3 6</pre>

<h3>Explicație</h3>

<p>Se folosesc <code>100</code> covoare de latură <code>1</code>. Dimensiunea maximă este <code>1</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int a , b;
    cin >> a >> b;
    cout << min(a , b) << ' ';
    int cnt = 0;
    while(a!=0 &amp;&amp; b!=0)
    {
        if(a > b)
        {
            int x = a/b;
            a-=(x*b);
            cnt += x;
        }
        else
        {
            int x = b/a;
            b-=(x*a);
            cnt += x;
        }
    }
    cout << cnt;
    return 0;
}</pre>
