<p>În urma apelului <code>FRadical(15000, x, y)</code>, se obține <code>x = 50</code>, <code>y = 6</code>.</p>

<h3>Important</h3>

<p>Soluţia propusă va conţine definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <cmath>
void FRadical(int n, int &amp;x, int &amp;y){
    int maxi = 0;
    for(int d = 1; d * d <= n; ++d){
        if(n % d == 0){
            int nr = d;
            int r = sqrt(nr);
            if(r * r == nr &amp;&amp; nr > maxi)
                maxi = nr, y = n / d;
            nr = n / d;
            r = sqrt(nr);
            if(r * r == nr &amp;&amp; nr > maxi)
                maxi = nr, y = d;
        }
    }
    x = sqrt(maxi);
}</pre>
