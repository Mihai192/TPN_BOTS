<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a subprogramului C++ <code>count</code> care are <code>2</code> parametri:  <code>a</code>, prin care primeşte un tablou unidimensional  cu maximum <code>100</code> de <strong>numere reale</strong> mai mici decât <code>1000</code> și <code>n</code>, numărul efectiv de elemente ale tabloului.</p>

<p>Subprogramul returnează numărul  de elemente din tablou care sunt mai mari sau cel puțin egale cu media aritmetică a tuturor elementelor din tablou.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 100</code></li>
	<li>numele subprogramului cerut este <code>count</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code></li>
	<li>elementele vectorului <code>a</code> sunt indexate de la zero</li>
	<li>pentru datele reale se va folosi tipul <code>double</code>. Utilizarea altor tipuri poate duce la erori de compilare.</li>
</ul>

<p>Dacă <code>n=6</code> și <code>a= (12, 7.5, 6.5, 3, 8.5, 7.5)</code> subprogramul va returna valoarea <code>4</code>, deoarece media aritmetică este <code>7.5</code>, iar în tablou sunt <code>4</code> elemente cel mult egale cu această valoare.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int count(double a[], int n){
    double sum = 0;
    for(int i = 0; i < n; ++i)
        sum+=a[i];
    sum = (double)sum / n;
    int cnt = 0;
    for(int i = 0; i < n; ++i)
        if(a[i] >= sum)
            cnt++;
    return cnt;
}</pre>
