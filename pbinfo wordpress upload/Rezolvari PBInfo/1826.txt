<h1>Cerinţa</h1>

<p>Să se scrie o funcție C++ care să returneze pentru un număr natural <code>n</code> transmis ca parametru numărul de cifre zero de la finalul lui <code>n! = 1•2•...•n</code>. </p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>zerof</code></li>
	<li>funcția va avea un singur parametru, <code>n</code></li>
	<li>valoarea calculată va fi returnată de către funcție</li>
	<li>prin definiție, <code>0!=1</code></li>
	<li><code>0 ≤ n ≤ 10000</code></li>
</ul>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce la erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int zerof (int n)
{
    int c2 = 0, c5 = 0;
    if (n == 0) return 0;
    else {
        for ( int i = 1; i <= n; i++)
        {
            int x=i;
            while ( x %5==0) {c5++;x=x/5;}
            while (x%2==0) {c2++;x=x/2;}
        }
        if (c2<= c5)return c2;
        else return c5;
    }
}</pre>
