<h1>Cerinţa</h1>

<p>Se dă un vector cu <code>n</code> elemente numere întregi, <code>n</code> fiind număr par. Să se ordoneze crescător elementele din prima jumătate a vectorului și descrescător elementele din a doua jumătate.</p>

<p>Se vor defini și apela următoarele subprograme:</p>

<ul>
	<li><code>citire</code>, care citește  valoarea lui <code>n</code> și cele <code>n</code> elemente ale tabloului</li>
	<li><code>afisare</code>, care afișează elementele tabloului, separate prin exact un spațiu</li>
	<li><code>sortare</code>, care ordonează elementele vectorului cuprinse între doi indici transmiși ca parametru. Criteriul de ordonare (crescător/descrescător) va fi transmis ca parametru.</li>
</ul>

<p><strong>În programele C/C++ nu se vor folosi variabile globale</strong>.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>halfsort1.in</code> conţine pe prima linie numărul <code>n</code> si pe a doua linie <code>n</code> numere întregi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>halfsort1.out</code> va conţine pe prima linie cele <code>n</code> elemente ale vectorului, ordonate conform cerinței, separate printr-un spațiu.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code>, <code>n</code> număr par</li>
	<li>valoarea absolută a numerelor de pe a doua linie a fişierului de intrare va fi mai mică decât <code>1.000.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("halfsort1.in");
ofstream cout("halfsort1.out");

void citire(int&amp; n, int a[]){
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
}

void sortare(int n, int a[], int st, int dr, char ch){
    if(ch == 'c'){
        for(int i = st; i < dr; ++i)
            for(int j = i + 1; j <= dr; ++j)
                if(a[i] > a[j])
                    swap(a[i], a[j]);
    }
    else{
        for(int i = st; i < dr; ++i)
            for(int j = i + 1; j <= dr; ++j)
                if(a[i] < a[j])
                    swap(a[i], a[j]);
    }
}

void afisare(int n, int a[]){
    for(int i = 1; i <= n; ++i)
        cout << a[i] << ' ';
}

int main(){

    int n, a[101];
    citire(n, a);
    sortare(n, a, 1, n / 2, 'c');
    sortare(n, a, n / 2 + 1, n, 'd');
    afisare(n, a);
    return 0;
}</pre>
