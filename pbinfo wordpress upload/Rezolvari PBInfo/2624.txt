<h1>Cerința</h1>

<p>Dat fiind un număr natural <code>n</code>, să se determine numărul natural <code>m</code> care are are proprietatea că are în reprezentarea în baza <code>2</code> biți de <code>1</code> pe pozițiile unde <code>n</code> are biți de <code>0</code> și are biți de <code>0</code> pe pozițiile unde <code>n</code> are biți de <code>1</code>. De exemplu, dacă <code>n = 346</code>, atunci <code>m = 165</code>, deoarece <code>n</code> se reprezintă în baza <code>2</code> prin <code>101011010</code>, iar <code>m</code> prin <code>010100101</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>m</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 ≤ n ≤ 1 000 000 000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

using namespace std;

int a[33] , cnt , f[33];

int main()
{
    int n , m = 0 , p = 1;
    cin >> n;
    if(n == 0) a[++cnt] = 0;
    while(n)
    {
       a[++cnt] = n % 2;
       n /= 2;
    }
    for(int i = 1 ; i <= cnt ; i++)
    {
        if(a[i]==0) f[i] = 1;
        else f[i] = 0;
    }
    for(int i = 1 ; i <= cnt ; i++)
    {
        m = m + p*f[i];
        p *= 2;
    }
    cout << m;
    return 0;
}</pre>
