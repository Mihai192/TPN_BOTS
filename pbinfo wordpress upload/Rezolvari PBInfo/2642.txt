<h1>Cerința</h1>

<p>Să se scrie funcția cu următorul antet:</p>

<pre contenteditable="true" editable="true">int Phi(int n)</pre>

<p>Funcția primește ca parametru un număr natural <code>n</code> și trebuie să returneze numărul de numere naturale nenule prime cu <code>n</code> și strict mai mici decât <code>n</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 2.000.000.000</code></li>
	<li>Numele funcției este <code>Phi</code>.</li>
	<li>Spunem că un număr natural <code>x</code> este prim cu <code>n</code> dacă <code>cmmdc(x, n) = 1</code>.</li>
</ul>

<p><code>Phi(12) = 4</code>, deoarece <code>12</code> este prim cu numerele <code>1</code>, <code>5</code>, <code>7</code>, <code>11</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <cmath>
bool prim(int n)
{
    int cnt=0;
    for(int i = 1 ; i * i <= n ; ++i)
        if(n % i == 0 &amp;&amp; i * i != n)
            cnt+=2;
        else if(i * i == n)
            cnt++;
    if(cnt == 2)
        return 1;
    return 0;
}

int Phi(int n)
{
    if(prim(n))
        return n-1;
    int d=2;
    int rez = 1;
    while(n>1)
    {
        int p = 0;
        while(n % d==0)
            n/=d , p++;
        if(p != 0)
            rez = rez * (d-1) * pow(d , p - 1);
        d++;
        if(d * d > n)
            d = n;
    }
    return rez;
}</pre>
