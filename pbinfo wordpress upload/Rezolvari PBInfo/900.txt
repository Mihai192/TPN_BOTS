<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a subprogramului C++ <code>ordonare</code> care are <code>2</code> parametri:  <code>a</code>, prin care primeşte un tablou unidimensional  cu maximum <code>1000</code> de numere naturale mai mici decât <code>1.000.000.000</code> și <code>n</code>, numărul efectiv de elemente ale tabloului.</p>

<p>Subprogramul ordonează descrescător elementele tabloului <code>a</code>, fără a returna valori.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt;= 1000</code></li>
	<li>numele subprogramului cerut este <code>ordonare</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code></li>
	<li>elementele tabloului <code>a</code> sunt indexate de la zero</li>
</ul>

<p>Dacă <code>n=6</code> și <code>a=(63,273,9,83,93,123)</code>,  după apelul subprogramului tabloul <code>a</code> devine <code>(273,123,93,83,63,9)</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <algorithm>
void ordonare(int a[] , int n)
{
    int b[1000];
    for(int i = 0 ; i < n ; ++i)
    {
        b[i]=a[i];
    }
    sort(b , b+n);
    int p=0;
    for(int i = n-1 ; i >= 0 ; --i)
    {
        a[p]=b[i];
        p++;
    }
}</pre>
