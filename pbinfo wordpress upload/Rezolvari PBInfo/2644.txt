<h1>Cerința</h1>

<p>Într-o școală sunt <code>n</code> clase, fiecare având un număr diferit de elevi. Școală primește <code>m</code> pachete cu cărți, fiecare cu un număr diferit de cărți. Pentru ca o clasa să primească un pachet, numărul elevilor din acea clasa trebuie să fie egal cu numărul cărților din pachet. Să se determine câte clase primesc un pachet de cărți.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>clase.in</code> va avea pe prima linie numărul <code>n</code>, iar pe următoarea linie <code>n</code> numere naturale ordonate crescător, separate prin spații, reprezentând numărul de elevi din fiecare clasa. Pe următoarea linie numărul <code>m</code>, și pe următoarea linie <code>m</code> numere, reprezentând numărul de cărți din fiecare pachet.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>clase.out</code> va conține pe prima linie numărul de clase care primesc pachete cu cărți.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n, m ≤ 10000</code></li>
	<li>într-un pachet sunt maxim <code>1.000.000.000.000</code> cărți.</li>
	<li>într-o clasa sunt maxim <code>1.000.000.000.000</code> elevi.</li>
</ul>

<h3>Explicație</h3>

<p>Doar clasa cu <code>20</code> de elevi și cea cu <code>17</code> elevi vor primi pachete cu cărți.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("clase.in");
ofstream cout("clase.out");

int main()
{
    int n, m;
    long long a[10001];
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    cin >> m;
    long long x;
    int cnt = 0;
    for(int i = 1; i <= m; ++i){
        cin >> x;
        int st = 1, dr = n;
        bool ok = false;
        while(st <= dr){
            int mij = (st + dr) / 2;
            if(a[mij] == x){
                ok = true;
                break;
            }
            else
                if(a[mij] < x)
                    st = mij + 1;
                else
                    dr = mij - 1;
        }
        if(ok)
            cnt++;
    }

    cout << cnt;

    return 0;
}</pre>
