<h1>Cerinţa</h1>

<p>Se dau <code>n</code>  şiruri cu elemente numere întregi. Să se verifice despre fiecare dacă este ordonat crescător.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>; iar apoi <code>n</code> şiruri – pentru fiecare, se citeşte mai întâi numărul de valori, apoi valorile din şir.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran <code>n</code> valori <code>0</code> sau <code>1</code>, separate prin spaţii. A <code>k</code>-a valoare este <code>1</code>, dacă al <code>k</code>-lea şir este ordonat crescător, respectiv <code>0</code> dacă şirul nu este ordonat crescător.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code></li>
	<li>fiecare şir va avea cel mult <code>500</code> elemente, numere întregi cu cel mult <code>4</code> cifre fiecare</li>
</ul>

<h1>Explicație</h1>

<p>S-au citit <code>3</code> şiruri; primul nu este ordonat crescător, iar al doilea şi al treilea sunt ordonate crescător.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <cmath>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int a[501];
    int rez[11];
    for(int i = 1; i <= n; ++i){
        int nelem;
        cin >> nelem;
        for(int j = 1; j <= nelem; ++j)
            cin >> a[j];
        bool crescator = true;
        for(int j = 2; j <= nelem; ++j)
            if(a[j] < a[j-1])
                crescator = false;
        if(crescator)
            rez[i] = 1;
        else
            rez[i] = 0;
    }

    for(int i = 1; i <= n; ++i)
        cout << rez[i] << ' ';

    return 0;
}</pre>
