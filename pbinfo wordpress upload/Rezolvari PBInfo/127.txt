<h1>Cerinţa</h1>

<p>Se dă un număr natural <code>n</code>, apoi <code>n</code> numere  naturale. Să se calculeze suma obținută prin adunarea primei cifre a celor <code>n</code> numere citite.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi cele <code>n</code> numere naturale, separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>S</code>, reprezentând suma obținută prin adunarea primei cifre a celor  <code>n</code> numere citite.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt; 1000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât 2000000</li>
</ul>

<p>1. Exemplu:</p>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">5
100 212 3 45 5021</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">15</pre>

<h1>Explicație</h1>

<p>S-au citit <code>5</code> numere. Trebuie adunate prima cifra a fiecarui numar, iar suma lor este <code>1+2+3+4+5=15</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int prim(int n)
{
    int c;
    while(n)
    {
        c=n%10;
        n/=10;
    }
    return c;
}

int main()
{
    int n;
    cin >> n;
    int a[1001];
    int s=0;
    for (int i = 0 ; i < n ; ++i)
    {
        cin >> a[i];
        s+=prim(a[i]);
    }
    cout << s;
}</pre>
