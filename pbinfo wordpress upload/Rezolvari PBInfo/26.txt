<h1>Cerinţa</h1>

<p>Să se scrie o funcție C++ care să returneze cel mai mic număr care se poate scrie cu cifrele unui număr natural transmis ca parametru.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>cmmnr</code></li>
	<li>funcția va avea un parametru reprezentând numărul care se dă</li>
	<li>numărul care se dă va fi mai mic decât <code>1.000.000.000</code></li>
</ul>


<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <algorithm>
int cmmnr(int n)
{
    long long int v[10] , o=0 ,p=0 , cnt=0;
    while(n)
    {
        if(n%10==0)
        cnt++;
        else
        {
        v[p]=n%10;
        p++;
        }
        n/=10;
    }
    sort(v , v+p);
    int ok=0;
    for(int i = 0 ; i < p ; ++i)
    {
        o=o*10+v[i];
        ok++;
        if(ok==1)
        {
            for(int i = 0 ; i < cnt ; ++i)
            {
                o=o*10;
            }
        }
    }
        return o;
}</pre>
