<h1>Cerinţa</h1>

<p>Să se scrie o funcție C++ care, pentru un număr natural <code>n</code> transmis ca parametru, determină și întoarce prin intermediul unor parametri de ieșire cel mai mare număr prim mai mic decât <code>n</code> și cel mai mic număr prim mai mare decât <code>n</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>sub</code></li>
	<li>funcția va avea exact trei parametri, în această ordine:
	<ul>
		<li>primul parametru, <code>n</code>, reprezintă un număr natural, <code>3 ≤ n &lt; 1.000.000.000</code></li>
		<li><code>a</code> și <code>b</code> sunt parametrii prin care funcția va întoarce cele două valori cerute, astfel:
		<ul>
			<li><code>a</code> este cel mai mare număr prim mai mic decât <code>n</code></li>
			<li><code>b</code> este cel mai mic număr prim mai mare decât <code>n</code></li>
		</ul></li>
		</ul></li>
		</ul>

<p>Dacă <code>n=28</code>, apelul subprogramului va furniza prin parametrul <code>a</code> valoarea <code>23</code>, iar prin <code>b</code> valoarea <code>29</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.
		</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int sub(int n , int&amp; a , int&amp; b)
{
    for(int i = n-1 ; 1 ; --i)
    {
        int cnt=0;
        for(int j = 1 ; j * j <= i ; ++j)
        {
            if(i%j==0)
                cnt+=2;
            if(j*j==i)
                cnt--;
        }
        if(cnt==2)
        {
            a=i;
            break;
        }
    }
    for(int i = n +1 ; 1 ; ++i)
    {
        int cnt=0;
        for(int j = 1 ; j * j <= i ; ++j)
        {
            if(i%j==0)
                cnt+=2;
            if(j * j == i)
                cnt--;
        }
        if(cnt==2)
        {
            b=i;
            break;
        }
    }
    return a;
    return b;
}</pre>
