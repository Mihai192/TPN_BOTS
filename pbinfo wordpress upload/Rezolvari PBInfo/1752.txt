<h3>Cerința</h3>

<p>Scrieţi în limbajul <code>C++</code> definiţia completă a <strong>funcţiei recursive</strong> <code>nr_aparitii</code> cu următorul antet:</p>

<pre class="code_c cm-s-default" data-lang="text/x-csrc" contenteditable="true" editable="true"><span class="cm-keyword">unsigned</span> <span class="cm-variable">nr_aparitii</span>(<span class="cm-keyword">char</span> <span class="cm-operator">*</span><span class="cm-variable">sir</span>, <span class="cm-keyword">char</span> <span class="cm-operator">*</span><span class="cm-variable">secventa</span>)</pre>

<p>ce returnează numărul de apariţii ale şirului de caractere <code>secventa</code> în şirul <code>sir</code>.</p>

<h3>Restricții și precizări</h3>

<ul>
	<li>cele două şiruri nu conţin spaţii şi au cel mult <code>1000</code> de caractere</li>
	<li>biblioteca <code>cstring</code> (<code>string.h</code>) este deja inclusă la verificare</li>
</ul>

<h3>Exemplu:</h3>

<p><code>nr_aparitii("maimuta_samurai", "mu") = 2</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">unsigned nr_aparitii(char *sir, char *secventa) 
{ 
    char *adr_prim = strstr(sir, secventa); 
    if(adr_prim == NULL) 
        return 0; 
    else
    return 1 + nr_aparitii(adr_prim + strlen(secventa), secventa); 
}</pre>
