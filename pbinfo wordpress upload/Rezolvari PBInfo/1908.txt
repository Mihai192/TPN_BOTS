<h1>Cerința</h1>

<p>Dându-se şirul de fracţii <code>1/N, 2/N, 3/N, ...,N/N</code>, să se afle câte fracţii sunt ireductibile. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>N</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul de fracţii ireductibile.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2.000.000.022</code></li>
</ul>

<h3>Explicație</h3>

<p>Fracţiile sunt <code>1/4, 3/4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    long long int n , d = 2;
    cin >> n;
    long long int nr = n;
    while(n > 1)
    {
        if(n % d == 0)
        {
            nr /= d;
            nr *= d - 1;
            while(n % d == 0)
                n /= d;
        }
        if(d == 2) 
            d = 3;
        else 
            d += 2;
        if(d * d > n) 
            d = n;
    }
    cout << nr;
    return 0;
}</pre>
