<h1>Cerința</h1>

<p>La un concurs participă <code>n</code> concurenți, numerotați de la <code>1</code> la <code>n</code>. Pentru jurizare s-a format o echipă din <code>m</code> arbitri, fiecare acordând fiecărui concurent un anumit punctaj. Pentru fiecare concurent se calculează media punctajelor, după eliminarea celui mai mare și celui mai mic punctaj acordat.</p>

<p>Afișați clasamentul concursului în ordinea descrescătoare a mediilor calculate. La medii egale se va afișa mai întâi concurentul cu numărul de ordine mai mic.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n m</code>, iar apoi <code>n</code> seturi de câte <code>m</code> numere naturale, setul <code>i</code> reprezentând punctajele acordate de arbitri concurentului numerotat cu <code>i</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele de ordine ale concurenților participanți, în ordinea clasamentului construit, separate prin exact un spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li><code>1 ≤ m ≤ 50</code></li>
	<li>punctajele acordate de cei <code>m</code> arbitri sunt numere naturale mai mici decât <code>1000</code></li>
	<li>la calculul mediei se exclud toate punctajele minime și toate punctajele maxime</li>
	<li>pentru fiecare concurent există cel puțin trei punctaje diferite</li>
</ul>

<h1>Explicație</h1>

<p>Sunt <code>6</code> concurenți și <code>5</code> arbitri.<br>
Pentru primul concurent se utilizează punctajele <code>5 4</code>, și se obține media <code>4.5</code><br>
Pentru al doilea concurent se utilizează punctajele <code>4 6 2</code>, și se obține media <code>4</code><br>
Pentru al treilea concurent se utilizează punctajele <code>7 8</code>, și se obține media <code>7.5</code><br>
Pentru al patrulea concurent se utilizează punctajele <code>6 7 5</code>, și se obține media <code>6</code><br>
Pentru al cincilea concurent se utilizează punctajele <code>6 8 4</code>, și se obține media <code>6</code><br>
Pentru al șaselea concurent se utilizează punctajele <code>3 5 2</code>, și se obține media <code>3.33</code></p>

<p>Astfel, clasamentul este <code>3 4 5 1 2 6</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
    int n , m;
    cin >> n >> m;
    double a[1001] , v[1001];
    int b[51];
    for(int i = 0 ; i < n ; ++i)
    {
        int min = 1001 , max=0;
        for(int j = 0 ; j < m ; ++j)
        {
            cin >> b[j];
            if(b[j]<min)
                min=b[j];
            if(b[j]>max)
                max=b[j];
        }
        int cnt=0 , sum=0;
        for(int j = 0 ; j < m ; ++j)
        {
            if(b[j]!=min &amp;&amp; b[j]!=max)
                cnt++ , sum+=b[j];
        }
        a[i]=(double)sum/cnt;
        v[i]=(double)sum/cnt;
    }
    sort(a , a+n);
    for(int i = n-1 ; i >= 0 ; --i)
    {
        for(int j = 0 ; j < n ; ++j)
        {
            if(a[i]==v[j])
            {
                cout << j+1 << ' ';
                v[j]=-1;
                break;
            }
        }
    }
    return 0;
}</pre>
