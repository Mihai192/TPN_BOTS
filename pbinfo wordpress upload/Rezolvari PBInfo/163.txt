<h1>Cerinţa</h1>

<p>Se dă un vector cu  <code>n</code> elemente numere  naturale. Să se șteargă din vector toate elementele care sunt numere prime.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spaţii, reprezentând elementele vectorul.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran, separate prin spații, elementele vectorului obținut prin ștergerea elementelor prime.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 1000</code></li>
	<li>elementele vectorului vor fi mai mici decât <code>2</code><sup><code>30</code></sup></li>
</ul>

int main()
{
    int n;
    cin >> n;
    int a[1001];
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    for(int i = 1; i <= n; ++i){
        int ca = a[i];
        int d = 2, cnt = 1;
        while(ca > 1){
            int p = 0;
            while(ca % d == 0)
                ca /= d, p++;
            cnt *= (p + 1);
            d++;
            if(d * d > ca)
                d = ca;
        }
        if(cnt == 2){
            for(int j = i + 1; j <= n; ++j)
                a[j-1] = a[j];
            n--;
            i--;
        }
    }

    for(int i = 1; i <= n; ++i)
        cout << a[i] << " ";

    return 0;
