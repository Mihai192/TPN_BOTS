<h1>Cerinţa</h1>

<p>Se dau două numere naturale <code>a</code> și <code>b</code>. Determinati numărul de palindromuri cuprinse în intervalul determinat de cele două numere.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură două numere naturale <code>a</code> și <code>b</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul afișează pe ecran numărul <code>P</code>, reprezentând numărul de palindromuri cuprinse în intervalul determinat de <code>a</code> și <code>b</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ a,b ≤ 1.000.000</code></li>
	<li>nu este obligatoriu ca valorile citite să respecte condiția <code>a ≤ b</code>.</li>
</ul>

<h1>Explicație</h1>

<p>Cele <code>5</code> palindromuri din intervalul <code>85 130</code> sunt <code>88</code>, <code>99</code>, <code>101</code>, <code>111</code>, <code>121</code>.</p>

    {

           {
               n=i;
               aux=i;
               ogl=0;
               while(n!=0)
               {
                   ogl=ogl*10+n%10;
                   n=n/10;
               }
              if(aux==ogl ) cate++;
           }
    }
    else if(b<a)
    {
        for(int i=b;i<=a;i++)
           {
               n=i;
               aux=i;
               ogl=0;
               while(n!=0)
               {
                   ogl=ogl*10+n%10;
                   n=n/10;
               }

           }
    }
cout<<cate;
    return 0;
