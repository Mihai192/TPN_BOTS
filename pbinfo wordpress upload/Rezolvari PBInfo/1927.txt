<h1>Cerința</h1>

<p>Se dă un vector cu <code>n</code> elemente, numere naturale nenule. Afișați termenii în ordine crescătoare.</p>

<p><b>Termenii care apar de mai multe ori se vor afișa o singură dată.</b></p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale nenule, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul afișează elementele vectorului, în ordine crescătoare, separate prin spații, respectând cerința cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici sau egale cu <code>300.000</code></li>
	<li><b>Atenție la limita de memorie!</b></li>
</ul>

<h3>Explicație</h3>

<p>Elementele distincte care apar in vectorul inițial sunt: <code>9</code>, <code>8</code>, <code>1</code>, <code>2</code></p>

<p>După sortare, vectorul se va reprezenta ca: <code>1</code>, <code>2</code>, <code>8</code>, <code>9</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <bitset>
using namespace std;

bitset<300001>f;

int main()
{
    int n;
    cin >> n;
    int x;
    for(int i = 0 ; i < n ; ++i)
        cin >> x , f[x]=1;
    for(int i = 1 ; i < 300001 ; ++i)
        if(f[i]!=0)
            cout << i << ' ';
    return 0;    
}</pre>
