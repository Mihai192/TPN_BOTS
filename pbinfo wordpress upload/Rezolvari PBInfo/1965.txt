<h1>Cerința</h1>

<p>Dându-se numărul natural <code>n</code>, să se afle al <code>n</code>-lea termen al şirului <code>1, 11, 21, 1211, 111221, 312211, 13112221,...</code></p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa al <code>n</code>-lea termen al șirului.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 30</code></li>
</ul>

<h3>Explicație</h3>

<p>Al <code>4</code>-lea termen al sirului este <code>1211</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int a[10001] , b[10001] , lung = 1 , n , t , cnt;
int main()
{
    a[0] = 1;
    cin >> n;
    for(int i = 1; i < n; i++)
    {
        cnt = 0 , t = 0;
        for(int j = 0; j < lung; j++)
        {
            if(cnt == 0) cnt++;
            else if(a[j] == a[j - 1]) cnt++;
            else
            {
                b[t++] = cnt;
                b[t++] = a[j - 1];
                cnt = 1;
            }
        }
        if(cnt)
        {
            b[t++] = cnt;
            b[t++] = a[lung - 1];
        }
        swap(a , b);
        lung = t;
    }

    for(int i = 0; i < lung; i++) cout << a[i];
}</pre>
