<h1>Cerința</h1>

<p>Se citeşte de la tastatură un număr natural <code>n</code>. Să se calculeze şi să se afişeze media aritmetică a tuturor divizorilor săi. Media va fi cu fix <code>2</code> zecimale dupa virgula.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>S</code>, reprezentand media aritmetica.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Divizorii lui 9 sunt: <code>1</code>, <code>3</code> si <code>9</code>. Media aritmetica este <code>4.33</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <iomanip>
using namespace std;


int main()
{
    int n;
    cin >> n;
    int cnt=0;

    long long sumaDiv = 0;
    for (int d = 1; d * d <= n; ++d)
        if (n % d == 0)
        {
            cnt++;
            sumaDiv += d;
            if (d * d != n)
                {sumaDiv += n / d;
                cnt++;}
        }
    cout << fixed << setprecision(2)<< (float)sumaDiv/cnt;
    return 0;
}</pre>
