<h1>Cerinţa</h1>

<p>Se dau <code>n</code> numere  naturale şi un număr natural <code>k</code>. Afişaţi acele numere date care au cel puţin <code>k</code> divizori.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>divk.in</code> conţine pe prima linie numerele <code>n</code> şi <code>k</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>divk.out</code> va conţine pe prima linie numerele care au cel puţin <code>k</code> divizori, separate printr-un spaţiu, în ordinea în care au fost citite.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numerele de pe a doua linie a fişierului de intrare, precum şi <code>k</code> vor avea cel mult <code>9</code> cifre</li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("divk.in");
ofstream fout("divk.out");
int main()
{
    long long n,x,k;
    fin>>n>>k;
    for(int i=1;i<=n;i++)
    {
        fin>>x;
        int cate=0;
        long long aux=x;
        for(int j=1;j*j<=x;j++)
        {
            if(x%j==0) cate+=2;
            if(j*j==x) cate--;
        }
        if(cate>=k) fout<<aux<<" ";
    }
    return 0;
}</pre>
