<h1>Cerinţa</h1>

<p>Să se scrie o funcție C++ care primește ca parametri două numere <code>n</code> și <code>k</code> și determină cel mai mare număr care poate fi scris cu <code>k</code> cifre ale lui <code>n</code>. Funcția va întoarce rezultatul prin intermediul unui parametru de ieşire.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>numar</code></li>
	<li>funcția va avea trei parametri: primul parametru reprezintă numărul <code>n</code>; al doilea parametru reprezintă numărul <code>k</code>, iar al treilea va reprezenta valoarea determinată de funcție</li>
	<li>toți parametrii vor fi numere naturale nenule mai mici decât <code>2.000.000.000</code></li>
	<li><code>n</code> va avea cel puțin <code>k</code> cifre</li>
</ul>

<p>După apelul </p>

<p><code>x</code> va avea valoarea <code>975</code>.</p>


<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void numar(int n , int k , int &amp;nr)
{
    int f[11]={0};
    int cn = n;
    nr=0;
    while(cn)
    {
        f[cn%10]++;
        cn/=10;
    }
    for(int i = 9 ; i >= 0 ; --i)
    {
        for(int j = 1 ; j <= f[i] &amp;&amp; k ; ++j , k--)
            nr = nr * 10 + i;
    }
}</pre>
