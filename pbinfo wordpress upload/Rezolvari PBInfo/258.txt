<h1>Cerinţa</h1>

<p>Se dau două numere naturale <code>a</code> şi <code>b</code>, <code>a≤b</code>. Să se determine, dacă există, un număr prim <code>x</code> care aparţine intervalului închis <code>[a,b]</code> pentru care valoarea expresiei <code>|b+a-2*x|</code> este minimă, unde <code>|m|</code> este valoarea absolută a numărului întreg <code>m</code>. Dacă nu există nici un astfel de număr, se va afişa mesajul <code>NU EXISTA</code>, iar dacă există mai multe se va afişa cel mai mic.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>a</code> şi <code>b</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>x</code> determinat, sau mesajul <code>NU EXISTA</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ a ≤ b ≤ 1.000.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int prim(int n)
{
    int cnt=0;
    for(int i = 1 ; i * i <= n ; ++i)
    {
        if(n % i == 0)
            cnt+=2;
        if(i * i == n)
            cnt--;
    }
    if(cnt==2)
        return 1;
    else
        return 0;
}

int main()
{
    int a , b;
    cin >> a >> b;
    int nr=(a+b)/2;
    int min=1000000000 , nprim=-1;
    for(int i = nr ; i >= a ; --i)
    {
        if(prim(i))
        {
            min=a+b-2*i;
            nprim=i;
            break;
        }
    }
    for(int i = nr+1 ; i <= b ; ++i)
    {
        if(prim(i))
        {
            if(a+b-2*i < min)
                nprim=i;
            break;
        }
    }
    if(nprim!=-1)
    cout << nprim;
    else
    cout << "NU EXISTA";
    return 0;    
}</pre>
