<h3>Explicație</h3>

<p>Primul exemplu este descris în enunț. În cel de-al doilea exemplu, în timpul primului pas, Căldărușe va muta cărțile <code>[3, 1, 4]</code>. După aceea, numai cărțile <code>2</code> și <code>5</code> vor rămâne în stack (cartea <code>2</code> fiind deasupra cărții <code>5</code>). În timpul celui de-al doilea pas, Căldărușe va lua și cărțile <code>2</code> și <code>5</code>. După aceea, stackul rămâne gol, și deci în timpul următorilor pași, Căldărușe nu va mai muta nicio carte.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <queue>
using namespace std;

queue<int> Q;
int f[200001];

int main()
{
    int n;
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        int x;
        cin >> x;
        Q.push(x);
    }
    for(int i = 1 ; i <= n ; ++i)
    {
        int x;
        cin >> x;
        int cnt = 0;
        if(f[x]==0)
        {
            while(!Q.empty() &amp;&amp; Q.front()!=x)
                f[Q.front()]=1 , Q.pop() , cnt++;
            Q.pop();
            cnt++;
            f[x]=1;
        }
        cout << cnt << ' ';
    }
    return 0;
}</pre>
