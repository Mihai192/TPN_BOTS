<h1>Cerința</h1>

<p>Se dau două șiruri cu câte <code>n</code>, respectiv <code>m</code> elemente. Dacă înmulțim fiecare element din primul șir cu fiecare element din al doilea șir, să se afle câte produse sunt mai mici decât <code>p</code>. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> și <code>p</code>, iar apoi <code>n</code> numere naturale, separate prin spații, reprezentând elementele primului șir, după care citește numărul <code>m</code> urmat de  <code>m</code> numere naturale, reprezentând elementele celui de-al doilea șir.  </p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>nr</code>, reprezentând numărul produselor mai mici decât <code>p</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ p ≤ 200.000</code></li>
	<li><code>1 ≤ n , m ≤ 50.000</code></li>
	<li>elementele celor două șiruri sunt numere naturale mai mici decât <code>10.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Efectuând înmulțirile dintre elementele primului șir și ale celui de-al doilea obținem rezultatele: <code>25</code>, <code>34</code>, <code>50</code>, <code>68</code>, <code>75</code>, <code>102</code>, <code>100</code>, <code>136</code>, <code>125</code>, <code>170</code>. Dintre acestea <code>5</code> sunt mai mici decât <code>99</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int f[10001]={0} , g[10001]={0} , n , m , p;
    long long int c=0;
    cin >> n >> p ;
    for(int i = 0 ; i < n ; ++i)
    {
        int x;
        cin >> x;
        f[x]++;
    }
    cin >> m;
    for(int i =0 ; i < m ; ++i)
    {
        int x;
        cin >> x;
        g[x]++;
    }
    for(int i = 0 ; i < 10000 ; ++i)
    {
        for(int j = 0 ; j < 10000 &amp;&amp; i * j < p ; ++j)
            c+=f[i]*g[j];
    }
    cout << c;

    return 0;
}</pre>
