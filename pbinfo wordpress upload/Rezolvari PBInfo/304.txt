<h1>Cerinţa</h1>

<p>Se dă un şir format din <code>n</code> elemente, numere  naturale. Determinaţi câte secvenţe maximale cu elementele ordonate crescător conţine şirul.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>secvente.in</code> conţine pe prima linie numărul <code>n</code>; urmează cele <code>n</code> elemente ale vectorului, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>secvente.out</code> va conţine pe prima linie numărul <code>C</code>, reprezentând numărul de secvenţe cu elementele ordonate crescător din şirul dat.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>elementele şirului vor avea cel mult <code>4</code> cifre</li>
	<li>o secvență cu elemente ordonate crescător este maximală dacă adăugând la secvență încă un element ea nu mai are elementele ordonate crescător</li>
</ul>

<h1>Explicație</h1>

<p>Secvenţele din şir cu elementele ordonate crescător sunt: <code>(12)</code>, <code>(10 15 17 17)</code>, <code>(10 12 14)</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("secvente.in");
ofstream cout("secvente.out");

int main()
{
    int n;
    cin >> n;
    int a[1001];
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    int cnt = 0;
    for(int i = 2; i <= n; ++i)
        if(a[i] < a[i-1])
            cnt++;

    if(a[n] > a[n-1])
        cnt++;

    cout << cnt;

    return 0;
}</pre>
