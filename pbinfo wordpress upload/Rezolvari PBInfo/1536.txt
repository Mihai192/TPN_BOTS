<p>Să considerăm ecuaţii de gradul I, de forma: <code>expresie_1=expresie_2</code>. Expresiile specificate sunt constituite dintr-o succesiune de operanzi, între care există semnul <code>+</code> sau semnul <code>-</code> (cu semnificaţia binecunoscută de adunare, respectiv scădere). Fiecare operand este fie un număr natural, fie un număr natural urmat de litera <code>x</code> (litera <code>x</code> reprezentând necunoscuta), fie doar litera <code>x</code> (ceea ce este echivalent cu <code>1x</code>). </p>

<p>De exemplu: <code>2x-5+10x+4=20-x</code>. Observaţi că în ecuaţiile noastre nu apar paranteze şi necunoscuta este întotdeauna desemnată de litera mică <code>x</code>.</p>

<h1>Cerința</h1>

<p>Scrieţi un program care să rezolve ecuaţii de gradul I, în formatul specificat în enunţul problemei.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>ecuatii.in</code> conține pe prima linie un număr natural <code>n</code>, reprezentând numărul de ecuaţii din fişier. Pe fiecare dintre următoarele n linii este scrisă câte o ecuaţie.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>ecuatii.out</code> va conține <code>n</code> linii, câte una pentru fiecare ecuaţie din fişierul de intrare. Pe linia <code>i</code> va fi scrisă soluţia ecuaţiei de pe linia <code>i+1</code> din fişierul de intrare.</p>

<p>Dacă soluţia ecuaţiei este un număr real, atunci acesta se va scrie cu <code>4</code> zecimale. Răspunsul este considerat corect dacă diferenţa în valoare absolută dintre soluţia corectă şi soluţia concurentului este <code>&lt; 0.001</code>. </p>

<p>În cazul în care ecuaţia admite o infinitate de soluţii, se va scrie mesajul <code>infinit</code> (cu litere mici). </p>

<p>Dacă ecuaţia nu admite soluţii, se va scrie mesajul <code>imposibil</code> (de asemenea cu litere mici).</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code></li>
	<li>Lungimea unei ecuaţii nu depăşeşte <code>255</code> caractere.</li>
	<li>Ecuaţiile nu conţin spaţii.</li>
	<li>Numerele naturale care intervin în ecuaţii sunt <code>≤1000</code>.</li>
</ul>

ifstream fin("ecuatii.in");
ofstream fout("ecuatii.out");

string s;
int n, egal;

int a1 = 0, b1 = 0, a2 = 0, b2 = 0, temp = 0, a = 0, b = 0;
double rez;
char semn = '+';

int main()
{
    fin >> n;
    while (n--)
    {
        fin >> s;
        egal = s.find('=', 0);

        for(int i = 0; i <= egal; i++)
        {
            if(i == egal)
            {
                if(semn == '-')
                    temp = 0 - temp;
                if(isdigit(s[i - 1]))
                    b1 += temp;
                else
                    if(temp == 0)
                    {
                        if(semn == '-')
                            a1 -= 1;
                        else
                            a1 += 1;
                    }
                    else
                        a1 += temp;
                temp = 0;
                break;
            }

            if(s[i] == '+' &amp;&amp; i != 0)
            {
                if(semn == '-')
                    temp = 0 - temp;
                if(isdigit(s[i - 1]))
                    b1 += temp;
                else
                    if(temp == 0)
                    {
                        if(semn == '-')
                            a1 -= 1;
                        else
                            a1 += 1;
                    }
                    else
                        a1 += temp;
                semn = '+';
                temp = 0;
            }

            if(s[i] == '-')
            {
                if(semn == '-')
                    temp = 0 - temp;
                if(i == 0)
                {
                    semn = '-';
                    continue;
                }
                if(isdigit(s[i - 1]))
                    b1 += temp;
                else
                    if(temp == 0)
                    {
                        if(semn == '-')
                            a1 -= 1;
                        else
                            a1 += 1;
                    }
                    else
                        a1 += temp;
                semn = '-';
                temp = 0;
            }

            if(isdigit(s[i]))
            {
                temp *= 10;
                temp += int(s[i]) - 48;
            }
    }
    if(s.find('x' , 0) > egal)
        a1 = 0;

    temp = 0;
    semn = '+';

    for(int i = egal + 1; i <= s.length();i++)
    {
        if(i == s.length())
        {
            if(semn == '-')
                temp = 0 - temp;
            if(isdigit(s[i - 1]))
                b2 += temp;
            else
                if(temp == 0)

                        a2 -= 1;
                    else
                        a2 += 1;
                }
                else
                    a2 += temp;
            temp = 0;
            break;

        if(s[i] == '+' &amp;&amp; i != egal + 1)

            if(semn == '-')
                temp = 0 - temp;
                if(isdigit(s[i - 1]))
