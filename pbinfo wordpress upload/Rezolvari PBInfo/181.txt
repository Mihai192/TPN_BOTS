<p>Se numește secvență într-un vector toate elementele din acel vector cu indici cuprinși între două valori date.</p>

<h1>Cerinţa</h1>

<p>Se dau <code>n</code> numere naturale, reprezentând elementele unui vector. Determinați cea mai lungă secvență de elemente ordonate strict crescător din vector. Dacă există mai multe astfel de secvențe se va determina cea mai din stânga.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>secvcresc.in</code> conține numărul <code>n</code> și <code>n</code> valori naturale, reprezentând elementele vectorului. Valorile din fișier pot fi dispuse pe mai multe linii.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>secvcresc.out</code> va conține pe prima linie numerele <code>st</code> și <code>dr</code>, reprezentând indicii de început și de sfârșit ai secvenței determinate.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 10.000</code></li>
	<li>elementele vectorului vor fi mai mici decât <code>1.000.000</code> și sunt numerotate de la <code>1</code></li>
</ul>

<h1>Explicație</h1>

<p>Sunt <code>4</code> secvențe de elemente ordonate strict crescător: <code>2 6</code>, <code>4 5 8 9</code>, <code>6</code> şi <code>3 4</code>. Dintre acestea, secvența <code>4 5 8 9</code> are lungimea mai mare, iar indicii de început și sfârșit ale ei sunt <code>3 6</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("secvcresc.in");
ofstream cout("secvcresc.out");

int main()
{
    int n;
    cin >> n;
    int a[10001];
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    int lmax = 1, l = 1, st = 1, dr = 1;

    for(int i = 2; i <= n; ++i){
        if(a[i] > a[i-1]){
            l++;
            if(l > lmax)
                lmax = l, st = i - l + 1, dr = i;
        }
        else
            l = 1;
    }

    cout << st << ' ' << dr;

    return 0;
}</pre>
