<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Dându-se totodată un număr <code>p</code>, apare o intrebare care poate fi de unul dintre următoarele două feluri posibile: care este suma cifrelor valorii minime dintre primele <code>p</code> ale șirului (tipul <code>1</code>) respectiv, care este numărul de divizori ai valorii maxime dintre primele <code>p</code> elemente ale șirului (tipul <code>2</code>). </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură mai întâi numerele <code>n</code>,  <code>t</code> și <code>p</code> iar apoi <code>n</code> numere naturale (elementele șirului).</p>

<h1>Date de ieșire</h1>

<p>Dacă valoarea <code>t</code>, este <code>1</code>, programul va afișa pe ecran răspunsul la cerința de tipul <code>1</code>.<br>
Dacă valoarea <code>t</code>, este <code>2</code>, programul va afișa pe ecran răspunsul la cerința de tipul <code>2</code>.</p>

<p><strong>Indiferent de tipul cerinței, pe linia următoare se va afișa suma elementelor șirului.</strong></p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code></li>
	<li>cele <code>n</code> numere citite vor fi naturale nenule și mai mici decât <code>1.000.000.000</code></li>
	<li>numerele șirului sunt numerotate începând cu <code>1</code></li>
	<li>valoarea lui <code>p</code> este cuprinsă între <code>1</code> și <code>n</code>, inclusiv</li>
	<li><code>1 ≤ t ≤ 2</code></li>
</ul>

<h1>Exemple:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">6 1 3
14 13 16 22 15 1</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">4
81</pre>

<h3>Explicație</h3>

<p>Este vorba despre o interogare de tipul <code>1</code> și minimul dintre primele trei elemente ale șirului este 13 iar suma cifrelor sale este 4.</p>


<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">6 2 3
14 13 16 22 15 1</pre>


<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">5
81</pre>

<h3>Explicație</h3>

<p>Este vorba despre o interogare de tipul <code>2</code> și maximul dintre primele trei elemente ale șirului este 16 iar numărul său de divizori este 5.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

long long int a[11] , sum;

int nrdiv(int n)
{
    int cnt = 0;
    for(int i = 1 ; i * i <= n ; ++i)
    {
        if(n % i == 0)
            cnt+=2;
        if(i * i == n)
            cnt--;
    }
    return cnt;
}

int main()
{
    int n , t , p;
    cin >> n >> t >> p;
    for(int i = 1 ; i <= n ; ++i)
        cin >> a[i] , sum += a[i];
    int min = 1000000000;
    for(int i = 1 ; i <= p ; ++i)
        if(a[i] < min)
            min = a[i];
    int max = 0;
    for(int i = 1 ; i <= p ; ++i)
        if(a[i] > max)
            max=a[i];
    if(t == 1)
    {
        int s = 0;
        while(min)
        {
            s+=min%10;
            min/=10;
        }
        cout << s << endl << sum;
    }
    else
    {
        cout<< nrdiv(max) << endl << sum;
    }
    return 0;
}</pre>
