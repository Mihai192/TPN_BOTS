<h1>Cerința</h1>

<p>Aky și Alex joacă un joc interesant. Acesta se desfășoară în felul următor: aceștia au cartonașe cu numere naturale până la <code>10.000.000</code> (se consideră că au un număr infinit de cartonașe pentru fiecare număr natural mai mic sau egal cu <code>10.000.000</code>). Ei aleg la întâmplare <code>n</code> cartonașe din cele date, iar pentru fiecare număr <code>x</code> de pe un cartonaș ales caută cartonașul pe care se află scris cel mai mare divizor prim al numărului <code>x</code>.</p>

<p>Astfel observă că pentru multe din numerele alese cel mai mare divizor prim coincide, deci se hotărăsc să creeze mai multe perechi de  cartonașe astfel: primul cartonaș al perechii va fi un număr prim, <code>P</code>, care este cel mai mare divizor prim al cel puțin unuia dintre numerele alese, iar numărul <code>C</code> de pe al doilea cartonaș reprezintă pentru câte din numerele din șirul numerelor alese numărul de pe primul cartonaș este cel mai mare divizor prim. De asemenea, perechile sunt ordonate crescător după <code>P</code>.</p>

<p>Cei doi băieți nu se descurcă singuri când numerele de pe cartonașe sunt foarte mari, deci vă roagă pe voi să realizați un program care să realizeze afișarea numarului de perechi formate precum și a acestora pentru un șir de <code>n</code> cartonașe alese.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>jocprim.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>jocprim.out</code> va conține pe prima linie numărul de perechi formate, iar pe următoarele linii, câte o pereche de numere <code>P</code> și <code>C</code>, separate printr-un spațiu, cu semnificațiile din enunț.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 20000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>10.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p><code>2</code> este cel mai mare divizor prim a <code>2</code> numere din fișierul de intrare, <code>3</code> și <code>5</code> a <code>4</code> numere, iar <code>11</code> și <code>13</code> a câte unui număr. </p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("jocprim.in");
ofstream cout("jocprim.out");

int n , x , cnt , f[10000001] , maxi , cate;

int divprim(int n)
{
    int d = 2 , rez;
    while(n > 1)
    {
        int p = 0;
        while(n % d == 0) n /= d , p++;
        if(p) rez = d;
        d++;
        if(d * d > n) d = n;
    }
    return rez;
}

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> x;
        f[divprim(x)]++;
        maxi = max(maxi , divprim(x));
    }
    for(int i = 1 ; i <= maxi ; i++)
        if(f[i]) cate++;
    cout << cate << '\n';
    for(int i = 1 ; i <= maxi ; i++)
        if(f[i]) cout << i << " " << f[i] << '\n';
    return 0;
}</pre>
