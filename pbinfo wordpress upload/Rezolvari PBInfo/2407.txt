<h1>Dorel</h1>

<p>a primit de ziua lui un tricou pe care era scris numărul <code>n</code>. </p>

<h1>Cerința</h1>

<p>Aflaţi câţi divizori ai lui <code>n</code> au mulţimea cifrelor din scrierea lor inclusă în mulţimea cifrelor din scrierea lui <code>n</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>odaoni.in</code> conține pe prima linie numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>odaoni.out</code> va conține pe prima linie numărul divizorilor lui <code>n</code> care au mulţimea cifrelor din scrierea lor inclusă în mulţimea cifrelor din scrierea lui <code>n</code>..</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numărul <code>12</code> are divizorii <code>1,2,3,4,6,12</code> iar dintre aceştia numai <code>1,2,12</code> au mulţimea cifrelor inclusă în mulţimea cifrelor lui <code>12</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("odaoni.in");
ofstream cout("odaoni.out");
bool  f[15];
int cnt;
int main()
{
   int n;
   cin >> n;
   int aux = n;
   while(aux!=0)
   {
       f[aux%10]=1;
       aux /= 10;
   }
   for(int j = 1 ; j*j <= n ; j++)
   {
       if(n%j==0)
       {
           int x = j , ok = 0 , y = n/j;
           bool fdiv1[11]={0};
           while(x > 0)
           {
               fdiv1[x%10]=1;
               if(f[x%10]==0 &amp;&amp; fdiv1[x%10]==1)ok=1;
               x /= 10;
           }
            if(ok==0) cnt++;
           ok = 0;
           bool fdiv2[11]={0};
            while(y > 0)
           {
               fdiv2[y%10]=1;
               if(f[y%10]==0 &amp;&amp; fdiv2[y%10]==1)ok=1;
               y /= 10;
           }
            if(ok==0) cnt++;
           if(j*j==n &amp;&amp; ok == 0) cnt--;
       }
   }
   cout << cnt;
}</pre>
