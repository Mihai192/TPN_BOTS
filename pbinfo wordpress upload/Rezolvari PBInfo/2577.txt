<h1>Cerința</h1>

<p>Se dau mai multe perechi de forma număr, bit. Afișați pentru fiecare pereche valoarea bitului corespunzător din reprezentarea internă pe <code>64</code> de biți a numărului dat.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>T</code>, iar apoi <code>T</code> linii care conțin, fiecare, câte două numere separate prin spațiu <code>n</code> și <code>b</code> cu semnificația că se cere valoarea bitului de pe poziția <code>b</code> din reprezentarea internă a numărului <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran, pe prima linie, neseparate prin spațiu, cele <code>T</code> valori, determinate, în ordinea dată.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>numerele <code>n</code> sunt memorate cu tipul de date pe <code>64</code> de biți, cu semn</li>
	<li>numărul <code>b</code> este natural, cuprins între <code>0</code> și <code>63</code>, inclusiv</li>
</ul>

<h3>Explicație</h3>

<p>Biții unui număr sunt numerotați de la dreapta începând cu poziția <code>0</code>. Astfel, ultimul bit al lui <code>6</code> este <code>0</code> iar antepenultimul bit al lui <code>5</code> este <code>1</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

int main()
{
    int t,b;
    unsigned long long n;
    cin>>t;
    for(int i=1;i<=t;i++)
    {
        cin>>n>>b;
        cout<<((n&amp;(1ull<<b))>>b);
    }
    return 0;
}</pre>
