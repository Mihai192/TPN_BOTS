<h1>Cerința</h1>

<p>Se dau două numere naturale <code>S</code> şi <code>K</code>. Să se afle cel mai mic număr natural <code>A</code> care are suma cifrelor egală cu <code>S</code>, precum şi restul împărţirii lui <code>A</code> la <code>K</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>numar9.in</code> conține pe prima linie numerele naturale <code>S</code> şi <code>K</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>numar9.out</code> va conține pe prima linie numărul <code>A</code>, iar pe a doua linie restul împărţirii lui <code>A</code> la <code>K</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ S ≤ 20.000.000</code></li>
	<li><code>1 ≤ K ≤ 1.000.000</code></li>
	<li>Pentru prima cerinţă se acordă <code>60p</code>, iar pentru a doua <code>40p</code></li>
	<li>Pentru a primi punctajul pentru a doua cerinţă, în fişierul <code>numar9.out</code> trebuie să afişaţi două numere</li>
</ul>

<h3>Explicație</h3>

<p>Cel mai mic număr natural care are suma cifrelor egală cu <code>25</code> este <code>799</code>. Restul împărţirii lui <code>799</code> la <code>13</code> este <code>6</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin ("numar9.in");
ofstream fout ("numar9.out");
int unsigned k,i,v[2200001];
int main()
{
    long long unsigned s , x , r , a , b;
    fin >> s >> k;
    r = s % 9;
    a = s / 9;
    b = a / 6;
    for (i = 1 ; i <= b ; i++) v[i]=999999;
    for (i = b + 1 ; i <= b + a % 6 ; i++) v[i]=9;
    if (r != 0 &amp;&amp; s != 0) v[0] = r;
    x = 0;
    if (v[0] != 0)
        for (i = 0 ; i <= b + a % 6 ; i++)
        {
            fout << v[i];
            if (v[i] < 10) x = (x * 10 + v[i])% k;
            else
            {
                while (v[i]!=0)
                {
                    x = (x * 10 + v[i] % 10)% k;
                    v[i] /= 10;
                }
            }
        }
    else
        for (i = 1 ; i <= b + a % 6 ; i++)
        {
            fout<<v[i];
            if (v[i]<10) x = (x * 10 + v[i])% k;
            else
            {
                while (v[i] != 0)
                {
                    x = (x * 10 + v[i] % 10)% k;
                    v[i] /= 10;
                }
            }
        }
        fout << " " << x;
        return 0;
}</pre>
