<h1>Cerința</h1>

<p>Fermierului Ion îi place să-și organizeze activitățile din timp. Chiar prea mult. Acesta ajunge să-și facă planuri și pentru câțiva ani. Singura dilemă pe care o are acesta este în ce zi a săptămânii va cădea data pe care își propune să efectueze respectiva activitate.</p>

<p>Preocuparea sa este atât de mare încât se întreabă și de evenimente care au avut loc în trecut.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul o dată calendaristică, formată din <code>3</code> numere naturale <code>Z</code>, <code>L</code> și <code>A</code> reprezentând ziua, luna și anul.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran una din zilele săptămânii (<code>Luni</code>, <code>Marti</code>, <code>Miercuri</code>, <code>Joi</code>, <code>Vineri</code>, <code>Sambata</code>, <code>Duminica</code>) semnificând ziua săptămânii corespunzătoare datei cerute.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ Z ≤ 31</code></li>
	<li><code>1 ≤ L ≤ 12</code></li>
	<li><code>1900 ≤ A ≤ 2100</code></li>
	<li>Nu se vor cere zile invalide precum <code>29 februarie 2015</code> sau <code>31 noiembrie 1980</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int x = 1 , zi ,  luna , an , luni[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

int bisect(int an)
{
    return (an % 400) == 0 || (an % 4 == 0  &amp;&amp;  an % 100 != 0);
}

int main()
{
    cin >> zi >> luna >> an;
    for(int a = 1900 ; a < an ; a++)
    {
        if(bisect(a))x += 366;
        else x += 365;
    }
    for(int l = 1 ; l < luna ; l++)
    {
        if(bisect(an) &amp;&amp; l == 2) x += 29;
        else x += luni[l];
    }
    x = (x + zi - 1) % 7;
    if(x == 0) cout << "Duminica";
    else if (x == 1) cout << "Luni";
    else if (x == 2) cout << "Marti";
    else if (x == 3) cout << "Miercuri";
    else if (x == 4) cout << "Joi";
    else if (x == 5) cout << "Vineri";
    else if (x == 6) cout << "Sambata";
}</pre>
