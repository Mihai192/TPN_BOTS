<h1>Cerința</h1>

<p>Scrieți funcția <code>inserare</code> care primește ca parametru un număr natural <code>n</code> și furnizează prin intermediul aceluiași parametru numărul obținut din <code>n</code> prin inserarea, între oricare două cifre alăturate ale sale, a valorii absolute a diferenței acestora.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>10 ≤ n ≤ 99999</code></li>
	<li>Puteți apela funcția matematică <code>abs</code>.</li>
</ul>

<p>Dacă <code>n = 7255</code>, atunci, după apelul <code>inserare(n)</code> valoarea lui n va fi <code>7523505</code>.</p>

<h3>Important</h3>

<p>Soluția propusă va conține definiția funcției cerute. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <cmath>
using namespace std;

void inserare(int &amp;n)
{
    int v[20] , p = 0;
    while(n)
        v[p]=n%10 , n/=10 , p++;
    int nr = 0;
    for(int i = p-1 ; i > 0 ; --i)
        n = n * 10 + v[i] , n = n * 10 + abs(v[i]-v[i-1]);
    n = n * 10 + v[0];

}</pre>
