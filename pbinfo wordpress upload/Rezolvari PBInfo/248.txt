<h1>Cerinţa</h1>

<p>Se dă un şir cu <code>n</code> numere  naturale distincte două câte două. Să se determine poziţia pe care s-ar afla primul element al şirului în şirul sortat.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>pozitie.in</code> conţine pe prima linie numărul <code>n</code>. Urmează <code>n</code> numere naturale, dispuse pe mai multe linii, separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>pozitie.out</code> va conţine pe prima linie numărul <code>p</code>, reprezentând poziţia în şirul sortat a primului element din şirul dat.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10000</code></li>
	<li>elementele şirului vor avea cel mult <code>9</code> cifre</li>
	<li>numerotarea elementelor se face de la <code>1</code></li>
</ul>

<h1>Explicație</h1>

<p>Şirul sortat este <code>7 13 45 79 267 628</code>. În acest şir, valoarea <code>267</code> (prima din şirul iniţial) se află pe poziţia a <code>5</code>-a.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("pozitie.in");
ofstream fout("pozitie.out");
int a[10];
int main()
{
    int n , x , a , p = 1;
    fin >> n >> x;
    for(int i = 2 ; i <= n ; ++i)
    {
        fin >> a;
        if(a < x) p++;
    }
    fout << p;
    return 0;
}</pre>
