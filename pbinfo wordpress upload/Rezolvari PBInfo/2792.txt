Subprogramul returnează cel mai mic număr natural <code>x</code> (<code>n&lt;x</code>) care NU este prim, cu proprietatea că în intervalul <code>[n,x]</code> există un singur număr prim.</p>

<h1>Important</h1>

<p>Soluția propusă va conține definiția subprogramului cerut. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">bool prim(int n)
{
    int cnt = 0;
    for(int d = 1 ; d * d <= n ; ++d)
    {
        if(n % d == 0)
            cnt+=2;
        if(d * d == n)
            cnt--;
    }
    if(cnt == 2)
        return 1;
    return 0;
}

int interval(int n)
{
    for(int i = n ; 1 ; ++i)
    {
        if(prim(i))
            return i+1;
    }
}</pre>
