<p>Un număr natural este pseudoperfect dacă este divizor al sumei divizorilor săi.</p>

<h1>Cerinţa</h1>

<p>Să se scrie un program care verifică dacă un număr natural citit de la tastatură este pseudoperfect.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code></p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran mesajul <code>DA</code>, dacă <code>n</code> este număr pseudoperfect, respectiv <code>NU</code>, în caz contrar.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt; 1.000.000.000</code></li>
</ul>


<h1>Exemplu1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">120</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">DA</pre>

<h1>Explicație</h1>

<p>Suma divizorilor lui <code>120</code> este <code>360</code>, care este multiplu al lui <code>120</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <cmath>
using namespace std;

int main ()
{
    int n;
    cin >> n;
    int s=0;
    for (int i = 1 ; i*i <= n;++i )
    {
        if (n%i==0)
        {
        s+=i; 
        s+=n/i;
        }
        if (i*i==n)
        s-=i;
    }
    if (s%n==0)
    cout << "DA";
    else 
    cout << "NU";
    return 0;    
}</pre>
