<h1>Cerința</h1>

<p>Scrieți funcția <code>sumaMinMax</code> care primește ca parametri un tablou unidimensional <code>v</code> format din <code>n</code> elemente și un număr natural <code>n</code>. Funcția va returna un vector cu două elemente, primul element reprezentând cea mai mică sumă formată din exact <code>n-1</code> elemente dintre cele <code>n</code> ale lui <code>v</code>, iar cel de-al doilea element reprezentând cea mai mare sumă formată din exact <code>n-1</code> elemente dintre cele <code>n</code> ale lui <code>v</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>sumaMinMax</code> primește doi parametri, în această ordine, un tablou unidimensional <code>v</code> și un număr natural <code>n</code> reprezentând numărul de elemente ale lui <code>v</code></li>
	<li>elementele lui <code>v</code> sunt indexate de la <code>0</code></li>
	<li><code>sumaMinMax</code> returnează un pointer la <code>int</code>, reprezentând un tablou alocat dinamic în cadrul funcției, tablou cu două elemente cu semnificațiile din cerință</li>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li><code>1 ≤ v[i] ≤ 50</code>, unde <code>v[i]</code> reprezintă al <code>i</code>-lea element din <code>v</code></li>
</ul>

<h3>Explicație</h3>

<p><code>10</code> reprezintă primul element din vectorul returnat, iar <code>14</code> reprezintă al doilea element.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int* sumaMinMax(int a[], int n){
    int *v = new int [2];
    int sum = 0;
    int mini = 51;
    int maxi = 0;
    for(int i = 0; i < n; ++i){
        sum += a[i];
        if(a[i] > maxi)
            maxi = a[i];
        if(a[i] < mini)
            mini = a[i];
    }
    v[0] = sum - maxi;
    v[1] = sum - mini;
    return v;
}</pre>
