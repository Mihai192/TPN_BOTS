<h1>Cerința</h1>

<p>Marian a fost foarte interesat de metoda divide et impera și a primit de la profesorul său o problemă : se dă o matrice de dimensiune <code>2</code><sup><code>n</code></sup> și ea trebuie parcursă după o anumită regulă bazată pe divide et impera . Pe baza a trei exemple , el trebuie să descopere regula și s-o aplice . Din păcate , acesta nu reusește și vă cere ajutorul . Vrea o rezolvare foarte eficienta atât din punct de vedere al timpului de execuție cât și a limitei de memorie .</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>matrice_div_et_imp.in</code> conține pe prima linie numărul <code>n</code>, iar pe celelalte <code>2</code><sup><code>n</code></sup> linii câte <code>2</code><sup><code>n</code></sup> numere naturale nenule separate prin spații .</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>matrice_div_et_imp.out</code> va conține o linie cu cele <code>2</code><sup><code>n</code></sup><code>*2</code><sup><code>n</code></sup> numere ale matricei , parcurse după respectiva regulă .</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 ≤ n ≤ 9 </code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<h6>Observație pentru ultimul exemplu : toate numerele sunt pe o singură linie </h6>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("matrice_div_et_imp.in");
ofstream cout("matrice_div_et_imp.out");

int m[2001][2001];

void afisare(int sus , int jos , int st , int dr)
{
    if(jos - sus >= 1)
    {
        int mij1=(sus + jos) / 2;
        int mij2=(st + dr) / 2;
        afisare(sus , mij1 , st , mij2);
        afisare(mij1+1 , jos , mij2+1 , dr);
        afisare(sus , mij1 , mij2+1 , dr);
        afisare(mij1+1 , jos , st , mij2);
    }
    else
        cout << m[sus][st] << ' ';
}

int main()
{
    int n;
    cin >> n;
    n = pow(2 , n);
    for(int i = 1 ; i <= n ; ++i)
        for(int j = 1 ; j <= n ; ++j)
            cin >> m[i][j];
    afisare(1 , n , 1 , n);
    return 0;
}</pre>
