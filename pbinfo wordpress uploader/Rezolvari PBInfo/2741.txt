<p>După ce ți-ai dat seama că nu poți învinge nici unul dintre monștrii (din problema <a href="https://www.pbinfo.ro/?pagina=probleme&amp;id=2729"><span class="caps">SAO</span></a>), ai decis să te retragi și să devii un fermier. Din banii pentru cumpărarea echipamentului ai cumpărat o parcelă codificată sub forma unei matrice de <code>n</code> linii și <code>m</code> coloane, pentru fiecare zonă cunoscându-se fertilitatea ei. Cum nu ai bani ca să cultivi pământul, dorești să selectezi o parcelă în care toate zonele să aibă aceeași fertilitate, iar fertilitatea totală să fie maximă. Fertilitatea totală a unei parcele este egală cu suma fertilităților zonelor care compun acea parcelă.</p>

<h1>Cerința</h1>

<p>Dându-se matricea codificărilor zonelor din teren, să se determine fertilitatea totală maximă a unei parcele în care  toate zonele au aceeași fertilitate.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>sao1.in</code> conține pe prima linie numerele <code>n</code> şi <code>m</code>, iar pe următoarele <code>n</code> linii câte <code>m</code> numere naturale, separate prin spaţii, reprezentând elementele matricei.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>sao1.out</code> va conține numărul <code>P</code>, reprezentând fertilitatea totală maximă a unei parcele, în condițiile date.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n,m ≤ 500</code></li>
	<li>fertilitatea totală se va încadra pe tipul de date <code>long long</code>.</li>
	<li>nu există sol infertil (a cărui fertilitate să fie mai mică sau egală cu <code>0</code>).</li>
	<li>acesta este <em>“bad ending”-ul</em> problemei <a href="https://www.pbinfo.ro/?pagina=probleme&amp;id=2729"><span class="caps">SAO</span></a></li>
</ul>

<h1>Exemplu 1:</h1>

<p><code>sao1.in</code></p>

<pre contenteditable="true" editable="true">4 4
1 1 1 3
1 1 1 3
1 1 1 3
3 3 3 3</pre>

<p><code>sao1.out</code></p>

<pre contenteditable="true" editable="true">21</pre>

<h3>Explicație</h3>

<p>Cele <code>7</code> valori de <code>3</code> formează zona cu profit maxim.</p>

<h1>Exemplu 2:</h1>

<p><code>sao1.in</code></p>

<pre contenteditable="true" editable="true">4 4
3 5 1 3
1 5 5 3
5 5 5 3
3 3 3 3</pre>

<p><code>sao1.out</code></p>

<pre contenteditable="true" editable="true">30</pre>

<h3>Explicație</h3>

<p>Cele <code>6</code> valori de <code>5</code> formează zona cu profit maxim.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("sao1.in");
ofstream cout("sao1.out");

long long a[501][501], c, f;
int n, m;

void fill(int i, int j){
    c++;
    a[i][j] = 0;
    if(a[i+1][j] == f)
        fill(i+1, j);
    if(a[i-1][j] == f)
        fill(i-1, j);
    if(a[i][j+1] == f)
        fill(i, j+1);
    if(a[i][j-1] == f)
        fill(i, j-1);
}

int main()
{
    cin >> n >> m;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            cin >> a[i][j];

    long long maxi = 0;

    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            if(a[i][j] != 0){
                f = a[i][j];
                c = 0;
                fill(i, j);
                if(c * f > maxi)
                    maxi = c * f;
            }

    cout << maxi;

    return 0;
}</pre>
