<p>Se consideră un șir de numere naturale nenule <code>a[1]</code>, <code>a[2]</code>, …, <code>a[n]</code>. Asupra șirului se efectuează <code>Q</code> interogări. Fiecare interogare este dată de o pereche <code>(x, s)</code>: care este indicele maxim <code>p</code> cu proprietatea că <code>a[i] ≤ x</code>, pentru orice <code>i=1..p</code> și în plus <code>a[1] + a[2] + ... + a[p] &lt;= s</code>? </p>

<h1>Cerința</h1>

<p>Trebuie să răspundeți la fiecare din cele <code>Q</code> întrebări.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații, reprezentând elementele șirului. Apoi se citește valoarea <code>Q</code> și la final se citesc Q perechi de forma <code>(x, s)</code> reprezentând întrebările. </p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe câte o linie la ecran <code>Q</code> valori reprezentând răspunsurile la întrebări.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100 000</code></li>
	<li><code>1 ≤ Q ≤ 100 000</code></li>
	<li><code>1 ≤ a[i] ≤ 1 000</code> pentru orice <code>i=1..n</code></li>
	<li>pentru fiecare întrebare, <code>1 ≤ x, s ≤ 1 000 000 000</code></li>
</ul>

<h3>Explicație</h3>

<p>La prima întrebare, <code>x=8</code>, <code>s=10</code>. Indicele maxim este <code>3</code> pentru că primele trei valori din șir sunt mai mici sau egale cu <code>8</code>, iar <code>5 + 3 + 1 ≤ 10</code>.<br>
La a doua întrebare, răspunsul este <code>0</code> deoarece primul număr din șir este <code>5</code> care este mai mare decât <code>x=4</code>.<br>
La a cincea întrebare, <code>x=10</code> și <code>s=100</code>. Răspunsul este chiar lungimea șirului.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int a[100001] , s[100001] , maxime[100001];

int cb(int a[] , int n , int val)
{
    int st = 1 , dr = n;
    while (st <= dr)
    {
        int mij = (st + dr) / 2;
        if(val < a[mij]) dr = mij - 1;
        else st = mij + 1;
    }
    return dr;
}

int main()
{
    int n;
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
        cin >> a[i] , s[i]=s[i-1]+a[i];
    for(int i = 1 ; i <= n ; ++i)
        if(a[i]>maxime[i-1])
            maxime[i]=a[i];
        else
            maxime[i]=maxime[i-1];
    int q;
    cin >> q;
    int x , p;
    for(int i = 1 ; i <= q ; ++i)
    {
        cin >> x >> p;
        int val1 = cb(maxime , n , x);
        int val2 = cb(s , n , p);
        cout << min(val1 , val2) << endl;
    }
    return 0;
