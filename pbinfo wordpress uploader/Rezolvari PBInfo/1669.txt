<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Aflaţi câte dintre ele sunt <strong>palindrom prim norocoase</strong>. Un număr este <strong>palindrom prim norocos</strong> dacă este <strong>palindrom</strong> (egal cu răsturnatul său, de exemplu <code>121</code>), <strong>prim</strong> (are exact <code>2</code> divizori, de exemplu <code>3</code>) şi <strong>norocos</strong> (pătratul numărului se poate scrie ca sumă de numere consecutive, exemplu <code>3</code>. <code>3 * 3 = 9 = 2 + 3 + 4</code>).</p>

<h1>Date de intrare</h1>

<p>Se va citi de la tastatura numărul <code>n</code> si <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Se va afisa numarul de numere care respecta conditiile.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere vor fi mai mici decât <code>2.000.000.000</code>.</li>
</ul>

<h3>Explicație</h3>

<p>Sunt <code>6</code> numere, dintre care doar <code>4</code> respectă condițiile: <code>3</code>, <code>5</code>, <code>7</code> şi <code>11</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int prim(int n)
{
    int cnt=0;
    for(int i = 1 ; i * i <= n ; ++i)
    {
        if(n%i==0)
            cnt+=2;
        if(i*i==n)
            cnt--;
    }
    if(cnt==2)
        return 1;
    else
        return 0;
}

int pal(int n)
{
    int ogl=0;
    int cn=n;
    while(n)
    {
        ogl=ogl*10+n%10;
        n/=10;
    }
    if(ogl==cn)
        return 1;
    else
        return 0;
}

int main()
{
    int n;
    cin >> n;
    int a[1000];
    int cnt=0;
    for(int i = 0 ; i < n ; ++i)
        cin >> a[i];
    for(int i = 0 ; i < n ; ++i)
    {
        if(pal(a[i])&amp;&amp;prim(a[i])&amp;&amp;a[i]%2==1)
            cnt++;
    }
    cout << cnt;
    return 0;
}</pre>
