<p>Împăratul Persiei, Seram dă de ştire în toată împărăția sa, că vrea să-şi aleagă vistiernic care să-i administreze averea. El precizează că visteria palatului are <code>n</code> încăperi numerotate cu numere naturale diferite de <code>0</code>. Suma de bani pe care o are în aceste încăperi este egală cu produsul numerelor cu care sunt numerotate încăperile visteriei. De asemenea împăratul dă  de ştire că va alege pe acel supus vistiernic, care ştie să calculeze în câte zerouri se termină numărul ce reprezintă averea sa.</p>

<h1>Cerința</h1>

<p>Scrieţi un program care determină  numărul de zerouri în care se termină averea  împăratului.</p>

<h1>Date de intrare</h1>

<p>De pe prima linie din fişierul de intrare  <code>vistiernic.in</code> se citeşte un număr natural <code>n</code>, ce reprezintă numărul de încăperi ale visteriei.</p>

<p>De pe linia următoare se citesc <code>n</code> numere naturale separate prin spaţii reprezentând numerele înscrise pe uşile încăperilor visteriei. </p>

<h1>Date de ieșire</h1>

<p>Pe prima linie în fişierul  <code>vistiernic.out</code> se va afişa un număr ce reprezintă numărul de  zerouri în care se termină averea împăratului.</p>

<h1>Restricții și precizări</h1>

<p><code>1 ≤ n ≤ 1000</code><br>
<code>1 ≤</code> numărul de pe uşă <code>≤ 32767</code>, pentru fiecare încăpere.</p>

<h3>Explicație</h3>

<p>Averea împăratului este egală cu numărul dat de produsul <code>8*225*64*15</code>. Acest număr se termină în <code>3</code> zerouri.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("vistiernic.in");
ofstream fout("vistiernic.out");

int main()
{
    int n;
    fin >> n;
    long long int tmp , cnt2=0 , cnt5=0;
    for(int i = 0 ; i < n ; ++i)
    {
        fin >> tmp;
        while(tmp%2==0)
        {
            cnt2++;
            tmp/=2;
        }
        while(tmp%5==0)
        {
            cnt5++;
            tmp/=5;
        }
    }
    if(cnt2>cnt5)
    fout << cnt5;
    else
    fout << cnt2;
    return 0;
}</pre>
