<h1>Cerința</h1>

<p>Să se scrie funcția cu următorul antet:</p>

<pre class="code_c cm-s-default" data-lang="text/x-csrc" contenteditable="true" editable="true"><span class="cm-keyword">void</span> <span class="cm-variable">Insert</span>(<span class="cm-keyword">int</span> <span class="cm-variable">a</span>[], <span class="cm-keyword">int</span> <span class="cm-operator">&amp;</span><span class="cm-variable">n</span>)</pre>

<p>Funcția primește ca parametri un vector <code>a</code> de numere naturale și <code>n</code> – numărul elementelor vectorului. Să se insereze după fiecare număr impar dublul său.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>după inserarea de elemente, este garantat că <code>n</code> va avea lungimea cel mult <code>800.000</code></li>
	<li>inițial vectorul are cel puțin un element</li>
	<li><code>0 ≤ a[i] ≤ 1.000.000</code>, pentru orice <code>i=0,n-1</code>.</li>
	<li>vectorul este indexat de la <code>0</code> la <code>n-1</code></li>
	<li>este garantat că vectorul <code>a</code> va putea memora în final toate elementele inițiale și cele inserate</li>
</ul>

<p>Dacă <code>n = 5</code> și <code>a = (22,5,7,8,11)</code>, atunci <code>Insert(a, n)</code> trebuie să obțină <code>n = 8</code> și <code>a = (22,5,10,7,14,8,11,22)</code></p>

<h3>Important</h3>

<p>Soluţia propusă va conţine definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void Insert(int a[], int &amp;n){
    int cnt = 0;
    for(int i = 0; i < n; ++i)
        if(a[i] % 2 == 1)
            cnt++;

    for(int i = n + cnt - 1, j = n - 1; i >= 0, j >= 0; --i, --j){
        if(a[j] % 2 == 1){
            a[i] = a[j] * 2;
            a[i-1] = a[j];
            i--;
        }
        else
            a[i] = a[j];
    }
    /*
        5
        1 2 3 4 5
        cnt = 3;
        i = 8, j = 5;
        daca a[j] este impar -> 5 10 i--;
        i--, j--;
        i = 6, j = 4;
        daca a[j] este par -> il pun pe 4 normal in sir, la pozitia i = 6;
        i--, j--;
        i = 5, j = 3;
        daca a[j] este impar -> 3 6 i--;
        i--, j--; 1 2 3 3 6 4 5 10
        i = 3, j = 2;
        daca a[j] este par -> il pun pe 2 normal in sir, la pozitia i = 3;
        i--, j--;
        i = 2, j = 1;
        daca a[j] este impar -> 1 2 i--;
        i--, j--;
        i = j = 0;
        1 2 2 3 6 4 5 10
    */

    n += cnt;

}</pre>
