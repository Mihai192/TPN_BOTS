<p>Un număr natural se numește aproape prim dacă poate fi scris ca produs de două numere naturale prime distincte.</p>

<h1>Cerinţa</h1>

<p>Se citește un număr natural. Să se verifice dacă este aproape prim.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran cuvântul <code>DA</code>, dacă <code>n</code> este aproape prim, respectiv <code>NU</code> în caz contrar.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000.000</code></li>
</ul>

<h1>Exemplu 1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">35</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">DA</pre>

<h1>Exemplu 2:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">50</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">NU</pre>

<h1>Explicație</h1>

<p><code>35=5*7</code>. <code>50</code> nu poate fi scris ca produs de două numere prime distincte.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main ()
{
    int n;
    cin >> n;
    int cnt=0;
    for (int i = 1; i * i <= n; ++i)
    {
        if (i*(n/i)==n)
        cnt+=2;
        if (i*i==n)
        cnt--;
    }
    if (cnt==4)
    cout << "DA";
    else
    cout << "NU";
}</pre>
