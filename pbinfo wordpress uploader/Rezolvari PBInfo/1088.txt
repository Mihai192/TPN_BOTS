<p><img align="right" alt="" height="134" src="/resurse/9dc152/p-1051-1075/zar.png" width="212"> Zarul folosit la diverse jocuri este un cub care are desenat pe fiecare faţă a sa <code>1</code>, <code>2</code>, <code>3</code>, <code>4</code>, <code>5</code> sau <code>6</code> puncte. Pe un zar nu există două feţe cu acelaşi număr de puncte şi suma punctelor de pe oricare două feţe opuse este egală cu <code>7</code>.</p>

<p>Pe o masă de joc este desenat un traseu în formă de pătrat, cu latura de dimensiune <code>n</code>. Fiecare latură a traseului este împărţită în <code>n</code> pătrăţele identice, care au latura egală cu cea a zarului. Zarul este aşezat iniţial în colţul din stânga sus al traseului şi apoi rostogolit de pe o faţă pe alta, din pătrăţel în pătrăţel, de-a lungul traseului parcurs în sensul acelor de ceasornic.</p>

<p>În orice moment ne-am uita la zar, putem vedea numărul punctelor desenate pe trei din feţele sale (aşa cum se vede în desenul de mai sus).</p>

<p>Notăm cu <code>f1</code> faţa cubului orientată spre noi, <code>f2</code> faţa superioară a cubului, respectiv cu <code>f3</code> faţa laterală din dreapta. Pentru exemplul din figură: <code>n=4</code>, faţa dinspre noi (<code>f1</code>) conţine trei puncte, faţa superioară (<code>f2</code>) conţine două puncte, faţa laterală din dreapta (<code>f3</code>) conţine un punct, iar sensul de deplasare este cel precizat prin săgeţi. </p>

<h1>Cerinţă</h1>

<p>Cunoscând dimensiunea <code>n</code> a traseului şi numărul punctelor de pe cele trei feţe ale zarului în poziţia iniţială, determinaţi  după <code>k</code> rostogoliri  numărul punctelor ce se pot observa pe fiecare din cele trei feţe ale zarului.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>zar.in</code> conține: </p>

<ul>
	<li>pe prima linie numerele naturale <code>n</code> şi <code>k</code> despărţite print-un spaţiu.</li>
	<li>pe linia a doua trei numere naturale separate printr-un spaţiu ce corespund numărului de puncte de pe feţele <code>f1</code>, <code>f2</code>, respectiv <code>f3</code> ale zarului în poziţia iniţială.</li>
</ul>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>zar.out</code> va conține o singură linie cu trei numere naturale separate prin câte un spaţiu, care reprezintă numărul punctelor ce se pot observa pe feţele <code>f1</code>, <code>f2</code> şi <code>f3</code> (în această ordine) după ce au fost efectuate <code>k</code> rostogoliri pe traseul dat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2≤n≤20000, 1≤k≤1000000</code></li>
</ul>

<h1>Explicație</h1>

<p>Fiecare latură a traseului este formată din <code>4</code> căsuţe şi se vor efectua <code>11</code> rostogoliri. </p>

<p>După prima rostogolire spre dreapta, valorile celor trei feţe (<code>f1</code>, <code>f2</code>, respectiv <code>f3</code>) ale zarului vor fi <code>3</code>, <code>6</code> şi <code>2</code>. După a doua rostogolire obţinem numerele <code>3</code>, <code>5</code>, <code>6</code>, iar după a treia rostogolire valorile feţelor vor fi <code>3</code>, <code>1</code> şi <code>5</code>.</p>

<p>În acest moment zarul a parcurs o latură a traseului. Următoarele trei rostogoliri se vor efectua în jos, de-a lungul traseului iar feţele vor avea succesiv valorile <code>1</code>, <code>4</code>, <code>5</code> apoi <code>4</code>, <code>6</code>, <code>5</code> şi <code>6</code>, <code>3</code>, <code>5</code>.</p>

<p>Urmează rostogolirile spre stânga, pe feţele zarului vom observa valorile <code>6</code>, <code>5</code>, <code>4</code> apoi <code>6</code>, <code>4</code>, <code>2</code> şi respectiv <code>6</code>, <code>2</code>, <code>3</code>. Ultimele două rostogoliri se vor efectua în sus de-a lungul laturii din stânga a traseului. După penultima rostogolire obţinem <code>5</code>, <code>6</code>, <code>3</code>, iar după ultima rostogolire valorile feţelor vor fi <code>1</code>, <code>5</code> şi <code>3</code>.</p>

int f1, f2, f3, n, k, dir=1, ind, aux;
int main()

    cin >> n >> k >> f1 >> f2 >> f3;
    ind = 1;
    for(int i = 1 ; i <= k; i++)
    {
        if(ind >= n) ind = 1 , dir++;
        if(dir > 4) dir = 1;
        if(dir == 1) aux = f3 , f3 = f2 , f2 = 7 - aux;
        else if (dir == 2) aux = f1 , f1 = f2 , f2 = 7 -aux;
        else if (dir == 3) aux = f2 , f2 = f3 , f3 = 7 - aux;
        else if (dir == 4) aux = f2 , f2 = f1 , f1 = 7 -aux;
        ind++;
    }
    cout << f1 << ' ' << f2 << ' ' << f3;
    return 0;
