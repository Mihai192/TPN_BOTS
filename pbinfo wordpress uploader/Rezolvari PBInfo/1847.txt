<h1>Cerinţa</h1>

<p>Scrieți definiția completă a subprogramului C++ <strong>recursiv</strong> <code>P</code> care primeşte prin intermediul parametrului <code>n</code> un număr natural nenul (<code>n≤100</code>), iar prin intermediul parametrului  <code>x</code> un tablou unidimensional cu  <code>n</code> componente întregi, de maximum șase cifre fiecare. </p>

<p>Subprogramul furnizează prin intermediul parametrului  <code>s</code> suma elementelor din tabloul <code>x</code> care au valori numere prime.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 100</code></li>
	<li>numele subprogramului cerut este <code>P</code></li>
	<li>parametrii sunt, în această ordine: <code>x</code>, <code>n</code>, <code>s</code></li>
	<li>elementele vectorului <code>x</code> sunt indexate de la zero</li>
	<li>se recomandă realizarea unei soluții recursive</li>
</ul>

<p>Dacă <code>n=6</code> și <code>x=(12, 7, 6, 3, 8, 5)</code>, după apel <code>s=15</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine definiţia subprogramului cerut și eventual al altor subprograme, apelate de acesta. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int P(int x[] , int n , int&amp; sum)
{
    sum=0;
    for(int i = 0 ; i < n ; ++i)
    {
        int cnt=1;
        int d=2;
        int c=x[i];
        while(x[i]>1)
        {
            int p = 0;
            while(x[i]%d==0)
                x[i]/=d , p++;
            if(p>0)
                cnt*=(p+1);
            d++;
            if(d*d>x[i])
                d=x[i];
        }
        if(cnt==2)
            sum+=c;
    }
}</pre>
