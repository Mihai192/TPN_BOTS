<h1>Cerința</h1>

<p>Se dă un vector cu <code>n</code> elemente, numere naturale. Afișați în ordine crescătoare elementele iar după fiecare element, inserați indicele poziției pe care acesta se afla înainte ca vectorul să fie sortat.</p>

<p>Daca există mai multe elemente cu aceeași valoare, indicii acestora se vor afișa în ordine crescatoare.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran șirul de numere separate prin spatiu, respectând cerința cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1.000.000.000</code></li>
	<li>numerotarea vectorului începe de la <code>1</code></li>
</ul>

<p><strong>Intrare</strong></p>

<h3>Explicație</h3>

<p></p><ul>
	<li>Vectorul sortat este <code>0 2 8 8 9 12 19</code>;</li>
	<li>Fostele poziții sunt <code>4</code> pentru <code>0</code> ; <code>6</code> pentru <code>2</code> etc.</li>
	<li>Elementul <code>8</code> apare de două ori, o dată pe poziția <code>2</code> și o dată pe poziția <code>7</code>. Astfel încât, indicii se afișează crescător : <code>8 2 8 7</code></li><br>
</ul><p></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int a[1000], b[1000];
    for(int i = 0 ; i < n ; ++i)
    {
        cin >> a[i];
        b[i]=a[i];
    }
    sort(b , b+n);
    for(int i = 0 ; i < n ; ++i)
    {
        for(int j = 0 ; j < n ; ++j)
        {
            if(b[i]==a[j])
            {
                cout << b[i] << ' ' << j+1 << ' ';
                a[j]=-1;
            }
        }
    }
    return 0;
}</pre>
