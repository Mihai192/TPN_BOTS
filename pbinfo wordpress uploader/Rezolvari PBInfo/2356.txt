<p>În vecinătatea lui Moș Crăciun se află o fâșie liniară formată din <code>N</code> zone identice de gheață. Pe această fâșie trăiesc mai mulți pinguini. Din cauza încălzirii globale zonele de gheața pe care nu se aflau pinguini s-au topit și au rămas mai multe grupuri de pinguini izolate. Moș Crăciun dorește să salveze pinguinii, dar mai întâi trebuie sa afle răspunsul la câteva întrebări.</p>

<h1>Cerința</h1>

<p>Cunoscând <code>N</code>, numărul zonelor din fâșia liniară și fiecare tip de zonă: apa (<code>1</code>), pinguin (<code>2</code>) să se determine:<br>
1) Câți pinguini trebuie să salveze Moș Crăciun;<br>
2) Câte grupuri izolate de pinguini sunt;<br>
3) Care este numărul maxim de pinguini care se află în același grup.</p>

<h1>Date de intrare</h1>

<p>În fișierul <code>pinguini.in</code> se află pe prima linie <code>c</code>, un număr natural reprezentând cerința, pe a doua linie se află <code>N</code>, un număr natural reprezentând numărul de zone din fâșia liniară și pe următoarea linie se află <code>N</code> numere naturale <code>a1</code>, <code>a2</code>, …, <code>aN</code> din mulțimea <code>{1, 2}</code> cu semnificația din enunț ( <code>1</code> – apă, <code>2</code> – pinguin).</p>

<h1>Date de ieșire</h1>

<p>În fișierul <code>pinguini.out</code> se va afla un singur număr natural <code>K</code>. <br>
Dacă cerința <code>c = 1</code>, <code>K</code> va reprezenta numărul de pinguini ce trebuie salvați.<br>
Dacă cerința <code>c = 2</code>, <code>K</code> va reprezenta numărul de grupuri izolate de pinguini. <br>
Dacă cerința <code>c = 3</code>, <code>K</code> va reprezenta numărul maxim de pinguini care se află în același grup.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n, k ≤ 10 000</code></li>
	<li>Pentru prima cerință se acordă <code>10</code> puncte</li>
	<li>Pentru a doua cerință se acordă <code>30</code> de puncte</li>
	<li>Pentru a treia cerință se acordă <code>50</code> de puncte</li>
	<li>În concurs s-au acordat <code>10</code> puncte din oficiu. Aici se acordă <code>10</code> puncte pentru cele trei exemple.</li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>pinguini.in</code></p>

<pre contenteditable="true" editable="true">1
12
1 2 1 2 2 2 2 1 1 2 2 2</pre>

<p><code>pinguini.out</code></p>

<pre contenteditable="true" editable="true">8</pre>

<h3>Explicație</h3>

<p><code>c = 1</code>. Numărul de pinguini ce trebuie salvați este <code>8</code>.</p>

<h1>Exemplul 2:</h1>

<p><code>pinguini.in</code></p>

<pre contenteditable="true" editable="true">2
12
1 2 1 2 2 2 2 1 1 2 2 2</pre>

<p><code>pinguini.out</code></p>

<pre contenteditable="true" editable="true">3</pre>

<h3>Explicație</h3>

<p><code>c = 2</code>. Numărul de grupuri izolate este <code>3</code>.</p>

<h1>Exemplul 3:</h1>

<p><code>pinguini.in</code></p>

<pre contenteditable="true" editable="true">3
12
1 2 1 2 2 2 2 1 1 2 2 2</pre>

<p><code>pinguini.out</code></p>

<pre contenteditable="true" editable="true">4</pre>

<h3>Explicație</h3>

<p><code>c = 3</code>. Numărul maxim de pinguini dintr-un grup este <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("pinguini.in");
ofstream cout("pinguini.out");
int main()
{
    int n , c , a[10002] , cnt = 0 , l = 0 , maxi = 0;
    cin >> c >> n;
    for(int i = 1 ; i <= n ; ++i)
        cin >> a[i];
    a[n + 1] = 1;
    n++;
    if(c == 1)
    {
        for(int i = 1 ; i < n ; ++i)
            if(a[i]==2) cnt++;
        cout << cnt;
    }
    if(c == 2)
    {
        for(int i = 1 ; i <= n ; ++i)
        {
            if(a[i]==1 &amp;&amp; l!=0)
            {
                cnt++;
                l = 0;
            }else if(a[i]!=1) l++;
        }
        cout << cnt;
    }
    if(c == 3)
    {
        for(int i = 1 ; i <= n ; ++i)
        {
            if(a[i] == 1)
            {
                if(l > maxi) maxi = l;
                l = 0;
            }else l++;
        }cout << maxi;
    }return 0;
}</pre>
