<h1>Cerinţa</h1>

<p>Să se scrie un program care să verifică dacă două numere naturale citite de la tastatură sunt prime între ele.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură două numere naturale <code>a</code> și <code>b</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran mesajul <code>PIE</code>, dacă <code>a</code> și <code>b</code> sunt prime între ele, respectiv <code>NOPIE</code> în caz contrar..</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; a, b &lt; 1.000.000.000</code></li>
	<li>două numere naturale sunt prime între ele dacă cel mai mare divizor comun al lor este <code>1</code></li>
</ul>

<h1>Exemplu 1</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">24 36</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">NOPIE</pre>

<h1>Exemplu 2</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">24 35</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">PIE</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    while(b){
        int r = a % b;
        a = b;
        b = r;
    }
    if(a == 1)
        cout << "PIE";
    else
        cout << "NOPIE";
    return 0;
}</pre>
