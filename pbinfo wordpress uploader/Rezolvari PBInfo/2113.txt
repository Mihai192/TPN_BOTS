<p>Nicoleta este pasionată de cifre. Fiind într-o bibliotecă, s-a întrebat dacă luând <code>n</code> cărţi din bibliotecă, cu cifrele cu care sunt numerotate paginile celor <code>n</code> cărţi, poate forma un număr care citit de la stânga la dreapta este identic cu cel citit de la dreapta la stânga <code>(un palindrom)</code>.</p>

<h1>Cerința</h1>

<p>Cunoscându-se numrul <code>n</code> de cărţi şi numărul <code>p</code> de pagini ale fiecărei cărţi să se determine dacă cu cifrele cu care sunt numerotate paginile cărţilor se poate forma un palindrom.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>pagini.in</code> conţine pe prima linie numărul de teste <code>T</code>. Fiecare dintre următoarele <code>T</code> linii conţin câte <code>n+1</code> numere naturale; primul dintre acestea reprezintă numărul de cărţi <code>n</code>, următoarele <code>n</code> valori reprezintă numărul <code>p</code> de pagini pentru fiecare carte dintre cele <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>pagini.out</code> conţine <code>T</code> linii. Pe fiecare linie se va afla mesajul <code>DA</code> dacă cu cifrele cu care sunt numerotate paginile cărţilor de pe linia respectivă din fişierul de intrare se poate forma un palindrom sau mesajul <code>NU</code> în caz contrar.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 &lt;= T &lt;= 3</code></li>
	<li><code>1 ≤ n ≤ 10.000</code></li>
	<li><code>10 ≤ p ≤ 1000</code></li>
	<li>Mesajele <code>DA/NU</code> sunt scrise cu majuscule</li>
	<li>Numerotarea paginilor începe pentru fiecare carte de la <code>1</code></li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>pagini.in</code></p>

<pre contenteditable="true" editable="true">3
1 19
2 19 19
1 18</pre>

<p><code>pagini.out</code></p>

<pre contenteditable="true" editable="true">DA
DA
NU</pre>

<h1>Exemplul 2:</h1>

<p><code>pagini.in</code></p>

<pre contenteditable="true" editable="true">1
6 11 73 243 53 33 604</pre>

<p><code>pagini.out</code></p>

<pre contenteditable="true" editable="true">NU</pre>

<h1>Exemplul 3:</h1>

<p><code>pagini.in</code></p>

<pre contenteditable="true" editable="true">2
5 199 929 325 299 325
6 11 73 243 53 33 604</pre>

<p><code>pagini.out</code></p>

<pre contenteditable="true" editable="true">NU
NU</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("pagini.in");
ofstream cout("pagini.out");
int main()
{
    int x , n , p;
    cin >> p;
    for(int i = 0 ; i < p ; ++i)
    {
        long long ok = 1 , cnt = 0 , f[10]={0};
        cin >> n;
        for(int j = 0 ; j < n ; ++j)
        {
            cin >> x;
            for(int k = 1 ; k <= x ; ++k)
            {
                int aux = k;
                while(aux != 0)
                {
                    f[aux%10]++;
                    aux /= 10;
                }
            }
        }
        for(int j = 0 ; j < 10 ; ++j)
        {
            if(f[j]%2!=0 &amp;&amp; cnt==0) cnt++;
            else if(f[j]%2!=0 &amp;&amp; cnt==1)ok = 0;
        }
        if(ok)cout << "DA" << endl;
        else cout << "NU" << endl;
    }
}</pre>
