<h1>Cerința</h1>

<p>Această problemă ar putea avea oricare dintre următoarele cerințe:</p>

<ul>
	<li>într-o clasă sunt <code>n</code> elevi. În câte moduri putem face o echipă din <code>k</code> elevi?</li>
	<li>în parcul auto al unei firme de taxi sunt <code>n</code> mașini. Într-o zi vin la lucru <code>k</code> șoferi. În câte moduri pot fi alese cele <code>k</code> mașini folosite?</li>
	<li>la o masă sunt <code>n</code> locuri. La masă vor sta <code>k</code> persoane. În câte moduri putem alege scaunele folosite?</li>
	<li>avem <code>n</code> bomboane diferite. În câte moduri putem alege <code>k</code> bomboane?</li>
	<li>o mulțime are <code>n</code> elemente. Câte submulțimi cu <code>k</code> elemente are?</li>
	<li>etc.</li>
</ul>

<p>Din lipsă de imaginație, cerința este: </p>

<p>Se dau numerele naturale <code>n</code> și <code>k</code>. <h1>Explicație</h1>

<p>Dacă o mulțime are <code>4</code> elemente, fie aceasta <code>A={a,b,c,d}</code>, atunci submulțimile cu câte <code>2</code> elemente sunt <code>{a,b}</code>, <code>{a,c}</code>, <code>{a,d}</code>, <code>{b,c}</code>, <code>{b,d}</code>, <code>{c,d}</code>, adică <code>6</code> submulțimi.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int n , k , v[1001];

void comb(int n , int k)
{
    for(int i = 1 ; i <= k ; i++)
        v[i] = n - i + 1;
    for(int i = 1 ; i <= k ; i++)
    {
        int x = i , d = 2;
        while(x > 1)
        {
            if(x % d == 0)
            {
                int p = 0;
                for(int j = 1 ; j <= k ; j++)
                    if(v[j] % d == 0) p = j;
                v[p] /= d;
                x /= d;
            }
            else d++;
            if(d * d > x) d = x;
        }
    }
    int r[1001];
    r[0] = 1 , r[1] = 1;
    for(int i = 1 ; i <= k ; i++)
    {
        int t = 0;
        for(int j = 1 ; j <= r[0] ; j++)
        {
            int x = t + r[j] * v[i];
            r[j] = x % 10;
            t = x / 10;
        }
        while(t) r[++r[0]] = t % 10 , t /=10;
    }
    for(int i = r[0] ; i > 0 ; i--)
        cout << r[i];
}

int main()
{
    cin >> n >> k;
    comb(n , k);
}</pre>
