<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale nenule. Aflaţi pentru fiecare număr dat <code>x</code>, câte numere naturale nenule mai mici sau egale cu <code>x</code> sunt prime cu <code>x</code>?</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>eratostene3.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>eratostene3.out</code> va conține pentru fiecare număr <code>x</code> din fişierul de intrare, numărul de numere naturale nenule mai mici sau egale cu <code>x</code>, prime cu <code>x</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numerele prime cu <code>4</code>, mai mici sau egale cu <code>4</code>, sunt: <code>1,3</code>.<br>
Numerele prime cu <code>7</code>, mai mici sau egale cu <code>7</code>, sunt: <code>1,2,3,4,5,6</code>.<br>
Numerele prime cu <code>12</code>, mai mici sau egale cu <code>12</code>, sunt: <code>1,5,7,11</code>.<br>
Vezi <a href="https://ro.wikipedia.org/wiki/Indicatorul_lui_Euler">Indicatorul lui Euler</a>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("eratostene3.in");
ofstream cout("eratostene3.out");

int E[1000001], n;
long long int t;

int main(){
    cin >> n;
    for(int i = 1; i <= 1000000; ++i)
        E[i] = i - 1;
    E[1] = 1;
    for(int i = 2; i <= 1000000; ++i)
        for(int j = 2 * i; j <= 1000000; j = j + i)
            E[j] = E[j] - E[i];
    for(int i = 1; i <= n; ++i){
        int x;
        cin >> x;
        cout << E[x] << ' ';
    }
    return 0;
}</pre>
