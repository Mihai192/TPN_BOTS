<h1>Cerinţa</h1>

<p>Să se ordoneze crescător elementele pătrat perfect ale unui şir dat, fără a afecta elementele care nu sunt pătrat perfect.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spaţii, reprezentând elementele vectorului.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran, separate prin spaţii, cele <code>n</code> elemente ale vectorului, după sortare.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât 50000</li>
</ul>

    for(int i = 1; i <= n; ++i)

    for(int i = 1; i <= n; ++i){
        int r = sqrt(a[i]);
        if(r * r == a[i]){
            int min = 10000000, poz = 0;
            for(int j = i; j <= n; ++j){
                int r1 = sqrt(a[j]);
                if(r1 * r1 == a[j] &amp;&amp; a[j] < min)
                    min = a[j], poz = j;
            }
            swap(a[i], a[poz]);
        }
    }

    for(int i = 1; i <= n; ++i)
        cout << a[i] << " ";

    return 0;
