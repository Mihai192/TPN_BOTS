<p>Se numește secvență într-un vector toate elementele din acel vector cu indici cuprinși între două valori date.</p>

<h1>Cerinţa</h1>

<p>Se dau <code>n</code> numere naturale, reprezentând elementele unui vector.</p>

<p>Determinați cea mai lungă secvență de elemente pare din vector. Dacă există mai multe secvențe cu lungime maximă se va determina cea cu suma elementelor maximă. Dacă există mai multe astfel de secvențe se va determina cea mai din stânga.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>secvmax.in</code> conţine numărul <code>n</code> și <code>n</code> valori naturale, reprezentând elementele vectorului. Valorile din fișier pot fi dispuse pe mai multe linii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>secvmax.out</code> va conţine pe prima linie numerele <code>st</code> și <code>dr</code>, reprezentând indicii de început și de sfârșit ale secvenței determinate.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n ≤ 10000</code></li>
	<li>elementele vectorului vor fi mai mici decât <code>1000</code> și sunt numerotate de la <code>1</code></li>
</ul>

<h1>Explicație</h1>

<p>Sunt <code>2</code> secvențe de elemente pare de lungime maximă, <code>2 6 4</code> și <code>8 8 6</code>. Dintre acestea, secvența <code>8 8 6</code> are suma elementelor mai mare.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("secvmax.in");
ofstream cout("secvmax.out");

int main()
{
    int n;
    cin >> n;
    int a[10001];
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    int lmax = 0, l = 0, st = 0, dr = 0, sum = 0, summax = 0;
    for(int i = 1; i <= n; ++i){
        if(a[i] % 2 == 0){
            sum += a[i];
            l++;
            if(l > lmax)
                lmax = l, st = i - l + 1, dr = i, summax = sum;
            else if(l == lmax)
                if(sum > summax)
                    st = i - l + 1, dr = i, summax = sum;
        }
        else
            l = 0, sum = 0;
    }

    cout << st << ' ' << dr;

    return 0;
}</pre>
