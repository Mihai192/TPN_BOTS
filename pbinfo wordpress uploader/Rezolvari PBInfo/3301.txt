<h1>Cerința</h1>

<p>Se dă un număr natural <code>n</code>.  Să se scrie un program care determină și afișează pe ecran numărul de numere mai mici sau egale cu <code>n</code> care au exact <code>9</code> divizori.</p>

<h1>Date de intrare</h1>

<p>Programul citește din fișierul <code>nrdiv9.in</code> numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul scrie în fișierul <code>nrdiv9.out</code> numărul <code>k</code>, reprezentând numărul de numere mai mici sau egale cu <code>n</code> care au exact <code>9</code> divizori.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000.000</code>;</li>
	<li>dacă nu există numere mai mici sau egale cu <code>n</code> care au exact <code>9</code> divizori se va afișa valoarea <code>0</code>.</li>
</ul>

<h1>Exemplul 1:</h1>

<p><strong>nrdiv9.in</strong></p>

<pre contenteditable="true" editable="true">100</pre>

<p><strong>nrdiv9.out</strong></p>

<pre contenteditable="true" editable="true">2</pre>

<h3>Explicație</h3>

<p>Sunt <code>2</code> numere mai mici sau egale cu <code>100</code> care au exact <code>9</code> divizori: <code>36</code> și <code>100</code>.</p>

<h1>Exemplul 2:</h1>

<p><strong>nrdiv9.in</strong></p>

<pre contenteditable="true" editable="true">1000</pre>

<p><strong>nrdiv9.out</strong></p>

<pre contenteditable="true" editable="true">8</pre>

<h3>Explicație</h3>

<p>Sunt <code>8</code> numere mai mici sau egale cu <code>1000</code> care au exact <code>9</code> divizori: <code>36</code>, <code>100</code>, <code>196</code>, <code>225</code>, <code>256</code>,  <code>441</code>, <code>484</code>, <code>676</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("nrdiv9.in");
ofstream cout("nrdiv9.out");

int main(){
    int n;
    cin >> n;
    int cc = 0;
    for(int i = 1; i * i <= n; ++i){
        int x = i;
        int  d = 2, cnt = 1;
        while(x > 1){
            int p = 0;
            while(x % d == 0)
                p++, x/=d;
            cnt=cnt * (2 * p + 1);
            d++;
            if(d * d > x)
                d = x;
        }
        if(cnt == 9)
            cc++;
    }
    cout << cc;
}</pre>
