<h1>Cerința</h1>

<p>În baza <code>16</code>, cifrele unui număr pot fi <code>0</code>, <code>1</code>, <code>2</code>, …, <code>9</code>, <code>A</code>, <code>B</code>, <code>C</code>, <code>D</code>, <code>E</code>, <code>F</code>. Fiind dat <code>n</code> un număr natural, afișați scrierea lui <code>n</code> în baza <code>16</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran scrierea în baza <code>16</code> a numărului <code>n</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2.000.000.000</code></li>
</ul>

<h1>Explicație</h1>

<p><code>3024 : 16 = 189</code> rest <code>0</code>; <code>189 : 16 = 11</code> rest <code>13</code>; <code>11 : 16 = 0</code> rest <code>11</code>. Cum <code>11 = B</code>, <code>13 = D</code>, <code>0 = 0</code>, rezultă <code>BD0</code>. </p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int a[10];
    int p=0;
    while(n)
    {
        a[p]=n%16;
        n/=16;
        p++;
    }
    for(int i = p-1 ; i >= 0 ; --i)
    {
        if(a[i]<10)
        cout << a[i];
        else
        {
            if(a[i]==10)
            cout << "A";
            if(a[i]==11)
            cout << "B";
            if(a[i]==12)
            cout << "C";
            if(a[i]==13)
            cout << "D";
            if(a[i]==14)
            cout << "E";
            if(a[i]==15)
            cout << "F";
        }
    }
}</pre>
