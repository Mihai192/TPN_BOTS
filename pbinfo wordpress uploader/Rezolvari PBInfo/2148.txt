<p>Pe Marte s-au descoperit <code>N</code>  marțieni, identificați de către oamenii de știință de pe Pământ prin numerele de la <code>1</code> la <code>N</code>. Cercetările au dovedit că <span class="caps">ADN</span>-ul oricărui marțian <code>X</code> este format din mulțimea factorilor primi din descompunerea lui <code>X</code>. De exemplu <code>ADN(6)={2,3}</code>.</p>

<p>Se știe că marțianul cu numărul de ordine <code>Y</code> îl moștenește pe marțianul cu numărul de ordine <code>X</code> dacă <code>ADN(X)</code> este inclus în <code>ADN(Y)</code>, adică mulțimea factorilor primi ai lui <code>X</code> este inclusă în mulțimea factorilor primi ai lui <code>Y</code>.</p>

<p>De exemplu, marțianul <code>6</code> îl moștenește pe marțianul <code>3</code> deoarece <code>ADN(3)={3}</code> este inclus în <code>ADN(6)={2,3}</code>.</p>

<p>Trebuie să specificăm că se pot întâlni situații extreme în care <code>X</code> îl moștenește pe <code>Y</code> dar și <code>Y</code> îl moștenește pe <code>X</code>, atunci când cei doi marțieni au <span class="caps">ADN</span>-urile egale. Este situația marțianului <code>12</code> care îl moștenește pe <code>6</code> dar și <code>6</code> îl moștenește pe <code>12</code>. </p>

<h1>Cerința</h1>

<p>Realizați un program care, considerând mulțimea celor <code>N</code> marțieni, determină numărul de perechi de marțieni <code>(Y, X)</code> pentru care <code>Y</code> îl moștenește pe <code>X</code>, unde <code>1 ≤ X ≤ N</code> și <code>1 ≤ Y ≤ N</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>adn.in</code> conține pe prima linie numărul <code> N</code>, reprezentând numărul de marțieni.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>adn.out</code> va conține pe prima linie numărul de perechi determinat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N ≤ 5 000 000</code></li>
	<li>Pe planeta Marte orice marțian <code>X</code> îl moștenește pe <code>X</code>.</li>
	<li>Orice marțian îl moștenește pe marțianul <code>1</code> deoarece <code>ADN(1)={}</code>, adică mulțimea vidă, care se consideră inclusă în orice mulțime nevidă.</li>
	<li>Se garantează că numărul de perechi determinat are cel mult nouă cifre.</li>
</ul>

<h1>Exemplul 1</h1>

<p><code>adn.in</code></p>

<pre contenteditable="true" editable="true">6</pre>

<p><code>adn.out</code></p>

<pre contenteditable="true" editable="true">16</pre>

<h3>Explicație</h3>

<p><code>ADN(1)={}</code>, <code>ADN(2)={2}</code>, <code>ADN(3)={3}</code>, <code>ADN(4)={2}</code>, <code>ADN(5)={5}</code>, <code>ADN(6)={2,3}</code>.</p>

<p>Perechile de marțieni determinate sunt <code>(1,1)</code>; <code>(2,2)</code>; <code>(3,3)</code>; <code>(4,4)</code>; <code>(5,5)</code>; <code>(6,6)</code>; <code>(4,2)</code>; <code>(2,4)</code>; <code>(6,2)</code>; <code>(6,3)</code>; <code>(6,4)</code>; <code>(2,1)</code> ; <code>(3,1)</code>; <code>(4,1)</code>; <code>(5,1)</code>; <code>(6,1)</code>.</p>

<h1>Exemplul 4</h1>

<p><code>adn.in</code></p>

<pre contenteditable="true" editable="true">99</pre>

<p><code>adn.out</code></p>

<pre contenteditable="true" editable="true">961</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("adn.in");
ofstream cout("adn.out");

int n , D[5000001];
long long c;

int main()
{
    cin >> n;

    for(int i = 1 ; i <= n ; i++)
        D[i] = 1;

    for(int i = 2 ; i <= n ; i++)
        if(D[i] == 1)
        {
            D[i] = i;
            for(int j = 2 ; j * i <= n ; j++)
                D[i*j] *= i;
        }
    for(int i = 1 ; i <= n ; i++)
        c += n / D[i];
    cout << c;
}</pre>
