<h1>Cerința</h1>

<p>Dorind să inventeze ceva nou Mate a inventat numerele “ecou”. Un număr natural <code>A</code> se numește număr-ecou dacă există un număr natural <code>B</code> cu proprietatea că numărul <code>A</code> se poate obține prin concatenarea de un număr de ori (cel puțin de două ori) a numărului <code>B</code>. De exemplu numărul <code>313313</code> este număr-ecou, iar <code>31313</code> nu este număr-ecou.</p>

<p>Mate și-a propus să afle câte numere-ecou sunt printre numerele naturale care au exact <code>N</code> cifre. Deoarece acest număr poate fi foarte mare, se va afișa doar restul împărțirii rezultatului la <code>10^9+17</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>nrecou.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>În fișierul de ieșire <code>nrecou.out</code> se va afișa, pe primul rând, valoarea cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>pentru 20% din teste <code>1 ≤ N ≤ 6</code></li>
	<li>pentru alte 20% din teste <code>N ≤ 12</code></li>
	<li>pentru alte 30% din teste <code>N ≤ 10.000</code></li>
	<li>pentru alte 30% din teste <code>N ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numerele ecou cu trei cifre sunt <code>111</code>, <code>222</code>, <code>333</code>, <code>444</code>, <code>555</code>, <code>666</code>, <code>777</code>, <code>888</code>, <code>999</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("nrecou.in");
ofstream cout("nrecou.out");

int n , D[10000000] ; 
long long nr , rez;

int main()
{
    cin >> n;
    for(int d = 2 ; d <= n &amp;&amp; n / d >= 2 ; d++)
        if(n % d == 0)
        {
            nr = 9;
            int x = d;
            for(int j = 1 ; j < x ; j++)
                nr = (nr * 10) % 1000000017;
            for(int p = 2 ; p <= x / 2 ; p++)
            {
                if(x % p == 0) nr -= D[p];
                if(nr < 0) nr += 1000000017;
            }
            nr -= 9;
            if(nr < 0) nr += 1000000017;
            D[d] = nr % 1000000017;
            rez = (nr + rez) % 1000000017;
        }
    cout<<(rez+9)%1000000017;
    return 0;
}</pre>
