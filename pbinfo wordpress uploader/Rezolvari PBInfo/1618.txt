<p>Un indicator numeric este un dispozitiv de afişaj electronic destinat afişării unei cifre zecimale. <br>
Acesta conține <code>7</code> segmente notate cu <code>a</code>, <code>b</code>, <code>c</code>, <code>d</code>, <code>e</code>, <code>f</code>, <code>g</code>, ca în figura de mai jos.</p>

<p>Afişarea unei cifre se face prin aprinderea unei combinații de segmente conform tabelului:</p>

<h1>Cerința</h1>

<p>Cunoscând un număr natural <code>N</code> afișat cu ajutorul mai multor indicatoare numerice, să se scrie un program care determină:</p>

<ol>
	<li>Numărul de segmente aprinse pentru afișarea numărului <code>N</code>.</li>
	<li>Numărul de numere distincte mai mari decât <code>N</code>,ce se pot forma prin aprinderea a cel puțin unui segment în plus, față de cele utilizate pentru afișarea numărului <code>N</code>, fără a folosi alte indicatoare numerice, și fără a stinge niciun segment dintre cele deja aprinse.</li>
</ol>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>cifre12.in</code> conține pe prima linie numărul natural <code>V</code> a cărui valoare poate fi doar <code>1</code> sau <code>2</code>, iar pe a doua linie numărul natural <code>N</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire este <code>cifre12.out</code>.</p>

<p>Dacă valoarea lui <code>V</code> este <code>1</code> atunci fişierul de ieşire va conţine pe prima linie un singur număr natural ce reprezintă numărul de segmente aprinse pentru afișarea numărului <code>N</code>.</p>

<p>Dacă valoarea lui <code>V</code> este <code>2</code> atunci fişierul de ieşire va conține pe prima linie un singur număr natural reprezentând numărul de numere distincte mai mari decât <code>N</code>, ce se pot forma prin aprinderea a cel puțin unui segment în plus, față de cele utilizate pentru afișarea numărului <code>N</code>, fără a folosi alte indicatoare numerice.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>10 ≤ N ≤ 10</code><sup><code>19</code></sup></li>
	<li>20% din teste vor avea valoarea <code>V = 1</code>, iar 80% din teste vor avea valoarea <code>V = 2</code>.</li>
</ul>

<h1>Exemplul 1</h1>

<p><code>cifre12.in</code></p>

<pre contenteditable="true" editable="true">1 
823</pre>

<p><code>cifre12.out</code></p>

<pre contenteditable="true" editable="true">17</pre>

<h3>Explicație</h3>

<p><code>V = 1</code>, deci se rezolvă <span class="caps">NUMAI</span> prima cerință. <code>N = 823</code>;</p>

<p>Pentru afișarea cifrei <code>8</code> s-au aprins <code>7</code> segmente, pentru cifra <code>2</code> s-au aprins <code>5</code> segmente și pentru cifra <code>3</code> tot <code>5</code> segmente. În total s-au aprins <code>17</code> segmente.</p>

<h1>Exemplul 2</h1>

<p><code>cifre12.in</code></p>

<pre contenteditable="true" editable="true">2
823</pre>

<p><code>cifre12.out</code></p>

<pre contenteditable="true" editable="true">5</pre>

<h3>Explicație</h3>

<p><code>V = 2</code>, deci se rezolvă <span class="caps">NUMAI</span> a doua cerință. <code>N = 823</code>;</p>

<p>Din cifra <code>8</code> nu se mai pot obține alte cifre prin aprinderea de noi segmente.<br>
Din cifra <code>2</code> se poate obține cifra <code>8</code> iar din cifra <code>3</code> se pot obține cifrele <code>8</code> și <code>9</code> prin aprinderea de noi segmente. Așadar, se pot obține <code>5</code> numere mai mari ca <code>823</code>: <code>828</code>, <code>829</code>, <code>883</code>, <code>888</code>, <code>889</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;


int c[]={6, 2, 5, 5, 4, 5, 6, 3, 7, 6 };
int t[]={2, 7, 2, 3, 3, 4, 2, 5, 1, 2};
int mm[]={1, 5, 1, 2, 2, 3, 1, 2, 0, 0 };
int cif[25];
char s[25];
unsigned long long p,sol;
ifstream fin ("cifre12.in");
ofstream fout ("cifre12.out");
int main()
{

    int t1,k=0,sum=0,g;
    fin >> t1;
    fin >> s;
    for (k=0;s[k];++k)
        cif[k]=s[k]-'0';
    if(t1==1)
    {
        for (int i=0;i<k;++i)
            sum +=c[cif[i]];
        fout << sum;
    }
    else
    {
        for (int i=0;i<k;++i)
        {
            p=1;
            for (int j=i+1;j<k;j++)
            {
                g=cif[j];
                p=p*t[g];
            }

            sol +=mm[cif[i]] *p;
        }
        fout << sol;
    }
    return 0;


}</pre>
