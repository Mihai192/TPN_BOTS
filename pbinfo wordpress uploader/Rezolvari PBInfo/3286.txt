Numerele sunt ordonate descrescător și sunt separate prin câte un spațiu. Se cere să se determine numărul valorilor naturale distincte din intervalul <code>[1,n]</code> care NU se găsesc în șir.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>nrnot.in</code> conține pe prima linie numărul <code>n</code>, iar pe următoarele linii numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>nrnot.out</code> va conține pe prima linie numărul <code>k</code>, reprezentând numărul de valori naturale distincte din intervalul <code>[1,n]</code> care nu se găsesc în șir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤  10.000.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>10.000.000</code></li>
	<li>se recomandă evitarea memorării numerelor date într-un tablou sau în altă structură similară!</li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare nu se găsesc valorile <code>10 9 7 6 4 2 1</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("nrnot.in");
ofstream cout("nrnot.out");

int main(){
    int x, n, cnt = 0, val;
    cin >> n;
    while(cin >> x){
        if(x < n)
            cnt += n - x;
        n = x - 1;
        val = x;
    }
    cnt += val - 1;
    cout << cnt;
    return 0;
}</pre>
