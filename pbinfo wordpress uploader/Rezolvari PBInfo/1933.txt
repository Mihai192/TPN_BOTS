<h1>Cerința</h1>

<p>Fie <code>N</code> un numar natural și un șir de <code>N</code> numere naturale <code>V[1]</code>, <code>V[2]</code>, …, <code>V[N]</code>. Pentru <code>M</code>  întrebări de forma <code>(i,j)</code>, să se calculeze suma termenilor <code>V[i]</code>, <code>V[i + 1]</code>, …, <code>V[j]</code>.</p>

<h1>Date de intrare</h1>

<p>Pe prima linie a fișierului <code>sume2.in</code> se găsește un număr natural <code>N</code>. Pe urmatoarea linie sunt <code>N</code> numere naturale, reprezentând valorile șirului <code>V</code>. Pe a treia linie se găsește un număr natural <code>M</code> reprezentând numărul de întrebări, iar pe următoarele M linii câte o pereche de numere <code>(i,j)</code>, reprezentând o întrebare pentru care se cere să se calculeze suma <code>V[i] + V[i + 1] + ... + V[j]</code>.</p>

<h1>Date de ieșire</h1>

<p>Pe prima linie din fişierul <code>sume2.out</code> se găseşte răspunsul la prima întrebare din fişierul de intrare, pe a doua linie se găseşte răspunsul la cea de-a doua întrebare, și așa mai departe.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N ≤ 100.000</code></li>
	<li><code>1 ≤ M ≤ 500.000</code></li>
	<li><code>0 ≤ V[i] ≤ 1.000.000.000</code></li>
	<li>Pentru 30% dintre teste <code>N, M ≤ 1.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Suma elementelor de pe pozițiile de la <code>1</code> la <code>3</code> este <code>6</code>.<br>
Suma elementelor de pe pozițiile de la <code>2</code> la <code>4</code> este <code>9</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("sume2.in");
ofstream cout("sume2.out");
int main()
{
    long long n , m , x , y , a[100001] , s[100001]={0};
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        cin >> a[i];
        s[i]=s[i-1]+a[i];
    }
    cin >> m;
    for(int i = 0 ; i < m ; ++i)
    {
        cin >> x >> y;
        cout << s[y]-s[x - 1] << endl;
    }
}</pre>
