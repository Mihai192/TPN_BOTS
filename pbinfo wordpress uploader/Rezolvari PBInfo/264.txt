<h1>Cerinţa</h1>

<p>Se dau mai multe numere naturale formate din exact o cifră. Determinaţi cifrele cu număr maxim de apariţii.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>maxcif.in</code> conţine cel mult <code>100.000</code> de numere naturale separate formate dintr-o singură cifră, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>maxcif.out</code> va conţine pe prima linie valorile din fişierul de intrare cu număr maxim de apariţii, în ordine crescătoare, separate prin exact un spaţiu.</p>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("maxcif.in");
ofstream fout("maxcif.out");

int main()
{
    int f[10]={0};
    int max=0;
    int tmp;
    while(fin >> tmp)
    {
        f[tmp]++;
        if(f[tmp]>max)
        max=f[tmp];
    }
    for(int i = 0 ; i <= 9 ; ++i)
    {
        if(f[i]==max)
        fout << i << ' ';
    }
    return 0;
}</pre>
