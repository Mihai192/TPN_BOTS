<h1>Cerința</h1>

<p>Scrieți un program care citește un cuvânt cu cel mult <code>15</code> litere mici ale alfabetului englez și apoi un text cu cel mult <code>255</code> de caractere (cuvinte formate din litere mici ale alfabetului englez separate prin unul sau mai multe spații) și care afișează șirul obținut prin inserarea în text a caracterului <code>?</code> după fiecare apariție a cuvântului citit. Dacă în text nu apare cuvântul citit, se va afișa mesajul <code>NU APARE</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>inserarechar.in</code> conține pe prima linie un cuvânt, iar pe a doua linie un text.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>inserarechar.out</code> va conține textul modificat prin inserarea caracterului <code>?</code> după fiecare apariție a cuvântului citit, sau mesajul <code>NU APARE</code> . </p>

<h1>Restricții și precizări</h1>

<ul>
	<li>textul va avea cel mult <code>255</code> de caractere litere mici din alfabetul englez și spații;</li>
	<li>cuvântul va avea cel mult <code>15</code> caractere, doar litere mici ale alfabetului englez.</li>
</ul>

#include <cstring>
using namespace std;

ifstream cin("inserarechar.in");
ofstream cout("inserarechar.out");

char m[255][255] , cnt = 0;

int main(){
    char cuv[20] , s[300];
    cin >> cuv;
    cin.get();
    cin.getline(s , 256);
    int i = 0;
    int j = 0;
    while(s[i]){
        if(s[i]==' ' &amp;&amp; s[i+1]!=' ')
            cnt++ , j = 0;
        else
            m[cnt][j]=s[i] , j++;
        i++;
    }
    int ok=0;
    for(int i = 0 ; i <= cnt ; ++i)
        if(!strcmp(m[i] , cuv))
            ok++;
    if(!ok)
        cout << "NU APARE";
    else
    for(int i = 0 ; i <= cnt ; ++i)
        if(!strcmp(m[i] , cuv))
            cout << m[i] << '?' << ' ';
        else
            cout << m[i] << ' ';
    return 0;
