<h1>Cerința</h1>

<p>După ce au trecut sărbătorile, ca în fiecare an, Moș Crăciun a început să facă inventarul cadourilor rămase pentru anul următor.  El are <code>N</code> cadouri și pe fiecare cadou este scris un număr natural. În fiecare an Moș Crăciun trebuie să noteze într-un carnețel cantitatea de fericire pe care o aduc aceste cadouri copiilor. </p>

<p>Pentru a calcula această valoare, prima dată el trebuie să înmulțească toate numerele înscrise pe cele <code>N</code> cadouri. Astfel el obține un număr foarte mare. Apoi el știe că numărul de divizori al acestui număr este cantitatea de fericire pe care el trebuie să o scrie în carnețel. Ajutați-l pe Moș Crăciun să afle cantitatea de fericire a celor <code>N</code> cadouri. Deoarece acest număr este foarte mare voi trebuie sa aflați doar restul împărțirii la <code>1.000.000.007</code>.</p>

<h1>Date de intrare</h1>

<p>Pe prima linie a fișierului de intrare <code>​cadouri2.in​</code> se află numărul natural <code>N</code>. Pe următoarea linie se află <code>N</code> numere naturale reprezentând numerele care sunt scrise pe cele <code>N</code> cadouri. </p>

<h1>Date de ieșire</h1>

<p>In fișierul de ieșire <code>​cadouri2.out</code> ​trebuie să se afle un singur număr care reprezintă cantitatea de fericire a celor <code>N</code> cadouri modulo <code>1.000.000.007</code></p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N ≤ 1000</code></li>
	<li>Numerele înscrise pe cadouri sunt numere naturale cuprinse între <code>1</code> și <code>10</code><sup><code>​6</code></sup></li>
	<li>Pentru teste în valoare de ​40 de puncte​, produsul celor <code>N</code> numere este ≤ <code>10​</code><sup><code>​9</code></sup></li>
	<li>Pentru alte ​10 puncte​, produsul celor <code>N</code> numere este ≤ <code>10​</code><sup><code>​12</code></sup></li>
	<li>Pentru alte ​20 de puncte, ​numerele înscrise pe cadouri sunt cuprinse între <code>1</code> și <code>1000</code></li>
</ul>

<h1>Exemplu 1:</h1>

<p><code>cadouri2.in</code></p>

<pre contenteditable="true" editable="true">3 
2 3 4</pre>

<p><code>cadouri2.out</code></p>

<pre contenteditable="true" editable="true">8</pre>

<h3>Explicație</h3>

<p>Produsul celor trei numere este <code>24</code>. Divizorii acestui număr sunt: <code>1, 2, 3, 4, 6, 8, 12, 24</code>. În total <code>8</code> divizori.</p>

<h1>Exemplu 2:</h1>

<p><code>cadouri2.in</code></p>

<pre contenteditable="true" editable="true">5 
12 24 3 7 15</pre>

<p><code>cadouri2.out</code></p>

<pre contenteditable="true" editable="true">120</pre>

<h3>Explicație</h3>

<p>Produsul celor <code>5</code> numere este <code>90720</code>. Acest număr are <code>120</code> de divizori.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("cadouri2.in");
ofstream cout("cadouri2.out");
int f1[500001] , f2[1000];
int main()
{
    int n , x;
    long long cnt = 1;
    int a = 0;
    cin >> x;
    for(int i = 1 ; i <= x ; ++i)
    {
        cin >> n;
        int d = 2;
        while(n > 1)
        {
            int p = 0;
            while(n % d == 0) n /= d , p++;
            if(d <= 500000) f1[d] += p;
            else
            {
                f2[a]+=p;
                a++;
            }
            d++;
            if(d * d > n) d = n;
        }
    }
    for(int i = 0 ; i <= 500000 ; ++i)
    {
        if(f1[i]!=0)
            cnt = (1LL * cnt * (f1[i]+1)) % 1000000007;
    }
    for(int i = 0 ; i < a ; ++i)
    {
        cnt = (1LL * cnt * (f2[i]+1)) % 1000000007;
    }
    cout << cnt;
    return 0;
}</pre>
