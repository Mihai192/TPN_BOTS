<h1>Cerința</h1>

<p>Se citesc mai multe numere naturale nenule. Fiecare astfel de număr <code>X</code> se va afișa, urmat de un triplet de numere <code>a b c</code> cu proprietatea că <code>a+b+c=X</code>, iar produsul lor este maxim.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>produsmaxim.in</code> conține mai multe numere naturale separate prin spații și scrise pe unul sau mai multe rânduri.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>produsmaxim.out</code> va conține pe mai multe linii valorile <code>X a b c</code> separate printr-un spațiu, reprezentând numărul și tripletul corespunzător acestuia, pentru  fiecare număr citit din fișierului de intrare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>vor fi cel mult <code>100.000</code> de numere în fișierul de intrare</li>
	<li>numerele citite vor fi mai mici decât <code>10.000.000</code></li>
	<li><code>a ≤ b ≤ c</code></li>
	<li>fiecare rând din fișierul de intrare va conține exact patru valori, conform cerinței</li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare se găsesc două numere <code>7</code> și <code>5</code>, iar fișierul de ieșire va avea două linii, cu valorile cerute (de exemplu, dintre toate tripletele în care se poate descompune <code>7</code>: <code>1+2+4</code>, <code>1+3+3</code>, <code>2+2+3</code>, produsul <code>2*2*3</code> este cel mai mare).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("produsmaxim.in");
ofstream cout("produsmaxim.out");
int n;
int main()
{
    while(cin >> n)
    {
        if (n % 3 == 0) cout << n << ' ' << n / 3 << ' ' << n / 3 << ' ' << n / 3 << '\n';
        else if (n % 3 == 1) cout << n << ' ' << n / 3 << ' ' << n / 3 << ' ' << n / 3 + 1 << '\n';
        else cout << n << ' ' << n / 3 << ' ' << n / 3 + 1 << ' ' << n / 3 + 1 << '\n';
    }
    return 0;
}</pre>
