<h1>Cerința</h1>

<p>Se dau două numere <code>b1 b2</code>, reprezentând două baze de numeraţie şi două şiruri de cifre <code>x y</code>, reprezentând două numere: <code>x</code> în baza <code>b1</code>, <code>y</code> în baza <code>b2</code>. Determinaţi suma numerelor <code>x</code> şi <code>y</code> în baza <code>10</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>baze1.in</code> conține pe prima linie numerele <code>b1 b2</code>, iar pe a doua linie cele două şiruri de cifre cu semnificaţia de mai sus.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>baze1.out</code> va conține pe prima linie numărul <code>S</code>, reprezentând suma dintre <code>x</code> şi <code>y</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ b1,b2 ≤ 10</code></li>
	<li>reprezentările în baza <code>10</code> ale lui <code>x</code>, <code>y</code> şi suma lor nu depăşesc <code>2 000 000 000</code></li>
</ul>

<h1>Explicație</h1>

<p><code>110</code><sub><code>(2)</code></sub><code> = 6</code><br>
<code>213</code><sub><code>(5)</code></sub><code> = 58</code><br>
<code>6 + 58 = 64</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;

ifstream fin("baze1.in");
ofstream fout("baze1.out");

char am[25] , an[25];
int b1 , b2 , n1 , n2 , v;

int main()
{
    fin >> b1 >> b2 >> am >> an;
    for(int i = 0 ; i < strlen(am) ; i++)
    {
        v = am[i] - '0';
        n1 = n1 * b1 + v;
    }
    for(int i = 0 ; i < strlen(an) ; i++)
    {
        v = an[i] - '0';
        n2 = n2 * b2 + v;
    }
    fout << n1 + n2;
}</pre>
