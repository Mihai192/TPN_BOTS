<h1>Cerința</h1>

<p>Se dă un vector cu <code>n</code> elemente, numere naturale, și un număr <code>k</code>, divizor al lui <code>n</code>. Se împarte vectorul în <code>k</code> secvențe disjuncte, numerotate de la <code>1</code> la <code>k</code>. Să se stabilească dacă există două secvențe identice.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> și <code>k</code>, iar apoi <code>n</code> numere naturale, reprezentând elementele vectorului.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele <code>i j</code>, <code>i &lt; j</code> reprezentând numerele de ordine a două secvențe identice, dacă există două astfel de secvențe, sau mesajul <code>NU</code>, dacă nu există două astfel de secvențe.</p>

<p>Dacă există mai multe perechi de secvențe identice se vor considera cele cu numerele de ordine minime.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ k &lt; n ≤ 1000</code>, <code>k</code> este divizor al lui <code>n</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1000</code></li>
</ul>

<h1>Explicație</h1>

<p>Vectorul se împarte în <code>5</code> secvențe: <code>(3 6 4)</code>, <code>(7 6 6)</code>, <code>(6 4 6)</code>, <code>(7 6 6)</code> și <code>(4 4 5)</code>, numerotate de la <code>1</code> la <code>5</code>. A doua și a patra secvență sunt identice.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int a[1000] , v[1000];
    int k;
    cin >> k;
    int ls=n/k;
    int st=0 , dr=0;
    for(int i = 0 ; i < n ; ++i)
    cin >> a[i];
    for(int i = 0 ; i < n ; i+=ls)
    {
        int p = 0;
        for(int j = i ; j < i+ls ; ++j)
        {
            v[p]=a[j];
            p++;
        }
        for(int j = i+ls ; j < n ; j+=ls)
        {
            int cnt=0;
            p=0;
            for(int m = j; m < j+ls ; ++m)
            {
                if(v[p]==a[m])
                cnt++;
                p++;
            }
            if(cnt==ls &amp;&amp; dr==0 &amp;&amp; st==0)
            st=i/ls+1 , dr=j/ls+1;
        }
    }
    cout << st << ' ' << dr;
}</pre>
