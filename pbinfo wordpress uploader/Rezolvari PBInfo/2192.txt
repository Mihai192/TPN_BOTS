<p><i>Românii au talent!</i>    Atraşi de marele premiu oferit de organizatorii concursului <i>Românii au talent</i>, la preselecţia organizată la Piatra Neamţ au venit foarte mulţi români să demonstreze că au talent. </p>

<p>La înscriere, fiecare participant a primit câte un număr de concurs, reprezentat de un număr natural nenul. Unii dintre participanţi pot avea statut special, fiind admişi direct în semifinale, ca urmare a rezultatelor deosebite obţinute la ediţiile anterioare, numărul de concurs primit de aceştia având proprietatea că toate cifrele lui pot fi aranjate astfel încăt să formeze un număr palindrom. </p>

<p>Printre numerele de concurs primite de participanţii cu statut special, există numere care au în scrierea lor zecimală un număr maxim de cifre distincte. Cel mai mic dintre aceste numere reprezintă numărul de concurs al participantului <span class="caps">VIP</span>. </p>

<h1>Cerinta</h1>

<p>Scrieţi un program care citeşte numărul natural <code>n</code> (reprezentând numărul de participanţi înscrişi la concurs), <code>n</code> numere naturale (reprezentând numerele de concurs ale celor n participanţi) şi determină:</p>

<p>a)	numărul <code>x</code> de participanţi admişi direct în semifinale;<br>
b)	numărul <code>y</code> de concurs al participantului <code>VIP</code>, dacă există un astfel de participant printre cei înscrişi.</p>

<h1>Date de intrare</h1>

<p>Fişierul <code>talent.in</code> conţine pe prima linie numărul n de participanţi, iar pe fiecare dintre următoarele n linii câte un număr natural reprezentând numărul de concurs al unui participant. </p>

<h1>Date de ieșire </h1>

<p>Fişierul <code>talent.out</code> va conţine pe prima linie cele două valori <code>x</code> şi <code>y</code>, separate printr-un singur spaţiu, în această ordine. Dacă nu există participanţi admişi direct în semifinală, atunci prima linie a fişierului va conţine două valori nule, separate printr-un singur spaţiu: <code>0 0</code>.</p>

<h2>Restricții și precizări</h2>

<p>•	<code>0 &lt;n ≤ 15000</code><br>
•	numerele de concurs sunt distincte două câte două şi sunt mai mici sau egale cu <code>2000000000</code><br>
•	un număr palindrom este un număr natural care este egal cu numărul natural obţinut prin scrierea în ordine inversă a cifrelor lui (de exemplu, numărul <code>12021</code> este un număr palindrom) <br>
•	pentru rezolvarea corectă a primei cerinţe se acordă <code>50%</code> din punctaj, iar pentru rezolvarea corectă a celei de a doua cerinţe se acordă <code>50%</code> din punctaj.</p>

<h3>Explicație</h3>

<p>Sunt <code>3</code> participanţi calificaţi direct în semifinală. Numerele lor de concurs sunt numere ale căror cifre pot fi aranjate astfel încât să formeze un palindrom : <code>9900</code>(format din <code>2</code> cifre distincte), <code>83181</code> (format din <code>3</code> cifre distincte) şi <code>46476</code> (format din <code>3</code> cifre distincte). Dintre acestea, cel mai mic număr format dintr-un număr maxim de cifre distincte este <code>46476</code> reprezentând numărul de concurs al participantului <code>VIP</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>

ifstream cin("talent.in");
ofstream cout("talent.out");

int pal(int n)
{
    int f[10]={0};
    int cn=0;
    while(n)
    {
        f[n%10]++;
        n/=10;
        cn++;
    }
    int cnt=0;
    for(int i = 0 ; i < 10 ; ++i)
    {
        if(f[i]%2==1)
        cnt++;
        if(f[0]%2==0 &amp;&amp; cn==f[0]+1 &amp;&amp; f[0]!=0)
        {
            cnt++;    
        }
    }
    if(cnt==0 || cnt==1)
    return 1;
    else
    return 0;
}

int mult(int n)
{
    int f[10]={0};
    while(n)
    {
        f[n%10]++;
        n/=10;
    }
    int ok=0;
    for(int i = 0 ; i < 10 ; ++i)
    {
        if(f[i]!=0)
        ok++;
    }
    return ok;
}

int main()
{
    int n;

    int a[15001];
    int cnt=0;
    int max=0;
    for(int i = 0 ; i < n ; ++i)
    {
    cin >> a[i];
    if(pal(a[i]))
    {
        cnt++;
        if(max < mult(a[i]))
        max=mult(a[i]);
    }
    }
    cout << cnt;
    int nmax=2000000001;
    if(cnt!=0)
    {
        for(int i = 0 ; i < n ; ++i)
        {
        if(mult(a[i])==max &amp;&amp; pal(a[i]))
        {
            if(a[i]<nmax)
            nmax=a[i];
        }
        }

    }
    else
