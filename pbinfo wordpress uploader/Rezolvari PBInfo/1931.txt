<h1>Cerința</h1>

<p>Definim un număr ca fiind <strong>fantastic</strong> dacă numărul de numere la care acesta se împarte exact este un număr prim.</p>

<p>Dându-se un șir cu <code>n</code> numere întregi strict pozitive, să se afișeze numărul de numere fantastice din șir.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>fantastice.in</code> conține pe prima linie numărul <code>n</code> de numere, iar pe cea de-a doua linie, separate prin câte un spaţiu, cele <code>n</code> numere.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>fantastice.out</code> va conține pe prima linie numărul de numere fantastice din șir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code><sup><code>6</code></sup></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici sau egale cu <code>10</code><sup><code>6</code></sup></li>
</ul>

<h3>Explicație</h3>

<p><code>21</code> are divizorii <code>1</code>, <code>3</code>, <code>7</code>, <code>21</code><br>
<code>19</code> are divizorii <code>1</code>, <code>19</code><br>
<code>25</code> are divizorii <code>1</code>, <code>5</code>, <code>25</code><br>
<code>16</code> are divizorii <code>1</code>, <code>2</code>, <code>4</code>, <code>8</code>, <code>16</code><br>
<code>27</code> are divizorii <code>1</code>, <code>3</code>, <code>9</code>, <code>27</code><br>
<code>729</code> are divizorii <code>1</code>, <code>3</code>, <code>9</code>, <code>27</code>, <code>81</code>, <code>243</code>, <code>729</code></p>

<p>Deci sunt <code>4</code> numere fantastice: <code>19 25 16 729</code></p>

    if(n==0 || n==1) return 0;

    if(n%2==0) return 0;
    else for(int d=3;d*d<=n;d+=2) if(n%d==0) return 0;
    return 1;
}
int nrdiv(int n)
{
    int cnt=0;
    for(int i=1;i*i<=n;i++)
    {
        if(n%i==0) cnt+=2;
        if(i*i==n) cnt--;
    }
    if(prim(cnt)) return 1;
    else return 0;
}

    fin>>n;

            fin>>x;

    }
