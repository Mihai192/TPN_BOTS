<h1>Cerința</h1>

<p>Se dă un șir cu <code>n</code> elemente numere întregi, numerotate de la <code>1</code> la <code>n</code> și <code>m</code> perechi de indici <code>i j</code>. Pentru fiecare pereche de indici se calculează suma elementelor din secvență determinată de cei doi indici. Afișați suma maximă obținută.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere întregi, apoi <code>m</code> și <code>m</code> perechi <code>i j</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>S</code>, reprezentând suma maximă determinată.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code>;</li>
	<li><code>1 ≤ m ≤ 500.000</code>;</li>
	<li><code>1 ≤ i,j ≤ n</code>;</li>
	<li>cele <code>n</code> elemente ale șirului vor avea cel mult nouă cifre.</li>
</ul>

<h3>Explicație</h3>

<ul>
	<li>suma elementelor din secvența delimitată de <code>1 3</code> este <code>10</code>;</li>
	<li>suma elementelor din secvența delimitată de <code>4 2</code> este <code>4</code>;</li>
	<li>suma elementelor din secvența delimitată de <code>2 5</code> este <code>-4</code>;</li>
	<li>suma maximă este <code>10</code>.</li>
</ul>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <cmath>
using namespace std;

int main()
{
    int n, m;
    cin >> n;
    int a[100001];
    long long sp[100001];
    sp[0] = 0;
    for(int i = 1; i <= n; ++i){
        cin >> a[i];
        sp[i] = sp[i - 1] + a[i];
    }

    int x, y;
    cin >> m;
    long long smax = -1000000000000000000;
    for(int i = 1; i <= m; ++i){
        cin >> x >> y;
        if(x > y)
            swap(x, y);
        long long sum = sp[y] - sp[x-1];
        if(sum > smax)
            smax = sum;
    }
    cout << smax;
    return 0;
}</pre>
