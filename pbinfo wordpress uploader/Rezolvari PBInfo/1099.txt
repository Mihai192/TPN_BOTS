<p>Arhipelagul <span class="caps">RGB</span> este format din insule care aparţin ţărilor <code>R</code>, <code>G</code> şi <code>B</code>. Putem reprezenta harta arhipelagului ca o matrice cu <code>n</code> linii şi <code>m</code> coloane cu elemente din mulţimea <code>{0, 1, 2, 3}</code>. Un element egal cu <code>0</code> reprezintă o zonă acoperită de apă; un element egal cu <code>1</code> reprezintă o zonă de pământ aparţinând unei insule din ţara <code>R</code>, iar un element egal cu <code>2</code> reprezintă o zonă de pământ aparţinând unei insule din ţara <code>G</code>, iar un element egal cu <code>3</code> reprezintă o zonă de pământ aparţinând unei insule din ţara <code>B</code>.</p>

<p>Se consideră că două elemente ale matricei sunt vecine dacă ele au aceeaşi valoare şi fie sunt consecutive pe linie, fie sunt consecutive pe coloană. Două elemente aparţin aceleiaşi insule dacă ele sunt vecine sau dacă se poate ajunge de la un element la celălalt pe un drum de-a lungul căruia oricare două elemente consecutive sunt vecine. </p>

<p>Pentru a încuraja relaţiile de colaborare dintre ţările <code>R</code> şi <code>G</code>, se doreşte construirea unui pod care să unească o insulă aparţinând ţării <code>R</code> de o insulă aparţinând ţării <code>G</code>. Podul trebuie să respecte următoarele condiţii:</p>

<ul>
	<li>să înceapă pe o zonă cu apă consecutivă pe linie sau coloană cu o zonă aparţinând ţării <code>R</code>;</li>
	<li>să se termine pe o zonă cu apă consecutivă pe linie sau coloană cu o zonă aparţinând ţării <code>G</code>;</li>
	<li>să traverseze numai zone acoperite cu apă;</li>
	<li>oricare două elemente consecutive ale podului trebuie să fie vecine;</li>
	<li>lungimea podului să fie minimă (lungimea podului este egală cu numărul de elemente traversate de pod).</li>
</ul>

<h1>Cerinţă</h1>

<p>Dată fiind harta arhipelagului să se determine câte insule aparţin fiecărei ţări, precum şi lungimea minimă a unui pod care să satisfacă condiţiile din enunț.</p>

<h1>Date de intrare </h1>

<p>Fişierul de intrare <code>insule.in</code> conţine pe prima linie numerele naturale <code>n</code> şi <code>m</code>, separate prin spaţiu. Pe următoarele <code>n</code> linii este descrisă harta arhipelagului. Pe fiecare dintre aceste <code>n</code> linii sunt scrise câte <code>m</code> valori din mulţimea <code>{0, 1, 2, 3}</code>; valorile nu sunt separate prin spaţii. </p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>insule.out</code> va conţine o singură linie pe care vor fi scrise patru numere naturale separate prin spaţii <code>NR NG NB Lg</code>, unde <code>NR</code> reprezintă numărul de insule aparţinând ţării <code>R</code>, <code>NG</code> numărul de insule aparţinând ţării <code>G</code>, <code>NB</code> numărul de insule aparţinând ţării <code>B</code>, iar <code>Lg</code> lungimea minimă a podului.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 &lt; n, m ≤ 100</code></li>
	<li>Se garantează că pe hartă există cel puţin un element <code>1</code>, un element <code>2</code> şi un element <code>0</code>.</li>
	<li>Se acordă 40% din punctaj pentru determinarea corectă a numărului de insule din fiecare ţară; se acordă punctaj integral pentru rezolvarea corectă a tuturor cerinţelor.</li>
	<li>Începutul şi sfârşitul podului pot să coincidă.</li>
	<li>Pentru datele de test există întotdeauna soluţie.</li>
</ul>

<h1>Explicație</h1>

<p>Lungimea minimă a unui pod care poate fi construit este <code>4</code>; de exemplu, podul traversează celulele <code>(6,5)</code>, <code>(6,4)</code>, <code>(6,3)</code>, <code>(6,2)</code>.</p>

ifstream fin("insule.in");
ofstream fout("insule.out");
struct POZ{int lin;int col; } ;
int di[]={1,-1,0,0},
    dj[]={0,0,-1,1};
int n , m , A[101][101], Nr[4];
void Lee(int i, int j, int vv, int vn)
{
    POZ Q[10001];
    int st = 1, dr = 0;
    dr ++;
    Q[dr] = {i,j};
    A[i][j] = vn;
    while(st <= dr)
    {
        int i = Q[st].lin , j = Q[st].col;
        for(int k = 0 ; k < 4 ; k ++)
        {
            int iv = i + di[k], jv = j + dj[k];
            if(iv >= 1 &amp;&amp; iv <= n &amp;&amp; jv >= 1 &amp;&amp; jv <= m &amp;&amp; A[iv][jv] == vv)
            {
                dr ++;
                Q[dr] = {iv , jv};
                A[iv][jv] = vn;
            }
        }
        st ++;
    }
}
int LangaInsula(int i, int j, int tipInsula)
{
    if(i<1 || i > n || j < 1 || j > m) return 0;
    if(A[i][j] != 0) return 0;
    for(int k = 0 ; k < 4 ; k ++)
    {
        int iv , jv;
        iv = i + di[k];
        jv = j + dj[k];
        if(iv >= 1 &amp;&amp; iv <= n &amp;&amp; jv >= 1 &amp;&amp; jv <= m &amp;&amp; A[iv][jv] == tipInsula) return 1;
    }
    return 0;
}
