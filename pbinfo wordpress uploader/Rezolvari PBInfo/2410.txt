<h1>Cerința</h1>

<p>Numim  <code>număr mare</code> un număr care conține prea multe cifre pentru a fi memorat în tipurile existente. <br>
Se dau două numere mari, ambele <code>≥0</code>. Calculați produsul lor.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>produsxxl.in</code> conține pe prima linie numărul mare <code>x</code>, iar pe a doua linie numărul mare <code>y</code>. Pentru fiecare dintre acestea se precizeaza mai intâi numărul de cifre și apoi cifrele numărului, separate prin câte un spațiu.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>produsxxl.out</code> va conține pe prima linie numărul <code>P</code>, reprezentând produsul celor două numere mari din fișierul de intrare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>ambele numere au cel mult 100 de cifre</li>
	<li>cifrele produsului nu vor fi separate prin spații</li>
</ul>

using namespace std;
ifstream cin("produsxxl.in");

int n , m , cnt , a[105] , b[105] , prod[205];

void citire(int a[] , int n)
{
    for(int i = n ; i >= 1 ; i--)
        cin >> a[i];
}
void afisare(int a[] , int n)
{
    for(int i = n ; i >= 1 ; i--)
        cout << a[i];
}
void inmultire_2nr_mari(int a[] , int b[])
{
    for(int i = 1 ; i <= n ; i++)
        for(int j = 1 ; j <= m ; j++)
            prod[i + j - 1] += a[i] * b[j];
    cnt = n + m - 1;
    int r = 0;
    for(int i = 1 ; i <= cnt ; i++)
    {
        r += prod[i];
        prod[i] = r % 10;
        r /= 10;
    }
    while(r)
    {
        prod[++cnt] = r;
        r /= 10;
    }
}

int main()
{
    cin >> n;
    citire(a , n);
    cin >> m;
    citire(b , m);
    inmultire_2nr_mari(a , b);
    afisare(prod , cnt);
    return 0;
