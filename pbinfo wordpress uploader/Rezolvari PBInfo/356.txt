<h1>Cerinţa</h1>

<p>La concursul “Miss &amp; Mister Boboc” participă <code>n</code> perechi de elevi, despre fiecare elev cunoscându-se înălțimea. Numim <em>grad de disproporționalitate al unei perechi</em> este diferența în valoare absolută dintre înălțimile elevilor care formează perechea.</p>

<p>Să se determine perechea cu grad de disproporționalitate maxim.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> perechi de numere naturale, separate prin spaţii, reprezentând înălțimile elevilor dintr-o pereche.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran două numere <code>x y</code>, separate prin exact un spațiu, reprezentând perechea cu grad maxim de disproporționalitate. Dacă există mai multe perechi cu grad maxim de disproporționalitate, se va afișa prima pereche în ordinea în care au fost date.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>înălțimile elevilor sunt numere naturale cuprinse între <code>1</code> și <code>1000</code> – unitatea de măsură folosită este de origine extraterestră, nu încercați să o înțelegeți !!</li>
</ul>

<h1>Explicație</h1>

<p>Există două perechi cu grad maxim de disproporționalitate: <code>8 12</code> și <code>10 6</code>. Dintre acestea, prima este <code>8 12</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int a, b, maxim = -1;
    for(int i = 1; i <= n; ++i){
        int x, y;
        cin >> x >> y;
        if(x > y){
            int grdis = x - y;
            if(grdis > maxim){
                maxim = grdis;
                a = x, b = y;
            }
        }
        else{
            int grdis = y - x;
            if(grdis > maxim){
                maxim = grdis;
                a = x, b = y;
            }
        }
    }
    cout << a << ' ' << b;
    return 0;
}
