O pereche <code>(a[j],a[k])</code> cu <code>1≤j&lt;k≤N</code> se numește pereche specială dacă are proprietatea că din <code>a[j]</code> și <code>a[k]</code> prin “lipire” se formează un număr <code>X</code> în care cifrele conținute apar de număr par de ori. De exemplu numerele <code>123</code> şi <code>21223</code> dacă se lipesc produc numărul <code>12321223</code> în care <code>1</code> apare de <code>2</code> ori, <code>2</code> apare de <code>4</code> ori și <code>3</code> apare de <code>2</code> ori.</p>

<p>Să se determine numărul perechilor speciale.</p>

<h1>Date de intrare</h1>

<p>Pe primul rând al fișierului text <code>parap.in</code> se află numărul natural <code>N</code> reprezentând numărul de elemente ale șirului dat. Pe al doilea rând, separate prin câte un spațiu se află elementele șirului dat.</p>

<h1>Date de ieșire</h1>

<p>Pe primul rând în fișierul de ieșire <code>parap.out</code> se va scrie un număr natural reprezentând numărul perechilor speciale.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ N ≤ 100.000</code></li>
	<li><code>1 ≤ a[k] ≤ 10</code><sup><code>18</code></sup></li>
</ul>

<h3>Explicație</h3>

<p>Perechile speciale sunt <code>(1,1),(1,212),(2,2),(2,112),(1,212),(2,112)</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;

ifstream cin("parap.in");
ofstream cout("parap.out");

long long n , x , cnt = 1 , a[100001] , sum;

long long frecv(long long x)
{
    long long f[15] = {0} , nr = 0;
    while(x != 0)
    {
        f[x % 10]++;
        x /= 10;
    }
    for(int i = 0 ; i <= 9 ; i++)
        nr = nr * 10 + f[i] % 2;
    return nr;
}

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> x;
        a[i] = frecv(x);
    }
    sort(a + 1 , a + n + 1);
    for(int i = 2 ; i <= n ; i++)
        if(a[i] == a[i - 1]) cnt++;
        else
        {
            sum = sum + cnt * (cnt - 1) / 2;
            cnt = 1;
        }
    sum = sum + cnt * (cnt - 1) / 2;
    cnt = 1;
    cout << sum;
}</pre>
