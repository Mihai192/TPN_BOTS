<p>La ora de matematică Georgică a învăţat o nouă operaţie: ridicarea la putere. În timpul orei de informatică aprofundează această noţiune considerând două numere naturale <code>m</code> şi <code>n</code>, cu acelaşi număr de cifre şi calculând:</p>

<p>a) puterea <code>p</code> =  <code>a</code><sup><code>b</code></sup>  , unde <code>a</code> este ultima cifră a lui <code>m</code>, iar <code>b</code> este ultima cifră a lui <code>n</code>;<br>
b) suma <code>s</code> a tuturor puterilor de forma <code>x</code><sup><code>y</code></sup>, unde <code>x</code> şi <code>y</code> sunt cifre din <code>m</code>, respectiv <code>n</code> de pe aceeaşi poziţie. De exemplu pentru <code>m=624</code> şi <code>n=253</code>, se va calcula <code>s=</code>  <code>4</code><sup><code>3</code></sup> + <code>2</code><sup><code>5</code></sup> + <code>6</code><sup><code>2</code></sup>  şi se obţine <code>s=132</code>.</p>

<h1>Cerinţă</h1>

<p>Scrieţi un program, care pentru două numere naturale date <code>m</code> şi <code>n</code> determină:</p>

<p>a) puterea <code>p</code> definită în enunţ;<br>
b) suma <code>s</code> definită în enunţ.  </p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>putere.in</code> conţine pe prima linie numerele <code>m</code> şi <code>n</code>, separate între ele printr-un spaţiu. </p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>putere.out</code> va conţine, pe prima linie <code>p</code> şi pe a doua linie <code>s</code>.  </p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt;m,n&lt; 2000000001</code></li>
	<li>Toate cifrele lui n sunt nenule.</li>
	<li><code>m</code> şi <code>n</code> au acelaşi număr de cifre.</li>
</ul>

<h1>Explicaţie</h1>

<p><code>m=624</code>, <code>n=253</code>.<br>
<code>p=</code>  <code>4</code><sup><code>3</code></sup> , adică <code>p=64</code>
 <code>s=</code>  <code>4</code><sup><code>3</code></sup> + <code>2</code><sup><code>5</code></sup> + <code>6</code><sup><code>2</code></sup> , adică <code>s=132</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("putere.in");
ofstream fout("putere.out");
int put(int a,int b)
{
    long long p = 1;
    for(int i = 1 ; i <= b ; ++i) p *= a;
    return p;
}
int main()
{
    long long n , m , s = 0;
    fin >> n >> m;
    fout << put(n % 10 , m % 10) << endl;
    while(n != 0 &amp;&amp; m != 0)
    {
        s = s + put(n % 10 , m % 10);
        n /= 10;
        m /= 10;
    }
    fout << s;

    fin.close();
    fout.close();

    return 0;
}</pre>
