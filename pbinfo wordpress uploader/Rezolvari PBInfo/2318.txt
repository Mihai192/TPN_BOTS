<h1>Cerința</h1>

<p>Se dau două numere <code>n</code> și <code>b</code>, unde <code>b</code> este o bază de numerație, iar <code>n</code> un număr reprezentat în baza <code>b</code>. Să se afișeze numărul obținut prin transformarea lui <code>n</code> din baza <code>b</code> în baza <code>10</code>. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>conversie_b_10.in</code> conține pe prima linie numerele <code>n</code> și <code>b</code>, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>conversie_b_10.out</code> va conține pe prima linie numărul <code>nb</code>, reprezentând transformarea lui <code>n</code> din baza <code>b</code> în baza <code>10</code></p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>n</code> este un număr pozitiv cu cel mult <code>7</code> cifre</li>
	<li><code>2 ≤ b ≤ 10</code></li>
	<li>datele de intrare sunt corecte</li>
</ul>

<h3>Explicație</h3>

<p> <code>13</code> reprezentat în baza <code>2</code> este <code>1101</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("conversie_b_10.in");
ofstream cout("conversie_b_10.out");

int main()
{
    int n , b;
    cin >> n >> b;
    int nr=1;
    int a[10];
    int p = 0;
    while(n)
    {
        a[p]=n%10;
        n/=10;
        p++;
    }
    nr=a[p-1];
    for(int i = p-2 ; i >= 0 ; --i)
    {
        nr=nr*b+a[i];    
    }
    cout << nr;
    return 0;
}</pre>
