<h1>Enunț</h1>

<p>Într-o școală sunt <code>n</code> clase a <span class="caps">VII</span>-a, fiecare de câte <code>k</code> elevi, și <code>m</code> clase a <span class="caps">VIII</span>-a, fiecare de câte <code>p</code> elevi. Directorul școlii vrea să facă o nouă repartizare a elevilor în clase astfel încât în fiecare clasă: a) să fie același număr de elevi; b) să fie doar elevi de clasa a <span class="caps">VII</span>-a sau doar elevi de clasa a <span class="caps">VIII</span>-a.</p>

<h1>Cerința</h1>

<p>Cunoscându-se valorile <code>n</code>, <code>k</code>, <code>m</code> și <code>p</code>, numere naturale nenule, scrieți un program pentru a determina <code>x</code> numărul de clase necesar acestei noi repartizări, noul număr <code>y</code> de clase a <span class="caps">VII</span>-a, respectiv <code>z</code> de clase a <span class="caps">VIII</span>-a, precum și numărul <code>t</code> maxim de elevi repartizați într-o clasă. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele naturale <code>n</code>, <code>k</code>, <code>m</code> și <code>p</code>, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele naturale:  <code>x y z t</code>, separate prin câte un spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>n, k, m, p, x, y, z, t ≤ 1.000.000</code></li>
</ul>

using namespace std;

int cmmdc(int a , int b)
{
    while(b)
    {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main()
{
    int n, k, m, p;
    cin >> n >> k >> m >> p;
    int ne7 = n * k;
    int ne8 = m * p;
    int x, y , z , t;

    y = ne7 / t;
    z = ne8 / t;
