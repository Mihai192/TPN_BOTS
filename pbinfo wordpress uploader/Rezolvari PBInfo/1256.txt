<p>Se consideră o listă liniară simplu înlănțuită, alocată dinamic, în care elementele sunt de tipul declarat mai jos:</p>

<pre class="code_c cm-s-default" data-lang="text/x-csrc" contenteditable="true" editable="true"><span class="cm-keyword">struct</span> <span class="cm-variable">node</span> {
  <span class="cm-keyword">char</span> <span class="cm-variable">key</span>;
  <span class="cm-variable">node</span> <span class="cm-operator">*</span><span class="cm-variable">next</span>;
};</pre>

<p>în care câmpul <code>key</code> memorează un caracter, iar câmpul <code>next</code> memorează adresa următorului element al listei</p>

<h1>Cerința</h1>

<p>Să se scrie o funcție C++ cu următorul prototip:</p>

<pre class="code_c cm-s-default" data-lang="text/x-csrc" contenteditable="true" editable="true"><span class="cm-variable">bool</span> <span class="cm-variable">palindrom</span>(<span class="cm-variable">node</span> <span class="cm-operator">*</span><span class="cm-variable">l</span>);</pre>

<p>care returnează <code>true</code> în caz ca lista al cărei prim element are adresa memorată în <code>l</code> formează un palindrom, iar <code>false</code> în caz contrar. </p>

<p>Un palindrom este un șir care citit de la stânga la dreapta sau de la dreapta la stânga rămâne neschimbat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>numele funcției va fi <code>palindrom</code></li>
	<li>lista va conține cel puțin <code>2</code> elemente</li>
	<li><code>2</code> caractere sunt considerate egale daca au acelaşi cod <strong><span class="caps">ASCII</span></strong></li>
</ul>

<h1>Important</h1>

<p>Soluţia propusă va conţine definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei. La ieșirea din subprogram lista <b>nu</b> va fi modificată.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">bool palindrom(node *l)
{
    node* pi = l;
    int a[10001];
    int n = 0;
    while (pi != NULL)
    {
        a[n] = pi ->key;
        n ++;
        pi = pi -> next;
    }
    for (int i = 0 ; i < n; i ++)
        if (a[i] != a[n - i - 1])return false;

    return true;
}</pre>
