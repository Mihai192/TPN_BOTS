<p>Se consideră un șir de numere naturale <code>a[1]</code>, <code>a[2]</code>, …, <code>a[n]</code>.</p>

<h1>Cerința</h1>

<p>Să se determine numărul tripletelor <code>(a[i], a[j], a[p])</code> cu <code>i &lt; j &lt; p</code>, iar <code>a[i] + a[j] + a[p]</code> este divizibil cu <code>5</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code>, <code>w</code>, <code>X</code>, <code>Y</code>, <code>Z</code>. Șirul de <code>n</code> numere se generează după relațiile: <code>a[1] = w</code>, <code>a[i] = (X * a[i-1] + Y) % Z</code>;</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul tripletelor cu suma divizibilă cu <code>5</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li><code>1 ≤ w, X, Y, Z ≤ 1.000.000.000</code></li>
</ul>

    for (long long i = 1; i <= k; i ++) rez /= i;

}
