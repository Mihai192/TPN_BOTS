<h1>Cerința</h1>

<p>Să se scrie o funcţie care primeşte ca argumente două numere naturale <code>a</code> şi <code>b</code> şi returnează numărul de elemente din intervalul <code>[a,b]</code> care au cifra de control egală cu <code>a</code>.<br>
Cifra de control a unui număr se obţine efectuând suma cifrelor  sale, apoi suma cifrelor acestei sume etc. până se obţine o sumă formată  dintr-o singură cifră. De exemplu, cifra de control a numărului <code>645</code> este <code>6</code>. (<code>6+4+5=15</code>, <code>1+5=6</code>)</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcţiei va fi <code>sum_cifra_control</code></li>
	<li>funcţia va avea doi parametri, <code>a</code> şi <code>b</code>, în această ordine</li>
	<li><code>1 ≤ a ≤ 9</code></li>
	<li><code>a &lt; b ≤ 10</code><sup><code>7</code></sup></li>
</ul>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int sum_cifra_control(long long a, long long b)
{
  long long p=0;
  if(a==9)
  {
      for(long long i=a;i<=b;i++)
      {
          if(i%9==0) p++;
      }
  }
  else
  for(long long i=a;i<=b;i++)
  {
      if(i%9==a) p++;
  }
  return p;
}</pre>
