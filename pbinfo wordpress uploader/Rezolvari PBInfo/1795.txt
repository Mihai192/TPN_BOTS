<h1>Cerința</h1>

<p>Gigel este elev în clasa a <span class="caps">XII</span>-a la Liceul Teoretic “Ion Luca” din Vatra Dornei. Acesta, știind că urmează examenul de Bacalaureat și că nu a învățat nimic, s-a hotărât să plece de acasă să își găsească un rost în lume. După zile bune de mers, lipsit de energie, flămând și însetat, acesta a făcut un popas și s-a gândit că era mai bine să nu plece de acasă, motiv pentru care s-a hotărât să se întoarcă. Este cunoscut faptul că în pădurile dornene locuiesc atât Yeti, cât și Bigfoot, precum și mulți vârcolaci. Gigel, fiind un dornean adevărat, cunoaște coordonatele zonelor unde aceștia locuiesc și dorește să se întoarcă acasă pe drumul cel mai scurt, evitându-i pe aceștia.</p>

<p>Cunoscând suprafața regiunii în care se află Gigel și casa acestuia (care poate fi reprezentată printr-un tablou bidimensional cu <code>n</code> linii și <code>m</code> coloane, în care fiecare zonă are coordonatele <code>x</code> și <code>y</code>), coordonatele casei (<code>X1</code>, <code>Y1</code>) și coordonatele locului de popas (<code>X2</code>, <code>Y2</code>), coordonatele zonelor în care locuiesc Yeti (<code>XY</code>, <code>YY</code>) și Bigfoot (<code>XB</code>, <code>YB</code>), precum și coordonatele (<code>X</code>, <code>Y</code>) ale celor <code>K</code> zone în care locuiesc vârcolacii, se cere să îl ajutați pe Gigel să găsească lungimea <code>D</code> a celui mai scurt drum spre casă.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>gigelajungeacasa.in</code> conține pe prima linie trei numere naturale <code>N</code>, <code>M</code> și <code>K</code>, pe linia a doua patru numere naturale <code>X1</code>, <code>Y1</code>, <code>X2</code> și <code>Y2</code>, pe a treia linie patru numere naturale <code>XY</code>, <code>YY</code>, <code>XB</code> și <code>YB</code>, iar pe următoarele <code>K</code> linii câte o pereche de două numere naturale <code>X</code> și <code>Y</code>. Semnificațiile sunt cele precizate în enunț.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>gigelajungeacasa.out</code> va conține un singur număr natural <code>D</code>, pe prima linie, reprezentând lungimea drumului minim spre casa lui Gigel.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N, M ≤ 1000</code></li>
	<li><code>0 ≤ K ≤ 14000</code></li>
	<li><code>Gigel poate merge pe oricare din direcțiile N, NE, E, SE, S, SV, V și NV.</code></li>
</ul>

<h3>Explicație</h3>

<p>Cel mai scurt drum din locul de popas al lui Gigel până la casa acestuia, ocolind vârcolacii, pe Yeti și pe Bigfoot, are lungimea <code>6</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("gigelajungeacasa.in");
ofstream cout("gigelajungeacasa.out");
const int di[]={-1 , -1 , -1 , 0 , 1 , 1 , 1 , 0 };
const int dj[]={-1 , 0 , 1 , 1, 1 , 0 , -1 , -1};
int n , m , a[1002][1002] , x[100001] , y[100001] , q , xy , yy , xb , yb , r , p;
int ip , jp , is , js;///coordonatele

bool inside(int i , int j)
{
    return i>=1 &amp;&amp; i<=n &amp;&amp; j>=1 &amp;&amp; j<=m;
}
int main()
{
    ///citire
    cin >> n >> m >> q;
    cin >> is >> js >> ip >> jp;
    cin >> xy >> yy >> xb >> yb;
    for(int i = 1 ; i <= q ; ++i) cin >> r >> p , a[r][p]=1;
    ///LEE
    int st = 1 , dr = 1;///capetele cozii
    a[ip][jp] = 1;///pornirea
    x[1]=ip;
    y[1]=jp;
    while(st <= dr &amp;&amp; a[is][js]==0)
    {
        int i= x[st] , j = y[st];///poz curenta
        for(int k = 0 ; k < 8 ; k++)
        {
            ///caculam pozitiie urmatoare
            int ii = i+di[k];
            int jj = j+dj[k];
            if(inside(ii , jj) &amp;&amp; a[ii][jj]==0)
            {
                dr++;///crestem coada
                x[dr]=ii;
                y[dr]=jj;
                a[ii][jj]=a[i][j]+1;///cu 1 mai departe
            }
        }
        st++;///avans in coada
    }
    cout << a[is][js]-1;
    return 0;
}</pre>
