<h1>Cerința</h1>

<p>Scrieți funcția <code>Kth</code> care primește ca parametri:</p>

<ul>
	<li><code>a</code> – un vector de numere întregi</li>
	<li><code>n</code> – un număr natural reprezentând numărul elementelor vectorului</li>
	<li><code>x</code> – un număr întreg</li>
	<li><code>k</code> – un număr natural</li>
</ul>

<p>Funcția va returna poziția din vector unde <code>x</code> apare pentru a <code>k</code>-a oară, sau <code>-1</code> dacă <code>x</code> nu apare de cel puțin <code>k</code> ori</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>elementele vectorului sunt indexate de la <code>0</code> la <code>n-1</code>.</li>
	<li><code>-1000 ≤ a[i] ≤ 1000</code>, pentru orice <code>i=0..n-1</code></li>
	<li><code>-1000 ≤ x ≤ 1000</code></li>
	<li><code>0 ≤ k ≤ 1000</code></li>
</ul>

<h1>Exemple:</h1>

<p>Dacă <code>a = (4,2,5,2,6,2)</code>, atunci <code>Kth(a, 6, 2, 2)</code> va returna valoarea <code>3</code>.<br>
Dacă <code>a = (4,2,5,2,6,2)</code>, atunci <code>Kth(a, 6, 5, 3)</code> va returna valoarea <code>-1</code>.<br>
Dacă <code>a = (4,2,5,2,6,2)</code>, atunci <code>Kth(a, 6, 5, 0)</code> va returna valoarea <code>-1</code>.</p>

<h3>Important</h3>

<p>Soluția propusă va conține definiția funcției cerute. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int Kth(int a[], int n, int x, int k){
    if(k == 0)
        return -1;
    int cnt = 0;
    for(int i = 0; i < n; ++i)
        if(a[i] == x)
            cnt++;
    if(cnt >= k){
        int poz = 0;
        for(int i = 0; i < n; ++i)
            if(a[i] == x){
                k--;
                if(k == 0)
                    poz = i;
            }
        return poz;
    }
    return -1;
}</pre>
