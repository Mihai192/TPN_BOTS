<h1>Cerința</h1>

<p>De-a lungul bulevardului sunt <code>n</code> copaci, numerotați de la <code>1</code> la <code>n</code>, pentru fiecare cunoscându-se înălțimea, exprimată în centimetri. Primarul dorește să taie copacii și apelează la un vrăjitor care va proceda astfel: alege o secvență cât mai lungă de copaci învecinați și aplică o vrajă prin care toți înălțimea tuturor copacilor din secvență scade cu o aceeași valoare, strict pozitivă. </p>

<p>Să se determine care este numărul minim de vrăji care trebuie aplicate astfel încât toți copacii să aibă înălțime zero.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, reprezentând  înălțimile copacilor.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>C</code>, reprezentând numărul minim de vrăji necesare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>înălțimile arborilor sunt numere naturale nenule mai mici decât <code>1000000</code></li>
	<li>secvența aleasă de vrăjitor la un moment dat nu poate conține copaci de înălțime zero</li>
</ul>

<p><strong>Intrare</strong></p>

<h1>Explicație</h1>

<ul>
	<li>prima vrajă se aplică pe întreg șirul de copaci (secvența <code>1 - 5</code>) și micșorează toți copacii cu <code>2</code>.<br>
Șirul devine: <code>3 0 1 3 1</code>.</li>
	<li>a doua vrajă se aplică pe secvența <code>1 - 1</code>) și micșorează copacii din secvență cu <code>3</code>.<br>
Șirul devine: <code>0 0 1 3 1</code>.</li>
	<li>a treia vrajă se aplică pe secvența <code>3 - 5</code>) și micșorează copacii din secvență cu <code>1</code>.<br>
Șirul devine: <code>0 0 0 2 0</code>.</li>
	<li>a patra vrajă se aplică pe secvența <code>4 - 4</code>) și micșorează copacii din secvență cu <code>2</code>.<br>
Șirul devine: <code>0 0 0 0 0</code>.</li>
</ul>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int a[1001] , cnt = 0;

void vraja(int st , int dr)
{
    cnt++;
    int min = 10000000;
    for(int i = st ; i <= dr ; ++i)
        if(a[i] < min)
            min = a[i];
    for(int i = st ; i <= dr ; ++i)
        a[i]-=min;
    for(int i = st ; i <= dr ; ++i)
        if(a[i]>0)
        {
            int j = i;
            while(j < dr &amp;&amp; a[j+1]>0)
                j++;
            vraja(i , j);
            i=j+1;
        }
}

int main()
{
    int n;
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
        cin >> a[i];
    vraja(1 , n);
    cout << cnt;
    return 0;
}</pre>
