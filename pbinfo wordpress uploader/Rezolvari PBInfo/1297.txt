<h1>Cerința</h1>

<p>Se dă următoarul triunghi de numere:</p>

<pre contenteditable="true" editable="true">    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
..........</pre>

<p>Deduceți regula după care a fost format triunghiul si afișați numerele aflate pe a <code>n</code>-lea linie a triunghiului.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>triunghiul.in</code> conține pe prima linie numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>triunghiul.out</code> va conține pe prima linie numerele care se află pe a <code>n</code>-lea linie din triunghi, separate prin câte un spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 ≤ n ≤ 19</code></li>
	<li>prima linie din triunghi se numerotează cu <code>0</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("triunghiul.in");
ofstream cout("triunghiul.out");

int n , a[50][50];

int main()
{
    cin >> n;
    n++;
    a[1][1] = 1;
    for(int i = 2 ; i <= n ; i++)
        for(int j = 1 ; j <= n ; j++)
            a[i][j] = a[i - 1][j] + a[i - 1][j - 1];

    for(int j = 1 ; j <= n ; j++)
        cout << a[n][j] << " ";
}</pre>
