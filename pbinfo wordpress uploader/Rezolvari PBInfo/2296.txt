<h1>Cerința</h1>

<p>Se dau două șiruri de câte <code>N</code> numere fiecare. Se cere să se găsească valoarea maximă a celui mai mare divizor comun a două numere <code>A</code> și <code>B</code>, astfel încât <code>A</code> să aparțină primului șir, iar <code>B</code> să aparțină celui de-al doilea șir.</p>

<h1>Date de intrare</h1>

<p>În fișierul <code>gcd.in</code> se va afla pe prima linie un număr reprezentând valoarea lui <code>N</code>. Pe cea de-a doua linie se vor afla <code>N</code> numere separate prin câte un spațiu, reprezentând elementele primului șir. Pe cea de-a treia linie se vor afla <code>N</code> numere separate prin câte un spațiu, reprezentând elementele celui de-al doilea șir.</p>

<h1>Date de ieșire</h1>

<p>În fișierul <code>gcd.out</code> se va afla pe primia linie un număr natural reprezentând valoarea maximă a celui mai mare divizor comun a două numere <code>A</code> și <code>B</code>, astfel încât <code>A</code> să aparțină primului șir, iar <code>B</code> să aparțină celui de-al doilea șir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>N</code> &lt;= <code>500.000</code></li>
	<li>Elementele celor două șiruri &lt;= <code>1.000.000</code></li>
	<li>Pentru <i>40%</i> din teste, <code>N</code> &lt;= <code>1.000</code></li>
</ul>

<h3>Explicație</h3>

<p><code>A</code> = <code>8</code>, <code>B</code> = <code>8</code>, iar <code>(A,B)</code> = <code>8</code> este valoarea maximă a celui mai mare divizor comun a vreunei perechi.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("gcd.in");
ofstream cout("gcd.out");

int n , a[1000001] , b[1000001] , c , maxi , x[1000001] , y[1000001];

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        cin >> c;
        a[c]++;
        if(c > maxi) maxi = c;
    }
    for(int i = 1 ; i <= n ; ++i)
    {
        cin >> c;
        b[c]++;
        if(c > maxi) maxi = c;
    }
    for(int i = 1 ; i <= maxi ; ++i)
        for(int j = i ; j <= maxi ; j += i)
        {
            if(a[j] != 0) x[i] = max(x[i] , j);
            if(b[j] != 0) y[i] = max(y[i] , j);
        }
    int ok = 0;
    for(int i = maxi ; i >= 0 ; --i)
        if(x[i]&amp;&amp;y[i]) break;
        else ok++;
    cout << maxi - ok;
    return 0;
}</pre>
