<h1>Cerinţa</h1>

<p>Să se scrie o <strong>funcție C++ recursivă</strong> care să returneze numărul de cifre egale cu zero ale unui număr natural transmis ca parametru.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>nr_cif_zero</code></li>
	<li>funcția va avea un parametru reprezentând numărul dat</li>
	<li>numărul pentru care se calculează numărul de cifre egale cu zero va fi mai mic decât <code>2.000.000.000</code></li>
</ul>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int cnt = 0;
int nr_cif_zero(int n){
    if(n == 0 &amp;&amp; cnt == 0)
        return 1;
    else if(n == 0)
        return 0;
    else{
        cnt++;
        if(n % 10 == 0)
            return nr_cif_zero(n / 10) + 1;
        else
            return nr_cif_zero(n / 10);
    }
}</pre>
