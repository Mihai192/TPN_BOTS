<p>Se consideră o expresie formată din numere naturale și operatorii <code>+</code>, <code>-</code>, <code>*</code>.</p>

<h1>Cerința</h1>

<p>Să se evalueze expresia.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură expresia.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran un singur număr reprezentând valoarea expresiei.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Lungimea expresiei va fi de cel mult <code>50</code> de caractere</li>
	<li>Rezultatul final precum și rezultatele intermediare se vor încadra în tipul de date <code>long long</code></li>
	<li>Expresia este corectă și nu conține alte caractere în afară de cifre și operatorii <code>+</code>, <code>-</code>, <code>*</code>.</li>
</ul>

    int semn = 1 , ori = 0 , i = 0;

        {
            if(s[i] == '+') semn = 1;

            x = 0;
            while(Cifra(s[i]))

                x = x * 10 + s[i] - '0';
                i++;
            }
            x *= semn;
            if(ori == 1) v[c] = v[c] * x;
            else v[++c] = x;
            semn = 1;
            ori = 0;
        }
    }

    cout << suma;
