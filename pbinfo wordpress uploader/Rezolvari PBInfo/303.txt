<h1>Cerinţa</h1>

<p>Se dau <code>n</code> numere naturale mai mici decât <code>1.000.000</code>. Determinaţi câte dintre ele sunt prime.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>eratostene.in</code> conţine pe prima linie numărul <code>n</code>; urmează cele <code>n</code> numere, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>eratostene.out</code> va conţine pe prima linie numărul <code>C</code>, reprezentând numărul valorilor citite care erau numere prime.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("eratostene.in");
ofstream cout("eratostene.out");

bool c[1000005];

int main()
{
    c[0] = c[1] = 1;
    for(int i = 2; i <= 1000; ++i)
        if(c[i] == 0)
            for(int j = 2; i * j <= 1000000; ++j)
                c[i * j] = 1;

    int n;
    cin >> n;
    int x;
    int cnt = 0;
    for(int i = 1; i <= n; ++i){
        cin >> x;
        if(c[x] == 0)
            cnt++;
    }

    cout << cnt;

    return 0;
}</pre>
