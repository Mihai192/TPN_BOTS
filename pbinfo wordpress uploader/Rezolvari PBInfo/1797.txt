<h1>Cerința</h1>

<p>Se dă următorul şir de numere naturale: <br>
<code>(1, 21, 321, 4321, 54321, 654321, 7654321, 87654321, 987654321, 10987654321, ...)</code>. Al <code>x</code>-lea termen este format prin alipirea primelor <code>x</code> numere naturale nenule. Determinaţi câte din primele <code>n</code> numere ale şirului sunt divizibile cu <code>3</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>c</code>, reprezentând câte din primele <code>n</code> numere din şir sunt divizibile cu <code>3</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Primele <code>3</code> numere sunt : <code>1, 21, 321</code>. Din cele <code>3</code> numere doar <code>2</code> sunt divizibile cu <code>3</code>. (<code>21</code> si <code>321</code>).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include<iostream>
using namespace std;

int main()
{
    unsigned long n;
    cin>>n;
    if(n%3==0 || n%3==1)
      cout<<2*(n/3);
    else cout<<2*(n/3)+1;  
    return 0;
}</pre>
