<p>Mai sunt câteva săptămâni și vine Crăciunul. Ajuns într-un magazin de jucării, Robert îl roagă pe tatăl său să-i cumpere cea mai frumoasă mașină cu telecomandă. Tatăl său îi spune că nu a fost cuminte în timpul anului și nu merită această jucărie, însă după dispute intense acesta hotaraste să-i mai acorde o sansă, doar dacă va rezolva următoarea problema: </p>

<p>Având un string <code>S</code>, putem să obținem un palindrom din acest șir ștergând un singur caracter? </p>

<p>Robert nu se prea descurca la algoritmică așa că vă roagă foarte mult să-i rezolvați problema pentru a se putea juca cu mașina cu telecomandă. În schimbul acestei mașini el vă va recompensa cu 100 puncte.</p>

<h1>Cerința</h1>

<p>Find dat un string <code>S</code>, se poate obține un palindrom din șirul inițial ștergând doar un singur caracter ?</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>constructpalindrom.in</code> conţine pe prima linie o valoare <code>T</code> reprezentând numărul de teste. Pe următoarele T linii vom avea cate un string reprezentând întrebarea adresata lui Robert de către tatăl sau.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>constructpalindrom.out</code> va conține <code>T</code> linii cu răspunsul <code>YES</code> daca se poate obține un palindrom ștergând un singur caracter și <code>NO</code> dacă nu se poate obține.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 &lt;= T &lt;= 100</code></li>
	<li>Dimensiunea stringului <code>&lt;= 100000</code></li>
	<li>Pentru 10% din punctaj dimensiunea stringului <code>&lt;= 1000</code></li>
	<li>String-ul conține caractere de la <code>a</code> la <code>z</code>.</li>
	<li>Dimensiunea string-ului după ștergerea unui caracter va fi mai mică decât a fost înainte.</li>
</ul>

<h3>Explicație</h3>

<p>Pentru primul exemplu (<code>aaa</code>): Putem șterge orice <code>a</code>, string-ul rezultat este <code>aa</code>, care este palindrom.</p>

<p>Pentru al doilea exemplu (<code>abc</code>): Nu este posibil să eliminăm exact un caracter și să obtinem un palindrom.</p>

<p>Pentru al treilea exemplu (<code>abdbca</code>): ștergem caracterul <code>c</code>, string-ul rezultat este <code>abdba</code> care este palindrom.</p>

<p>Pentru exemplul al patrulea (<code>abba</code>): Ștergem <code>b</code>, obținem <code>aba</code> care este palindrom.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;

char s[100001];

ifstream cin("constructpalindrom.in");
ofstream cout("constructpalindrom.out");

int main()
{
    int t;
    cin >> t;
    for(int q = 1 ; q <= t ; ++q)
    {
        cin >> s;
        int n = strlen(s);
        int cnt = 0;
        bool ok = true;
        for(int i = 0 , j = n - 1 ; i < j ; ++i , j--)
            if(s[i]!=s[j])
            {
                j--;
                cnt++;
                if(s[i]!=s[j])
                    j++,i--;
                if(s[i]!=s[j])
                {
                    ok=false;
                    break;
                }
            }
        if(ok == true &amp;&amp; cnt <= 1)
            cout << "YES" << endl;
        else
            cout << "NO" << endl;
    }
    return 0;
}</pre>
