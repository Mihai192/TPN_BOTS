<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale distincte. Pentru oricare două numere date, distincte, <code>x</code> şi <code>y</code>,  se determină ultima cifră a numărului <font size="6"><code>3</code><sup><code>3</code><sup><code>3</code><sup><code>x•y</code></sup></sup></sup></font> şi se însumează cifrele obţinute. Aflaţi valoarea acestei sume.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>3lan.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie cele <code>n</code> numere date, despărţite prin spaţii.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>3lan.out</code> va conține pe prima linie suma cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere date sunt cel mult egale cu <code>10.000</code></li>
</ul>

<h3>Explicație</h3>

<p> Singura pereche de numere dintre cele date este <code>9 , 0</code>, iar ultima cifră a numărului <font size="6"><code>3</code><sup><code>3</code><sup><code>3</code><sup><code>9•0</code></sup></sup></sup></font> este <code>7</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("3lan.in");
ofstream cout("3lan.out");

int n , a[1001] , s;

int treilan(int x , int y)
{
    int rest = (x * y) % 4 , nr1 , nr2;
    if(rest == 0) nr1 = 1;
    else if(rest == 1) nr1 = 3;
    else if(rest == 2) nr1 = 9;
    else if(rest == 3) nr1 = 7;
    nr1 %= 4;
    if(nr1 == 0) nr2 = 1;
    else if(nr1 == 1) nr2 = 3;
    else if(nr1 == 2) nr2 = 9;
    else if(nr1 == 3) nr2 = 7;
    nr2 %= 4;
    if(nr2 == 0) return 1;
    else if(nr2 == 1) return 3;
    else if(nr2 == 2) return 9;
    else if(nr2 == 3) return 7;
}

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
        cin >> a[i];

    for(int i = 1 ; i <= n ; i++)
        for(int j = i + 1 ; j <= n ; j++)
            s += treilan(i , j);
    cout << s;
    return 0;
}</pre>
