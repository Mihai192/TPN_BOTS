<p>Un număr natural nenul se numește <strong>perfect</strong> dacă este egal cu suma divizorilor săi naturali strict mai mici decât el.<br>
<strong>Exemplu</strong>: <code>28</code> este număr perfect pentru că <code>28=1+2+4+7+14</code>.</p>


<h1>Cerinţa</h1>

<p>Să se scrie o funcție C++ care, pentru doi parametri, <code>a</code> și <code>b</code>, afișează pe ecran, separate prin câte un spațiu, în ordine descrescătoare, toate numerele perfecte din intervalul <code>[a,b]</code>. Dacă în interval nu există astfel de numere, subprogramul afișează pe ecran mesajul <code>nu exista</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>perfect</code> și va avea exact doi parametri, <code>a</code> și <code>b</code>, cu semnificația de mai sus</li>
	<li><code>1 ≤ a ≤ b ≤10000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int sumdiv(int n){
    if(n == 1)
        return 0;
    int sum = 1;
    for(int d = 2; d * d <= n; ++d){
        if(n % d == 0)
            sum += d + (n / d);
        if(d * d == n)
            sum -= d;
    }
    return sum;
}

void perfect(int a, int b){
    bool gasit = false;
    if(a > b)
        swap(a, b);
    for(int i = b; i >= a; --i)
        if(i == sumdiv(i))
            cout << i << ' ', gasit = true;
    if(!gasit)
        cout << "nu exista";
}</pre>
