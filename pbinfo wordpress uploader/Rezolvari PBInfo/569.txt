<h1>Cerința</h1>

<p>Gigel a învățat la matematică adunarea mai multor termeni și acum lucrează la temă. Aceasta constă în scrierea a <code>n</code> egalități, fiecare fiind formată din <code>m</code> termeni, dispuși în ordine în stânga și în dreapta semnului =. Gigel a reușit să-și facă tema, însă maleficul Costel a profitat de neatenția lui Gigel și a șters toate semnele dintre numerele scrise, atât cele de adunare(+) cât și semnul egal(=). Ajutați-l pe Gigel să reconstituie egalitățile la care a lucrat.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n m</code>, iar apoi <code>n</code> șiruri cu câte <code>m</code> numere naturale, fiecare șir reprezentând termenii dintr-o egalitate.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>n</code> perechi de numere <code>s d</code>. Acestea corespund, în ordine, fiecărei egalități și anume <code>s</code> reprezintă numărul de termeni din stângă semnului =, iar <code>d</code> reprezintă numărul de termeni din dreapta semnului =.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li><code>1 ≤ m ≤ 1000</code></li>
	<li>pentru fiecare egalitate termenii sunt numere naturale nenule mai mici decât <code>1000</code></li>
	<li>toate egalitățile scrise de Gigel au fost inițial corecte</li>
	<li>pentru fiecare egalitate <code>s+d=m</code></li>
	<li>maleficul Costel nu a modificat ordinea termenilor</li>
</ul>

<h1>Explicație</h1>

<p>Pentru prima egalitate, <code>1 + 2 + 3 = 1 + 5</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    int n , m , a[1000] , st[1000] , dr[1000];
    cin >> n >> m;
    for(int i = 1 ; i <= n ; i++)
    {
        int s = 0 , s1 = 0;
        for(int j = 1 ; j <= m ; j++)
        {
            cin >> a[j];
            s += a[j];
        }
        for(int j = 1 ; j <= m ; j++)
        {
            s1 += a[j];
            if(s1 == s / 2) st[i] = j , dr[i] = m - j;
        }
    }
    for(int i = 1 ; i <= n ; i++)
    {
        cout << st[i] << ' ' << dr[i] << '\n';
    }
}</pre>
