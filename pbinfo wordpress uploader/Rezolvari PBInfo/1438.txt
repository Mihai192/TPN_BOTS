<p>În Regatul Numerelor, a început războiul civil. Se dau <code>n</code> soldați, reprezentați prin <code>n</code> numere naturale, nu neapărat distincte. Cei <code>n</code> soldați sunt recrutați în două batalioane adverse, după o lege de recrutare. Această lege are un număr asociat, care este egal cu <code>1</code> sau <code>2</code>. Dacă legea este <code>1</code>, atunci soldații care au ultima cifră egală cu <code>0</code>, <code>2</code>, <code>4</code>, <code>6</code> și <code>8</code> sunt recrutați de primul batalion, iar ceilalți de cel de-al doilea. Dacă legea e <code>2</code>, atunci soldații care au suma divizorilor număr par sunt recrutați de primul batalion, iar restul de cel de-al doilea.</p>

<h1>Cerința</h1>

<p>Dându-se <code>n</code>, numărul de soldați, <code>L</code>, legea de recrutare, și identificatorii celor <code>n</code> soldați, să se afișeze numărul soldaților din primul, respectiv al doilea batalion.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>razboi.in</code> conține pe prima linie numerele <code>n</code> şi <code>L</code>, cu semnificaţiile din enunţ, iar pe a doua linie se găsesc <code>n</code> numere reprezentând identificatorii celor <code>n</code> soldați.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>razboi.out</code> va conține pe prima linie cele două numere cerute, separate printr-un spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 60 000</code></li>
	<li>Numerele din şir sunt mai mici decât <code>100 000</code></li>
</ul>

<h1>Exemplul 1</h1>

<p><code>razboi.in</code></p>

<pre contenteditable="true" editable="true">3 2
6 17 2</pre>

<p><code>razboi.out</code></p>

<pre contenteditable="true" editable="true">2 1</pre>

<h3>Explicație</h3>

<p>Suma divizorilor lui <code>6</code> este <code>12</code>.<br>
Suma divizorilor lui <code>17</code> este <code>18</code>.<br>
Suma divizorilor lui <code>2</code> este <code>3</code>.</p>

<p>Așadar, primul batalion are doi soldați, iar al doilea batalion are unul singur.</p>

<h1>Exemplul 2</h1>

<p><code>razboi.in</code></p>

<pre contenteditable="true" editable="true">10 1
15 150 12 36 45 78 85 95 93 141</pre>

<p><code>razboi.out</code></p>

<pre contenteditable="true" editable="true">4 6</pre>

<h1>Exemplul 3</h1>

<p><code>razboi.in</code></p>

<pre contenteditable="true" editable="true">10 2
15 150 12 36 45 78 85 95 93 14</pre>

<p><code>razboi.out</code></p>

<pre contenteditable="true" editable="true">9 1</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("razboi.in");
ofstream cout("razboi.out");

int sd(int n)
{
    int s=0;
    for(int i = 1 ; i * i <= n ; ++i)
    {
        if(n%i==0)
        s+=i+n/i;
        if(i*i==n)
        s-=i;
    }
    return s;
}

int main()
{
    int n;
    cin >> n;
    int c;
    cin >> c;
    if(c==1)
    {
        int x , cnt=0;
        for(int i = 0 ; i < n ; ++i)
        {
            cin >> x;
            if(x%2==0)
            cnt++;
        }
        cout << cnt << ' ' << n-cnt;
    }
    if(c==2)
    {
        int x , cnt=0;
        for(int i = 0 ; i < n ; ++i)
        {
            cin >> x;
            if(sd(x)%2==0)
            cnt++;
        }
        cout << cnt << ' ' << n-cnt;
    }
    return 0;    
}</pre>
