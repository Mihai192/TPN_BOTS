<h1>Cerința</h1>

<p>Se dă harta unui lac de formă dreptunghiulară, împărțit în <code>n*m</code> zone dispuse sub forma unei matrice cu <code>n</code> linii și <code>m</code> coloane. Zonele pot fi acoperite cu apă, sau pot fi zone de uscat. Zonele de uscat care sunt învecinate pe linie sau pe coloană formează insule sau peninsule. Peninsule conțin cel puțin o zonă de uscat pe marginea lacului (matricei), în timp ce insulele sunt situate în întregime în  interiorul lacului.</p>

<p>Cunoscând harta lacului, determinați numărul de insule și numărul de peninsule.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>lac.in</code> conține pe prima linie numerele <code>n m</code>. Următoarele <code>n</code> linii conțin câte <code>m</code> valori <code>0</code> sau <code>1</code>, separate prin câte un spațiu. Valoarea <code>0</code> reprezintă o zonă acoperită cu apă, iar valoarea <code>1</code> reprezintă o zonă acoperită cu uscat.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>lac.out</code> va conține pe prima două numerele <code>nrI nrP</code>, separate prin exact un spațiu, reprezentând numărul de insule, respectiv numărul de peninsule existente pe lac.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n , m ≤ 1000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("lac.in");
ofstream cout("lac.out");
const int di[]={-1 , 0 , 1 , 0};
const int dj[]={0 , 1 , 0 , -1};
int n , m , a[1002][1002] , x[1000001] , y[1000001];
int ip , jp , is , js;///coordonatele

bool inside(int i , int j)
{
    return i>=1 &amp;&amp; i<=n &amp;&amp; j>=1 &amp;&amp; j<=m;
}
void lee(int ip , int jp)
{
    int st = 1 , dr = 1;
    a[ip][jp] = 2;
    x[1]=ip;
    y[1]=jp;
    while(st <= dr)
    {
        int i= x[st] , j = y[st];
        for(int k = 0 ; k < 4 ; k++)
        {
            int ii = i+di[k];
            int jj = j+dj[k];
            if(inside(ii , jj) &amp;&amp; a[ii][jj]==1)
            {
                dr++;
                x[dr]=ii;
                y[dr]=jj;
                a[ii][jj]=2;
            }
        }
        st++;
    }
}
int main()
{
    int p = 0 , k = 0;
    cin >> n >> m;
    for(int i = 1 ; i <=n ; ++i)
        for(int j = 1 ; j <=m ; ++j)
            cin >> a[i][j];
    for(int i = 1 ; i <=n ; ++i)
    {
        if(a[i][1]==1){p++;lee(i,1);}
        if(a[i][m]==1){p++;lee(i,m);}
    }
    for(int i = 1 ; i <=m ; ++i)
    {
        if(a[1][i]==1){p++;lee(1,i);}
        if(a[n][i]==1){p++;lee(n,i);}
    }
    for(int i = 2 ; i <n ; ++i)
    {
        for(int j = 2 ; j <m ; ++j)
        {
            if(a[i][j]==1)
            {
                k++;
                lee(i,j);
            }
        }
    }
    cout << k << " "<<p;
    return 0;
}</pre>
