<h1>Cerința</h1>


<p>Să se scrie un program care citește un număr natural <code>k</code> și afișează cel mai mic număr natural <code>n</code> mai mare decât <code>1</code>, care nu este divizibil cu primele <code>k</code> numere prime și nu este prim.</p>

<p>Fișierul de intrare <code>kprim.in</code> conține pe prima linie numărul <code>k</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>kprim.out</code> va conține pe prima linie numărul <code>n</code>, reprezentând cel mai mic număr natural <code>n</code>,care nu este divizibil cu primele <code>k</code> numere prime și nu este prim.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ k ≤ 10000</code></li>
</ul>

<p><code>kprim.in</code></p>

<pre contenteditable="true" editable="true">3</pre>

<h3>Explicație</h3>

<p>Primele <code>3</code> numere prime sunt <code>2</code>, <code>3</code>, <code>5</code>. </p>

<p>Numerele care nu sunt divizibile cu <code>2</code>, <code>3</code> sau <code>5</code> sunt : <code>7</code>, <code>11</code>, <code>13</code>, <code>17</code>, <code>19</code>, <code>23</code>, <code>31</code>, <code>37</code>, <code>41</code>, <code>47</code>, <code>49</code>, … </p>

<p><code>49</code> este cel mai mic număr care nu este prim.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("kprim.in");
ofstream fout("kprim.out");
unsigned long long prim (unsigned long long n)
{
    if (n==0 || n==1) return 0;
    if(n==2) return 1;
    if(n%2==0) return 0;
    for (unsigned int i=3;i*i<=n;i += 2)
        if(n%i==0) return 0;
    return 1;
}
int main()
{
     unsigned long long n,x,r;
     fin>>n;
     if(n==1) fout<<9;
     else
     {x=3;
     while(n>0)
     {
        if(prim(x))
        {
            r=x;
            n--;
        }
        x++;
     }
     unsigned long long p;
     p=r*r;
     fout<<p;
     }
    return 0;
}</pre>
