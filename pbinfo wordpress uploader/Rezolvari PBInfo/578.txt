<h1>Cerința</h1>

<p>Se dă un vector cu <code>n</code> elemente, numere naturale. Determinați câte secvențe ale vectorului au toate elementele egale.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, apoi cele <code>n</code> elemente ale vectorului.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>C</code>, reprezentând valoarea cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li>elementele vectorului vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Cele 8 secvențe sunt: <code>(4)</code>, <code>(3)</code>, <code>(3 3)</code>, <code>(3 3 3)</code>, <code>(3 )</code>, <code>(3 3)</code>, <code>(3)</code>, <code>(5)</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
long long v[100005];
int main()
{   
    long long n,nrsecv=0,c;  
    cin >> n;  
    for (int i = 1; i<=n; i++)    
    cin >> v[i];   
    for (int i = 1; i<=n; i++)  
    {        
        c = 1;     
        while (v[i] == v[i+1] &amp;&amp; i+1<=n)    
        {        
            c++;    
            i++;     
        }     
        nrsecv+=c*(c+1)/2;  
    }  
    cout << nrsecv;
}</pre>
