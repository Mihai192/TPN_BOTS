<h1>Cerința</h1>

<p>Alin este un elev pasionat de înmulțirea numerelor. Știe că produsul a două sau mai multe numere naturale poate avea ultima cifră <b>0</b>.</p>

<p>Se dau <code>n</code> numere naturale nenule. Ajutați-l pe Alin să determine:<br>
a.	numărul de cifre <code>0</code>  de la sfârşitul produsului celor <code>n</code> numere; <br>
b.	care este ultima cifră nenulă a acestui produs. </p>


<h1>Date de intrare</h1>

<p>De pe prima linie a fișierului <code>cifre15.in</code> se citește un număr natural <code>n</code> iar de pe următoarea linie din fișier se citesc <code>n</code> numere naturale nenule, separate prin câte un spaţiu.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>cifre15.out</code> conține o singură linie pe care vor fi scrise cele două valori determinate, separate printr-un singur spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare au cel mult <code>9</code> cifre.</li>
</ul>

<h3>Explicație</h3>

<p><code>50∙12∙18=10800</code>, la finalul produsului sunt <code>2</code> cifre <code>0</code>, iar ultima cifră nenulă este <code>8</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;
ifstream fin("cifre15.in");
ofstream fout("cifre15.out");

int main()
{
    int n;
    fin >> n;
    int d2 = 0, d5 = 0, p = 1 , m , k;
    for(int i = 1 ; i<=n ; ++i)
    {
        fin >> m;
        while(m % 5 == 0)
            m /= 5, d5++;
        while(m % 2 == 0)
            m /= 2, d2++;
        p = p * (m%10) % 10;
    }
    fout << min(d2,d5)<<" ";
    if(d2 > d5)
    {
        k = d2 - d5;
        for(int i=1;i<=k;++i)
            p = (p*2) % 10;
    }
    else if(d5 >= d2)
    {
        k = d5 - d2;
        for(int i=1;i<=k;++i)
            p = (p*5) % 10;
    }
    fout << p;

    fin.close();
    fout.close();
    
    return 0;
}</pre>
