<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Să se afle pentru câte dintre numere, suma cifrelor împărţită la <code>9</code> dă restul <code>r</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> şi <code>r</code>, iar apoi cele <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul de numere citite pentru care suma cifrelor dă restul <code>r</code> la împărţirea cu <code>9</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 200.000</code></li>
	<li><code>0 ≤ r ≤ 8</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>10</code><sup><code>18</code></sup></li>
</ul>

<h3>Explicație</h3>

<p>Se dă <code>n=3</code> şi <code>r=2</code>. S-au citit <code>3</code> numere : <code>11</code> are suma cifrelor <code>2</code> şi dă restul <code>2</code> la împărţirea cu <code>9</code>, <code>200</code> are suma cifrelor <code>2</code> şi dă restul <code>2</code> la împărţirea cu <code>9</code>, iar <code>3310</code> are suma cifrelor <code>7</code>, care dă restul <code>7</code> la împărţirea cu <code>9</code>. Deci două numere au proprietatea cerută.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n, r;
    cin >> n >> r;
    int cnt = 0;
    for(int i = 1; i <= n; ++i){
        long long x;
        cin >> x;
        if(x % 9 == r)
            cnt++;
    }
    cout << cnt;
    return 0;
}</pre>
