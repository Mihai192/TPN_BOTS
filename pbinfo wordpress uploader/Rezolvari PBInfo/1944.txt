<p>La ultima oră de matematică, Ionel a învățat despre <strong>numere speciale</strong>. Acestea sunt numere naturale cu număr impar de cifre care au prima cifră egală cu ultima. Ionel a primit ca temă să analizeze un șir format din numere având număr impar de cifre. El trebuie să determine suma cifrelor din mijloc, de la numerele speciale care se găsesc în șirul dat.</p>

<h1>Cerința</h1>

<p>Se citește numărul natural <code>n</code> și apoi se citesc <code>n</code> numere naturale având fiecare număr impar de cifre. Să se calculeze suma cifrelor din mijlocul numerelor speciale din șirul dat.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>suma6.in</code> conține pe prima linie numărul natural <code>n</code>. Pe a doua linie în fișier, se găsesc separate prin câte un spațiu <code>n</code> numere naturale, fiecare având număr impar de cifre.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>suma6.out</code> va conține pe prima linie un număr ce reprezintă suma cifrelor din mijlocul numerelor speciale din șirul dat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>cele <code>n</code> numere au cel mult nouă cifre</li>
	<li>pentru 30% dintre teste cele <code>n</code> numere au exact trei cifre</li>
</ul>

ifstream fin("suma6.in");

{
    int aux = n , cnt = 0 , p = 0;
    while(n != 0)
    {
        cnt++;
        p = n % 10;
        n /= 10;
    }
    if(cnt % 2 == 1 &amp;&amp; p == aux % 10) return 1;
    else return 0;
}
int nrcif(int n)
{
    int cnt = 0;
    while(n != 0)
    {
        cnt++;
        n /= 10;
    }
    return cnt;
}
int mij(int n)
{
    int cnt = nrcif(n) , p = 1;
    for(int i = 1 ; i <= cnt/2 ; ++i) p *=10;
    int cmij = (n / p) % 10;
    return cmij;
}
int main()
{
    int n , x , s = 0;
    fin >> n;
    for(int i = 0 ; i < n; ++i)
    {
        fin >>x;
