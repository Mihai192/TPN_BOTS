<h1>Cerința</h1>

<p>Să se scrie o funcție C++ care are ca parametri două numere naturale <code>n</code> și <code>m</code> și o matrice <code>A(n , m)</code> avȃnd elemente numere întregi și returnează numărul de elemente „șa” din matrice. Un element <code>A(i,j)</code> din matrice se numește element „șa” dacă este maximul de pe coloana <code>j</code> si minimul de pe linia <code>i</code> sau invers.</p>

<p>Subprogramul se va numi <code>nr_sa</code> va avea trei parametri: <code>a</code>, un tablou bidimensional cu elemente întregi, <code>n</code>, <code>m</code>, reprezentând numărul de linii, respectiv de coloane ale tabloului <code>a</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n,m ≤ 100</code></li>
	<li>numele subprogramului cerut este <code>nr_sa</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code>, <code>m</code></li>
	<li>indicii de linie și de coloana ai tabloului <code>a</code> încep de la <code>0</code> și sunt de tip <code>int</code>;</li>
	<li>matricea folosită în apelul din funcția  <code>main</code> va fi declarată <code>int a[100][100];</code> și are elementele indexate de la <code>0</code>.</li>
</ul>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int nr_sa(int a[][100], int n, int m){
    int minlin[100], maxlin[100], mincol[100], maxcol[100];
    for(int i = 0; i < n; ++i){
        int mini = 2000000001;
        int maxi = 0;
        for(int j = 0; j < m; ++j){
            if(a[i][j] < mini)
                mini = a[i][j];
            if(a[i][j] > maxi)
                maxi = a[i][j];
        }
        minlin[i] = mini;
        maxlin[i] = maxi;
    }
    for(int j = 0; j < m; ++j){
        int mini = 2000000001;
        int maxi = 0;
        for(int i = 0; i < n; ++i){
            if(a[i][j] < mini)
                mini = a[i][j];
            if(a[i][j] > maxi)
                maxi = a[i][j];
        }
        maxcol[j] = maxi;
        mincol[j] = mini;
    }
    int cnt = 0;
    for(int i = 0; i < n; ++i)
        for(int j = 0; j < m; ++j)
            if((a[i][j] == maxlin[i] &amp;&amp; a[i][j] == mincol[j]) ||
               (a[i][j] == minlin[i] &amp;&amp; a[i][j] == maxcol[j]))
                cnt++;
    return cnt;
}</pre>
