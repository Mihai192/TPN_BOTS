<h1>Cerința</h1>

<p>Scrieţi definiţia completă a subprogramului C/C++ <code>modificare</code>, care are doi parametri, <code>v</code> şi <code>n</code>, prin care primeşte un tablou unidimensional cu maximum <code>10000</code> de numere naturale nenule şi, respectiv, numărul de elemente din tablou. Subprogramul rearanjează elementele tabloului astfel încât toate valorile prime să se afle pe primele poziţii, iar valorile care nu sunt prime, în continuarea celor prime. <br>
<strong>Ordinea în cadrul ambelor secvenţe (numerele prime şi cele care nu sunt prime), poate fi oricare.</strong> </p>

<p>Pentru <code>n=10</code> şi <code>v=(42,36,7,53,27,15,29,40,39,2)</code>, după apel, <code>v</code> ar putea fi <code>(7,53,29,2,27,15,42,40,39,36)</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void modificare(int v[], int n)
{
    int i, j, k1 = 0, k2 = 0, x, v1[10005], v2[10005];
    bool prim;
    for(j = 0; j < n; j ++)
     {
         x = v[j];
         prim = true;
         for (i = 2; i * i <= x; i ++)
          if(x % i == 0)
           {
               prim = false;
               break;
           }
          if(prim) v1[k1 ++] = x;
              else v2[k2 ++] = x;
     }
     for(j = 0; j < k1; j ++)
      v[j] = v1[j];
     for(j = k1; j < n; j ++)
      v[j] = v2[j - k1];
}</pre>
