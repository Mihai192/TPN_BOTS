<h1>Cerința</h1>

<p>Se dă un șir <code>a</code><sub><code>1</code></sub>, <code>a</code><sub><code>2</code></sub>, …, <code>a</code><sub><code>n</code></sub> format din <code>n</code> numere naturale. Determinați numărul de perechi de elemente din șir <code>(a</code><sub><code>i</code></sub><code>,a</code><sub><code>j</code></sub><code>)</code> cu <code>i &lt; j</code>, care au suma număr prim.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul rezultatelor care sunt numere prime.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 100.000</code></li>
	<li>elementele șirului vor fi mai mici decât <code>10.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Sumele obţinute sunt <code>2+5=7 , 2+9=11 , 5+9=14</code>. Dintre rezultatele obţinute, două sunt prime, <code>7</code> şi <code>11</code>. </p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    std::ios::sync_with_stdio(false);
    int n , E[20001]={0} , max = 20000;
    unsigned long long c = 0,F[10001]={0};
    for(int i = 2 ; i <= max ; i++)
        E[i] = 1;
    for(int i = 2 ; i * i <= max; i++)
        if(E[i] == 1)
            for(int j = i * i ; j <= max ; j= j + i)
                E[j] = 0;
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        int x;
        cin >> x;
        F[x]++;
    }
    c += F[1]*(F[1]-1)/2;
    c += F[0]*F[2];
    for(int i = 1 ; i <= 10000 ; i = i + 2)
        if(F[i] != 0)
            for(int j = i + 1 ; j <= 10000 ; j = j + 2)
                if(F[j] != 0)
                    if(E[i+j] == 1) c += F[i]*F[j];
    for(int i = 0 ; i <= 10000 ; i = i + 2)
        if(F[i] != 0)
            for(int j = i + 1 ; j <= 10000 ; j = j + 2)
                if(F[j] != 0)
                    if(E[i+j] == 1) c += F[i]*F[j];
    cout<<c;
    return 0;
}</pre>
