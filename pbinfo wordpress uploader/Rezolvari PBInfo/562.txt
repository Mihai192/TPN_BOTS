<h1>Cerința</h1>

<p>Fast-food-ul de la colțul străzii are  <code>n</code> clienți. Pentru fiecare client se cunoaște momentul în care intră în fast-food și momentul în care iese, ambele fiind exprimate în minute, numărate de la începutul perioadei de funcționare. Gigel, patronul acestui fast-food, dorește să afle perioada totală în care în local nu a fost nici un client, știind că acesta se deschide la momentul <code>t1</code> și se închide la momentul <code>t2</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n t1 t2</code>, iar apoi <code>n</code> perechi de numere naturale <code>x y</code> , reprezentând montul intrării, respectiv momentul ieșirii fiecărui client din restaurant.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>P</code>, reprezentând valoarea cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li><code>1 ≤ t1 &lt; t2 ≤ 10.000</code></li>
	<li>pentru fiecare client, <code>t1 ≤ x &lt; y ≤ t2</code></li>
</ul>

<h1>Explicație</h1>

<p>Intervalele în care nu este nici un client în fast-food sunt: <code>2-5</code>, <code>9-10</code>, <code>13-15</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int a[10001];
int main()
{
    int n , t1 , t2 , x , y , cnt = -1;
    cin >> n >> t1 >> t2;
    for(int i = 1; i <= n ; i++)
      {
         cin >> x >> y;
         a[x]++;
         a[y]--;
      }
    for(int i = t1 ; i <= t2 ; ++i)
            a[i]+=a[i-1];
    for(int i = t1 ; i <= t2 ; ++i)
        if(a[i]==0) cnt++;
    cout << cnt;
    return 0;
}</pre>
