<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a subprogramului C++ <code>sub</code> care are <code>3</code> parametri: <code>n</code> – prin care primește un număr natural, <code>v</code>, prin care primeşte un tablou unidimensional cu <code>n</code> elemente, numere naturale cu cel mult <code>4</code> cifre și <code>x</code>, prin care primeşte un număr natural. Cel puțin un element al tabloului <code>v</code> are valoarea <code>x</code>.</p>

<p>Subprogramul modifică ordinea valorilor din tablou, astfel încât toate valorile egale cu <code>x</code> să ocupe primele poziţii din <code>v</code>, iar celelalte valori să se regăsească în continuarea acestora, în ordinea inițială. Tabloul modificat este furnizat tot prin parametrul <code>v</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>2&lt;n&lt;50</code></li>
	<li>numele subprogramului cerut este <code>sub</code></li>
	<li>parametrii sunt, în această ordine: <code>n</code>, <code>v</code>, <code>x</code></li>
	<li>elementele tabloului <code>v</code> sunt indexate de la zero</li>
</ul>

<p>Dacă <code>n=9</code>, <code>v=(2, 1, 0, 1, 7, 0, 1, 4, 5)</code> şi <code>x=1</code>, atunci, după apel, <code>v=(1, 1, 1, 2, 0, 7, 0, 4, 5)</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void sub(int n, int v[], int x){
    int a[61], ind = 0;
    for(int i = 0; i < n; ++i)
        if(v[i] == x)
            a[++ind] = x;
    for(int i = 0; i < n; ++i)
        if(v[i] != x)
            a[++ind] = v[i];
    for(int i = 1; i <= ind; ++i)
        v[i-1] = a[i];
}</pre>
