<p>Se consideră o secvență de <code>N</code> numere naturale nenule.</p>

<h1>Cerința</h1>

<p>Determinați numărul de divizori pentru fiecare număr din secvența dată.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>nrdiv.in</code> conține pe prima linie numărul natural <code>N</code>, care reprezintă numărul de valori din secvență. Pe următoarele <code>N</code> linii se află cele <code>N</code> numere naturale din secvență, câte un număr pe o linie.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>nrdiv.out</code> va conține <code>N</code> linii. Pe linia <code>i</code> va fi scris numărul de divizori ai celui de-al <code>i</code>-lea număr din secvență (considerând ordinea din fișierul de intrare).</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ N ≤ 50</code></li>
	<li><code>1 ≤ numerele din secvență ≤ 10</code><sup><code>13</code></sup></li>
</ul>

<h3>Explicație</h3>

<p><code>13</code> are doi divizori (<code>1</code> şi <code>13</code>)<br>
<code>1</code> are un divizor (<code>1</code>)<br>
<code>24</code> are <code>8</code> divizori (<code>1</code>, <code>2</code>, <code>3</code>, <code>4</code>, <code>6</code>, <code>8</code>, <code>12</code>, <code>24</code>).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("nrdiv.in");
ofstream cout("nrdiv.out");

unsigned long long n , x;

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> x;
        unsigned long long cnt = 1 , d = 2;
        while(x > 1)
        {
            unsigned long long p = 0;
            while(x % d == 0) p++ , x /= d;
            if(p) cnt *= (p + 1);
            d++;
            if(d * d > x) d = x;
        }
        cout << cnt << '\n';
    }
    return 0;
}</pre>
