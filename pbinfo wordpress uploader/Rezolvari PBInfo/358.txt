<h1>Cerinţa</h1>

<p>De-a lungul principalei străzi din orașul nostru există <code>n</code> plopi, pentru fiecare cunoscându-se înălțimea. Primarul orașului dorește să știe care este diferența de înălțime maximă între doi plopi învecinați, și câte perechi de plopi învecinați au această diferență de înălțime.</p>

<p>Ajutați-l pe primar să afle răspunsul la întrebare.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul de plopi <code>n</code>, iar apoi <code>n</code> numere naturale nenule, separate prin spaţii, reprezentând înălțimile plopilor.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numerele <code>D</code> și <code>nrp</code>, separate prin exact un spațiu, reprezentând diferența de înălțime maximă între doi plopi învecinați, și câte perechi de plopi învecinați au această diferență de înălțime.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 1000</code></li>
	<li>înălțimile plopilor vor fi mai mici decât <code>5000</code></li>
</ul>

<h1>Explicație</h1>

<p>S-au citit înălțimile a <code>7</code> plopi. Diferența maximă de înălțime este <code>4</code>, și sunt <code>3</code> perechi de plopi învecinați care au această diferență de înălțime, și anume cei cu înălțimile <code>3 7</code>, <code>6 2</code> și <code>2 6</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

{
    int n,max=0,cnt=0;
    cin >> n;
    int a[1002];
    for (int i = 1 ; i <= n; ++i)
    {
        cin >> a[i];
        a[0]=a[1];
        if (a[i]-a[i-1] < 0)
        {
            if (a[i-1]-a[i]> max)
            max=a[i-1]-a[i];
        }
        else
        {
            if (a[i]-a[i-1]>max)
            max=a[i]-a[i-1];
        }
    }
    for (int i = 1 ; i <= n; ++i)
    {
        if (a[i]-a[i-1] < 0)
        {
            if (a[i-1]-a[i]== max)
            cnt++;
        }
        else
        {
            if (a[i]-a[i-1]==max)
            cnt++;
        }
    }
    cout << max << ' ' << cnt;
}</pre>
