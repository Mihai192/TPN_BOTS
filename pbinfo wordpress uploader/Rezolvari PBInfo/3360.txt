<h1>Cerința</h1>

<p>Se dă un șir de <code>n</code> numere naturale. Determinați o pereche de numere consecutive în șir, cu proprietatea că au același număr de factori primi. Dacă sunt mai multe asemenea perechi se va determina aceea cu suma numerelor mai mică. Dacă în continuare sunt mai multe perechi, se va determina prima din șir.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran două numere <code>a b</code>, separate prin câte un spațiu, reprezentând perechea determinată. Ordinea numerelor <code>a b</code>  este cea din șir.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>numerele din șir vor fi mai mici decât <code>1.000.000.000</code></li>
	<li>dacă nu există nicio pereche care respectă cerința se va afișa <code>NU EXISTA</code></li>
</ul>

<h3>Explicație</h3>

<p>Perechea <code>40 63</code> respectă regula, dar are suma mai mare (<code>40+63 &gt; 64 + 27</code>).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int desc(int n){
    int cnt = 0, d = 2;
    while(n > 1){
        int p = 0;
        while(n % d == 0)
            p++, n/=d;
        if(p)
            cnt++;
        d++;
        if(d * d > n)
            d = n;
    }
    return cnt;
}

bool ok(int a, int b){
    return desc(a) == desc(b);
}

int main(){
    int n, a, b, rez1 = 1000000000, rez2 = 1000000000;
    cin >> n;
    cin >> a;
    for(int i = 2; i <= n; ++i){
        cin >> b;
        if(ok(a, b) &amp;&amp; a + b < rez1 + rez2)
            rez1 = a, rez2 = b;
        a = b;
    }
    if(rez1 != 1000000000)
        cout << rez1 << ' ' << rez2;
    else
        cout << "NU EXISTA";
    return 0;
}</pre>
