<p>Un număr natural este perfect dacă dublul său este egal cu suma divizorilor săi.</p>

<h1>Cerinţa</h1>

<p>Să se scrie un program care verifică dacă un număr natural citit de la tastatură este perfect.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code></p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran mesajul <code>n este perfect</code>, dacă <code>n</code> este număr perfect, respectiv <code>n nu este perfect</code>, în caz contrar.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt; 2.000.000.000</code></li>
</ul>


<h1>Exemplu1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">6</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">6 este perfect</pre>

<h1>Explicație</h1>

<p>Divizorii lui <code>6</code> sunt <code>1</code>, <code>2</code>, <code>3</code>, <code>6</code>. Suma lor este <code>12</code>, iar <code>2*6=12</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

using namespace std;


int main()
{
    int n;
    cin >> n;
    int cnt=0;

    long long sumaDiv = 0;
    for (int d = 1; d * d <= n; ++d)
        if (n % d == 0)
        {
            sumaDiv += d;
            if (d * d != n)
                {sumaDiv += n / d;
                }
        }
        if (sumaDiv == 2*n)
        cout << n << " este perfect";
        else
        cout << n << " nu este perfect";
    return 0;
}</pre>
