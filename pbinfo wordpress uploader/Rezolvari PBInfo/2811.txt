<h1>Cerința</h1>

<p>Subprogramul <code>triplete</code> are trei parametri, <code>a</code>, <code>b</code> și <code>c</code>, în această ordine, prin care primește câte un număr natural din intervalul <code>[0,10</code><sup><code>4</code></sup><code>]</code> (<code>a&lt;b</code>). Subprogramul afișează pe ecran în ordine lexicografică toate tripletele de numere naturale, <code>x</code>, <code>y</code> și <code>z</code>, soluții ale ecuației <code>x+y+z=c</code>, cu proprietatea că <code>a≤ x≤ y≤ z≤ b</code>.</p>

<p>Fiecare triplet afișat pe câte o linie și este încadrat între acolade, iar numerele <code>x</code>, <code>y</code> și <code>z</code> sunt afișate în această ordine, separate prin câte o virgulă. Dacă nu există niciun triplet cu proprietatea cerută, se afişează mesajul <code>nu exista</code>.</p>

<p>Dacă <code>a=2</code>, <code>b=4</code> și <code>c=8</code>, se afișează pe ecran tripletele:</p>

<pre contenteditable="true" editable="true">{2,2,4}

<p>Dacă <code>a=5</code>, <code>b=8</code>, <code>c=8</code> se afișează pe ecran mesajul <code>nu exista</code>.</p>

<h1>Important</h1>

<p>Soluția propusă va conține definiția subprogramului cerut. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void triplete(int a , int b , int c)
{
    int cnt = 0;
    for(int i = a ; i <= b ; ++i)
        for(int j = i ; j <= b ; ++j)
                if(c - (i + j) >= j &amp;&amp; c - (i + j) <= b)
                    cout << '{' << i << ',' << j << ',' << c - (i + j) << '}' << endl, cnt++;
    if(cnt==0)
        cout << "nu exista";
}</pre>
