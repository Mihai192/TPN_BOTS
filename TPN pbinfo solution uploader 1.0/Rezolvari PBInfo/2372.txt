<h1>Cerința</h1>

<p>Se citește un număr natural <code>n</code>. Afișați toate tripletele de numere naturale <code>x</code>, <code>y</code>, <code>z</code> (<code>1&lt;x&lt;y&lt;z&lt;n</code>) care sunt divizori ai lui <code>n</code> și care au proprietatea că <code>x+y+z=n</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran pe rânduri separate câte un triplet conform cerinței, numerele din triplet fiind in ordine strict crescatoare și separate prin cate un spațiu. Dacă nu există nici un astfel de triplet, atunci se va afișa pe ecran <code>nu exista</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000000000</code></li>
</ul>

<h1>Exemple:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">24</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">4 8 12</pre>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">20</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">nu exista</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    long long int n;
    cin >> n;
    if(n%6==0)
        cout << 1LL * n/6 << ' ' << 1LL * n/3 << ' ' << 1LL* n/2 << endl;
    else cout << "nu exista";
}</pre>
