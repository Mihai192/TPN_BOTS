<h1>Cerința</h1>

<p>Subprogramul <code>Impare</code> are un singur parametru, <code>n</code>, prin care primește un număr natural. Subprogramul înlocuiește fiecare cifră impară a lui <code>n</code> cu cea mai mare cifră pară strict mai mică decât ea (astfel cifra <code>1</code> se înlocuieşte cu cifra <code>0</code>, cifra <code>3</code> cu cifra <code>2</code> etc.) și furnizează numărul obținut tot prin parametrul <code>n</code>. </p>

<p>Scrieți definiția completă a subprogramului.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000.000</code></li>
	<li>Numele funcției va fi <code>Impare</code></li>
	<li>Funcția va primi un singur parametru, numărul <code>n</code>, iar la finalul funcției <code>n</code> va avea valoarea cerută.</li>
</ul>

<p>Dacă <code>n=235690</code>, atunci, după apel, <code>n=224680</code>, iar dacă <code>n=15690</code>, atunci, după apel, <code>n=4680</code>.</p>

<h1>Important</h1>

<p>Soluția propusă va conține doar definiția funcției cerute. Prezența în soluție a altor instrucțiuni poate duce la erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void Impare(int &amp;n)
{
    int a[11], x = 0;
    while(n)
    {
        a[x] = n % 10;
        n/=10;
        x++;
    }
    int p = 1;
    for(int i = 0 ; i < x; ++i)
    {
        if(a[i] % 2 == 1)
            a[i] --;
        n = n + a[i] * p;
        p *= 10;
    }
}</pre>
