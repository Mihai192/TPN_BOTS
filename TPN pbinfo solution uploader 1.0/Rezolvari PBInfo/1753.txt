<h1>Cerința</h1>

<p>La un concurs de programare s-au înscris <code>n</code> elevi. Concursul se desfăşoară pe două secţiuni, secţiunea <code>1</code> pentru începători şi secţiunea <code>2</code> avansaţi. Proba de concurs se desfăşoară pe parcursul a <code>3</code> ore şi elevii au de rezolvat <code>2</code> probleme. <br>
Fiecare problemă poate avea punctajul minim de <code>0</code> puncte şi punctajul maxim de <code>100</code> de puncte. Punctajul final al concurentului este format din suma punctajelor celor două probleme. Să se scrie un program care citeşte numărul de elevi înscrişi şi apoi date despre fiecare elev înscris (secţiunea la care s-a înscris, punctajul obţinut pentru prima problema şi punctajul obţinut pentru a două problemă) și rezolvă următoarele cerinţe:</p>

<p>1.  Afișează mesajul <code>DA</code> dacă toţi cei <code>N</code> elevi înscrişi au reuşit să obţină un punctaj nenul la ambele probleme propuse, respectiv <code>NU</code>” în caz contrar.<br>
2. Afişează  pentru  fiecare  secţiune  numărul  de  elevi  înscrişi.  Afişarea  se  va  face în ordinea crescătoare a numărului secţiunii, prin perechi de numere de forma <code>nr_secţiune nr_elevi</code>.<br>
3. Afişaţi pentru fiecare secţiune punctajul maxim obţinut şi numărul de elevi care au obţinut acest punctaj. Afişarea se va face în ordinea crescătoare a numărului secţiunii, prin triplete de numere de forma <code>nr_secţiune punctaj_maxim nr_elevi</code>. Ştiind ca premiile se acordă doar celor care au luat punctaj maxim, afişaţi şi numărul de premii care vor fi acordate.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>concurs2.in</code> conţine pe prima linie un număr natural <code>C</code>. Pentru toate testele de intrare, numărul <code>C</code> are una din valorile <code>1</code>, <code>2</code> sau <code>3</code>. Pe linia a doua a fișierului se găseşte numărul natural <code>N</code> – numărul de elevi înscrişi  iar pe următoarele <code>N</code> linii, separate prin spaţiu,trei numere <code>S P1 P2</code> cu semnificaţia: <code>S</code> secţiunea la care participă elevul, <code>P1</code> punctajul obţinut la prima problemă şi <code>P2</code> punctajul obţinut la a doua problemă.</p>

<h1>Date de ieșire</h1>

<p>Dacă <code>C=1</code>, se va rezolva numai punctul 1). În acest caz, în fişierul de ieşire <code>concurs2.out</code> se va scrie un singur mesaj <code>DA</code> sau <code>NU</code>.</p>

<p>Dacă <code>C=2</code>, se va rezolva numai punctul 2). În acest caz, în fişierul de ieşire <br>
<code>concurs2.out</code> se vor scrie <code>2</code> linii, pe fiecare linie o pereche de numere, separate prin spaţiu reprezentând valorile cerinţei 2.</p>

<p>Dacă <code>C=3</code>, se va rezolva numai punctul 3). În acest caz, în fişierul de ieşire <br>
<code>concurs2.out</code> se vor scrie <code>3</code> linii, pe primele două linii câte un triplet de numere separate prin spaţiu reprezentând valorile cerinţei 3 iar pe ultima linie numărul de premii.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 &lt;= C &lt;= 3</code>,</li>
	<li><code>1 &lt;= S &lt;= 2</code></li>
	<li><code>0 &lt; n &lt;= 300</code></li>
	<li><code>0 &lt;= P1,P2 &lt;= 100</code></li>
</ul>

<h3>Explicație</h3>

<p>La secţiunea <code>1</code> punctajul maxim (<code>200</code>) a fost obţinut de <code>3</code> elevi. La secţiunea <code>2</code> punctajul maxim (<code>200</code>) a fost obţinut de <code>1</code> elev. Număr de premii <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("concurs2.in");
ofstream cout("concurs2.out");

int main()
{
    int c, n;
    cin >> c >> n;
    if(c==1)
    {
        int a , b , c;
        int cnt=0;
        for(int i = 0 ; i < n ; ++i)
        {
            cin >> a >> b >> c;
            if(b > 0 &amp;&amp; c > 0)
            cnt++;
        }
        if(cnt==n)
            cout << "DA";
        else
            cout << "NU";
    }
    if(c==2)
    {
        int a , b , c;
        int cnt1=0 , cnt2=0;
        for(int i = 0 ; i < n ; ++i)
        {
            cin >> a >> b >> c;
            if(a==1)
            cnt1++;
            if(a==2)
            cnt2++;
        }
        cout << 1 << ' ' << cnt1;
        cout << endl;
        cout << 2 << ' ' << cnt2;
    }
    if(c==3)
    {
        int a[500] , b[500] , c[500];
        int cnt1=0 , cnt2=0;
        int max1=0 , max2=0;
        for(int i = 0 ; i < n ; ++i)
        {
            cin >> a[i] >> b[i] >> c[i];
            if(a[i]==1)
            {
                if(b[i]+c[i]>max1)
                    max1=b[i]+c[i];
            }
            if(a[i]==2)
            {
                if(b[i]+c[i]>max2)
                    max2=b[i]+c[i];
            }
        }
        for(int i = 0 ; i < n ; ++i)
        {
            if(a[i]==1 &amp;&amp; b[i]+c[i]==max1)
                cnt1++;
            if(a[i]==2 &amp;&amp; b[i]+c[i]==max2)
                cnt2++;
        }
        cout << 1 << ' ' << max1 << ' ' << cnt1 << endl;
        cout << 2 << ' ' << max2 << ' ' << cnt2 << endl;
        cout << cnt1+cnt2;
    }
    return 0;    
}</pre>
