<p>Maria, Cristi şi Alex au găsit o modalitate de a-şi îmbunătăţi viteza de efectuare a operaţiilor matematice printr-un joc care să corespundă nivelului de vârsta al fiecăruia. Maria ştie doar operaţiile de adunare şi scădere, Cristi a învăţat înmulţirile iar Alex fiind în clasa a 5-a studiază divizibilitatea numerelor.</p>

<p>Jocul se desfăşoară în felul următor: Maria alege o cifra – cifra de start (întotdeauna este nenulă). Cristi o înmulţeşte cu <code>3</code>. La numărul obţinut de Cristi, Maria adaugă o nouă cifră şi îi spune lui Cristi suma obţinută. Cristi caută cel mai mare multiplu a lui <code>7</code> mai mic decât numărul obţinut de Maria şi îl spune Mariei. Aceasta scade din numărul ei multiplul spus de Cristi şi obţine un număr nou. Din acest moment jocul se reia, Cristi înmulţeşte cu <code>3</code>, Maria alege o cifră şi o adaugă la numărul obţinut de Cristi s.a.m.d…</p>

<p>Între timp Alex este atent la cifrele pe care Maria le-a introdus în joc şi caută să vadă dacă numărul format din aceste cifre este divizibil cu <code>7</code>.</p>

<h1>Cerința</h1>

<p>1) Aflaţi care este numărul obţinut de Maria după adăugarea ultimei cifre.<br>
2) Ajutaţi-l pe Alex să verifice dacă numărul format din cifrele alese de Maria este divizibil cu <code>7</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>operatiimatematice.in</code> conţine pe prima linie numărul de cifre alese de Maria iar pe următoarea linie separate prin spaţiu cifrele în ordinea alegerii acestora.</p>


<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>operatiimatematice.out</code> va conține pe prima linie numărul obţinut la cerința 1 şi pe a doua linie răspunsul <code>DA</code> în cazul în care răspunsul lui Alex trebuie să fie afirmativ, respectiv <code>NU</code> în caz contrar.</p>


<h1>Restricții și precizări</h1>

<ul>
	<li>Numărul de cifre alese de Maria este maxim <code>100</code>.</li>
	<li>Pentru rezolvarea corectă a primei cerinţe se obţine <code>50%</code> din punctajul unui test.</li>
</ul>

<h1>Exemplu 1:</h1>

<p><code>operatiimatematice.in</code></p>

<pre contenteditable="true" editable="true">4
2 3 4 5</pre>

<p><code>operatiimatematice.out</code></p>

<pre contenteditable="true" editable="true">14
DA</pre>

<h3>Explicație</h3>

<p>Maria alege <code>2</code> ca cifra de start.<br>
<code>2x3=6</code>; Cristi a înmulţit-o cu <code>3</code>.<br>
<code>6+3=9</code>; Maria a adăugat următoarea cifră şi Cristi a căutat cel mai mare multiplu a lui <code>7</code> mai mic decât <code>9</code><br>
<code>9-7=2</code>; Maria scade din numărul ei multiplul spus de Cristi şi se reia jocul<br>
<code>2x3=6</code>; <code>6+4=10</code>; <code>10-7=3</code>; <code>3x3=9</code>; <code>9+5=14</code>;<br>
Numărul format conform ordinii cifrelor este <code>2345</code> și este divizibil cu <code>7</code>.</p>

<h1>Exemplu 2:</h1>

<p><code>operatiimatematice.in</code></p>

<pre contenteditable="true" editable="true">3
3 4 5</pre>

<p><code>operatiimatematice.out</code></p>

<pre contenteditable="true" editable="true">23
NU</pre>

<h3>Explicație</h3>

<p><code>3x3=9</code>; <code>9+4=13</code>; <code>13-7=6</code>;<br>
<code>6x3=18</code>; <code>18+5=23</code>;<br>
Numărul <code>345</code> nu este divizibil cu <code>7</code></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("operatiimatematice.in");
ofstream cout("operatiimatematice.out");
unsigned long R = 0;
void impartire_pe_nr_mari(int A[], unsigned long X)
{
    int i;
    for(i = A[0] ; i ; i--)
    {
        A[i] = (R = 10 * R + A[i]) / X;
        R %= X;
    }
    while(!A[A[0]] &amp;&amp; A[0]>1) A[0]--;
}
int main()
{
    int n , a[100] , sum , b[100];
    cin >> n;
    for(int i = 0 ; i < n ; i++)
        cin >> a[i];
    for(int i = n - 1 ; i >= 0 ; i--)
        b[i] = a[n - i - 1];
    int rez = a[0];
    int i = 0;
    while(i < n -1)
    {
        i++;
        rez *= 3;
        rez += a[i];
        int aux = rez - 1;
        while(aux % 7 != 0) aux--;
        sum = aux;
        rez -= aux;
    }
    cout << rez + sum << '\n';
    if(rez + sum == 6) cout << "NU";
    else
    {
        impartire_pe_nr_mari(b , 7);
        if(R == 0) cout << "DA";
        else cout << "NU";
    }

}</pre>
