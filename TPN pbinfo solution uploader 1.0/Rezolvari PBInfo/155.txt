<h1>Cerinţa</h1>

<p>Scrieţi un program care citeşte de la tastatură un şir de cel mult 50 de caractere (litere mici şi mari ale alfabetului englez, cifre, puncte, virgule şi spaţii) şi afişează pe ecran cifra care apare de cele mai multe ori în şirul citit.</p>


<h1>Date de intrare</h1>

<p>Programul citeşte de la tastatură un şir de cel mult 50 de caractere.</p>

<h1>Date de ieşire</h1>

<p>Programul va afişa pe ecran cifra care apare de cele mai multe ori în şir.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>dacă şirul conţine mai multe cifre cu număr maxim de apariţii si va afişa cea mai mică</li>
	<li>dacă şirul nu conţine cifre se va afişa <code>NU</code>.</li>
</ul>

<p><strong>Explicaţie</strong></p>

<p>Cifrele <code>0</code> şi <code>1</code> apar de număr maxim de ori, dar <code>0</code> este mai mică.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int f[10];

int main()
{
    char ch[300];
    cin.getline(ch , 300);
    int i = 0;
    while(ch[i]!='\0')
    {
        if(ch[i]=='0')
            f[0]++;
        if(ch[i]=='1')
            f[1]++;
        if(ch[i]=='2')
            f[2]++;
        if(ch[i]=='3')
            f[3]++;
        if(ch[i]=='4')
            f[4]++;
        if(ch[i]=='5')
            f[5]++;
        if(ch[i]=='6')
            f[6]++;
        if(ch[i]=='7')
            f[7]++;
        if(ch[i]=='8')
            f[8]++;
        if(ch[i]=='9')
            f[9]++;  
        i++;  
    }
    int max=0 , val=-1;
    for(int i = 0 ; i <= 10 ; ++i)
    {
        if(f[i]>max)
            max=f[i] , val=i;
    }
    if(val==-1)
        cout << "NU";
    else
        cout << val;
}</pre>
