<p>Andrei a făcut într-o zi un șir de <code>N</code> numere. În a doua zi a lăsat în acel șir doar numerele prime. În a treia zi a calculat pentru fiecare număr rămas în șir suma cifrelor, iar apoi a adunat toate aceste sume în <code>S</code>. După ce a obținut numărul <code>S</code> a început să adune toate cifrele din care este format <code>S</code> și tot așa până când ajunge la o cifră terminală <code>C</code>.</p>

<h1>Cerința</h1>

<p>Ajutați-l pe Andrei să calculeze cifra terminală <code>C</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>vterminal.in</code> conține pe prima linie numărul <code>N</code>, iar pe a doua linie <code>N</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>vterminal.out</code> va conține pe prima linie numărul <code>C</code>, reprezentând valoarea cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici decât <code>1.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Doar <code>11</code>, <code>13</code> și <code>97</code> sunt prime, deci <code>11 + 13 + 97 = 121</code><br>
<code>1 + 2 + 1 = 4</code>, deci <code>4</code> este cifra terminală</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("vterminal.in");
ofstream fout("vterminal.out");
int prim(int n)
{
    if(n==0 || n==1) return 0;
    if(n==2) return 1;
    if(n%2==0) return 0;
    else
        for(int d=3;d*d<=n;d+=2)
    {
        if(n%d==0) return 0;
    }
    return 1;
}
int main()
{
    int n,x,s=0;
    fin>>n;
    for(int i=1;i<=n;i++)
        {
            fin>>x;
            if(prim(x)) s=s+x;
        }
    fout<<s%9;
    return 0;

}</pre>
