<h1>Cerința</h1>

<p>Se consideră un text format din cel mult <code>200</code> de caractere litere mici și spații. Cuvintele din text sunt separate prin unul sau mai multe spații. Scrieti un program care citește textul și care:<br>
<code>a)</code> determină numărul de consoane din textul citit; <br>
<code>b)</code> înlocuiește prima literă a fiecărui cuvânt din textul citit cu litera X (mare); <br>
<code>c)</code> modifică textul citit prin aranjarea în ordine lexicografică a tuturor cuvintelor din text și separarea lor prin câte un singur spațiu.</p>


<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>prosirx.in</code> conține pe prima linie textul.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>prosirx.out</code> va conține:<br>
- pe prima linie răspunsul la cerința <code>a)</code><br>
- pe a doua linie răspunsul la cerința <code>b)</code><br>
- pe a treia linie răspunsul la cerința <code>c)</code></p>

<h1>Restricții și precizări</h1>

<ul>
	<li>textul este format din cel mult <code>200</code> de caractere, doar litere mici sau spații</li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
#include <map>
using namespace std;

ifstream cin("prosirx.in");
ofstream cout("prosirx.out");

char a[200][200];

int main()
{
    char s[300];
    cin.getline(s , 300);
    int i = 0;
    int cnt = 0;
    while(s[i]!='\0')
    {
        if(s[i]!='a' &amp;&amp; s[i]!='A' &amp;&amp; s[i]!='e' &amp;&amp; s[i]!='E' &amp;&amp; s[i]!='i' &amp;&amp; s[i]!='I' &amp;&amp; s[i]!='o' &amp;&amp; s[i]!='O' &amp;&amp; s[i]!='u' &amp;&amp; s[i]!='U' &amp;&amp; s[i]!=' ')
            cnt++;
        i++;
    }
    cout << cnt << endl;
    i=0;
    while(s[i]!='\0')
    {
        if(i == 0 &amp;&amp; s[i]!=' ')
            cout << 'X';
        else if(s[i-1]==' ' &amp;&amp; s[i]!=' ')
            cout << 'X';
        else
            cout << s[i];
        i++;
    }
    s[i]=' ' , s[i+1]='\0';
    i = 0 , cnt = 0;
    int j = 0;
    while(s[i]!='\0')
    {
        if(s[i]!=' ')
        {
            a[cnt][j]=s[i];
            j++;
        }
        else
        {
            j=0;
            if(s[i+1]!=' ')
                cnt++;
        }
        i++;
    }
    cout << endl;
    for(int i = 0 ; i < cnt ; ++i)
        for(int j = i + 1 ; j < cnt ; ++j)
            if(strcmp(a[i] , a[j]) > 0)
                swap(a[i] , a[j]);
    for(int i = 0 ; i < cnt ; ++i)
        if(a[i][0]!=' ')
            cout << a[i] << ' ';
    return 0;
}</pre>
