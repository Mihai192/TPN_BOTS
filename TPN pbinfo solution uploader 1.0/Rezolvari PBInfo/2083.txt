<h1>Cerința</h1>

<p>Scrieți definiția completă a subprogramului C++ <code>ordon123</code> care are <code>2</code> parametri: <br>
</p><ul>
	<li><code>n</code> – prin care primește un număr natural</li>
	<li><code>a</code> – prin care primește un tablou unidimensional cu <code>n</code> elemente, numere naturale din mulțimea <code>{1,2,3}</code>.</li><br>
</ul><p></p>

<p>Subprogramul ordonează crescător tabloul <code>a</code> fără a returna valori.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000</code></li>
	<li>numele subprogramului cerut este <code>ordon123</code></li>
	<li>parametrii sunt, în această ordine: <code>n, a</code></li>
	<li>elementele tabloului <code>a</code> sunt indexate de la zero.</li>
</ul>

<h2>Exemplu:</h2>

<p>Dacă <code>n=6</code> și <code>a=(3,1,2,1,3,1)</code>, după apelul subprogramului tabloul <code>a</code> devine  <code>(1,1,1,2,3,3)</code>.</p>


<h3>Important</h3>

<p>Soluția propusă va conține doar definiția subprogramului cerut. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void ordon123(int n , int a[])
{
    int cnt1=0 , cnt2=0 , cnt3=0;
    for(int i = 0 ; i < n ; ++i)
    {
        if(a[i]==1)
            cnt1++;
        if(a[i]==2)
            cnt2++;
        else
            cnt3++;
    }
    for(int i = 0 ; i < cnt1 ; ++i)
        a[i]=1;
    for(int i = cnt1 ; i < cnt1+cnt2 ; ++i)
        a[i]=2;
    for(int i = cnt1+cnt2 ; i < n ; ++i)
        a[i]=3;
}</pre>
