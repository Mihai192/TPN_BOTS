<h1>Cerința</h1>

<p>Scrieți funcția cu următorul antet:</p>

<pre contenteditable="true" editable="true">int Path(const char cale[], char d[101][101])</pre>

<p>Parametrul <code>cale</code> memorează un șir de caractere format din litere mici și semnul <code>\</code> și reprezintă o succesiune de foldere incluse unul în altul. Funcția trebuie să memoreze fiecare folder pe câte o linie a matricei <code>d</code> și să returneze numărul de foldere conținute în <code>cale</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Lungimea șirului <code>cale</code> nu va depăși <code>100</code></li>
	<li><code>cale</code> este nevid, deci va exista cel puțin un folder</li>
	<li>Cei doi parametri sunt indexați de la <code>0</code></li>
	<li>Nu vor exista alte caractere în afară de cele menționate în enunț</li>
</ul>

<p>Dacă <code>cale = "pbinfo\clx\teste\azi"</code>, atunci <code>Path(cale, d)</code> va returna <code>4</code>. Linia <code>0</code> din <code>d</code> va memora <code>pbinfo</code>, linia <code>1</code> va memora <code>clx</code>, linia <code>2</code> va memora <code>teste</code> și linia <code>3</code> va memora <code>azi</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">bool lit(char ch){
    return (ch >= 'a' &amp;&amp; ch <= 'z') || (ch >= 'A' &amp;&amp; ch <= 'Z');
}

int Path(const char cale[], char d[101][101]){
    int i = 0, c = 0, j = 0;
    while(cale[i]){
        if(!lit(cale[i]))
            c++, j = 0;
        else{
            d[c][j] = cale[i];
            j++;
        }
        i++;
    }
    return c + 1;
}</pre>
