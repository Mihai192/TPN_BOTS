<p>Lumea este în pericol. Încălzirea globală  are efecte profunde în cele mai diferite domenii.  Ea determină apariția extremelor climatice, topirea ghețarilor, ridicarea nivelului mărilor și, cel mai grav, dispariția a numeroase specii de animale. Cercetătorii au observat că cele mai afectate sunt furnicile. </p>

<p>Pentru a stabili cât de rapid este procesul de dispariție, ei și-au propus să le studieze timp de <code>n</code> zile, notând în fiecare zi numărul de furnici lucrătoare care ies după hrană.  Ei au constatat că ele sunt afectate dacă, în zile consecutive, numărul de divizori ai numărului de furnici lucrătoare descrește. Vor lua în considerare doar perioadele de cel puțin două zile consecutive de descreșteri. </p>

<p>Pentru finalizarea studiului, trebuie să știe câte astfel de perioade au existat. Pentru că la calculele matematice pot greși, vă roagă să-i ajutați.</p>

<h1>Cerința</h1>

<p>Dându-se un număr natural <code>n</code>, reprezentând numărul de zile în care se face studiul și apoi un șir <code>x</code> de <code>n</code> numere naturale, ce reprezintă șirul numerelor de furnici lucrătoare care ies după hrană, unde <code>x</code><sub><code>i</code></sub> reprezintă numărul de furnici lucrătoare din ziua <code>i</code>. Se cere să se determine numărul de secvențe maximale, ordonate strict descrescător după numărul de divizori, de lungime minim <code>2</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>furnici.in</code> conţine două linii. Pe prima linie este scris numărul natural <code>n</code>, cu semnificația din cerință iar pe linia a doua, elementele șirului <code>x</code>, cu semnificația din cerință.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>furnici.out</code> va conţine un număr natural reprezentând numărul de secvențe maximale, de lungime minim <code>2</code>, ordonate strict descrescător după numărul de divizori.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 100000</code></li>
	<li><code>10</code> ≤ <code>x</code><sub><code>i</code></sub> ≤ <code>1.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Șirul numărului de divizori este: <code>2 3 8 4 3 2 4 12 12 6</code>.<br>
Se observă că sunt două secvențe maximale, de lungime minim doi, strict descrescătoare <code>8 4 3 2</code> și <code>12 6</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("furnici.in");
ofstream cout("furnici.out");
int a[101001];
int nrdiv(int n)
{
    int d = 2 , prod = 1;
    while(n > 1)
    {
        int p = 0;
        while(n % d == 0) p++,n /= d;
        if(p) prod *= (p+1);
        d++;
        if(d*d > n) d = n;
    }
    return prod;
}
int main()
{
    int n , x , l = 0 , lmax = 0;
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        cin >> x;
        a[i] = nrdiv(x);
    }
    for(int i = 1 ; i <= n ;++i)
    {
        if(a[i] < a[i-1]) l++;
        else if(a[i] >= a[i-1])
        {
            if(l) lmax++;
            l = 0;
        }
    }
    if(l) cout << lmax+1;
    else cout << lmax;
}</pre>
