<p>Pornind de la numărul <code>1</code>,orice număr natural se poate obţine aplicând repetat în mod convenabil operaţii din cele de mai jos:</p>

<ul>
	<li>înmulţire cu <code>3</code></li>
	<li>adunare cu <code>1</code></li>
	<li>scădere cu <code>1</code></li>
</ul>

<p>De exemplu numărul <code>24</code> se poate obţine astfel:</p>

<p>Adunăm <code>1</code>: <code>1 + 1 = 2</code><br>
Adunăm <code>1</code>: <code>2 + 1 = 3</code><br>
Înmultim cu <code>3</code>: <code>3 * 3 = 9</code><br>
Scădem <code>1</code>: <code>9 - 1 = 8</code><br>
Înmulțim cu <code>3</code>: <code>8 * 3 = 24</code></p>

<p>Urmărind operaţiile de la stânga la dreapta pentru exemplul de mai sus, şirul de operaţii se codifică cu <code>1</code>, <code>1</code>, <code>3</code>, <code>-1</code>, <code>3</code>. </p>

<h1>Cerinţă</h1>

<p>Cunoscând numărul natural <code>n</code>,să se tipărească şirul de operaţii prin care se poate ajunge de la numărul iniţial <code>1</code> la numărul final <code>n</code>.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>n311.in</code> conţine pe prima linie valoarea numărului natural <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>n311.out</code> va conţine pe prima linie şirul de operaţii format din numere întregi separate prin câte un spaţiu, cu semnificaţia de mai sus.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 &lt; n &lt;= 2 000 000000</code>;</li>
	<li>Pot exista mai multe soluţii, se acceptă oricare, dacă se încadrează în timpul de execuţie;</li>
	<li>Nu este obligatorie folosirea tuturor tipurilor de operaţii.</li>
</ul>

<h1>Exemple</h1>

<h2>Exemplul 1</h2>

<p><code>n311.in</code></p>

<pre contenteditable="true" editable="true">24</pre>

<p><code>n311.out</code></p>

<pre contenteditable="true" editable="true">1 1 3 -1 3</pre>

<h2>Exemplul 2</h2>

<p><code>n311.in</code></p>

<pre contenteditable="true" editable="true">4</pre>

<p><code>n311.out</code></p>

<pre contenteditable="true" editable="true">1 1 1</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("n311.in");
ofstream cout("n311.out");

int a[1000];

int main ()
{
    int n,i=0;
    cin >> n;
    while (n>1)
    {
        if (n%3==0)
            n/=3 , a[++i]=3;
        else if(n%3==1 &amp;&amp; n !=0)
            n-- , a[++i]=1;
        if (n%3==2)
            n++ , a[++i]=-1;
    }
    for(int j=i;j>=1;--j)
        cout << a[j] << " ";
    return 0;
}
