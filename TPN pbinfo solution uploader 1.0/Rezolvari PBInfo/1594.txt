<p>După ce și-a cumpărat biscuiți, Costy, eroul nostru, ajunge acasă și se apucă de teme. Astfel, dă peste următoarea problemă:</p>

<p>“La o probă de maraton participă <code>N</code> maratonişti. Ştiind că la secunda <code>0</code>, un maratonist se află la <code>Xi</code> metri de linia de sosire și aleargă cu o viteză de <code>Yi</code> metri/secundă, să se răspundă la <code>Q</code> întrebări de tipul:</p>

<ul>
	<li>Câți maratonişti au trecut linia de sosire după <code>Qi</code> secunde ? “</li>
</ul>

<h1>Cerința</h1>

<p>Ajutați-l pe Costy să răspundă la cele <code>Q</code> întrebări.</p>

<h1>Date de intrare</h1>

<p>Fișierul maraton.in conține:</p>

<ul>
	<li>pe prima linie numărul <code>N</code>, reprezentând numărul de maratoniști;</li>
	<li>pe următoarele <code>N</code> linii, câte <code>2</code> numere, <code>Xi Yi</code>, reprezentând distanța fată de linia de sosire și viteza fiecărui maratonist;</li>
	<li>pe următoarea linie, numărul <code>Q</code> reprezentând numărul de întrebări;</li>
	<li>pe următoarele <code>Q</code> linii se află câte un număr <code>Qi</code> reprezentând întrebarea <code>i</code>;</li>
</ul>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire maraton.out conţine:<br>
</p><ul>
	<li><code>Q</code> linii, linia <code>i</code> reprezentând răspunsul la întrebarea <code>i</code>;</li><br>
</ul><p></p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 &lt;= N, Q, Qi, Xi &lt;= 100 000</code></li>
	<li><code>1 &lt;= Yi &lt;= 1000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp"># include <iostream>
# include <fstream>

# define MAX_X 100000

using namespace std;

ifstream fin("maraton.in");
ofstream fout("maraton.out");

int f[MAX_X + 5];

int main() 
{

    int N , Q , x , y , q , i;

    fin >> N;

    for ( i = 0; i < N; i ++ ) 
    {
        fin >> x >> y;
        f[x / y + ( x % y > 0 )] ++;
    }

    for ( i = 1; i <= MAX_X; i ++ )
        f[i] += f[i - 1];

    fin >> Q;

    for ( i = 0; i < Q; i ++ ) 
    {
        fin >> q;
        fout << f[q] << '\n';
    }

    fin.close();
    fout.close();

    return 0;
}</pre>
