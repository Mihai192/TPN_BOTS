<h1>Cerința</h1>

<p>Se dă un cuvânt format din litere ale alfabetului englez și cifre. Afișați toate permutările circulare spre stânga ale sale.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>permutarecuvant.in</code> conține cuvântul pentru care se vor afișa permutările circulare spre stânga.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>permutarecuvant.out</code> va conține pe o singura linie toate permutările circulare ale cuvântului, separate prin câte un spațiu.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Cuvântul va avea maxim <code>20</code> de caractere.</li>
</ul>

using namespace std;

ifstream cin("permutarecuvant.in");

int main()
{
    char ch[30];
    cin >> ch;
    int i = 0;
    while(ch[i]!='\0')
    {
        int j = i;
        while(ch[j]!='\0')
        {
            cout << ch[j];
            j++;
        }
        for(int k = 0 ; k < i ; ++k)
            cout << ch[k];
        cout << ' ';
        i++;
    }
    return 0;
