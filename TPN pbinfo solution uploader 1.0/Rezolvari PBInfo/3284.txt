<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Considerăm poziţiile lor numerotate de la <code>1</code> la <code>n</code>.<br>
Să se determine cea mai mare valoare memorată pe o poziţie pară şi cea mai mică valoare de două cifre aflată pe o poziţie impară.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numerele <code>x</code> şi <code>y</code>, pe același rând și separate prin spațiu, reprezentând, în ordine, cele două valori calculate.<br>
În cazul în care nu sunt valori de două cifre pe poziții impare se va afișa <code>-1</code> în contul valorii <code>y</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n ≤ 20</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Cea mai mare valoare de pe o poziție pară este <code>4</code> și nu avem numere de două cifre pe poziții impare.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int nrcif(int n){
    int cnt = 0;
    while(n){
        n/=10;
        cnt++;
    }
    return cnt;
}

int main(){
    int n, x, min = 100, max = -1;
    cin >> n;
    for(int i = 1; i <= n; ++i){
        cin >> x;
        if(i % 2 == 0){
            if(x > max)
                max = x;
        }
        else{
            if(nrcif(x) == 2 &amp;&amp; x < min)
                min = x;
        }
    }
    cout << max << ' ';
    if(min != 100)
        cout << min;
    else
        cout << -1;
}</pre>
