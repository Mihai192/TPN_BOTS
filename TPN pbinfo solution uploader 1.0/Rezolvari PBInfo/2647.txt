<h1>Cerința</h1>

<p>Să se scrie funcția cu următorul antet:</p>

<pre contenteditable="true" editable="true">long long SecvBiti(char s[])</pre>

<p>Funcția primește ca parametru un șir de caractere din mulțimea <code>{'0', '1'}</code> și returnează numărul secvențelor cu proprietatea că numărul biților de <code>1</code> din secvență este egal cu numărul biților de <code>0</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ lungimea șirului ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Secvențele sunt: <code>1001</code>, <code>10</code>, <code>01</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei. De asemenea, e posibil ca folosirea funcțiilor predefinite pentru șiruri de caractere să dea erori de compilare</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <cstring>
int f[2000001];
long long SecvBiti(char s[])
{
    int l = strlen(s);
    f[l]=1;
    long long int cnt = 0;
    for(int i = 0 ; i < l ; ++i)
        if(s[i]=='1') cnt++ , f[l+cnt]++;
        else cnt-- , f[l+cnt]++;
    cnt=0;
    for(int i = 0 ; i < 2*l ; ++i)
        cnt = cnt + f[i] * (f[i]-1LL) / 2;
    return cnt;
}</pre>
