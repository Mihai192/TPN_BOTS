<h1>Cerința</h1>

<p>Scrieți o funcție operand cu doi parametri care returnează poziția maximului dintr-un șir dat de numere întregi. Se presupune că elementele șirului sunt distincte și memorate începând cu poziția <code>1</code>. Parametrii funcției sunt: tabloul în care este memorat șirul și numărul de elemente ale acestuia.</p>

<p>Scrieți o funcție procedurală cu trei parametri care oferă poziția minimului dintr-un șir dat de numere întregi. Se presupune că elementele șirului sunt distincte și memorate începând cu poziția <code>1</code>. Parametrii funcției sunt: tabloul în care este memorat șirul, numărul de elemente ale acestuia precum și rezultatul.</p>

<p>Scrieți o funcție procedurală care interschimbă două valori întregi.</p>

<p>Scrieți o funcție procedurală care are trei parametri, în ordine: <code>v</code> (tablou unidimensional cu elemente int), <code>pozInit</code> (de tip int), <code>pozFin</code> (de tip int), și care afișează în ordine descrescătoare a indicilor (separate prin spații) elementele secvenței delimitate de <code>pozInit</code> și <code>pozFin</code>, inclusiv. Se garantează că primul parametrul <code>pozInit</code> se transmite funcției mai mic strict decât <code>pozFin</code>. Se garantează că tabloul are cel puțin <code>pozFin</code> elemente și că ambii indici se transmit funcției mai mari sau egali cu <code>1</code>.</p>

<p>Scrieți un program care, apelând util <span class="caps">TOATE</span> funcțiile de mai sus, preia dintr-un fișier dimensiunea (notată de noi mai departe <code>n</code>) și elementele unui vector și le stochează în acesta începând cu poziția <code>1</code>. Programul va scrie într-un fișier, în ordine descrescătoate a indicilor, elementele cuprinse între valoarea minimă și valoarea maximă. Se garantează că elementele șirului se dau distincte.</p>

<h1>Date de intrare</h1>

<p>Programul citește din fișierul <code>calcfunct.in</code> mai întâi un număr <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va scrie în fișierul <code>calcfunct.out</code>, pe un singur rând separate prin spații, elementele secvenței specificate, în ordinea cerută.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>cele <code>n</code> numere citite vor fi distincte, naturale, și mai mici decât <code>1.000.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("calcfunct.in");
ofstream cout("calcfunct.out");

int pmax(int a[], int n){
    int maxi = 0, pmaxi = 1;
    for(int i = 1; i <= n; ++i)
        if(a[i] > maxi)
            maxi = a[i], pmaxi = i;
    return pmaxi;
}

void pmin(int a[], int n, int&amp; pmini){
    int pozz = 0;
    for(int i = 1; i <= n; ++i)
        if(a[i] < pmini)
            pmini = a[i], pozz = i;
    pmini = pozz;
}

void inter(int&amp; pmini, int&amp; pmaxi){
    swap(pmini, pmaxi);
}

void solve(int a[], int a1, int b1){
    for(int i = b1; i >= a1; --i)
        cout << a[i] << " ";
}

int main()
{
    int n, a[101], pozmin, pozmax;
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
    pozmax = pmax(a, n);
    pozmin = 1000000000;
    pmin(a, n, pozmin);
    if(pozmin > pozmax)
        inter(pozmin, pozmax);
    solve(a, pozmin, pozmax);
    return 0;
