<h1>Cerința</h1>

<p>Se dau două numere naturale <code>a</code> și <code>b</code>. Calculați suma numerelor echilibrate din intervalul <code>[a,b]</code>. Un număr este echilibrat dacă are suma cifrelor pare egală cu suma cifrelor impare. De exemplu <code>3467</code> este echilibrat, iar <code>8333</code> nu este.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>a</code> și <code>b</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>s</code>, reprezentând suma numerelor echilibrate din intervalul <code>[a,b]</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ a ≤ b ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>În intervalul <code>[301, 402]</code> numerele echilibrate sunt <code>314</code>, <code>336</code>, <code>341</code>, <code>358</code>, <code>363</code>, <code>385</code>, iar suma lor este <code>2097</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

long long int s;

bool echi(int n){
    int sp = 0, si = 0;
    while(n){
        if(n % 2 == 0)
            sp += n % 10;
        else
            si += n % 10;
        n/=10;
    }
    return sp == si;
}

int main(){
    int a, b;
    cin >> a >> b;
    for(int i = a; i <= b; ++i)
        if(echi(i))
            s += i;
    cout << s;
    return 0;
}</pre>
