<h1>Cerința</h1>

<p>Se consideră două evenimente a căror durată este exprimată fiecare prin câte trei numere naturale: ore (<code>h</code>), minute (<code>m</code>) şi secunde (<code>s</code>).<br>
Să se scrie în fișierul de ieșire:<br>
a) pe primele două linii, duratele în formatul <code>h: m: s</code><br>
b) pe următoarele două linii, duratele exprimate în secunde, corespunzătoare fiecărui<br>
eveniment, pe rânduri separate;<br>
c) pe următoarea linie suma obţinută din adunarea duratelor celor două evenimente, exprimată în<br>
ore, minute, secunde, în formatul <code>h: m: s</code>.</p>


<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>ore.in</code> conține două linii pe care sunt scrise câte trei numere naturale, separate prin câte un spaţiu, reprezentând duratele a două evenimente exprimate în: <code>ore (h), minute (m) şi secunde (s)</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>ore.out</code> va conține pe primele două linii răspuncul de la cerința <code>a)</code>, apoi pe liniile <code>3</code> și <code>4</code> răspunsul la cerința <code>b)</code>, iar pe a <code>5</code>-a linie răspunsul la cerința <code>c)</code>.</p>


<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ h ≤ 24</code></li>
	<li><code>0 ≤ m ≤ 60</code></li>
	<li><code>0 ≤ s ≤ 60</code></li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare sunt detaliile referitoare la duratele cele două evenimente, iar suma lor este <code>6: 16: 3</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("ore.in");
ofstream fout("ore.out");
int main()
{
    long long h1 , h2 , m1 , m2 , s1 , s2 , h , m , s;
    fin >> h1 >> m1 >> s1 >> h2 >> m2 >> s2;
    fout << h1 << ": " << m1 << ": " << s1 << endl;
    fout << h2 << ": " << m2 << ": " << s2 << endl;
    long long x1 = h1 * 3600 + m1 * 60 + s1;
    long long x2 = h2 * 3600 + m2 * 60 + s2;
    fout << x1 << endl << x2 << endl;
    long long y = x1 + x2;
    s = y % 60;
    m = ((y - 3) / 60) % 60;
    y = (y - 3) / 60;
    h = y / 60;
    fout << h << ": " << m << ": " << s;
    return 0;
}</pre>
