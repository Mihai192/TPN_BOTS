<h1>Cerința</h1>

<p>Gigel a fost prins de doamna profesoară de matematică în timp ce se juca Art of conquest pe telefon în timpul orei de matematică. Drept pedeapsă, aceasta i-a dat să calculeze cel mai mic multiplu comun al unui șir de n numere naturale. Dorind să termine cât mai repede ceea ce are de făcut și să reia jocul fără să fie dat afară din alianță, Gigel apelează la ajutorul tău și te roagă să scrii un program care să îi rezolve problema primită ca pedeapsă.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>aoc2020.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>aoc2020.out</code> va conține pe prima linie numărul <code>x</code>, reprezentând cel mai mic multiplu comun ale celor n numere citite .</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi mai mici sau egale cu <code>1.000</code></li>
	<li>x are cel mult 18 cifre</li>
</ul>

<h3>Explicație</h3>

<p>Cel mai mic multiplu comun al numerelor <code>18</code>, <code>54</code>, <code>24</code> și <code>42</code> este <code>1512</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("aoc2020.in");
ofstream cout("aoc2020.out");

long long multiplu = 4000000000000000001;

long long cmmdc(long long a, long long b){
    while(b){
        long long r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main(){
    int n;
    cin >> n;
    long long a, b;
    cin >> multiplu;
    for(int i = 2; i <= n; ++i){
        cin >> b;
        multiplu = multiplu / cmmdc(multiplu, b) * b;
    }
    cout << multiplu;
    return 0;
}</pre>
