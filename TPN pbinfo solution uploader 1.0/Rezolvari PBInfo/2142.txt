<h1>Cerința</h1>

<p>Se consideră un vector cu <code>n</code> elemente numere naturale. Calculați suma sumelor tuturor <strong>subsecvențelor</strong> ce se pot forma cu elementele vectorului. Pentru că suma poate fi foarte mare, afișați suma <strong>modulo</strong> <code>1.000.000.007</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>easy_sum.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>easy_sum.out</code> va conține pe prima linie numărul <code>S</code>, reprezentând suma sumelor tuturor subsecvențelor <strong>modulo</strong> <code>1.000.000.007</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi cuprinse în intervalul <code>[1,1.000.000]</code></li>
	<li>prin <strong>subsecvență</strong> înțelegem orice înșiruire de elemente din vector aflate pe poziții consecutive.</li>
</ul>

<h3>Explicație</h3>

<p>Subsecvențele formate: <code>{1}</code>, <code>{2}</code>, <code>{3}</code>, <code>{1,2}</code>, <code>{2,3}</code>, <code>{1,2,3}</code>.<br>
Suma sumelor tuturor subsecvențelor este <code>20</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("easy_sum.in");
ofstream fout("easy_sum.out");
int main()
{
    int n;
    long long x , s = 0;
    fin>>n;
    for(int i = 0; i < n; i++)
    {
        fin >> x;
        s = (s + x * (i + 1) % 1000000007 * (n - i) % 1000000007) % 1000000007;
    }

    fout << s;
    return 0;
}</pre>
