<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale. Aflați câte dintre acestea se pot scrie ca sumă de puteri distincte ale unui număr natural <code>k</code>. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>n</code> și <code>k</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul de numere din șirul dat care se pot scrie ca sumă de puteri distincte ale lui <code>k</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li><code>2 ≤ k ≤ 10</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<p><strong>Intrare</strong></p>

<h1>Explicație</h1>

<p>Avem <code>37=3</code><sup><code>3</code></sup><code>+3</code><sup><code>2</code></sup><code>+3</code><sup><code>0</code></sup> , <code>12 = 3</code><sup><code>2</code></sup><code>+3</code><sup><code>1</code></sup>, <code>9 = 3</code><sup><code>2</code></sup> , deci <code>3</code> numere se pot scrie ca sumă de puteri distincte ale lui <code>3</code> ( pentru <code>6</code> avem <code>6 = 3 + 3</code>, iar <code>5 = 3 + 1 + 1</code> nu convin ).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int n , k , x , nr;
int main()
{
    cin >> n >> k;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> x;
        int c = 1;
        while(x != 0)
        {
            if(x % k > 1)
            {
                c = 0;
                break;
            }
            x /= k;
        }
        if(c) nr++;
    }
    cout << nr;
    return 0;

}</pre>
