<h1>Cerința</h1>

<p>Scrieți funcția <code>Trim</code> care primește ca parametru un șir de caractere <code>s</code>, elimină eventualele spații de la începutul și sfârșitul șirului și returnează tot prin intermediul lui <code>s</code> șirul de caractere rezultat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ lungimea șirului ≤ 100</code></li>
	<li>Numele funcției trebuie să fie <code>Trim</code></li>
	<li>Se garantează că șirul inițial conține cel puțin un caracter diferit de spațiu</li>
</ul>

<p>Dacă <code>s="   ana   are   mere        "</code>, atunci, după apelul <code>Trim(s)</code>, șirul va fi <code>s="ana   are   mere"</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void Trim(char s[])
{
    int i = 0;
    int ok = false;
    int poz1 , poz2;
    while(s[i]!='\0')
    {
        if(s[i]!=' ' &amp;&amp; ok == false)
            ok=true , poz1=i;
        if(s[i]!=' ')
            poz2=i;
        i++;
    }
    for(int i = poz1 ; i <= poz2 ; ++i)
        s[i-poz1]=s[i];
    s[poz2-poz1+1]='\0';
}</pre>
