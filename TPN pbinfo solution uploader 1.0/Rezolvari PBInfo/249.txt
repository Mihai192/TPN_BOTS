<h1>Cerinţa</h1>

<p>Se dă un număr natural <code>X</code> şi un şir cu <code>n</code> numere  naturale distincte două câte două. Să se determine poziţia pe care s-ar afla numărul <code>X</code> în şirul ordonat crescător.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>pozitiex.in</code> conţine pe prima linie numerele <code>X</code> şi <code>n</code>. Urmează <code>n</code> numere naturale, dispuse pe mai multe linii, separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>pozitiex.out</code> va conţine pe prima linie numărul <code>p</code>, reprezentând poziţia în şirul ordonat crescător a primului element din şirul dat. Dacă numărul <code>X</code> nu apare în şir, se va afişa mesajul <code>NU EXISTA</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10000</code></li>
	<li><code>X</code> şi elementele şirului vor avea cel mult <code>9</code> cifre</li>
</ul>

<h1>Explicație</h1>

<p>Şirul sortat este <code>7 13 45 79 267 628</code>. În acest şir, valoarea <code>79</code> se află pe poziţia a <code>4</code>-a.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;
ifstream fin("pozitiex.in");
ofstream fout("pozitiex.out");
int main()
{
    int x , n , cnt = 0 , a[10001];
    fin >> x >> n;
    for (int i = 0 ; i < n ; ++i) fin >> a[i];
    sort(a,a+n);
    for (int i = 0 ; i < n ; ++i)
    {
        if (a[i] == x)
        {
            fout << i+1;
            cnt++;
        }
    }
    if (cnt == 0)
    fout << "NU EXISTA";

    fin.close();
    fout.close();

    return 0;
}</pre>
