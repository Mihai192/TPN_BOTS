<p>Ionel i-a dat numărul său de telefon <code>N</code> lui Vasile, dar a greșit exact o cifră de pe o anumită poziție. Se cunoaște că pe acea poziție cifra corectă este o cifra pară.</p>

<h1>Cerința</h1>

<p>Determinați numărul minim <code>NR</code> de numere de telefon pe care trebuie să le încerce Vasile astfel încât printre ele să se afle cu siguranță numărul corect de telefon al lui Ionel.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>numar4.in</code> conține pe prima linie numărul <code>N</code> greșit de<br>
telefon transmis de Ionel.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>numar4.out</code> va conține pe prima linie numărul <code>NR</code> de<br>
combinații necesare lui Vasile pentru a nimeri cu siguranță și numărul corect al lui Ionel.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Numărul <code>N</code> nu poate începe cu cifra <code>0</code>. Numărul corect poate incepe cu cifra <code>0</code>.</li>
	<li>Numărul <code>N</code> are cel mult <code>9</code> cifre.</li>
</ul>

<h1>Exemplul 1</h1>

<p><code>numar4.in</code></p>

<pre contenteditable="true" editable="true">72361</pre>

<p><code>numar4.out</code></p>

<pre contenteditable="true" editable="true">23</pre>

<h3>Explicație</h3>

<p>In numarul corect:</p>

<ul>
	<li>Pe prima poziție poate fi una din cifrele: <code>0</code>, <code>2</code>, <code>4</code>, <code>6</code>, <code>8</code> sau</li>
	<li>Pe a doua poziție poate fi una din cifrele: <code>0</code>, <code>4</code>, <code>6</code>, <code>8</code> sau</li>
	<li>Pe a treia poziție poate fi una din cifrele: <code>0</code>, <code>2</code>, <code>4</code>, <code>6</code>, <code>8</code> sau</li>
	<li>Pe a patra poziție poate fi una din cifrele: <code>0</code>, <code>2</code>, <code>4</code>, <code>8</code> sau</li>
	<li>Pe a cincea poziție poate fi una din cifrele: <code>0</code>, <code>2</code>, <code>4</code>, <code>6</code>, <code>8</code>.</li>
</ul>

<p>Așadar numărul total de combinații posibile este <code>5+4+5+4+5=23</code>.</p>

<h1>Exemplul 2</h1>

<p><code>numar4.in</code></p>

<pre contenteditable="true" editable="true">27544</pre>

<p><code>numar4.out</code></p>

<pre contenteditable="true" editable="true">22</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("numar4.in");
ofstream fout("numar4.out");
int main()
{
    int n,cnt=0;
    fin>>n;
    while(n!=0)
    {
        if(n%2==0) cnt+=4;
        else
        if(n%2==1) cnt+=5;
        n=n/10;
    }
    fout<<cnt;
    return 0;
}</pre>
