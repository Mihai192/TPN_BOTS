<h1>Cerinţa</h1>

<p>Se dau două numere naturale <code>x</code> şi <code>n</code>. Calculaţi  <code>x</code><sup><code>n</code></sup> .</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele <code>x</code> şi <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>P</code>, valoarea calculată.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ x ≤ 1000</code></li>
	<li><code>1 ≤ n ≤ 1000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int main()
{
    int n, a[10001]={0}, c=1 , m;
    cin >> m >> n;
    a[1]=1;
    for(int i = 1; i <= n; i++)
    {
        int t = 0;
        for(int j = 1; j <= c; j++)
        {
            int cif = a[j] * m + t;
            a[j] = cif % 10;
            t = cif / 10;
        }
        while(t)
        {
            a[++c] = t % 10;
            t /=10;
        }
    }
    for(int i = c ; i >= 1; i--)
        cout << a[i];
    return 0;
}</pre>
