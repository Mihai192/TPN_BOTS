<h1>Cerința</h1>

<p>Georgiana a mai primit o problemă de la doamna profesor. Se dau <code>n</code> triplete de forma <code>m</code>, <code>b</code>, <code>r</code>, iar pentru fiecare triplet Georgiana trebuie să afle care este cel mai mic număr natural format cu <code>m</code> cifre, care împărţit la <code>b</code> dă restul <code>r</code>.  </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> triplete de numere naturale.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran, pe linii separate, cele <code>n</code> numere cerute.</p>


<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10</code></li>
	<li><code>1 ≤ m ≤ 100.000</code></li>
	<li><code>2 ≤ b ≤ 10</code></li>
	<li><code>0 ≤ r &lt; b</code></li>
</ul>

<h3>Explicație</h3>

<p>Cel mai mic număr de două cifre care dă restul <code>1</code> la împarţirea cu <code>3</code> este <code>10</code>.<br>
Cel mai mic număr de trei cifre care dă restul <code>4</code> la împarţirea cu <code>5</code> este <code>104</code>.<br>
Cel mai mic număr de patru cifre care dă restul <code>0</code> la împarţirea cu <code>6</code> este <code>1002</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;

int n , a[20] , b , r , cif , nu[20] , rest;

int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
    {
        cin >> a[i] >> b >> r;
        rest = 1;
        for(int j = 1 ; j < a[i] ; j++)
            rest = (rest * 10) % b;
        if(r >= rest) cif = r - rest;
        else cif = b + r - rest;
        nu[i] = cif;
    }
    for(int i = 1 ; i <= n ; i++)
    {
        cout << 1;
        for(int j = 1 ; j <= a[i] - 2 ; j++) cout << 0;
        cout << nu[i] << '\n';
    }
}</pre>
