<p>Gigel, mare amator de probleme de matematică şi informatică, a observat că unele numere prime au o proprietate interesantă: orice cifră ar elimina dintr-un astfel de număr, numărul obţinut este tot număr prim. A numit astfel de numere numere <strong>extraprime</strong>. De exemplu, numărul <code>317</code> este un număr extraprim: el este număr prim şi, în plus, dacă eliminăm cifra <code>3</code>, obţinem <code>17</code>, care este prim; dacă eliminăm <code>1</code>, obţinem <code>37</code>, care este prim; dacă eliminăm <code>7</code>, obţinem <code>31</code>, care este şi el număr prim.</p>

<h1>Cerință</h1>

<p>Spunem că <code>x</code> este între <code>a</code> şi <code>b</code> dacă <code>x≥a</code> şi <code>x≤b</code>. Fiind date două valori naturale <code>a</code> şi <code>b</code>, să se determine câte numere <strong>extraprime</strong> există între <code>a</code> şi <code>b</code>, precum şi cel mai mic şi cel mai mare număr <strong>extraprim</strong> dintre <code>a</code> şi <code>b</code>.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>extraprime.in</code> conține pe prima linie cele două valori naturale <code>a</code> şi <code>b</code>, separate printr-un spaţiu.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>extraprime.out</code> va conține <code>3</code> linii. Pe prima linie se va scrie un număr natural <code>nr</code> reprezentând numărul de numere  <strong>extraprime</strong> dintre <code>a</code> şi <code>b</code>. Pe linia a doua a fişierului de ieşire se va scrie cel mai mic număr <strong>extraprim</strong> dintre <code>a</code> şi <code>b</code>, iar pe linia a treia a fişierului de ieşire se va scrie cel mai mare număr <strong>extraprim</strong> dintre <code>a</code> şi <code>b</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>10 &lt; a ≤ b &lt; 10000000</code>;</li>
	<li>Numărul <code>1</code> nu este prim;</li>
	<li>Pentru datele de test există întotdeauna soluţie;</li>
</ul>

<p>Se află <code>4</code> numere <strong>extraprime</strong> mai mari decât <code>10</code> şi mai mici decât <code>100</code>: <code>23</code>, <code>37</code>, <code>53</code> şi <code>73</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("extraprime.in");
ofstream cout("extraprime.out");
bool ciur[10000005];
int a , b , cif[10] , ncif;
void Ciur()
{
    ciur[1] = 1;
    for(int i = 4 ; i <= 10000000; i += 2) ciur[i] = 1;
    for(int i = 3; i * i <= 10000000; i += 2)
        if(!(ciur[i]))
            for(int j = i * i ; j <= 10000000; j += 2 * i)
                ciur[j] = 1;

}
int extra(int x)
{
    ncif=0;
    if(ciur[x]) return false;
    while(x)
    {
        cif[++ncif]=x%10;
        x/=10;
    }
    for(int i = ncif ; i >= 1 ; i--)
    {
        int x = 0;
        for(int j = ncif ; j >= 1 ; j--)
            if(i!=j) x = x * 10 + cif[j];
        if(ciur[x]) return false;
    }
    return true;
}
int main()
{
    int s = 0;
    Ciur();
    cin >> a >> b;
    int maxim = -1000000000 , minim = 1000000000;
    for(int i = a ; i <= b ; i++)
    {
        int nr = i;
        if(extra(nr))
        {
            maxim=max(maxim , nr);
            s++;
            minim=min(minim , nr);
        }
    }
    cout<<s<<'\n' << minim << '\n' << maxim;
    return 0;
