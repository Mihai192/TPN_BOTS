<h1>Cerința</h1>

<p>Se dă următorul șir de numere naturale:</p>

<p><code>1, 3, 9, 25, 65, 161, 385, 897, 2049, 4609, 10241, 22529, 49153, 106497…</code></p>

<p>Pentru un număr natural <code>n</code>, citit de la tastatură, afișati numărul de divizori pentru fiecare dintre primii <code>n</code> termeni ai șirului.  </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>;</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran cele <code>n</code> numere, reprezentând numărul de divizori ai fiecarui numar dintre cele <code>n</code>, separate prin spații.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 &lt; n &lt; 60</code></li>
</ul>

<h3>Explicație</h3>

<p></p><ul>
	<li><code>1</code> are <code>un</code> divizor,</li>
	<li><code>3</code> are <code>2</code> divizori,</li>
	<li><code>9</code> are <code>3</code> divizori,</li>
	<li><code>25</code> are <code>3</code> divizori.</li><br>
</ul><p></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <bits/stdc++.h>

using namespace std;

int n;

unsigned long long Count(unsigned long long a);

int main()
{

    cin >> n;

    for(unsigned long long i=0;i<n;i++)
    {
        unsigned long long nr=i*(1ULL<<i)+1;
            cout << Count(nr) << " ";
    }

    return 0;
}

unsigned long long Count(unsigned long long a)
{
   unsigned long long count = 1, k = 0, i;
   if (a == 1 || a == 2)
      return a;
   while ((a &amp; 1) == 0)
   {
      k++;
      a >>= 1;
   }
   if (a == 1)
      return k + 1;
   else
      count = k + 1;
   for(i = 3; i*i <= a; i += 2)
   {
      k = 0;
      while(a % i == 0)
      {
         k++;
         a /= i;
      }
      count *= (k + 1);
   }
   if (a > 1)
      count <<= 1;
   return count;
}</pre>
