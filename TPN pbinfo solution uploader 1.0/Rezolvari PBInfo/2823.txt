<p>Dacă <code>n=23541</code>, subprogramul returnează valoarea <code>24</code>, iar dacă <code>n=28</code>, subprogramul returnează valoarea <code>-1</code>.</p>

<h1>Important</h1>

<p>Soluția propusă va conține definiția subprogramului cerut. Prezența în soluție a altor instrucțiuni poate duce erori de compilare sau de execuție care vor avea ca efect depunctarea soluției.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int cifreImpare(int n)
{
    int a[11] , p = 1;
    int cn = n;
    while(n)
    {
        a[p]=n%10;
        n/=10;
        p++;
    }
    p--;
    int nr = 0;
    for(int i = p ; i >= 1 ; --i)
        if(a[i]%2==0)
        nr = nr * 10 + a[i];
    if(nr == 0 || nr == cn)
        return -1;
    return nr;
}</pre>
