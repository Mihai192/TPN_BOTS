<h1>Cerința</h1>

<p>Se dau două numere naturale diferite. Afişaţi cel mai mare număr care poate fi scris folosind toate cifrele celor două numere date.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>numere6.in</code> conține pe prima linie cele două numere.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>numere6.out</code> va conține pe prima linie numărul cerut.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>cele două numere date vor fi mai mici decât <code>2 000 000 000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;

ifstream fin("numere6.in");
ofstream fout("numere6.out");

int main()
{
    int n , m;
    fin >> n >> m;
    int v[20];
    int p = 0;
    if(n==0)
        v[p]=0 , p++;
    if(m==0)
        v[p]=0 , p++;
    while(n)
    {
        v[p]=n%10;
        p++;
        n/=10;
    }
    while(m)
    {
        v[p]=m%10;
        p++;
        m/=10;
    }
    for(int i = 9 ; i >= 0 ; --i)
    {
        for(int j = 0 ; j < p ; ++j)
        {
            if(i==v[j])
            fout << i;
        }
    }
    return 0;
}</pre>
