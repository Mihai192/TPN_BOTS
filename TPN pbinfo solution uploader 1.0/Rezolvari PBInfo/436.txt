<h1>Cerinţa</h1>

<p>Să se scrie un program care citeşte de la tastatură un număr natural <code>n</code> şi apoi un şir de <code>n</code> numere naturale şi determină cel mai mare număr prim din șir și de câte ori apare.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numerele <code>M nrap</code>, separate printr-un spațiu, reprezentând cel mai mare număr prim din șir și de câte ori apare acest număr în șir.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt; 1000</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>2.000.000.000</code></li>
	<li>pentru toate testele de evaluare, datele de intrare vor conține cel puțin un număr prim</li>
</ul>

int prim(int n)
{
    int cnt=0;
    for (int i = 1 ; i * i <= n; ++i)
    {
        if (n%i==0)
            cnt+=2;
        if (i*i==n)
            cnt--;
    }
    if (cnt == 2)
        return 1;
