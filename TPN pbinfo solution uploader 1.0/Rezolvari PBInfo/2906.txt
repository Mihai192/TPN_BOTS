<h1>Cerința</h1>

<p>Gigel a găsit un șir de <code>n</code> cifre minunate. A adormit cu ele în brațe și a visat <code>m</code> numere naturale. Nedumerit, a cerut părerea vrăjitoarei Ghiocica. Acesta i-a spus:</p>

<blockquote>
	<p>- Gigele, ești norocos. Suma numerelor distincte visate care sunt scrise cu cifre consecutive în șirul de cifre minunate este suma pe care o vei câștiga la Loto. </p>
</blockquote>

<p>Nerăbdător, Gigel vă roagă să scrieți un program care să citească cele <code>n</code> cifre și cele <code>m</code> numere și să determine suma pe care o va câștiga la Loto.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>potrivire.in</code> conține pe prima linie numărul <code>n</code>; a doua linie conține șirul de <code>n</code> cifre. A treia linie conține numărul <code>m</code>, iar a patra linie conține cele <code>m</code> numere.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>potrivire.out</code> va conține pe prima linie numărul <code>S</code>, reprezentând suma pe care o va câștiga Gigel.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n, m ≤ 100000</code></li>
	<li>pentru 50% din teste, <code>1 ≤ n, m ≤ 1000</code></li>
	<li>cele <code>m</code> numere visate sunt numere naturale și au cel mult cinci cifre</li>
</ul>

<h3>Explicație</h3>

<p>Numerele distincte visate care se pot scrie cu cifre consecutive din șir sunt: <code>456</code>, <code>2607</code> și <code>97</code>, iar suma lor este <code>3160</code>. <code>2607</code> a fost visat de două ori, dar se ia în considerare o singură dată.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("potrivire.in");
ofstream cout("potrivire.out");

long long n , m , a[100001] , x , f[100001];
long long s;


int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; i++)
        cin >> a[i];

    for(int i = 1 ; i <= n ; i++)
    {
        int num = 0;
        for(int j = i ; j <= i + 4 &amp;&amp; j <= n ; j++)
        {
            num = num* 10 + a[j];
            f[num]++;
        }
    }
    cin >> m;
    for(int i = 1 ; i <= m ; i++)
    {
        cin >> x;
        if(f[x] != 0 &amp;&amp; f[x] != -1) s+= x;
        f[x] = -1;
    }
    cout << s;
}</pre>
