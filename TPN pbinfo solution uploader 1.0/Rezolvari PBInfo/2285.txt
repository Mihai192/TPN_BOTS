<h1>Cerința</h1>

<p>Fie un număr natural <code>n</code>, <code>n</code> &lt; <code>2^62</code>. Să se afişeze toate numerele naturale mai mici sau egale cu <code>n</code>, cu proprietatea că reprezentarea lor in baza <code>2</code> are exact două cifre <code>1</code>.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>b2.in</code> conţine pe prima linie numărul natural <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>În fişierul de ieşire <code>b2.out</code> se vor afişa în ordine crescătoare numerele cerute, pe o singură linie cu spaţiu între ele.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>3</code> ≤ <code>n</code> &lt; <code>2^62</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin ("b2.in");
ofstream fout ("b2.out");

int main ()
{
    int nc=0;
    unsigned long long int x,n;
    fin >> n;
    x=n;
    while(x)
    {
        x/=2;
        nc++;
    }
    for (int i = 1; i <nc; ++i)
        for(int j = 0 ; j <i; ++j)
    {
        x=(1ull<<i)+(1ull<<j);
        if (x<=n) fout << x << ' ';
    }
    return 0;
}</pre>
