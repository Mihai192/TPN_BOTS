<h1>Cerința</h1>

<p>Se dă un  șir cu <code>n</code> elemente, numere întregi. Determinați secvența de elemente cu suma maximă.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>secvsummax.in</code> conține pe prima linie numărul <code>n</code>; urmează cele <code>n</code> elemente ale șirului, dispuse pe mai multe linii și separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>secvsummax.out</code> va conține pe prima linie numerele <code>p</code> și <code>u</code>, separate printr-un spațiu, reprezentând poziția de început și de sfârșit a secvenței determinate.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100.000</code></li>
	<li>elementele șirului vor avea cel mult <code>4</code> cifre și sunt numerotate de la <code>1</code> la <code>n</code></li>
	<li>dacă șirul conține mai multe secvențe de suma maximă, se va determina cea cu indicele de început cel mai mic, iar în caz de egalitate, cea mai scurtă</li>
	<li>șirul va conține cel puțin un element pozitiv</li>
</ul>

<h1>Explicație</h1>

<p>Secvența <code>1 4 -2 2 3</code> are suma <code>8</code> și este suma maximă pentru toate secvențele care se pot forma.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("secvsummax.in");
ofstream cout("secvsummax.out");
int n, a[100002];
int main()
{
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
        cin >> a[i];
    int s = 0 , d = 1 ,maxi = -2000000010 , sum=a[1] , p = 1;
    for(int i=2;i<=n;++i)
    {
        if(sum < 0) sum=a[i], p = i;
        else sum += a[i];
        if(sum > maxi)
        {
            maxi = sum;
            s = p;
            d = i;
        }
    }
    cout << s << " " << d;
    return 0;
}</pre>
