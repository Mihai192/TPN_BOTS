<h1>Cerinţa</h1>

<p>Se dă un şir format din <code>n</code> elemente, numere  naturale. Calculaţi suma elementelor din secvenţa determinată de primul şi ultimul element prim.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>sumsec.in</code> conţine pe prima linie numărul <code>n</code>; urmează cele <code>n</code> elemente ale vectorului, dispuse pe mai multe linii şi separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>sumsec.out</code> va conţine pe prima linie numărul <code>S</code>, reprezentând suma calculată.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1000</code></li>
	<li>elementele şirului vor avea cel mult <code>9</code> cifre</li>
	<li>şirul va conţine cel puţin un element prim</li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin("sumsec.in");
ofstream fout("sumsec.out");

int prim(int n)
{
    int cnt=1;
    int d=2;
    while(n>1)
    {
        int p = 0;
        while(n%d==0)
        {
            p++;
            n/=d;
        }
        if(p>0)
            cnt*=(p+1);
        d++;
        if(d*d>n)
            d=n;
    }
    if(cnt==2)
    return 1;
    else
    return 0;
}

int main()
{
    int n;
    fin >> n;
    long long int a[1000];
    int cnt=0;
    long long int p = 0 , u = 0 , s = 0;
    for(int i = 0 ; i < n ; ++i)
    {
        fin >> a[i];
        if(prim(a[i]) &amp;&amp; cnt==0)
        {
        p=i;
        cnt++;
        }
        if(prim(a[i]))
        u=i;
    }
    for(int i = p ; i <= u ; ++i)
    {
        s+=a[i];    
    }
    fout << s;
    
    fin.close();
    fout.close();
    
    return 0;
}</pre>
