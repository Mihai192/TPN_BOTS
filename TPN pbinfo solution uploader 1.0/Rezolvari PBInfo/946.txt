<h1>Cerința</h1>

<p>Se dă  un număr <code>n</code> scris în baza <code>2</code>. Să se afișeze valoarea acestuia în baza <code>4</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură scrierea în baza <code>2</code> a numărului <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran scrierea în baza <code>4</code> a numărului <code>n</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>n</code> are cel mult <code>100</code> de cifre în baza <code>2</code></li>
</ul>

<h1>Explicație</h1>

<p><code>100011</code><sub><code>(2)</code></sub> este scrierea binară a numărului <code>35</code><sub><code>(10)</code></sub> . Numărul <code>35</code> trecut în baza <code>4</code> are scrierea <code>203</code><sub><code>(4)</code></sub></p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include<iostream>
#include<string>
using namespace std;

string n;

void modf(int v)
{
    for(int i=v; i<n.length(); i+=2) {
        if(n[i] == '0' &amp;&amp; n[i+1] == '0') cout << 0;
        else if(n[i] == '0' &amp;&amp; n[i+1] == '1') cout << 1;
        else if(n[i] == '1' &amp;&amp; n[i+1] == '0') cout << 2;
        else cout << 3;
    }
}

int main()
{
    cin >> n;
    if(n.length() &amp; 1) { cout << 1; modf(1); }
    else modf(0);

    return 0;
}</pre>
