<h1>Cerința</h1>

<p>La un campionat de șah participa <code>n</code> persoane. Știind că la fiecare rundă se fac grupe de câte <code>2</code> persoane, iar câștigătorul trece în runda următoare, aflați după câte runde se va află câștigătorul.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul de runde.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>n ≤ 4.444.444.444</code></li>
	<li>Dacă nu se poate realiza campionatul în conformitate cu regulile de mai sus, se va afișa <code>-1</code>.</li>
</ul>

<h1>Exemplul 1:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">4</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">2</pre>

<h3>Explicație</h3>

<p>În prima rundă se formează <code>2</code> grupe de <code>2</code> participanți. În runda următoare va fi o grupa de <code>2</code>, după care se va află câștigătorul.</p>

<h1>Exemplul 2:</h1>

<p><strong>Intrare</strong></p>

<pre contenteditable="true" editable="true">3</pre>

<p><strong>Ieșire</strong></p>

<pre contenteditable="true" editable="true">-1</pre>

<h3>Explicație</h3>

<p>Nu se poate realiza campionatul cu regulile din enunț.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    unsigned long long n , cnt = 0;
    cin >> n;
    if(n == 1) cout << 0;
    else
    if(n % 2 == 1 || n == 0) cout << -1;
    else
    {
        while(n % 2 == 0)
        {
            cnt++;
            n /= 2;
        }
        if(cnt > 0 &amp;&amp; n == 1) cout << cnt ;
        else cout << -1;
    }
    return 0;
}</pre>
