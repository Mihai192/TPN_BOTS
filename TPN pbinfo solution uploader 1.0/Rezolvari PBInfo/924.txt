<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a <strong>subprogramului recursiv C++</strong> <code>multiplu</code> care are <code>3</code> parametri:  <code>a</code>, prin care primeşte un tablou unidimensional  cu maximum <code>100</code> de numere naturale mai mici decât <code>1000</code>,  <code>n</code>, numărul efectiv de elemente ale tabloului  şi  <code>k</code>, un număr natural.</p>

<p>Subprogramul returnează numărul  de elemente din tablou care sunt multipli ai numărului <code>k</code> şi au ultima cifră egală cu <code>k</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt;= 100</code></li>
	<li><code>k &lt; 10</code></li>
	<li>numele subprogramului cerut este <code>multiplu</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code>, <code>k</code></li>
	<li>elementele vectorului <code>a</code> sunt indexate de la zero</li>
</ul>

<p class="notice"><strong>Se recomandă realizarea unei soluții recursive.</strong></p>

<p>Dacă <code>n=6</code>, <code>a=(9,273,63,83,93,123)</code>, iar <code>k=3</code>, subprogramul va returna valoarea <code>4</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int multiplu(int a[] , int n , int k)
{
    int cnt=0;
    for(int i = 0 ; i < n ; ++i)
    {
        if(a[i]%k==0 &amp;&amp; a[i]%10==k)
            cnt++;
    }
    return cnt;
}</pre>
