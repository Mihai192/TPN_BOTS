<h1>Cerința</h1>

<p>Se dă numărul natural <code>n</code> și se cere să se afișeze cel mai mic număr natural format din cifrele sale. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>mincifre.in</code> conține pe prima linie numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>mincifre.out</code> va conține pe prima linie numărul <code>m</code>, reprezentând cel mai mic număr ce se poate forma cu cifrele numărului din fișierul de intrare.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Numerele citite vor avea cel mult <code>10.000</code> de cifre</li>
</ul>

<h3>Explicație</h3>

<p>Cel mai mic număr ce se poate forma cu cifrele 3,2,1 este 123.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <algorithm>
using namespace std;
ifstream cin("mincifre.in");
ofstream cout("mincifre.out");
int main()
{
    char s[10001];
    int a[10001]={0} , n = 0;
    cin >> s;
    for(int i = 0 ; s[i] ; i++)
        a[++n]=s[i]-'0';
    sort(a+1 , a+n+1);
    int cnt =0 , nr = 0;
    while(a[++nr]==0) cnt++;
    cout << a[cnt+1];
    for(int i = 1 ; i <= cnt ; ++i)
        cout << a[i];
    for(int i = cnt+2 ; i <= n ; ++i)
        cout << a[i];
}</pre>
