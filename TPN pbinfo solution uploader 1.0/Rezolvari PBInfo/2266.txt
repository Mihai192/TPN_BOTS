<p>Un celebru (sperăm) rezolvitor de pe acest site își va lua un calculator nou dacă intră la liceul la care dorește să intre. Dar momentan rămâne pe calculatorul său rezolvând probleme. Una dintre ele era foarte simplă și constă în adunarea/scăderea a două numere naturale.</p>

<h1>Cerința</h1>

<p>Se dau <code>2</code> numere naturale. Calculați suma / diferenţa lor.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>calcmare.in</code> conține pe prima linie numărul <code>a</code>, pe a doua linie numărul <code>b</code>, iar pe a treia linie un semn <code>+</code> sau <code>-</code> care arată dacă numerele trebuie adunate / scăzute.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>calcmare.out</code> va conține pe prima linie numărul <code>R</code>, reprezentând rezultatul determinat.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤</code> nr. cifre număr <code>≤ 1000</code>.</li>
	<li>Pentru <code>50%</code> din punctaj semnul va fi <code>+</code> , iar pentru restul de <code>50%</code>, semnul <code>-</code>.</li>
	<li>Pentru <code>20%</code> din punctaj <code>nr. cifre număr ≤ 15</code>.</li>
	<li>Se garantează că <code>a≥b</code> pentru scăderi.</li>
</ul>

<h3>Explicație</h3>

<p>În fișierul de intrare sunt <code>2</code> numere care trebuie adunate, iar suma lor este <code>444</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include<fstream>
using namespace std;

ifstream cin("calcmare.in");
ofstream cout("calcmare.out");

int main()
{
    char s[1001];
    int A[100001],B[100001],m=0,n=0;
    char c;
    cin>>s;
    for(int i=0;s[i];i++)
        A[++n]=s[i]-'0';
    for(int i=1;i<=n/2;i++)
        swap(A[i],A[n-i+1]);
    cin>>s;
    for(int i=0;s[i];i++)
        B[++m]=s[i]-'0';
    for(int i=1;i<=m/2;i++)
        swap(B[i],B[m-i+1]);
    cin>>c;
    if(c=='+')
    {
        //adun pe B la A
        if(m>n)
            n=m;
        int t=0;//transporturi
        for(int i=1;i<=n;i++)
        {
            int c=A[i]+B[i]+t;
            A[i]=c%10;//cifra
            t=c/10;//transport
        }
        if(t>0)
            A[++n]=t;
        for(int i=n;i>=1;i--)
            cout<<A[i];
    }
    else
    {
        for(int i=1;i<=n;i++)
        {
            if(A[i]-B[i]<0)
                A[i]+=10 , A[i+1]--;
            int c=A[i]-B[i];
            A[i]=c;
        }
        while(A[n]==0)
            n--;
        int ok=0;
        for(int i=n;i>=1;i--)
        {
            cout<<A[i];
            ok++;
        }
        if(!ok)
            cout << 0;
    }
    return 0;
}</pre>
