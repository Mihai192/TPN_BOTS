<h1>Cerinţa</h1>

<p>Scrieți definiția completă a subprogramului <code>numar</code>, care primește prin intermediul parametrului <code>n</code> un număr natural nenul (<code>1≤n≤100</code>), iar prin intermediul parametrului <code>a</code>, un tablou unidimensional care conţine <code>n</code> valori naturale.</p>

<p>Subprogramul furnizează prin parametrul <code>k</code> o valoare naturală egală cu numărul obţinut prin concatenarea valorii maxime cu valoarea minimă din tablou.</p>


<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numele subprogramului cerut este <code>numar</code></li>
	<li>parametrii sunt, în această ordine: <code>n</code>, <code>a</code>, <code>k</code></li>
	<li>elementele vectorului <code>a</code> sunt indexate de la zero</li>
</ul>

<p>Dacă <code>n=6</code> și <code>a= (120, 34, 51, 26, 403, 71)</code>, după apel <code>k=40326</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <algorithm>
int numar(int n , int a[] , int&amp; k)
{
    sort(a , a+n);
    int b=a[0];
    int cnt=1;
    while(b)
    {
        cnt*=10;
        b/=10;
    }
    k=a[n-1]*cnt+a[0];
}</pre>
