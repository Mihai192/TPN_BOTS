<h1>Cerinţa</h1>

<p>Să se scrie o <strong>funcție C++ recursivă</strong> cu trei parametri <code>n</code>, <code>k</code>, <code>c</code> și întoarce prin parametrul <code>c</code> numărul de cifre ale lui <code>n</code> care sunt mai mari sau egale decât <code>k</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numele funcției va fi <code>cnt_cif</code></li>
	<li>funcția va avea trei parametri: <code>n</code>, <code>k</code>, <code>c</code>, cu semnificația de mai sus</li>
	<li><code>0 ≤ n ≤ 2.000.000.000</code></li>
	<li><code>0 ≤ k ≤ 9</code></li>
</ul>

<p>După apelul <code>cnt_cif(2750, 4, c)</code>, valoarea lui <code>c</code> va fi <code>2</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia funcţiei cerute. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">int cnt = 0;
void cnt_cif(int n, int k, int&amp; c){
    if(n == 0 &amp;&amp; cnt == 0 &amp;&amp; k == 0)
        c = 1;
    else if(n == 0)
        c = 0;
    else{
        cnt++;
        if(n % 10 >= k)
            cnt_cif(n/10, k, c), c++;
        else
            cnt_cif(n/10, k, c);
    }
}</pre>
