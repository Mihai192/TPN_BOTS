<p>Se consideră o propoziţie formată din litere mici ale alfabetului englez şi eventual spaţii. Cuvintele sunt formate numai din litere şi sunt separate între ele prin unul sau mai multe spaţii. </p>

<p>Definim <strong>numărul asociat</strong> unui cuvânt <code>c</code><sub><code>1</code></sub><code>c</code><sub><code>2</code></sub><code>...c</code><sub><code>k</code></sub> ca fiind un număr natural <code>nc</code>, obţinut ca produsul puterilor de forma <code>p</code><sup><code>i</code></sup>, unde <code>p</code> este poziţia în alfabet a literei <code>c</code><sub><code>i</code></sub>. Astfel cuvântului <code>badab</code> i se asociază numărul <code>nc=2</code><sup><code>1</code></sup><code>∙1</code><sup><code>2</code></sup><code>∙4</code><sup><code>3</code></sup><code>∙1</code><sup><code>4</code></sup><code>∙2</code><sup><code>5</code></sup>, adică <code>nc=4096</code>.</p>

<p>Definim <strong>gradul unui cuvânt</strong> <code>c</code><sub><code>1</code></sub><code>c</code><sub><code>2</code></sub><code>...c</code><sub><code>k</code></sub> ca fiind numărul <code>nr modulo k</code>, unde <code>nr</code> este numărul de divizori al lui <code>nc</code>. Gradul cuvântului <code>badab</code> este <code>3</code>, pentru că <code>nr=13</code> (cei <code>13</code> divizori ai lui <code>4096</code> sunt: <code>1</code>, <code>2</code>, <code>4</code>, <code>8</code>, <code>16</code>, <code>32</code>, <code>64</code>, <code>128</code>, <code>256</code>, <code>512</code>, <code>1024</code>, <code>2048</code> şi <code>4096</code>), <code>k=5</code> (cuvântul conţine <code>5</code> litere) şi <code>13 modulo 5=3</code>.</p>

<p>Definim <strong>gradul unei propoziţii</strong> ca fiind suma gradelor cuvintelor existente în ea.</p>

<h1>Cerința</h1>

<p>Să se scrie un program care pentru o propoziţie dată determină gradul ei.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>grad.in</code> va conţine pe prima linie o propoziţie.</p>

<h1>Date de ieșire</h1>

<p>Fişierul de ieşire <code>grad.out</code> va conţine pe prima linie gradul propoziţiei.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>Propoziţia are cel mult <code>255</code> de caractere.</li>
	<li>Propoziţia începe şi se termină cu literă, şi este urmată de sfârşit de linie.</li>
	<li><code>n modulo k</code> înseamnă restul împărţirii lui <code>n</code> la <code>k</code>.</li>
	<li>În alfabet litera <code>a</code> este pe poziţia <code>1</code>, <code>b</code> este pe poziţia <code>2</code>, şi aşa mai departe.</li>
</ul>

<h1>Explicație</h1>

<p>Gradul cuvântului <code>de</code> este <code>1</code>, iar gradul cuvântului <code>badab</code> este <code>3</code>, deci gradul propoziţiei citite va fi <code>1+3=4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;

ifstream cin("grad.in");
ofstream cout("grad.out");

char s[256];

int grad(char s[])
{
    int k = strlen(s);
    int div[27] = {0} , g = 1;
    for(int i = 1 ; i <= k ; i++)
    {
        int n = s[i - 1] - 'a' + 1;
        int d = 2;
        while(n > 1)
        {
            int p = 0;
            while(n % d == 0) p++ , n /= d;
            if(p)
            {
                p *= i;
                div[d] += p;
                //cout << d << " " << p << endl;
            }
            d++;
        }
    }
    //for(int i = 1 ; i <= 26 ; i++)
        //cout << div[i] << endl;
    for(int i = 1 ; i <= 26 ; i++)
        g = g * (div[i] + 1) % k;
        //cout << g << endl;
    return g;
}

int main()
{
    char s[256] , *p;
    cin.getline(s , 256);
    int g = 0;
    p = strtok(s , " ");
    while(p != 0)
    {
        g += grad(p);
        p = strtok(NULL , " ");
    }
    cout << g;
    return 0;
