<h1>Cerința</h1>

<p>Anul <code>2017</code> tocmai s-a încheiat, suntem trişti deoarece era număr prim, însă avem şi o veste bună, anul <code>2018</code> este produs de două numere prime, <code>2</code> şi <code>1009</code>. Dorel, un adevărat colecţionar de numere prime, şi-a pus întrebarea: “Câte numere dintr-un interval <code>[a,b]</code> se pot scrie ca produs de două numere prime? “. </p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul natural <code>t</code>, iar apoi <code>t</code> perechi de numere naturale <code>a,b</code> cu <code>a≤b</code>, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran, pentru fiecare pereche <code>a,b</code>, numărul numerelor din intervalul <code>[a,b]</code> care se pot scrie ca produs de două numere prime. </p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ t ≤ 100.000</code></li>
	<li><code>1 ≤ a ≤ b ≤ 1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>În intervalul  <code>[1,7]</code> sunt <code>2</code> numere ce se pot scrie ca produs de două numere prime:  <code>4</code> şi  <code>6</code>. În intervalul <code>[10,30]</code> sunt <code>8</code> numere: <code>10,14,15,21,22,25,26</code>. În intervalul <code>[88,100]</code> sunt <code>4</code> numere: <code>91,93,94,95</code>. </p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int n , a , b , p[1000000] , E[1000000] , s[1000000] , rez[1000000];
int main()
{
    p[0] = p[1] = 1;
    for(int i = 2 ; i <= 500000 ; ++i)
        for(int j = 2 ; i*j <= 1000000 ; ++j)
            p[i*j]=1;
    for(int i = 2 ; i <= 500000 ; ++i)
        for(int j = 2 ; i*j <= 1000000 ; ++j)
            if(p[i]==0 &amp;&amp; p[j]==0)
                E[i*j]=1;
    for(int i = 1 ; i <= 1000000 ; ++i)
    {
        s[i] = s[i-1];
        if(E[i] == 1) s[i]++;
    }
    cin >> n;
    for(int i = 1; i <= n ; ++i)
    {
        cin >> a >> b;
        if(E[a] == 1) rez[i] = s[b]-s[a]+1;
        else rez[i] = s[b]-s[a];
    }
    for(int i = 1 ; i <= n ;i++)
        cout << rez[i] << " ";
    return 0;
}</pre>
