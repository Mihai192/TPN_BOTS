<p>La ora de educație tehnologică a clasei a V-a profesorul Forus, pasionat de matematică, a adus pentru fiecare dintre cei <code>N</code> elevi câte un carton pe care este scris câte un număr natural nenul. Fiecare elev poate folosi cartonul așa cum l-a primit sau poate să taie o singură dată cartonul între două cifre și să lipească partea stângă la finalul părții drepte. Elevul NU are voie să facă o tăietură în fața cifrei <code>0</code>, deci niciunul dintre numerele obținute NU poate să înceapă cu cifra <code>0</code>. Dintre toate numerele pe care le poate obține, elevul îl alege pe cel care are număr minim de divizori, iar dacă poate obține mai multe astfel de numere, îl alege pe cel mai mic dintre ele. La sfârșitul orei, profesorul strânge cartoanele cu numerele alese, în ordinea distribuirii lor. De exemplu, dacă inițial elevul primește cartonul cu numărul <code>25082</code> atunci el are doar următoarele trei variante de tăiere și lipire:</p>

<p><img alt="" src="/resurse/16a4b0/forus.jpg"></p>

<h1>Cerința</h1>

<p>Scrieţi un program care citeşte numărul natural <code>N</code> și cele <code>N</code> numere scrise pe cartoanele aduse de profesorul Forus, apoi rezolvă următoarele două cerinţe:<br>
1. determină numărul de cartoane pe care elevii au voie să le taie de oriunde (NU conțin cifre în fața cărora NU au voie să taie);<br>
2. determină, în ordinea strângerii cartoanelor, numerele preluate de către profesorul Forus la finalul orei.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>forus.in</code> conține pe prima linie un număr natural <code>C</code> reprezentând cerința din problemă care trebuie rezolvată (<code>1</code> sau <code>2</code>). A doua linie din fișier conține un număr natural <code>N</code>, reprezentând numărul de elevi, iar a treia linie din fișier conţine <code>N</code> numere naturale, separate prin câte un spațiu, reprezentând numerele scrise pe cartoanele aduse de profesor, în ordinea distribuirii lor.</p>

<h1>Date de ieșire</h1>

<p>Dacă <code>C = 1</code>, fişierul de ieşire <code>forus.out</code> conţine pe prima linie un număr natural reprezentând răspunsul la cerinţa 1.<br>
Dacă <code>C = 2</code>, fişierul de ieşire <code>forus.out</code> conţine pe prima linie <code>N</code> numere naturale, separate prin câte un spațiu, reprezentând răspunsul la cerința 2; numerele sunt scrise în ordinea în care au fost strânse.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ N ≤ 30</code></li>
	<li><code>1 ≤ numărul natural de pe carton &lt; 1 000 000 000</code></li>
	<li>Pentru rezolvarea corectă a cerinţei 1 se acordă <code>20</code> de puncte; pentru rezolvarea corectă a cerinței 2 se acordă <code>70</code> de puncte.</li>
	<li>În concurs s-au acordat <code>10</code> puncte din oficiu. Aici se acordă <code>10</code> puncte pentru exemplele din enunț.</li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>forus.in</code></p>

<pre contenteditable="true" editable="true">1
5
1234 25082 543 52 150</pre>

<p><code>forus.out</code></p>

<pre contenteditable="true" editable="true">3</pre>

<h3>Explicație</h3>

<p>Cerinţa este <code>1</code>. Sunt <code>3</code> numere care pot fi tăiate de oriunde: <code>1234</code>, <code>543</code>, <code>52</code>.</p>

<h1>Exemplul 2:</h1>

<p><code>forus.in</code></p>

<pre contenteditable="true" editable="true">2
5
51 1234 50822 345 150</pre>

<p><code>forus.out</code></p>

<pre contenteditable="true" editable="true">15 2341 25082 453 501</pre>

<h3>Explicație</h3>

<p>Cerinţa este <code>2</code>. Pentru cartonul cu numărul <code>51</code> se pot obţine numerele <code>15</code> și <code>51</code>. Ambele numere au câte <code>4</code> divizori. Astfel, se va alege numărul <code>15</code>, fiind cel mai mic. Pentru cartonul cu numărul <code>1234</code> (<code>4</code> divizori) pot fi obținute numerele: <code>2341</code> (<code>2</code> divizori), <code>3412</code> (<code>6</code> divizori) și <code>4123</code> (<code>8</code> divizori). Se va alege numărul <code>2341</code> pentru că are numărul minim de divizori. Analog se va proceda pentru toate celelalte numere din șir.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cmath>
using namespace std;

ifstream cin("forus.in");
ofstream cout("forus.out");

int n , cer , x , p , cnt , y;
int  put , nrmax , val , nrc;

int nrdiv(int n)
{
    int d=2 , prod = 1;
    while(n > 1)
    {
        int p = 0;
        while(n % d == 0) n /= d , p++;
        if(p) prod *= (p + 1);
        d++;
        if(d * d > n) d = n;
    }
    return prod;
}

int p10(int n)
{
    int p = 1;
    for(int i = 1 ; i <= n ; ++i) p *= 10;
    return p;
}

int main()
{
    cin >> cer >> n;
    if(cer == 1)
    {
        for(int i = 1 ; i <= n ; i++)
        {
            cin >> x;
            int zero = 0;
            while(x != 0)
            {
                if(x % 10 == 0) zero++;
                x /= 10;
            }
            if(zero == 0) cnt++;
            }
        cout << cnt;
    }
    else
    {
        for(int i = 1 ; i <= n ; i++)
        {
            cin >> x;
            y = log10(x);
            put = p10(y);
            nrmax = nrdiv(x);
            val = x;
            for(int j = 1 ; j <= y; ++j)
            {
                x = x % put * 10 + x / put;
                if(x / put != 0)
                {
                    nrc = nrdiv(x);
                    if(nrc < nrmax) nrmax = nrc , val = x;
                    else if(nrc == nrmax &amp;&amp; val > x) val = x;
                }
            }
            cout << val << " ";
        }
    }
    return 0;
}</pre>
