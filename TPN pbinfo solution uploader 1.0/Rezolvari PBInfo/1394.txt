<h1>Enunț</h1>

<p>Într-o zi, Gigel a găsit pe masa tatălui său o foaie A4 pe care era trecut șirul denumit “devt” sub forma <code>1, 4, 6, 8, 9, 10, 12, 14, 15, 16, 18, 20, 21, 22, 24, 25, ... , n</code>. Dedesubtul acestui șir găsește un text alcătuit din <code>k</code> întrebări de forma <code>a</code>, <code>b</code> cu semnificația “Câte numere din acest șir se află în intervalul <code>[a,b]</code>?”.</p>

<h1>Cerința</h1>

<p>Ajutați-l pe Gigel să răspundă corect la toate cele <code>k</code> întrebări.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>devt.in</code> conține pe prima linie numerele naturale <code>n</code> și <code>k</code>, iar pe următoarele <code>k</code> linii numerele <code>a</code>, <code>b</code> cu semnificația din enunț.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>devt.out</code> va conține <code>k</code> linii, pe fiecare linie <code>i</code> aflându-se un număr natural, reprezentând răspunsul întrebării <code>i</code>. </p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 ≤ a ≤ b ≤ n ≤ 100000</code></li>
	<li><code>1 ≤ k ≤ 5000</code></li>
	<li><code>n</code> aparține șirului devt</li>
</ul>

<h3>Explicație</h3>

<p>În intervalul <code>[3,7]</code> se află <code>2</code> numere din șir. <br>
În intervalul <code>[12,20]</code> se află <code>6</code> numere din șir.<br>
În intervalul <code>[3,4]</code> se află <code>1</code> număr din șir.<br>
În intervalul <code>[16,24]</code> se află <code>6</code> numere din șir.<br>
În intervalul <code>[12,24]</code> se află <code>9</code> numere din șir.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream fin ("devt.in");
ofstream fout ("devt.out");

int x[100001];
int s[100001];

int main ()
{
    int n,q,a,b;
    x[1]=1;
    fin >> n;
    for (int i=2;i<=n;++i)
        if(x[i]==0)
            for (int j=i+i;j<=n;j=j+i)
                x[j]=1;
    for (int i=1;i<=n;++i)
        s[i]=s[i-1]+x[i];
    fin >> q;
    for (int i=1;i<=q;++i)
    {
        fin>>a>>b;
        fout <<s[b]-s[a-1]<< endl;
    }
    
    fin.close();
    fout.close();
    
    return 0;
}</pre>
