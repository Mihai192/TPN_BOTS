<h1>Cerinţa</h1>

<p>Se citește un număr natural <code>n</code>. Să se determine numărul de divizori ai oglinditului lui <code>n</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>D</code>, reprezentând rezultatul cerut.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 1.000.000.000</code></li>
</ul>

<h1>Explicație</h1>

<p>Oglinditul lui <code>63</code> este <code>36</code>, care are <code>9</code> divizori.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

using namespace std;

int main()
{
    int n,r=0;
    cin >> n;
    while(n)
    {
    r=r*10+n%10;
    n=n/10;
    }
    int cnt=0;
    for (int i = 1; i*i <= r ; ++i)
    {
        if (r%i==0)
            cnt+=2;
        if (i*i==r)
            cnt-=1;;
    }
    cout << cnt;
    return 0;
}</pre>
