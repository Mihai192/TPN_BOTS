<h1>Cerinţa</h1>

<p>Se dau două şiruri cu câte <code>n</code> elemente, numere  naturale nenule. Să se verifice dacă este posibilă rearanjarea elementelor celor două şiruri astfel încât acestea să fie direct proporţionale.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi cele <code>n</code> elemente ale şirului, separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran mesajul <code>DA</code>, dacă este posibilă rearanjarea elementelor celor două şiruri astfel încât acestea să fie direct proporţionale, respectiv <code>NU</code> în caz contrar.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>elementele şirurilor vor avea cel mult <code>4</code> cifre</li>
	<li>se recomandă evitarea lucrului cu numere reale</li>
</ul>

<h1>Explicaţie</h1>

<p>Numerele <code>(10 10 15 30 25 15)</code> şi <code>(6 6 9 18 15 9)</code> sunt proporţionale. Raportul este <code>5/3</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    int n , a[101] , b[101] , s1 = 0 , s2 = 0 , cate = 0;
    cin >> n;
    for(int i = 0 ; i < n ; ++i)
    {
        cin >> a[i];
        s1+=a[i];
    }
    for(int i = 0 ; i < n ; ++i)
    {
        cin >> b[i];
        s2+=b[i];
    }
    for(int i = 0 ; i < n ; ++i)
    {
        int cnt=0;
        for(int j = 0 ; j < n ; ++j)
        {
            if((double)a[i] / b[j] == (double)s1 / s2)
            cnt++;
        }
        if(cnt!=0) cate++;
    }
    if(cate == n) cout << "DA";
    else cout << "NU";
    return 0;
}</pre>
