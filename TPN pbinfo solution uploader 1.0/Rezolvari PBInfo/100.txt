<p>Se citeşte <code>n</code> şi un şir de <code>n</code> numere naturale.</p>

<h1>Cerinţa</h1>

<p>Să se determine câte elemente din şir sunt numere prime.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>nrapprime.in</code> conţine pe prima linie numărul <code>n</code> şi pe a doua linie <code>n</code> numere naturale separate prin spaţii.</p>

<h1>Date de ieşire</h1>

<p>Fişierul de ieşire <code>nrapprime.out</code> va conţine pe prima linie un singur număr natural, reprezentând numărul de elemente prime din şir.</p>

<h1>Restricţii şi precizări</h1>

<ul>
    <li><code>0 < n < 100</code></li> 
    <li>numerele de pe a doua linie a fişierului de intrare vor fi mai mici decât <code>1.000.000.000</code></li>
</ul>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
int prim (int n)
{
    if(n==0 || n==1) return 0;
    if(n==2) return 1;
    if(n%2==0) return 0;
    for (int i=3;i*i<=n;i += 2)
    {
        if(n%i==0) return 0;
    }
    return 1;
}
int main ()
{
    ifstream fin ("nrapprime.in");
    ofstream fout ("nrapprime.out");
    int n,x,cate=0;
    fin >> n;
    for (int j=1;j<=n;++j)
    {
        fin >> x;
        if(prim(x)) cate++;
    }
    fout << cate;
    fin.close();
    fout.close();
    return 0;
}</pre>
