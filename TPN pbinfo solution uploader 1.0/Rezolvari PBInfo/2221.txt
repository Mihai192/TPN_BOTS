<h1>Cerința</h1>

<p>Se dă un număr natural <code>n</code>. Să se afle câte dintre numerele obținute din <code>n</code> prin ștergerea unei cifre, sunt divizibile cu <code>3</code>.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul numerelor obţinute din <code>n</code> prin ștergerea unei cifre, care sunt divizibile cu <code>3</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2.000.000.000</code></li>
	<li>dacă  <code>n</code> are o cifră, prin ştergerea unei cifre el devine <code>0</code></li>
	<li>dacă prin ştergerea unei cifre numărul rămas începe cu <code>0</code>, zerourile de la început se elimină</li>
</ul>

<h3>Explicație</h3>

<p>Prin eliminarea cifrei <code>1</code> se obţine numărul <code>2370</code> divizibil cu <code>3</code>, iar prin eliminarea cifrei <code>7</code> se obţine <code>2301</code> divizibil cu <code>3</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    int n,x,cate=0;
    cin>>n;
    x=n%3;
    while(n>0)
    {
        if((n%10)%3==x) cate++;
        n=n/10;
    }
    cout<<cate;
    return 0;
}</pre>
