<p>Se consideră o matrice <code>A</code> cu următoarele proprietăţi:</p>

<ul>
	<li>conţine <code>n</code> linii şi <code>m</code> coloane;</li>
	<li>conţine doar valorile <code>0</code> şi <code>1</code>;</li>
	<li>pe fiecare linie valorile sunt plasate în ordine crescătoare.</li>
</ul>

<p>Definim o submatrice determinată de perechea de linii <code>L1</code> şi <code>L2</code> (<code>L1 ≤ L2</code>) şi de perechea de coloane <code>C1</code> şi <code>C2</code> (<code>C1 ≤ C2</code>) ca fiind totalitatea elementelor matricei <code>A[i,j]</code> pentru care <code>L1 ≤ i ≤ L2</code> şi <code>C1 ≤ j ≤ C2</code>.<br>
Dacă toate elementele unei submatrice sunt egale cu <code>0</code>, atunci submatricea se numeşte nulă.<br>
Asupra matricei <code>A</code> putem efectua una sau mai multe operaţii de interschimbări de linii. Prin astfel de interschimbări liniile matricei pot fi rearanjate astfel încât matricea <code>A</code> să conţină cel puţin o submatrice nulă cu număr maxim de elemente.</p>

<h1>Cerința</h1>

<p>Fiind dată o astfel de matrice se cere să se determine numărul maxim de zerouri dintr-o submatrice nulă ce se poate obţine printr-o rearanjare a liniilor matricei date.</p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>submat1.in</code> conţine pe prima linie două numere naturale <code>n m</code>, separate printr-un spaţiu, reprezentând numărul de linii, respectiv numărul de coloane ale matricei <code>A</code>. Pe următoarele <code>n</code> linii ale fişierului sunt descrise cele <code>n</code> linii ale matricei <code>A</code>; pe fiecare linie din cele <code>n</code> vor fi scrise câte <code>m</code> valori din mulţimea <code>{0, 1}</code>, separate prin spaţii, reprezentând în ordine elementele liniei respective.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>submat1.out</code> va conţine o singură linie pe care va fi scris un număr natural reprezentând numărul maxim de elemente pe care îl conţine o submatrice nulă rezultată în urma rearanjărilor liniilor matricei <code>A</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>2 ≤ n, m ≤ 1000</code></li>
	<li>Pentru <code>60%</code> din teste <code>n, m ≤ 100</code>.</li>
</ul>

<h3>Explicație</h3>

<p>Dacă rearanjăm liniile astfel încât matricea rezultată să fie următoarea:<br>
<code>0 0 0 1 1</code><br>
<code>0 0 0 0 1</code><br>
<code>0 1 1 1 1</code><br>
atunci se observă că submatricea determinată de prima şi a doua linie şi de prima şi a treia coloană conţine <code>6</code> valori de <code>0</code> (<code>6</code> fiind numărul maxim posibil).</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream cin("submat1.in");
ofstream cout("submat1.out");
int main()
{
    int n , m , x , cntmax = 0 , f[1001]={0};
    cin >> n >> m;
    for(int i = 0 ; i < n; ++i)
        for(int j = 0 ; j < m ; ++j)
        {
            cin >> x;
            if(x==0) f[i]++;
        }
    for(int i = 1 ; i <= m ; ++i)
    {
        int cnt = 0;
        for(int j = 0  ; j < n ; ++j)
            if(f[j]>=i) cnt+=i;
        if(cnt > cntmax) cntmax = cnt;
    }
    cout << cntmax;
    return 0;
}</pre>
