<h1>Cerința</h1>

<p>Se dau <code>n</code> numere naturale prime. Pentru <code>t</code> perechi de numere naturale <code>s</code> şi <code>d</code> să se afle câte numere naturale din intervalul <code>[s,d]</code> sunt divizibile prin cel puţin unul dintre cele <code>n</code> numere prime. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>eratostene8.in</code> conține pe prima linie numerele <code>n</code> şi <code>t</code>, pe a doua linie cele <code>n</code> numere prime, iar pe următoarele <code>t</code> linii câte o pereche de numere <code>s</code> şi <code>d</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>eratostene8.out</code> va conține pe linia <code>i</code> răspunsul la întrebarea <code>i</code>, pentru orice <code>i</code> de la <code>1</code> la <code>t</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 10.000</code></li>
	<li><code>1 ≤ t ≤ 100.000</code></li>
	<li><code>1 ≤ s ≤ d ≤ 10.000.000</code></li>
	<li>numerele prime sunt mai mici sau egale cu <code>1.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numerele prime date sunt <code>2</code> şi <code>3</code>. În intervalul <code>[1,5]</code> sunt 3 numere divizibile cu 2 sau 3, în intervalul <code>[4,6]</code> sunt 2 numere divizibile cu 2 sau 3, iar în intervalul <code>[5,20]</code> sunt 10 numere divizibile cu 2 sau 3.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;

ifstream cin("eratostene8.in");
ofstream cout("eratostene8.out");

int n,t,s,d,E[10000001], P[10001];
bool F[1000001];

int main(){
    cin >> n >> t;
    for(int i = 1; i <= n; ++i)
        cin >> P[i];
    for(int i = 1; i <= n; ++i){
        E[P[i]]=1;
        if(F[P[i]] == 0){
            for(int j = 2; j * P[i] <= 10000000; ++j)
                E[P[i] * j] = 1;
            F[P[i]] = 1;
        }
    }
    for(int i = 1; i <= 10000000; ++i)
        E[i] = E[i-1] + E[i];
    for(int i = 1; i <= t; ++i){
        cin >> s >> d;
        cout << E[d] - E[s-1] << '\n';
    }
}</pre>
