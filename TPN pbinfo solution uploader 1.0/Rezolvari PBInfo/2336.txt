<h1>Cerința</h1>

<p>Dorel are o nouă pasiune, să coloreze divizorii unui număr natural. Primind cadou de ziua lui un număr natural <code>n</code>, s-a gândit să coloreze toate numerele naturale de la <code>1</code> la <code>n</code> cu câte o culoare, astfel încât toţi divizorii proprii ai unui număr să aibă aceeaşi culoare cu numărul. Vă roagă pe voi să aflaţi care este numărul maxim de culori care pot fi folosite. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>primcolor.in</code> conține pe prima linie numărul <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>primcolor.out</code> va conține pe prima linie numărul maxim de culori ce pot fi folosite.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 50.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Numerele <code>1,2,3,4,5</code> pot fi colorate astfel : <font color="red">1</font>, <font color="green">2</font>,<font color="blue">3</font>,<font color="green">4</font>,<font color="yellow">5</font>. Se folosesc astfel <code>4</code> culori. Numerele <code>2</code> şi <code>4</code> trebuie colorate la fel deoarece <code>2</code> este divizor propriu al lui <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <bitset>
using namespace std;
ifstream cin("primcolor.in");
ofstream cout("primcolor.out");
bitset<50000000> ciur;
int main()
{
    for(int i = 2 ; i * i <= 50000000 ; i++)
        if(ciur[i]==0)
            for(int j = i * i; j <= 50000000; j += i) ciur[j]=1;
    int n , cnt=2;
    cin >> n;
    if(n == 1)cout << 1;
    else if(n == 2)cout << 2;
    else if(n == 3)cout << 3;
    else if(n == 4)cout << 3;
    else
    {
        for(int i = n/2 + 1 ; i <= n ; ++i)
            if(ciur[i]==0) cnt++;
        cout << cnt;
    }
    return 0;
}</pre>
