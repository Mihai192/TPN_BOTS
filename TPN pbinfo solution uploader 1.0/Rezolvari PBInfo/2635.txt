<h1>Cerința</h1>

<p>Marcel a primit o tastatura noua de ziua lui. El vrea sa scrie un text folosind doar litere ale alfabetului englez si spatii. El a descoperit tasta <code>CapsLock</code>, care atunci cand este apasata se activeaza. Daca tasta este activata, atunci toate literele care le va scrie in continuare, pana in momentul in care tasta este dezactivata (apasand <code>CapsLock</code> inca o data) vor fi majuscule. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>capslock.in</code> conține pe prima linie tastele apasate de Marcel (daca tasta e litera, atunci este codificata prin litera mica a acestia, daca e spatiu atunci va fi codificata printr-un spatiu iar daca e <code>CapsLock</code> va fi codificata prin simbolul <code>#</code>), fara spatii intre ele;</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>capslock.out</code> va conține pe prima linie textul scris de Marcel.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li>textul este mai scurt de <code>255</code> de caractere</li>
	<li>el va apăsa tasta <code>CapsLock</code> de maxim <code>35</code> ori</li>
	<li>la început tasta <code>CapsLock</code> este dezactivata.</li>
	<li>spatiile nu sunt afectate de <code>CapsLock</code></li>
</ul>

<h3>Explicație</h3>

<p>Tasta CapsLock devine activa de la început și se dezactivează după tasta <code>s</code> → el a scris majuscula <code>S</code>. Tasta CapsLock se activează și la al <code>2</code>-lea cuvânt și rămâne activă → restul literelor vor fi majuscule.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <cstring>
using namespace std;
ifstream fin("capslock.in");
ofstream fout("capslock.out");
char s[256];

int main()
{
    fin.getline(s,256);
    for(int i=0; i<strlen(s); i++)
        if(s[i] == '#')
    {
        int j = i+1;
        while(s[j] &amp;&amp; s[j] != '#')
            {
                s[j] = toupper(s[j]);
                j++;
            }

        i = j;
    }

    for(int i=0; i<strlen(s); i++)
        if(s[i]!='#')
          fout << s[i];
    return 0;
}</pre>
