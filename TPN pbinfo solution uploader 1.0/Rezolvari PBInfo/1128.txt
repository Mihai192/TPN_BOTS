<h1>Cerința</h1>

<p>La o grădiniță, cei <code>m</code> copii de la grupa mică s-au trezit în fața a <code>n</code> jucării diferite. Cel mai isteț dintre ei vă întreabă în câte moduri ar putea să-și aleagă fiecare câte o jucărie ?</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numerele naturale <code>m</code> și <code>n</code>.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran numărul <code>q</code>, reprezentând numărul de moduri în care pot să-și aleagă fiecare câte o jucărie. Pentru că acest număr poate fi foarte mare, se va afișa modulo <code>224737</code>.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n , m ≤ 100.000</code></li>
	<li>grădinița este virtuală</li>
</ul>

<h1>Explicație</h1>

<p>Avem <code>2</code> copii și <code>4</code> jucării ( notate <code>a</code>, <code>b</code>, <code>c</code>, <code>d</code> ). Cei doi copii pot lua jucăriile <code>(a,b)</code>, <code>(a,c)</code>, <code>(a,d)</code>, <code>(b,c)</code>, <code>(b,d)</code>, <code>(c,d)</code>, <code>(b,a)</code>, <code>(c,a)</code>, <code>(d,a)</code>,  <code>(c,b)</code>, <code>(d,b)</code>, <code>(d,c)</code>, în total <code>12</code> variante de alegere.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main()
{
    int n , m;
    cin >> m >> n;
    if(m > n) cout << 0;
    else
    {
        long long p = 1;
        for(int i = 1 ; i <= m ; ++i)
        {
            p *= (n - m + i);
            if(p > 224737) p %= 224737;
        }
        cout << p%224737;
    }
}</pre>
