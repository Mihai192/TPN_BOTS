<h1>Cerinţa</h1>

<p>Scrieţi definiția completă a subprogramului C++ <strong>recursiv</strong> <code>ordonare</code> care are <code>2</code> parametri:  <code>a</code>, prin care primeşte un tablou unidimensional  cu maximum <code>1000</code> de numere naturale mai mici decât <code>1.000.000.000</code> și <code>n</code>, numărul efectiv de elemente ale tabloului.</p>

<p>Subprogramul ordonează crescător elementele tabloului <code>a</code>, fără a returna valori.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>0 &lt; n &lt;= 1000</code></li>
	<li>numele subprogramului cerut este <code>ordonare</code></li>
	<li>parametrii sunt, în această ordine: <code>a</code>, <code>n</code></li>
	<li>elementele tabloului <code>a</code> sunt indexate de la zero</li>
	<li><strong>se recomandă realizarea unei soluții recursive</strong></li>
</ul>

<p>Dacă <code>n=6</code> și <code>a=(63,273,9,83,93,123)</code>,  după apelul subprogramului tabloul <code>a</code> devine <code>(9,63,83,93,123,273)</code>.</p>

<h1>Important</h1>

<p>Soluţia propusă va conţine doar definiţia subprogramului cerut. Prezenţa în soluţie a altor instrucţiuni poate duce erori de compilare sau de execuţie care vor avea ca efect depunctarea soluţiei.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">void ordonare(int a[] , int n)
{
    for(int i = 1 ; i < n ; ++i)
    {
        if(a[i]<a[i-1])
            swap(a[i],a[i-1]);
        
    }
    if(n > 0)
        ordonare(a , n - 1);
}</pre>
