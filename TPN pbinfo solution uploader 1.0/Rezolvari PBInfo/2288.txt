<h1>Cerinţă</h1>

<p>Să se scrie un program care să determine cea mai mare cifră impară a unui număr natural citit de la tastatură.</p>

<h1>Date de intrare</h1>

<p>Programul va citi de la tastatură un număr natural.</p>

<h1>Date de ieşire</h1>

<p>Programul va afişa pe ecran cea mai mare cifră impară a numărului citit.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numărul citit va fi mai mic decât <code>2.000.000.000</code></li>
	<li>dacă numărul citit nu conține cifre impare se va afișa mesajul <code>nu exista</code></li>
</ul>

<h1>Exemplul 1</h1>

<p><code>Intrare</code></p>

<pre contenteditable="true" editable="true">2785</pre>

<p><code>Ieşire</code></p>

<pre contenteditable="true" editable="true">7</pre>

<h1>Exemplul 2</h1>

<p><code>Intrare</code></p>

<pre contenteditable="true" editable="true">2064</pre>

<p><code>Ieşire</code></p>

<pre contenteditable="true" editable="true">nu exista</pre>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>

using namespace std;

int main()
{
    int n;
    cin >> n;
    int cnt=0;
    int max=0;
    while (n)
    {
        if ((n%10)%2==1 &amp;&amp; n%10>max)
        {
            max=n%10;
            cnt++;
        }
        n/=10;
    }
    if (cnt==0)
        cout << "nu exista";
    else
        cout << max;
    return 0;
}</pre>
