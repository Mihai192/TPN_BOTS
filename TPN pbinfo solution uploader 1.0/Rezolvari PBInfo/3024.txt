<h1>Cerința</h1>

<p>Pe strada lui Dorel casele sunt aşezate doar de o parte a străzii. Cu ocazia sărbătorilor de Paşti, fiecare proprietar împarte ouă roşii vecinilor cei mai apropiaţi de casa lui. Se ştie că pe strada lui Dorel sunt <code>n</code> case, fiecare proprietar <code>i</code> are <code>O[i]</code> ouă, fiecare proprietar împarte ouă la un număr egal de case situate în stânga şi în dreapta lui, de asemenea fiecare vrea să împartă ouă la un număr maxim de case, un număr egal de ouă, cât mai mare, la fiecare casă. Dacă obiceiul de împărţire a ouălor are loc simultan, aflaţi câte ouă va avea fiecare la sfârşitul zilei. </p>

<h1>Date de intrare</h1>

<p>Fișierul de intrare <code>ou.in</code> conține pe prima linie numărul <code>n</code>, iar pe a doua linie <code>n</code> numere naturale separate prin spații, reprezentând numărul ouălor din fiecare casă.</p>

<h1>Date de ieșire</h1>

<p>Fișierul de ieșire <code>ou.out</code> va conține pe prima linie <code>n</code> numere, reprezentând numărul ouălor avute de fiecare proprietar la sfârşitul zilei.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>3 ≤ n ≤ 100.000</code></li>
	<li>numerele de pe a doua linie a fișierului de intrare vor fi naturale nenule, mai mici decât <code>1.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Al doilea proprietar împarte câte un ou la câte un vecin stânga-dreapta, al treilea câte un ou la cei patru vecini iar al patrulea câte 22 de ouă la cei doi vecini cei mai apropiaţi.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
#include <vector>

using namespace std;

ifstream cin ("ou.in");
ofstream cout ("ou.out");

#define Max 100001
long long n;
long long a[Max], anou[Max];
int main()
{
    cin >> n;
    for (long long i = 1, nro, bpo; i <= n; i ++)
    {
        cin >> a[i];
        nro = min(i - 1, n - i);
        if (nro * 2 > a[i])
            nro = a[i]/2;

        if (nro)
        {
            bpo = a[i] / (nro * 2);

            anou[i] += (a[i] - bpo*nro*2);
            anou[i + 1] -= (a[i] - bpo*nro*2);

            anou[i - nro] += bpo;
            anou[i] -= bpo;

            anou[i + 1] += bpo;
            anou[i + nro + 1] -= bpo;
        }
        else
        {
            anou[i] += a[i];
            anou[i + 1] -= a[i];
        }
    }
    for (int i = 1; i <= n; i ++)
        anou[i] += anou[i - 1];
    for (int i = 1; i <= n; i ++)
        cout << anou[i] << ' ';
    return 0;
}</pre>
