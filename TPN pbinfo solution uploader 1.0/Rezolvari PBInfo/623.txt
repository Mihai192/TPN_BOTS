<p>Având o pasiune pentru numere, Răzvan a inventat un nou joc pentru două persoane. Regulile jocului sunt simple: se aleg trei numere naturale <code>a</code>, <code>b</code> şi <code>c</code> astfel încât <code>a&lt;b&lt;c</code>. La fiecare pas, primul jucător adună la <code>a</code> suma cifrelor lui <code>a</code>, iar cel de-al doilea jucător scade din <code>c</code> suma cifrelor lui <code>c</code>, la pasul următor fiind luate în considerare noile valori obţinute pentru <code>a</code> şi <code>c</code>. Jocul se termină când, la un anumit pas, <code>a</code> devine mai mare sau egal decât <code>b</code>, caz în care primul jucător este declarat câştigător, sau când <code>c</code> devine mai mic sau egal cu <code>b</code>, situaţie în care câştigă al doilea jucător. Dacă la un pas se îndeplinesc ambele codiţii, jocul se termină la egalitate.</p>

<h1>Cerința</h1>

<p>Cunoscând <code>a</code>, <code>b</code> şi <code>c</code> se cere:</p>

<p>a)	Suma cifrelor celor trei numere.<br>
b)	Valorile numerelor <code>a</code> şi <code>c</code> la fiecare pas al jocului, precum şi câştigătorul jocului.</p>

<h1>Date de intrare</h1>

<p>Fişierul de intrare <code>numere3.in</code> conţine pe prima linie un număr natural <code>p</code>. Pentru toate testele de intrare, numărul <code>p</code> poate avea doar valoarea <code>1</code> sau valoarea  <code>2</code>.</p>

<p>Pe a doua linie se găsesc cele trei numere naturale <code>a</code>, <code>b</code> şi <code>c</code> descrise în enunţ.</p>

<h1>Date de ieșire</h1>

<p>Dacă valoarea lui <code>p</code> este <code>1</code>,  se va rezolva numai punctul a) din cerință. </p>

<p>În acest caz, în fişierul de ieşire <code>numere3.out</code> se va scrie un singur număr natural <code>s</code>, reprezentând suma cifrelor lui <code>a</code>, <code>b</code> şi <code>c</code>.</p>

<p>Dacă valoarea lui <code>p</code> este <code>2</code>, se va rezolva numai punctul b) din cerință. </p>

<p>În acest caz, fişierul de ieşire <code>numere3.out</code> va conține mai multe linii pe care se vor găsi câte două numere naturale separate printr-un spaţiu: valorile lui <code>a</code> şi <code>c</code> la fiecare pas al jocului. Pe ultima linie se va găsi <code>1</code>, dacă primul jucător câştigă, <code>2</code> dacă cel de-al doilea jucător câştigă sau <code>0</code> dacă jocul se termină la egalitate.</p>


<h1>Restricții și precizări</h1>

<ul>
	<li><code>0 &lt; a &lt; b &lt; c &lt; 10.000.000</code></li>
	<li>Pentru rezolvarea corectă a primei cerinţe se acordă 20 de puncte, iar pentru cerința a doua se acordă 80 de puncte.</li>
</ul>

<h1>Exemplul 1:</h1>

<p><code>numere3.in</code></p>

<pre contenteditable="true" editable="true">1
25 51 75</pre>

<p><code>numere3.out</code></p>

<pre contenteditable="true" editable="true">25</pre>

<h1>Explicație</h1>

<p><code>p = 1</code><br>
<code>2+5+5+1+7+5=25</code></p>

<p><strong>Atenție! Pentru acest test se rezolvă doar cerința a)</strong></p>

<h1>Exemplul 2:</h1>

<p><code>numere3.in</code></p>

<pre contenteditable="true" editable="true">2

<p><code>numere3.out</code></p>

<pre contenteditable="true" editable="true">25 75
32 63

2</pre>

<h1>Explicație</h1>

<p><code>p = 2</code></p>

<p>La cel de-al treilea pas, <code>c</code> devine mai mic decât <code>b</code>, jocul fiind câștigat de către al doilea jucător.</p>

<p><strong>Atenție! Pentru acest test se rezolvă doar cerința b).</strong></p>

<h1>Exemplul 3:</h1>

<p><code>numere3.in</code></p>

<pre contenteditable="true" editable="true">2
25 46 75</pre>

<p><code>numere3.out</code></p>

<pre contenteditable="true" editable="true">25 75
32 63
37 54
47 45
0</pre>

<h1>Explicație</h1>

<p><code>p = 2</code></p>

<p>La cel de-al treilea pas, <code>a</code> devine mai mare decât <code>b</code>, dar și <code>c</code> devine mai mic decât <code>b</code>. Astfel, jocul se termină la egalitate.</p>

<p><strong>Atenție! Pentru acest test se rezolvă doar cerința b).</strong></p>

ofstream fout("numere3.out");

{
    int s = 0;
    while(n != 0)
    {
        s += n % 10;
        n /= 10;
    }
    return s;
}
int main()
{
    int a , b , c , p;
    fin >> p >>a >> b >> c;
    if(p == 1)
    {
        fout << sum(a) + sum(b) + sum(c);
    }
    else
    {
        fout << a <<" " << c << endl;
        while(a < b &amp;&amp; c > b)
        {
            a = a + sum(a);
            c = c - sum(c);
            fout << a <<" "<< c << endl;
        }
        if(a >= b &amp;&amp; c <=b) fout << 0;
        else if(a >= b &amp;&amp; c > b) fout << 1;
        else fout << 2;
    }

    fout.close();

    return 0;
