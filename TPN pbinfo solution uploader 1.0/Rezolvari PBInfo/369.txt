<h1>Cerinţă</h1>

<p>Se dă un număr natural <code>n</code>. Calculaţi oglinditul sumei cifrelor sale.</p>

<h1>Date de intrare</h1>

<p>Programul va citi de la tastatură un număr natural <code>n</code>.</p>

<h1>Date de ieşire</h1>

<p>Programul va afişa pe ecran oglinditul sumei cifrelor lui <code>n</code>.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li>numărul citit va fi mai mic decât <code>2</code><sup><code>31</code></sup>.</li>
</ul>

<h1>Explicație</h1>

<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int main ()
{
    int n,s=0,ogl=0;
    cin >> n;
    while (n != 0)
    {
        s=s+n%10;
        n=n/10;
    }

    while (s)
    {
        ogl=ogl*10+s%10;
        s=s/10;
    }
    cout << ogl;
    return 0;


}</pre>
