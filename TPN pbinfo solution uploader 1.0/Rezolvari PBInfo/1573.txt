<h1>Cerința</h1>

<p>Se dau <code>n</code> valori naturale. Stabiliți dacă există o progresie aritmetică cu rația număr natural mai mare decât <code>1</code> din care să facă parte toate aceste valori.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> numere naturale, separate prin spații.</p>

<h1>Date de ieșire</h1>

<p>Programul va afișa pe ecran mesajul <code>DA</code> , dacă există progresia aritmetică cerută, respectiv <code>NU</code> în caz contrar.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 2500</code></li>
	<li>cele <code>n</code> numere citite vor fi mai mici decât <code>2.000.000.000</code></li>
</ul>

<h3>Explicație</h3>

<p>Rația este <code>2</code> , lipsesc termenii <code>4</code>, <code>6</code>, <code>8</code>, <code>14</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
#include <algorithm>
using namespace std;
int cmmdc(int a , int b)
{
    int r;
    while(b!=0)
    {
        r=a%b;
        a=b;
        b=r;
    }
    return a;
}
int main()
{
    int n , a[3001];
    cin >> n;
    for(int i = 1 ; i <= n ; ++i) cin >> a[i];
    sort(a+1 , a+n+1);
    int max = a[2]-a[1];
    for(int i = 2 ; i < n ; ++i)
        max=cmmdc(max,a[i+1]-a[i]);
    if(max>1) cout << "DA";
    else cout << "NU";
}</pre>
