<p>Puteţi defini şi alte subprograme care vă ajută să rezolvaţi mai uşor problema.</p>

    while (n%3==0){n=n/3;}
    while (n%5==0){n=n/5;}
    if (n==1) return 1;
    else return 0;
}</pre>
