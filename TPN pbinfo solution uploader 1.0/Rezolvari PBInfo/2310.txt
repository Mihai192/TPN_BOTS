<h1>Cerința</h1>

<p>Pacală şi Tândală mergeau printr-o pădure. Într-un copac, ei au  văzut o pasăre.<br>
- Măi, Tândală, uite o pasăre! Hai s-o prindem, s-o frigem şi s-o mâncăm boiereşte, zise Păcală lingându-şi degetele. <br>
- Că bine zici. Dar cum s-o mâncăm fără sare? Stai tu s-o păzeşti ca să nu zboare şi eu dau fuga în sat să aduc nişte sare, zise Tândală.<br>
- Du-te în pivniţa mea şi acolo vei găsi <code>N</code> saci, pe fiecare sac vezi că e o etichetă pe care scrie un număr; dacă <strong>numărul are toate cifrele în ordine strict crescătoare, de la stânga la dreapta</strong>, înseamnă că acel sac conţine sare, altfel conţine zahăr. Încarcă în căruţă toţi sacii cu sare şi hai cu ei încoace până păzesc eu pasărea, zise Păcală.</p>

<p>Ajutaţi-l pe Tândală să numere câţi saci cu sare trebuie să încarce în căruţă.</p>


<h1>Date de intrare</h1>

<p>De pe prima linie a fişierului <code>saci.in</code> se citeşte un număr natural <code>N</code> ce reprezintă numărul sacilor.<br>
De pe următoarele <code>N</code> linii se citesc valori naturale, câte una de pe fiecare linie, reprezentând numărul scris pe eticheta unui sac.</p>


<h1>Date de ieșire</h1>

<p>În fişierul <code>saci.out</code> se va afişa, pe prima linie, un număr natural ce  reprezintă <strong>numărul de saci</strong> cu sare ce trebuie încărcaţi în căruţă.</p>

<h1>Restricții și precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>numărul scris pe fiecare sac este un număr natural nenul, cu cel mult <code>8 cifre</code></li>
</ul>

<h3>Explicație</h3>

<p>Sunt 5 saci. Dintre numerele cu care sunt etichetaţi saci doar <code>159</code>, <code>18</code> şi <code>7</code>, au cifrele în ordine strict crescătoare de la stânga la dreapta.<br>
Astfel doar <code>3</code> saci conţin sare şi trebuie încărcaţi în căruţă. </p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <fstream>
using namespace std;
ifstream fin("saci.in");
ofstream fout("saci.out");

int cub(long long n)
{
    int ok = 0 , p = 10 , c = 0 , cnt = 0;
    while(n != 0)
    {
        if(n % 10 < p) c++;
        cnt++;
        p = n % 10;
        n /= 10;
    }
    if(c == cnt) return 1;
    else return 0;
}
int main()
{
    int n , cnt = 0 , x;
    fin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        fin >> x;
        if(cub(x)) cnt++;
    }
    fout << cnt;

    fin.close();
    fout.close();

    return 0;
}</pre>
