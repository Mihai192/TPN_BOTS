<h1>Cerinţa</h1>

<p>Într-un grup sunt <code>n</code> persoane, numerotate de la <code>1</code> la <code>n</code> și fiecare are cont pe Facebook. Ele îsi dau like-uri unul altuia, dar o persoană nu-și poate da like ei însăși. Despre fiecare pereche de persoane <code>i j</code> se știe câte like-uri a dat persoana <code>i</code> persoanei <code>j</code>.</p>

<p>Să se determine persoana care a primit cele mai multe like-uri.</p>

<h1>Date de intrare</h1>

<p>Programul citește de la tastatură numărul <code>n</code>, iar apoi <code>n</code> șiruri cu câte <code>n-1</code> numere naturale. Al <code>i</code>lea șir reprezintă numărul de like-uri date de persoana <code>i</code>, celeorlate persoane, în ordine.</p>

<h1>Date de ieşire</h1>

<p>Programul afișează pe ecran numărul <code>K</code>, reprezentând numărul de ordine al persoanei cu număr maxim de like-uri primite.</p>

<h1>Restricţii şi precizări</h1>

<ul>
	<li><code>1 ≤ n ≤ 100</code></li>
	<li>dacă sunt mai multe persoane cu număr maxim de like-uri primite, se va afișa cea cu numărul de ordine mai mic</li>
</ul>

<h1>Explicație</h1>

<p>Persoana <code>1</code> a primit <code>3</code> like-uri (<code>2</code> de la persoana <code>2</code>, <code>1</code> de la persoana <code>3</code>).<br>
Persoana <code>2</code> a primit <code>4</code> like-uri (<code>1</code> de la persoana <code>1</code>, <code>3</code> de la persoana <code>3</code>).<br>
Persoana <code>3</code> a primit <code>2</code> like-uri (<code>1</code> de la persoana <code>1</code>, <code>1</code> de la persoana <code>2</code>).</p>

<p>Persoana <code>2</code> a primit cele mai multe like-uri, <code>4</code>.</p>						
<pre class="EnlighterJSRAW" data-enlighter-language="cpp">#include <iostream>
using namespace std;
int v[101];
int main()
{
    int n , x , max = 0 , rez;
    cin >> n;
    for(int i = 1 ; i <= n ; ++i)
    {
        int p = 1;
        for(int j = 1 ; j < n ; ++j)
        {
            if(i == j)p++;
            cin >> x;
            v[p]+=x;
            p++;
        }
    }
    for(int i = 1 ; i <= n ; ++i)
        if(v[i]>max)
        {
            max=v[i];
            rez = i;
        }
    cout << rez;
    return 0;
}</pre>
